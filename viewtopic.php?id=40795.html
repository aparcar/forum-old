Hello

I am using OpenWrt Attitude Adjustment 12.09-rc1 / LuCI 0.11 Branch (0.11+svn9425) on my Router Tl-Wr841ND V7

I installed upnpdeamon and it is working, after that i tryed to install Luci-app-Ddns but i reloaded and rebooted and the tab SERVICES does not display.

Can somebody see if DDNS is displaying in Luci?

How can i solve?


EDIT:

SOLVED, i found this comment in the www:

ez-ipupdate is required to be installed for the DDNS option to display in the WebUI.

I did not remeber if i installed ez-ipupdate the last time but now it works.

If someone has the same problem here is the solution.

Post #2
written_direcon
29 Nov 2012, 09:56 
opkg update && opkg install luci-app-ddns
rm -f /tmp/luci-indexcache

Reload LuCI in browser.
