Hi guys,

I looked here and there for PCB images of TP-Link TL-MR11U, but did not find any.... so I decide to open mine and to take some photos:
A new toy, I like it more than TL-WR703N one , because it have internal battery I do not need to modify WR703 now. It has better exposure on some of the GPIO pins 4+1 Leds + OSS and Reset Buttons. I have not played yet with GPIO. It will be easy to solder these GPIO than on WR703N
I want to mode first  4M flash and put 8M instead, and 32M RAM to 64M RAM. the only thing that is not very clear if I'll manage to put new Uboot?

On the 4 led side there is enough space to easily add antenna connector and will still have space left for connector for UART and some of the GPIO's....

It will can become ultimate penetration testing device! Chip inside is AR9331; FLASH - 4M - S24FL032P (32-Mbit CMOS 3.0 Volt Flash Memory with 104-MHz SPI (Serial Peripheral Interface) Multi I/O Bus) - 8-pin SO package (208 mils); MEMORY - 32M - ZENTEL A3S56D40FTP-G5 - TSOP66

Battery life is very good. I need to do more test to say how much time exactly will last ( yesterday with standard firmware - acting like WAN-AP, no USB attached it was running all afternoon on battery power )
