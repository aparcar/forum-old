You don't need to compile anything. Just use http://downloads.openwrt.org/kamikaze/8 … uashfs.trx

If you want kernel 2.6, you should compile yourself since the current Kamikaze 8.09 kernel 2.6 image doesn't work well with broadcom wifi. It's because broadcom didn't release specs or an open source driver. Kernel 2.4 use a proprietarty, binary driver for wireless.

If you want compile OpenWRT trunk on Ubuntu you should install this packages:

build-essential bison flex gettext subversion libncurses-dev unzip automake zlib1g-dev gawk

If you need USB to work stable and fast you need kernel 2.6, because on 2.4 this USB in Asus could working not properly.
Wi-Fi works fine with kernel 2.6.28/29 with b43 driver. So, compile it.
