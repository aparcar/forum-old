if I understand correct and your  TL-MR3020 has OpenWrt and have LuCi interface,and you want to Repeat your ftiends wifi in your room
to set a Client (repeater) + AP
set TL-MR3020 to a dif. subnet from your friends wifi
now, set the AP, SSID, password etc. go to Networ/WiFi opn you have a Scan tab scan for networks, select your friends then Click Join network, next secreen opens, first line Replace network configuration UNTICK check mark,

hosting imagenes
next line write network to join password, if you want to see your writing click the 2 green arrows, go to bottom SUBMIT, on next screen, is just to check you had set the right password, if you need to see it, click the 2 green arrows again SAVE, and SAVE APPLY.
thas all you need to do.

jmurdoch wrote:

Is it possible to set up OpenVPN on just the access point and NOT the main router?

OpenVPN only has to be installed on the tunnel-endpoints.

The OpenVPN server has to be reachable in the internet.
If you set up an OpenVPN-server on your AP you have to configure port-forwarding on the main router.
Of course, your main-router needs a puclic IP address.

If you want to install an OpenVPN client on your AP, no configuration tasks should be neccessary on the main router.
