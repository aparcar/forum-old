Hello all,

I found what I believe to be a bug, but would appreciate anyone who can confirm it also as an issue.


Running 15.05.1 on a Bullet M2 device.

After a clean install, or reset, when I perform a WiFi scan, Hidden networks are displayed in the list, as they should be.

After I connect to a network (non-hidden), and perform another scan, hidden networks are no longer displayed in the scan results.  All the non-hidden networks show up as expected.

If I disable the wifi-iface that I connected to/created, hidden networks once again show up.


So, there seems to be some issue with iwinfo when a wifi-iface has been defined, at least on Atheros devices.

I also installed the iwinfo bin file to run that manually.  Same results.

When I run iwinfo radio0 scan it will show hidden networks if I have wifi-iface not defined, or disabled.  Once it exists, or enabled, iwinfo radio0 scan does not show the hidden networks.

Anyone have any insight into this?

Thanks,
John
