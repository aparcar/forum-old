I used to modify /var/run/wpa_supplicant-wlan0.conf.  But when I restart the network or system, this file will generate automatically. How can I use my own wpa_supplicant configuration file when the system init?

Post #2
hnyman
12 Aug 2014, 13:53

That file in /var/run is generated again every time when wpa-supplicant starts.

You modify the script that generates the file:
https://dev.openwrt.org/browser/trunk/p … plicant.sh

That script first parses the config from uci and from line 153 onward writes the config file for the new wpa-supplicant instance.

(probably /lib/wifi/wpa_supplicant.sh in a live system. Can't check now :-(


Ps. same happens naturally also for hostapd/wpad itself: https://dev.openwrt.org/browser/trunk/p … hostapd.sh
