Post #32
bkil
27 Mar 2015, 19:20

Okay, forget SCP then and we'll just download the image on the router. Connect to the router via telnet and type (or copy&paste) the following:

cd /tmp &&
wget -O o.bin http://downloads.openwrt.org/barrier_breaker/14.07/ar71xx/generic/openwrt-ar71xx-generic-tl-wr740n-v4-squashfs-sysupgrade.bin &&
[ "`md5sum o.bin`" = "2cf3c1d2a4af44403a8a559edcd45da7  o.bin" ] &&
sysupgrade -v o.bin ||
echo error $?

Then it's only a matter of minutes before it automatically finishes. Watch out for the line breaks in this post. If you first copy&paste this box to a text file, you will see it more clearly.

Edit: removed backslash to enable copy&pasting to single line
