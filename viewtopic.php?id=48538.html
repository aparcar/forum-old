I am trying to use the block-mount package to mount a usb disk automatically after its plugin.

But there's a problem :
if the system boot with usb disk connected, it will create the /etc/config/fstab automatically.

config 'global'
        option  anon_swap       '0'
        option  anon_mount      '0'
        option  auto_swap       '1'
        option  auto_mount      '1'
        option  delay_root      '5'
        option  check_fs        '0'

config 'mount'
        option  target  '/mnt/sda1'
        option  uuid    '241d-8e30'
        option  enabled '0'

but it won't mount the disk until i change the option "enable" to 1.
I found the source code in ubox/block.c, i changed the default value to 1, it seems work fine.

but another problem showed up, when I plugged out the first usb disk and insert another one, it didn't mount at all.
the "block detect" command showed the right config, but the /etc/config/fstab did not get updated.


Did I miss something?
