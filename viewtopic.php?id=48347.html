So, where after running firstboot did you reboot the router?
It seems, that this firmware is already pretty huge, so it will be pretty difficult to install packages (as it says: just about 300 kb left). So, since you already have some usb-storage connected, you may try to use an extroot-solution (see the wiki for details). Or you may build your own firmware image with only the packages you need. Or you could try a trunk snapshot, which seems to be 1 mb smaller.
