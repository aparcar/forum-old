Can you ping the gateway from either the laptop or the tplink? "ping 192.168.1.1"
Can you access the internet from the tplink? "ping 8.8.8.8"
Try pinging 8.8.8.8 from the laptop as well, in case it is just a DNS issue.
What did you set the default route to on the linux laptop? Did you remember to actually SET the default route? Without a default route, it won't know to send packets to the gateway in order to get outside your local network. Run the command "route" on your laptop.
What did you set the DNS server to on the laptop? Are you able to resolve internet hostnames? "dig google.com"
