Usually when there is a build for a specific model, features like that are hooked up.

Note that open support for Broadcom wifi chips is incomplete.  You may not get very good wifi performance.    You could try ddwrt as a third party firmware, because they have licenses with Broadcom for their closed-source drivers.

It should have a CFE recovery mode which makes it pretty simple to flash back to stock if you aren't satisfied.
