Old topic, but still persists in some cases. Mostly caused by errors in read/write of the sectors.

Solution which works every time:

1: rm -rf /usr/lib/lua/luci/

2: opkg --force-reinstall install luci-lib-sys liblua libuci-lua lua libubus-lua luci-lib-core luci-lib-nixio luci-sgi-cgi luci-lib-web luci-proto-core luci-i18n-english luci-mod-admin-core libiwinfo libiwinfo-lua luci-theme-base luci-theme-openwrt luci-app-firewall luci-lib-ipkg luci-proto-ppp luci-mod-admin-full uhttpd luci

I hope this helps you.
