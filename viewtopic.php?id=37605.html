oh my god,... I didn't brick it :-)

boot with serial interface connected, press "2" to load from TFTP server. Enter device IP (ASUS WL-330N3G), TFTP-Server IP (eg. other OpenWRT Router with atftpd running) and filename. The original filename didn't work (maybe too long?) so I renamed to test.bin and it worked well!

I updated the Wiki accordingly:
http://wiki.openwrt.org/toh/asus/wl-330n


best,
joky
