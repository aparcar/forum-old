I probably wouldn't use an RPi 3 as a router. The 4-cores aren't going to be much help. The single USB2 bus for most IO is a problem for all sorts of things, but it is probably fine of your internet connection is less than 100Mbps.

I also wouldn't use that TP-Link router for downloading torrents. 32MB of RAM doesn't sound like enough to me. The RPi's CPU is probably overkill for torrents, but I think the extra memory will be useful.

I don't generally like having my router doing anything besides providing basic services for internet connectivity, so: Firewall/NAT, DHCP, UPnP and caching DNS. Anything more than that, I run on a separate device(s). Its more reliable, makes it easier to manage upgrades and easier to keep applications from getting in the way of the network functions.

raspberry pi in general is not a good idea as a router thing. unless youre running a internet connection lower than 10mbps. the reason for this is the USB is emulated, and emulation is nightmare CPU resource wise.

so while raspberry pi can handle 10mbps connection assuming you had 2 ethernet port (1 connected via USB port using a random USB LAN card or something) and 1 wifi dongle. Mr3020 can handle 60mbps connection just fine.

if your using as a get-ho torrent downloader thing, well raspberry pi is not a bad idea, I think. At least it can manage to download torrent at 3MB/s at best.

E ----

I still have my raspberry pi b+ that I put aside long time ago and decided to test it, torrent downloading is much more worst than the base raspbian installation, it works much better on arch linux though. Managed to get 4.7MB/s download speed with CPU maxed out, raspbian in somewhere 4.5MB/s while OpenWRT didnt even managed to get over 900KB/s, all based downloading a Ubuntu x64 iso torrent which is well seeded, tried downloading it on my NAS and it literally just gaves up at download speed of 35MB/s. Based on my 300mbit/s symmetric fiber line connection.

OpenWRT is no just optimized on this kind of miscellaneous things, but it does do excellent job on networking tasks.
