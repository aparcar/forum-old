Ok, so figured it out.
Using the above link I was able to get to Failsafe mode (blinking power and responding to pings).
I couldn't scp, tftp, or ssh to the router, but telnet worked.

From there I followed the info in this link:
http://cyberforat.squat.net/openwrt/Ope … /x338.html
The mount command didn't show any mtd mounts so I ran 'firstboot' as it said and rebooted. 

Router now has it's web interface up, responds to ssh etc...

Now to figure out the wireless interface...
