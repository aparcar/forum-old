Post #7
mikma
15 Feb 2013, 03:56

You need to configure the WAN PC and WAN interface of the router with static IP addresses in the same IP network (and you can't use the same IP network as on the LAN interface, or another interface of the router). For example:

Router WAN IP: 10.1.0.1
WAN PC IP: 10.1.0.2
netmask: 255.255.255.0
