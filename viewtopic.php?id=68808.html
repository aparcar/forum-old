Check on the main status screen that you have received an IP address from the ISP, and that it is not in the 192.168.1.X range.  This is prone to happen with cable or DSL modems in their default modem/router configuration.  The simplest work-around is just to change your OpenWrt router's IP to something like 192.168.2.1.

If you have DSL and are not getting any wan IP in the default dhcp mode, you may need to set up your wan as pppoe.
