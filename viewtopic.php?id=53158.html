drawz thanks for the idea.  Went back to the wiki, and found this:

Note: Luci for Barrier Breaker does not supporting configuring 802.11ac via the web GUI (supported in Chaos Calmer trunk branch as of 10/2014). To enable 802.11ac, run the following commands from the command line (telnet/ssh):

uci set wireless.radio0.htmode=VHT80
uci commit
wifi

This is almost certainly the magic I was looking for. I will try it out tonite when I'm back home.
