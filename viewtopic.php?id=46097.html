http://wikidevi.com/wiki/Linksys_E2500_v2
http://wikidevi.com/wiki/Linksys_E2500_v3

It seems to be BCM5358, which probably uses high-rev N-PHY radio for 2GHz. It's not supported yet
5GHz radio seems to be BCM43236, unfortunately we don't know the chip revision. It's probably V1 or V2, so it won't be supported, Broadcom decided to support V3 only in their brcmfmac driver [0]
