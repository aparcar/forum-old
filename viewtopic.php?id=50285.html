Asus expects TRX format but somehow magically modified. It needs to contain a MAGIC at the end of binary.

Currently OpenWrt doesn't generate Asus-compatible TRX images. The only reason to install OpenWrt on Broadcom based Asus device is to use CFE directly, see http://wiki.openwrt.org/doc/techref/bootloader/cfe
