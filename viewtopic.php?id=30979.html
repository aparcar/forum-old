Post #8
jow
23 Jul 2011, 00:08

With scp:

scp backup_file.tar.gz root@192.168.1.1:/tmp/
ssh root@192.168.1.1
cd /
tar -xzf /tmp/backup_file.tar.gz
reboot

On windows you can use pscp.exe:

pscp.exe -scp C:\backup_file.tar.gz root@192.168.1.1:/tmp/
