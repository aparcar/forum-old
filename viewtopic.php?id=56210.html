My own experience is that the toolchain for 10.03.1/brcm-2.4 is broken.
I failed to compile simple c-programs with it, but using a later toolchain (12.09) worked. That is, both compilation and running the program worked.

I suggest you try to use "everything" from 12.09 and just innocently install to 10.03.1.
