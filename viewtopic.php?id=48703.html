Hi all!

I have a TP-Link TL-WR1043N/ND with Attitude Adjustment 12.09 on it.
The WAN-Port is connected via two PowerLAN-adapters and a switch to my Fritzbox 7390.
It should be used as an access point for WLAN. The four LAN-ports will not be used.

So...first I need to access ssh and the OpenWrt-WebGUI over the WAN-port.
I found some lines for /etc/config/firewall, that make this work.
And after setting System/Administartion/SSH Access/Interface = wan, I was also able
to block ssh for connections coming from the LAN-Ports (or wireless).
But I do not find any (working) infos, how to block the Web-GUI for connections,
coming in over WLAN.
How do I disable the OpenWrt-WebGUI for all ports except the WAN-port ?

Dede

Post #2
jow
29 Jan 2014, 18:49

You need to edit /etc/config/uhttpd on the cli and set the listen_http line 1.2.3.4:80 where 1.2.3.4 is the IP of your wan interface.
