preferably both routers should be same brand and Chipset and firwware, if you have made changes, I advise to System/back up flash firmware/reset to defaults/perform reset
This are the steps, once first router is set with WDS enabled, set second router on same subnet dif ip, out of DHCP server from main router, then go to Network/interfaces/Lan, Protocol by default is set to Static, at the bottom ignore interface, tick disable dhcp for this interface, before you save settings, as you disable dhcp, set static ip in your Lap and check you did it right, now save & apply
Now go to Network/wifi/Scan after seconds a new window with all networks at your reach, join the one from your main router, in the new window write password from router to join, below in create and assign firewall zone, select Lan the green one, at the bottom right submit, in the new window in mode select Client WDS, and Lan, save, you are in General Tab,  click on right Adavanced settings, check HT mode is same as main router, wireless security is set just check, click the two green arrows to view password, Save & Apply
Don't forget you have static ip in your Lap reset to dinamyc and now you are able to surf Internet, now add the AP go to Network/wifi/Add set SSID any you wish, in Mode Access Point WDS in Network Lan, set security, Save & Apply,  end.
In case I made a mistake you can set Routed Client mode, read my post it's last one here https://forum.openwrt.org/viewtopic.php?id=61025

I have two WDR3600 in bridge/WDS-configuration.

In Luci:

The "host" Wifi:
  "Mode=Access Point (WDS)".

The "client" Interfaces:
  Only LAN (br-lan) (I deleted the other interfaces)
  I use static IP on LAN

The "client" Wifi:
  Mode="Client (WDS)"
  BSSID="Mac Address of host"
  otherwise, identically configured as host
