Well, digitemp seems to be in the "oldpackages" repository:
http://git.openwrt.org/?p=packages.git;a=summary
Package: http://git.openwrt.org/?p=packages.git; â€¦ c5;hb=HEAD

That repository contains all packages that have no active maintainer and are thus abandoned.

There was a big package cleanup two years ago: currently all packages need a maintainer who keeps the package updated. Packages that nobody has been interested enough, have been left on the oldpackages repo. But those packages are not compiled for releases (like CC15.05) or for trunk snapshots.

If you want the package brought to the DD trunk, pledge yourself as the new maintainer and create an up-to-date pull request at the new Github packages repo. https://github.com/openwrt/packages/pulls

Alternatively, activate the oldpackage feed in your build system and build the package for yourself. (however, as the package has not received any fixes for two years, it might not compile in the current trunk.)
