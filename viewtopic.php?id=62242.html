I'm not much of a script writer, but, here is what I came up with before testing:

#!/bin/sh

if [ "$ACTION" = wifi up ]; then
        if [ "$INTERFACE" = "radio1" ] ; then
                echo "0" > /sys/class/leds/ath9k-phy1/brightness
        else
                echo "0" > /sys/class/leds/ath9k-phy1/brightness

        fi
fi
