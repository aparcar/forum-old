I don't think that WDS itself halves the speed. Your speed is halved when your unit receives and transmits all the packets via wifi, this is when the speed is halved, because wifi is half-duplex. But this is what you have already, unless you have your WA connected by cable, which you don't by the picture. If you set everything up correctly (mainly option noscan 1, 40MHz channels) you can expect about the same speeds when you enable WDS. I'm using WDS bridges too, but 90mbit is a dream, I don't believe you can get 90mbit of real throughput on a wifi client connected to the retransmitting unit, on your computer connected by cable 192.168.0.100 it is possible though. Some math: 300mbit wifi /2 is 150mbit of real data throughput MAX, when retransmitted we are at 75mbit, which is the max speed you can get under ideal conditions...
