That's pretty basic knowledge I'd say...

802.11an: 5 GHz radio (802.11n is a protocol on both 2,4 GHz and 5 GHz band)
802.11bgn: 2,4 GHz radio.

Hence the 'dual band'. It's up to you do find out which is better suited to your usage.

HT means 'high throughput', the numbers indicate the channel width. Ie HT20 is a 20 MHz wide channel, HT40 double.

This is very public information, a Google search would turn this up as well, along with all the background info if you're interested in how the spec works.
