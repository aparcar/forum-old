Just to clarify: you are talking about openWRT running on an embedded device? If that is so, after you installed it, you will have needed to telnet into it in order to activate ssh. Now normally it is at this point people create a password. That is the one you need.

Edit: not sure, but dropbear may not accept nothing as a password (ie blank). Try ssh into it and then passwd to set a password.

There is NO default password set in OpenWrt by default!
1) telnet at 192.168.1.
2) set password with passwd
3) reconnect with SSH root@192.168.1.1 and the password you just se
After setting a password telnet login is disabled!!
