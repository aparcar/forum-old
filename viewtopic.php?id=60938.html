1. You are definitely not using the JTAG interface, but an UART (serial console).
2. Probably the TXD pin is used by the SoC for some low level configuration (sometimes called boot strap) - for example, memory type (SDR/DDR1/DDR2) or boot device (NAND, NOR) is selected based on initial value of that pin. Your USB-UART adapter probably changes the default value because of the internal pull-up or pull-down resistor.
