I don't use luci at all, but if I want to enable ssh access from wan, i need to make a rule in the firewall, so you're correct. I never used luci, but it should work over https, right? So opening 443 and using https seems like a better idea, you never know, who's listening

this is my /etc/config/firewall rule to accept ssh (I'm listening on 2222 to avoid at least some of the bot probes)

config rule
        option src              wan
        option dest_port        2222
        option target           ACCEPT
        option proto            tcp
