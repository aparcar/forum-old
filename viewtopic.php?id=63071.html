Hubs are no problem for OpenWrt. I have a cascade of 4port -> 10 port -> 2x 7port in heavy use. 4port is bus powered, 10 + 2x7 port are self powered.

If a single USB device is working on your router, then USB hubs will work too. Just install the standard packages for USB support, and if you need storage, additional packages for USB storage.

https://wiki.openwrt.org/doc/howto/usb.essentials
https://wiki.openwrt.org/doc/howto/usb.storage
