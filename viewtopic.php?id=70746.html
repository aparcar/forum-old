Maybe try this:

cat /proc/mtd

Check which one is your rootfs.
Than: 

cat /dev/mtdx > /tmp/mtdx.bin

    (x is your mtd partition number from the first command.
Than copy that file from /tmp through scp and voil√†, you've got it.
