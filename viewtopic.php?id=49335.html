I have a bunch of OpenWRT routers behind a main OpenWRT router/firewall.

These secondary routers are set up as wifi access points and get their br-lan IP from the main router via DHCP.

Thsi works fine except that these routers do not honor the DNS server info sent out by the main OpenWRT router; instead they go out directly to the net.  This means than on those routers internal names do not resolve.

How do I get these routers to use the dns server IP sent out by the main router?

root@ts7200:/etc/config# cat /etc/resolv.conf
search lan
nameserver 127.0.0.1

This needs to show the main router IP, not localhost.
