download image http://downloads.openwrt.org/attitude_a … actory.bin
Turn off the device
Press the reset button
Turn on the device
Release the reset button ~10 seconds (but not longer) after turning the device on. You will know it's ready when the LEDs change.
from command prompt 

tftp -i 192.168.1.20 put openwrt-ar71xx-generic-ubnt-nano-m-squashfs-factory.bin

wait until OpenWRT start working, install aircrack-ng.
