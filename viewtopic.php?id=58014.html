Hello,

I would create a forward between your lan and vpn and specific ACCEPT rules for ports 80,53 and 22 from vpn to lan.  I don't know how to do this through the gui but via the /etc/config/firewall file:

config forwarding
  option src 'lan'
  option dest 'vpn'

config rule
  option name 'Allow-vpn-lan-22'
  option src 'vpn'
  option dest 'lan'
  option proto 'any'
  option dest_port '22'
  option target 'ACCEPT'

Make sense?
