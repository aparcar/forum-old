I believe that whiterussian was the old version before backfire

backfire is the 'stable' branch, but is getting pretty old and so does not support a lot of current stuff.

kamikaze is another name for trunk, which is the current development branch, which is what many people are using because it supports the more current stuff.

I would start by looking your router up in the hardware compatibility table and see what version it says is needed to support that router. anything newer than that should work.

If you are working to support a new router, you should we working on the trunk development branch.
