Well, I've got me a working native build chain on a Netgear WGT634.

Just compiled Lighttpd 1.4.9 on the Netgear itself..l

All without a console cable...   


1.   Installed OpenWGT 0.06 from the Netgear WebGUI
2.  Created a USB rootfs by combining Mathia's USB fs and the pre-compiled Uclibc
     Mipsel toolchain onto a 256Meg partition on a thumb drive.  Made that partition
     read-only.
3.   Used the other half of the 512Meg thumbdrive as a work partion for building
     programs.   

The compiler works like a champ..

next to add some real storage and live filesystem on 80gig seagate USB
drive.

Makes a pretty nice build environment...  And I don't need a Linux PC..
