Hi,

The fact you get garbage is quite encouraging. I would as a first step check that your terminal settings are correct.

They should be 115200 baud, 8 bits no parity, and 1 stop bit. Try with no hardware flow control, and with and without software flow control. (I don't think I use software flow control).

This should bring up the CFE bootloader, if it does you can follow the instructions here:
http://wiki.openwrt.org/OpenWrtDocs/Har â€¦ ar/WGT634U

In future, never use the web interface for upgrading, use tftp or serial console triggered updates.

Hope this helps.

Mike
