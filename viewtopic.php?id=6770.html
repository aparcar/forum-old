If you read posts in this thread, there seems to be some works in progress to unlock this PAP2v2. Apparently, a PAP2v2 is a TI-chipset based ATA device running @160MHz with 8/16MB Flash/RAM. Looking through its firmware, I believe one can swear that this PAP2v2 firmware is based on a Linksys WRTP54G firmware source code. If this is the case and the device can be unlocked, I believe it is time to start porting the WiP WRTP54G firmware code to this PAP2v2 along with an asterisk. A PAP2v2 + asterisk should be able to outrun a Linksys SPA9000 in term of price and performance.
