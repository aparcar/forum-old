I'm not familiar with DD-WRT software. Inside OpenWrt / ddns-scripts the following url is used:
http://www.dtdns.com/api/autodns.cfm?id=[DOMAIN]&pw=[PASSWORD]&ip=[IP]
where [DOMAIN] is replaced with your hostname as FQDN (host.example.com),
[PASSWORD] with your password or security token at dtdns and
[IP] with your current IP.
I don't know which placeholders DD-WRT is using.
Additional ask i.e. Google or any other search-engine you like.

Thank's that's it! I overread this in the help!
I tried it now exactly like in the example of the help and now, in the DDNS Status there is a successful update visible.
I tried that before with the whole url in the URL field, that was the wrong thing.
So just important is:

DDNS Service: Custom
DYNDNS Server: www.dtdns.com
Username: anything but not empty
Password: anything but not empty
Host Name: myhostname.dtdns.net
URL: /api/autodns.cfm?id=myhostname.dtdns.net&pw=mypassword

Save > Apply Settings
and it is done
