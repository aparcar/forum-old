Post #2

try adding

option multicast_to_unicast 0

to br-lan section in /etc/config/network

This will disable ap isolation option which is by default = 1 (despite in docs its declared = 0) because of multicast_to_unicast default=1

Then restart wifi with "wifi" command

Also can try

       option multicast_querier '0'
       option igmp_snooping '0'

Sometimes on my systems I had problems with ipv6 neighbor discovery without it
