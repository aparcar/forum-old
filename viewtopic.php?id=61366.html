maybe its a cable/wire  thing.. it should be ok since you reverted to OEM fw
read owrtwiki http://wiki.openwrt.org/toh/tp-link/tl-wr1043nd
if the wan port is broke you can transform a lan port into a wan port

from luci>network>switch assign one port from lan to wan and disable real wan switch port
considering you already have the wan interface defined it should work ok

make sure you can enter failsafe or have a ttl usb serial adapter close by before messing network/switch settings
