When selecting a router, pay some attention to the failsafe methods available and described in wiki. If the router contains a built-in TFTP functionality in the boot process, it is easy to recover from bad flashes or bad firmwares (which you will probably have at some point).

I have been happy with my WNDR3700(v2) which seems to match your functionality requirements.

Well, the very old WNDR3700 is based on the very old SoC: Atheros AR7161 rev2 which contains a MIPS 24Kc CPU running at 680MHz.

The newer TP-Link TL-WDR4300 is based on the SoC: Atheros AR9344 which contains a MIPS 74Kc CPU running at 560MHz.

I'd go with the newer technology.
