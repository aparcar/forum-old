When working on RB751U/G-2HnD I managed to "fix" MikroTik's Netinstall. The fix allows to install custom made npk's on the MikroTik's devices.

With the release of Attitude Adjustment 12.09-beta I created ar71xx compliant package which can be used to install newest OpenWrt release on all supported devices (ie. RB4xx, RB7xx). Package is built from: http://downloads.openwrt.org/attitude_a … 71xx/nand/

What to do:
1. Read this page: http://wiki.mikrotik.com/wiki/Manual:Netinstall
Instead of Netinstall provided by MikroTik use the fixed one: http://openwrt.ayufan.eu/netinstall-5.14-fixed.exe

2. Select and install OpenWrt 12.09beta package: http://openwrt.ayufan.eu/openwrt-12.9beta1-mipsbe.npk

3. Wait for reboot and telnet 192.168.1.1. You may need to switch ethernet cable from port 1 into any other.

I hope that someone will find it useful.

Kamil
