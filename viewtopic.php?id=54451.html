Post #2
mwarning
6 Dec 2014, 22:44

Maybe something like this added?

/etc/config/network:

config interface
       option ifname 'wlan0'
       option proto 'static'
       option ipaddr '192.168.2.1'
       option netmask '255.255.255.0'

/etc/config/dhcp

config dhcp
    option interface 'wlan0'
    option start '100'
    option limit '150'
    option leasetime '12h'
    option dhcpv6 'server'
    option ra 'server'

This might work if you have wlan0 configured as AccessPoint.
