For reference sake, this is solved by replacing the socket.so file, found at /usr/lib/ruby/1.9/arm-linux/socket.so
I used the file from Backfire 10.03 (NOT 10.03.1, it doesn't work) in the ruby-core package. Make sure it's version 1.9.1
In order for "gem update --system" to work, I simply resorted to installing ruby*.
And yes, OpenWRT is installed on external storage
