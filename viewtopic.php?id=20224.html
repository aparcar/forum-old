Hi.

The command to pull the package feed is:

./scripts/feeds update packages

To make a package appear in menuconfig:

./scripts/feeds install packagename

Or the equivalent to package/symlinks (put all packages in menuconf):

./scripts/feeds install -a

~ JoW
