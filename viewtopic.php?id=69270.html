Yes you can store many STA configurations in /etc/config/wireless and set the 'disabled' flag on the ones you are not using.  This can be automated with a script that scans for APs in range and enables the one that is found.  The process would be the same for any type of hardware.

Post #4
mk24
7 Jan 2017, 20:52

You can leave BSSID unspecified, and it will connect to any AP found with its SSID matching.

The WR802N would not be my first choice among those you mentioned, as it only has 4 MB flash space. That is a limitation if you want to do much beyond a very basic installation.  A lot of people are using the GL's.
