Ok, I figured it out. Thinking back when I first started, I installed the latest snapshot dev builds which was v4.1 of the kernel. Since the mains went down due to HDD issues, the snap shot builds have not been posted or not available again. So I unknowingly installed the v3.1 version of the 5Ghz file which would not work with v4.1. Going back and starting all over again with the stable version of OpenWRT v3.1, I was able to install the 5Ghz package and it's now available and displayed in the Luci UI.
