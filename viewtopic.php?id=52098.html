OK...

1.) install php5, php5-cgi, nmap

opkg update
opkg install php5 php5-cgi nmap

2.) enable php interpreter in uhttpd

Edit the file /etc/config/uhttpd and enable php support, that means remove the "#" from the start of the line.

list interpreter '.php=/usr/bin/php-cgi'

3.) Restart uhttpd and you are ready.

/etc/init.d/uhttpd restart

4.) Go to /www/ and create a file nmap.php
Copy-paste the code (you better use nano editor (not included by default, you have to install it) because vi is messy and will destroy the text layout), and you should be able to run the script by typing http://192.168.1.1/nmap.php , or whatever your router's IP is.

That was all.
