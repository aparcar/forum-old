Hi All,

After many months of testing finally we released GUI interface for OpenWRT. Our main focus was to build GUI which is fast, easy to use, different than most of the current solutions and allows easily to make desired configurations.

Now you can connect different devices to the same Mesh network and activate billing services with dynamic address IP.

Some features:

* An open interface to add or remove packages
* External Web server, AAA RADIUS and Chillispot support
* Support for Authentication, Authorization & Accounting   based on an   industry standard RADIUS/EAP accounting system
* Support for secure authentication, encryption and access control  through Web-based captive portal
* WISPr
* Dynamic and static IP to manage hotspot and wireless mesh network
* 802.11 abg (madWiFi) and 802.11 abgn (ath9K)

Hi aport,

Thank you for visual evaluation. Took us almost 18 months to built one and 4 months to test. Works great with RouterStation and is very fast! Some of our customers are managing legacy abg networks with RouterStation because easily they can manage up to 200 simultaneous users online.

The GUI is available as a standalone or with skyspot access points or Picostations2/2HP.  The GUI is in two editions: Enterprise and Professional.

The Enterprise version is free of charge and is synchronized with skyspot dashboard srvices. You can use own devices like RouterStation, Accton or PicoStation2/2HP to connect them to WLAN Cloud Controller.

The Professional Edition was built for organizations and ISP's with own AAA Radius servers and own Web captive portal. By default also has a possibility to work with skyspot dashboard. SkySpot devices with Professional Edition are starting with number 7, e.g. billing gateway SBG700, indoor access point ST712 and outdoor access point SR752. These devices can be managed by your own infrastructure.

Both versions have an interface open for making changes or add/remove packages as required. More info you can find in the skyspot STORE section.

Miro
