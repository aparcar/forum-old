This should be fairly straightforward to do.  You'll need to do the pseudo-bridge stuff in the client wireless wiki pages as well as re-assigning the WAN port back to the LAN VLAN.

All of this should be available from Luci but since there is only trunk snapshots available you'll either need to install Luci after flashing or build an image with Luci embedded.
