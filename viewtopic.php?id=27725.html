Backfire is the next "branch" in the source code SVN system after the Kamikaze.
http://wiki.openwrt.org/about/history

The following is a complete history of post White Russian releases (follow the links for release notes):

•Backfire 10.03 (April 2010)
•Kamikaze 8.09.2 (January 2010)
•Kamikaze 8.09.1 (June 2009)
•Kamikaze 8.09 (September 2008)
•Kamikaze 7.09 (September 2007)
•Kamikaze 7.07 (July 2007)
•Kamikaze 7.06 (June 2007)

Backfire is the most recent "stable" released version, while development continues at "trunk" and some bug fixes are still being implemented to the Backfire branch for the next "interim release".

Currently Backfire 10.03.1-rc4 is the most recent official beta/rc build of the next "interim release".
https://forum.openwrt.org/viewtopic.php?id=27466

23 November 2010

The OpenWrt Release Team would like to announce a fourth release candidate (RC4) for Backfire Interim Release 1 (10.03.1).  Testing of this release candidate will allow further refinement of 10.03.1

Binaries can be downloaded at http://downloads.openwrt.org/backfire/10.03.1-rc4/
