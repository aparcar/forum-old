Try this:

make distclean
svn update
./scripts/feeds update packages luci
./scripts/feeds install -a -p luci
./scripts/feeds install
make menuconfig
make V=99

This should clean your build and get the latest from SVN (I'm trying to recall these commands from memory... sorry if they don't work exactly as written).  Also, make sure you have all the required packages for LuCI configured in menuconfig.  The only thing you need to select in menuconfig (for basic LuCI functionality) is the main LuCI package (which will automatically select luci-admin-full and all required packages).  Sorry, I don't recall the exact menu option for this (I'm not at my Linux box right now), but it's a top level LuCI package that is not in the LuCI Components menu.

I just double checked at home.  Go to the "LuCI --> Collections" sub-menu in menuconfig and select "luci", "luci-light" or "luci-medium".  This will auto-select all the required packages (such as luci-admin-core, luci-admin-full, etc).  You may then add optional LuCI Components if desired, but not necessary.
