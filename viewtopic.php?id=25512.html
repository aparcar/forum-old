Post #13
p4trykx
18 Apr 2011, 13:01

The "M" option seems to be easier because OpenWrt has automated build environment it downloads everything(kernel, libraries ...) you need. Router has a different processor architecture than x86 PC and some other differences. So following instructions form http://www.saillard.org/linux/pwc/ and crosscompiling it for OpenWrt wouldn't be easy.
If you have OpenWrt installed and you don't want to flash it again just download the exact version of backfire you have from svn.
svn checkout svn://svn.openwrt.org/openwrt/tags/backfire_10.03
The whole OpenWrt buildsystem take about 4-5 GB on disk and takes few hours to compile(on my 5 yo Pentium M) the first time so be prepared.
