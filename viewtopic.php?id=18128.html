It requires the foo2zjs printer driver which needs compiling as there is no publicly available binary for Kami.

-ash: make: not found

Has anyone done this?  If so, would they be kind enough to share their foo2zjs package with me?

I've solved it.
I didn't read the HOWTO properly.  It said "install foo2zjs drivers...  on linux box".
I didn't need to install them on my wl-500g premium v1, after all.
All I had to do was get a copy of the printer's "sihp1018.dl" firmware file on to the wl-500g and cat it to /dev/usb/lp0.
I can now print from my Ubuntu netbook on to the HP1018 attached to my wl-500g.
