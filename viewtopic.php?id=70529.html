How significant is it to run vnstat, in terms of flash wear on devices such as the TP-LINK and other mini routers that use only flash storage?

Post #2
slh
9 Apr 2017, 21:13

The vnstat package, by default, only logs to RAM, which means it doesn't write to flash at all (this also means that all values are lost over a reboot).

Post #4
mk24

No, everything under /var and /tmp is on a RAM disk (tmpfs).  It is in the RAM chip.  Writing a file there does not affect the flash chip at all.  But it will be lost when power is cut off or the router is rebooted.
