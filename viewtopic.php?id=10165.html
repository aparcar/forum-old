I've run the tests and indeed kamikaze's routing speed it's quite low. Here are the results.

Kamikaze:

akira ~ # iperf -c 10.0.0.3
------------------------------------------------------------
Client connecting to 10.0.0.3, TCP port 5001
TCP window size: 16.0 KByte (default)
------------------------------------------------------------
[  3] local 10.10.13.2 port 1098 connected with 10.0.0.3 port 5001
[  3]  0.0-10.0 sec  47.3 MBytes  39.7 Mbits/sec

WhiteRussian 1.0

akira ~ # iperf -c 10.0.0.3
------------------------------------------------------------
Client connecting to 10.0.0.3, TCP port 5001

------------------------------------------------------------
[  3] local 10.10.13.2 port 1899 connected with 10.0.0.3 port 5001
[  3]  0.0-10.0 sec  54.9 MBytes  46.1 Mbits/sec

I've used 2 ASUS WL-500gP routers, one with Kamikaze and the other one with WhiteRussian.
