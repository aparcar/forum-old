void_runner wrote:

In the previous versions of the firmware this section is NULL.

So what I did was to grab the here published firmware and to add this content on address 0x120, then recalculate checksum, and ... miracle... it was flashed successfully.

Hi,

Awesome job, I understand that you put 00000000;45550000; at address 0x120, but how did you recalculate the checksum and at what location did you write it? I want to do the same for wa701nd (I saw that the same string is added at 0x120)

Edit:
NVM I got it! Here's what I did:
I used the mktplinkfw binary from openwrt imagebuilder on the old firmware (which allows uploading other firmware).
mktplinkfw -i ~/Downloads/wa701nv2_en_3_17_0_up_boot\(140324\).bin

That command told me this checksum is wrong and should be that. Then I searched the old checksum in the file and wrote correct one instead.
