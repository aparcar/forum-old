I am trying to use the usbnet module on kamikaze 7.09 on a WRTSL54GS using the 2.4.34 kernel. When I do insmod usbnet I get the message "insmod: unresolved symbol generic_mii_ioctl".

Can anyone tell me what kernel module provides this symbol? I'm currently building the buildroot to see if I can find it, however in the make menuconfig I don't see any obvious choices which would provide this.

Post #2
abercoder
29 Jan 2008, 17:33

Turns out that mii.o is in build_mipsel/linux/drivers/net in the buildroot, just copying this to /lib/modules/2.4.34 on the router solved the problem.
