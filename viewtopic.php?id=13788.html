For the Wl-500gP the prebuild image (brcm-2.4) works, so no need to compile one yourself.

According to the Table of Hardware the only difference between the gP and the W is the wireless card and the number of antennas, so theoretically the image should also work for the 500W. But as I don't have one I can't test it, so use it at your own risk.

But as the n-mode for the wireless isn't supported yet, there is currently no difference in function between the two when using openwrt, so its up to you. Though you might get a higher wireless range with the three antennas of the 500W.
