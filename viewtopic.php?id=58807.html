In "config wifi-device 'radio0'", I have the following parameters different to yours:

    option hwmode '11ng'
    option htmode 'HT40'
    option noscan '1'

I also have the following parameters in that section, but I am not sure that these are too hardware-specific:

    list ht_capab 'LDPC'
    list ht_capab 'SHORT-GI-40'
    list ht_capab 'SHORT-GI-40'
    list ht_capab 'TX-STBC'
    list ht_capab 'RX-STBC1'
    list ht_capab 'DSSS_CCK-40'

Have you tried different channels?  Use both the router and your wireless client computer to search for different networks on the 5ghz channels (AC and N), then make sure you're using a channel that is 'in the clear' from the others.  Also try changing the channel width from 40 to 20 (and 80 and 160) and see if that makes any difference.
