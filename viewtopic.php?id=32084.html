Thats neither OpenWrt nor uhttpd specific, search the internet for the basics of CGI scripting. It is a bad idea to do such low level data processing in shell scripts as it is very hard to do securely.
Best is to use a framework like "haserl" which does the heavy lifting for you, but again this is in no way OpenWrt specific.
