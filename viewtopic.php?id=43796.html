old_router wrote:

Most factory Firmware have tab to see all the domains that clients visit.

I very much doubt that. Most factory firmwares have a setting to block certain domains, which is something entirely different and much easier to do.

So how to enable the same after installing OpenWrt ? ( data need not be persistent)

You would have to run an http proxy that logs all accesses. Unless you don't want to set up each software on each client individually this proxy would have to be a "transparent proxy". These are the keywords you can use to find various approaches on the OpenWRT website and in this forum.
