if you use backfire it can be done with hotplug2
to checkout whats going on "behind the scene" edit /etc/hotplug2-common.rules and uncomment 2nd & 3rd line looking like this

# uncomment me to log hotplug events
DEVPATH is set {
        exec logger -s -t hotplug -p daemon.info "name=%DEVICENAME%, path=%DEVPATH%"
}

reboot

logread or logread -f now shows all hotplug triggers with parameters.

read about hotplug in the wiki  @ http://wiki.openwrt.org/doc/techref/hotplug and the forum here

come back for any questions

good luck
regards
3zl
