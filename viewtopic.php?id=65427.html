Post #2
milankocvara

Available control you can see here.

cat /proc/sys/net/ipv4/tcp_available_congestion_control

to see, which control is used,

cat /proc/sys/net/ipv4/tcp_congestion_control

to change you can echo or you can use sysctl
net.ipv4.tcp_congestion_control = {reno,cubic,westwood,yeah}

and yeah is not in 15.05 in default, but you can compile support in own kernel(openwrt)
