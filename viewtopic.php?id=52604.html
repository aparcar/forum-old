If you have a serial to USB adapter with 5V, you can adopt it with a voltage divider to 3V. But with some interfaces it might not work. I have done this successfully with an Arduino Board, which I am using as a serial interface.

Asterisk only works on this box, if you extend the root. But I have no experience with ISDN.

Both Barrier Breaker and the trunk version Chaos Calmer work with my 803, but probably you have to build your own images.

If the box is going to be bricked, there is still a way to load a firmware as described in the wiki. I recommend to buy a second 803 to play with it. It is very cheap now.

I have no idea what an EasyBOX 803 is. However, when it comes to a USB/Serial console adapter, there are lots of inexpensive USB/Serial adapters you can buy from China. For me, I just re-purpose the serial port on my old and free FON2100 WiFi router.
