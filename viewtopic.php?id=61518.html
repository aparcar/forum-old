hkapanen, i got the USB-Ethernet dongle to work on the Raspberry Pi Zero. I compiled from the latest trunk and included the following from menuconfig:

Kernel modules -> USB Support -> kmod-usb-net (Kernel modules for USB-to-Ethernet converters). I then selected kmod-usb-net-asix and asix-ax88179. Mine isnt an Apple dongle, but this worked for me. There are other usb-Ethernet drivers under kmod-usb-net that might work with the Apple device if Asix doesn't work.
