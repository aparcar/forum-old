Strongly suggest buying usb-to-serial converter. No soldering, very easy to connect and cost few bux.
Also can connect to HDMI and watch monitor output. Not sure if keyboard works (for me it did not work) but at least will see u-boot and console output. If nothing is displayed then probably sdcard boot structure is wrong or device cant read the card.
