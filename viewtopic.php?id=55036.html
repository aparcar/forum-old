Yes and No...

You can use fqdn as value for options src_ip and dest_ip. However they will be resolved when mwan3 loads its rules to only one ip address at the moment of resolving. It will work for dns lookups with only one A record. Sites like google and youtube (which operate under multiple ip addresses) will not work this way.
