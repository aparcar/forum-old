Found the solution and cause ....

In case anyone else have the same problem - Asterisk14 v1.4.23 from http://downloads.openwrt.org/kamikaze/8 … 4/packages is apparently broken and causing AGI failures.  You need to go down to version 1.4.22 available at http://downloads.openwrt.org/kamikaze/o … 4/packages. I may try to recompile v.1.4.23 myself and see if the problem is fixed.
