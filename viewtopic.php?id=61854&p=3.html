If you can connect with SSH ( 192.168.1.1 )

1 - place a good firmware in a folder with SCP

2 - start SSH

3 - cd in the firmware folder

4 - run -> mtd write "name of the good  firmware" firmware ( firmware for the BS-U25 is mtd3 )
after some minutes you have a prompt if the file transfert is finished

5 - run ->  firstboot -> yes

6 - run -> reboot
