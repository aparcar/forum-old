Post #447
avico
22 Jun 2009, 23:21

ok, problem resolved.

added MTU settings to wan interface

config 'interface' 'wan'
        option 'ifname' 'eth1.1'
        option 'proto' 'dhcp'
        option 'mtu' '1492'

by default it was 1500.

also double-check that in firewall config you have mtu_fix enabled

config forwarding
        option src      lan
        option dest     wan
        option mtu_fix  1
