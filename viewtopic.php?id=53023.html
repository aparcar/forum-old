How to set up IPv6 NAT and assign fc00::/7 IPs to users?

The openwrt router has multiple IPv6 WAN and one or more routed prefix on each WAN. If I use auto configuration, all the routed prefix will be assigned to clients by DHCP. Therefore, I need to set a static IP for each LAN and enable the DHCP.

However, one of my IPv6 WAN is using 6to4 with a dynamic IP assigned by the ISP. It is not a good idea to configure static IPv6 on the router as the prefix may change. Therefore, how to set up IPv6 NAT and assign fc00::/7 IPs to users?

NAT ipv6? Are you right about that?
What 6to4 service are you using?

Why not use a 6in4 (he.net or sixxs)?
There we receive multiple static ipv6 networks.

However if you really want it, include a ip6tables rule in your firewall.
ip6tables -t nat -A POSTROUTING -o 6to4-wan6 -s fc00::/7 -j MASQUERADE

You need install kmod-ipt-nat6 before…
