Post #4
hellord
12 Jan 2013, 15:08

change network config like this:

config interface 'wan'
        option ifname 'ppp0'
        option proto '3g'
        option maxwait '0'
        option service 'evdo'
        option username 'ctnet@mycdma.cn'
        option password 'vnet.mobi'
        option auto '1'
        option device '/dev/ttyUSB0'


DO NOT modify other  files.

you can run this command:
gcom -s /etc/gcom/getstrength.gcom -d /dev/ttyUSB0

if you get reply like this:

+CSQ: 36,99
OK

then ifup wan, your 3g should up..
if you have problems, maybe i can help you.
