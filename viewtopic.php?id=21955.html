Hi, if you wont need an ADSL modem, you can give the Huawei D100 a try. still, USB is OHCI (1.1) unter linux2.4 only, and when using the 2.6 image you will lose wifi, which can be worked around using a rtl8187 adapter you will have a pretty and stable little device. (the board itself is around 7x10 cm, the whole router ~ 13x10x3cm) though, it ships with a linux firmware, which you are able to access using telnet, you can also install your own software, but still upgrading to openwrt is a bit of tricky, though not impossible.
