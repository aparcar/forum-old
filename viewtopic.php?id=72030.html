Do you want to install these packages on openWrt?
Does your openWrt have a web interface (LuCI)?

For libusb: log into the web interface > log in > system > software > update list (optional) > put "libusb-1.0" into the textbox next to the lable "Download and install" and press ok. OpenWrt should install now the up to date version of libusb (if it's not installed already).
