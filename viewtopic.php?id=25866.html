I think you can traceroute to a explicit ip only ... (e.x. 192.168.1.1 - depending on your subnet and ips). the "***" are no hops - this indicates that nothing could be reached ... a good traceroute looks like this (shortend ...):

traceroute www.google.de
traceroute to www.google.de (66.249.92.104), 30 hops max, 38 byte packets
1  modem.sog (172.22.26.7)  1.365 ms  1.112 ms  1.073 ms
2  62.214.64.210 (62.214.64.210)  50.549 ms  50.315 ms  112.474 ms
3  62.214.109.245 (62.214.109.245)  48.699 ms  48.306 ms  48.168 ms
4  10g-8-4.ber023isp005.versatel.de (62.214.110.45)  49.778 ms  48.834 ms  48.741 ms

and so on
