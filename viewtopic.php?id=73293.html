Is it possible to override my ISP provided DNS servers while my WAN interface is set to DHCP client mode? I understand I can override DNS servers per device but I would like to just be able to do it for all devices at once in my router options.

Post #2
AnKat

I found the solution myself under Network>Interfaces>WAN on the Advanced settings tab, I was able to uncheck "Use DNS servers advertised by peer" and add my own DNS servers.
