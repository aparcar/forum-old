As long as there is no gateway conflict, you should have Internet connection
I don't know if LuCi is going to fit
opkg update
opkg install luci
/etc/init.d/uhttpd restart
Once installed, you need to enable it
/etc/init.d/uhttpd start
/etc/init.d/uhttpd enable
If you get it log in http://192.168.1.1/cgi-bin/luci and enable Wi-Fi by default is disable

Plug the wan port into your main router and it should dhcp an address and be connected to the internet.

If your main router is running in 192.168.1.X you need to change either that or change the LAN network on the 1043 to a different subnet so they don't conflict.
