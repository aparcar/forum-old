1M x 4 x 16 = 64 MBit = 8 MByte

albert_xi wrote:

Flash : winbond 25Q168VSIG 0946 -> not yet found the datasheet

25Q16 => 16 Mbit = 2 MByte

Unfortunately, while 2 MByte flash is already only usable with tricks, 8 MByte RAM makes it (afaik) completely unable to run OpenWrt (OpenWrt needs 16 at minimum).
