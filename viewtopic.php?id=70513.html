Well, you are trying to install the ancient wshaper that has already been deleted from the Openwrt/LEDE master package repo due to being deprecated.

wshaper was cool maybe 10 years ago. But it is not ok any more.
https://www.bufferbloat.net/projects/bl … _Must_Die/

It was succeeded by qos-scripts, and that has also been superseded by sqm-scripts (and luci-app-sqm).

If you need bandwidth shaper, use SQM with a current LEDE or Openwrt build.
SQM is the current de-facto standard bandwidth shaper.
