The memory chip needs to have the same type, the same layout (or configuration), e.g. 16Mbit x 16bit or 32Mbit x 8bit, etc. and of course the same package (TSOP II).

Put the number on the chip into google, e.g. "Hynix HY5DU121622DTP-D43" and find out it's layout. Then order a chip with the same layout and frequency. Should work.
