Do you need another server or do you just need to direct to the appropriate html file?

echo bingo > /www/control.html

chmod 666 /www/control.html

http://192.168.10.55/control.html

This should give you a screen with the text, "bingo" (assuming that your device is 192.168.10.55).

CGI is more complicated, but the same principle applies.

This thread has some CGI information:  https://forum.openwrt.org/viewtopic.php?id=36382

In the thread I pointed to, post 7 has a form (/www/furnace.html) which allows the user to click one of 9 options.  The action script for this form, (/www/cgi-bin/set-furnace.sh, in the same post), sends two-character codes to the serial port, /dev/ttyUSB0, (which in my case was connected to a picaxe microprocessor, but which could be connected to an arduino).
