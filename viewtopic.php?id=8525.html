Ok... I installed the following.

updatedd                        2.5-1    A tool to update dynamic dns services
updatedd-mod-dyndns    2.5-1    dyndns.org
updatedd-mod-noip        2.5-1    no-ip.com

As you can see I use 2 services... NO-IP.COM en DYNDNS.ORG

Then I set up cron by the following command

crontab -e

I configured the following

0 0 * * * updatedd noip <account>:<password> bulky.bounceme.net
0 0 * * * updatedd dyndns <account>:<password> bulky.homeftp.org

This will update both accounts once a day at 00:00.

I hope this helps
