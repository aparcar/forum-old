WZR-600DHP2(both Japanese version and other version) are currently not supported on OpenWRT. Can anyone adds support for WZR-600DHP2?
WZR-600DHP2 is supported on DD-WRT. device spec is below.

CPU:Broadcom BCM4707 (armv7, single core, 800MHz)
Wireless: Broadcom BCM43228 & BCM43217
RAM: 256MB
Flash: 128MB Zentel A5U1GA31ATS
details can be read in this page.
http://wikidevi.com/wiki/Buffalo_WZR-600DHP2

There are some differences between Japanese version and other version.
the differences are described in below.
http://www.dd-wrt.com/phpBB2/viewtopic.php?p=773377
