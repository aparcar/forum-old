I don't think it is a new model?  The AR9130 is same as always, emulates the older ar71xx instruction set.  I think the AR8216 is the wired network switch, it really isn't of concern.

If you are really worried it is a new model, you need to remove the metal lid to the RF shield to see the radio chip (ar9102).
