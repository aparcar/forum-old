Post #3
mich181189
19 Jul 2011, 20:16

After further investigation, it seems that the VLANing on this is a bit suspect. For example, after making the above change to add port 5, it comes up fine if you ping it all the time it's coming up. If you wait until it's up before pinging it does not work.

Essentially it seems that when the switch is in VLAN mode, packets only get through if you ping from both sides at the same time, and even then they only get through one way!
