Is port 1 the CPU port?  If you want eth0.x CPU ports you need to tag into the CPU.  An untagged port into the CPU will just be eth0, the switch strips off the VLAN tag.

Modules 802.1q and swconfig are almost always built into the kernel itself.  Check dmesg to see if they activated at boot-up.

VLAN 1 : 0 1 2 4t
VLAN 2 : 3 4t

You will need to change the lan network from eth0 to eth0.1.  The new VLAN is eth0.2 and it goes to port 3 of the switch, which is labeled "1" on the case.
