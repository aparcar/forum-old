Hi,
I'm currently using openwrt on my Asus 500gp and it works great. I've just got myself a MacBook Pro with a 802.11n and my gf also has a macbook with 802.11n. We're both using iTunes for playing music and only have one set of speakers. The Apple's Airport Express has the very nice feature of streaming audio from iTunes via wifi. I would like to buy the AE for this feature + for the 802.11n bandwidth.

I still want to keep  my openwrt box cos I'm using a lot the ssh access when I'm not at home.

my current set up is the following:

CABLE MODEM ------- (public IP via DHCP) OPENWRT (192.168.2.1)--------- 192.168.2.0/24 WLAN

What network configuration would I have if I add the AE which is another router. Something like this ?

CABLE MODEM ------- (public IP via DHCP) OPENWRT (192.168.2.1)---------  (192.168.2.2) AirportExpress 192.168.1.0/24 WLAN

The problem with this config is that I will lose all the OpenWRT services (dhcp, QoS, samba HD,..) on my LAN since the latter will be managed by the Airport Express.

So what do you advice ?

Thanks,
Tex

Well I will answer my own question

Just just got myself the Airport Express it supports the bridge mode. So my config is as follows

CABLE MODEM ------- (public IP via DHCP) OPENWRT (192.168.2.1)---------  (192.168.2.2) AirportExpress  bridged 192.168.2.0/24 WLAN

So my wlan is on the same lan as the openwrt and I can use all the services of openwrt
