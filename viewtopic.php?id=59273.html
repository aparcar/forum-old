Or reflash your router with Barrier Braker it has LuCi included here https://downloads.openwrt.org/barrier_b … x/generic/
You will use WinScp
5 On WinSCP:
5.1 Select New Site;
5.2 File protocol: SCP;
5.3 Hostname: 192.168.1.1 (or your router IP);
5.4 Username: root
5.5 Password: your admin pass
5.6 Drag downloaded firmware from Download folder and drop in /tmp
5.7 (Important, I think...) On Transfer Settings, select Binary
5.8 Transfer; I never do this last 2 steps, 5.7 & 5.8

6 On putty;
6.1 Login on 192.168.1.1 (user root and pass);
6.2 Type:
cd /tmp (Enter)
mtd -r write xxxxx.bin  (Enter) (Sure, replace xxxx with your firmware filename...)
Easier to rename the file to tplink.bin just make sure you don't end up with tplink.bin.bin with double .bin some OS do that.
You end up with a short name as this example mtd -r write tplink.bin firmware
