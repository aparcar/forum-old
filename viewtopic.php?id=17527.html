sandreas wrote:

is it possible to transfer files via serial and if so, how do i do that?

Sure, it is. I personally use x or z modem, running  the command line programs rx, rz on the target from the lrzsz package (OpenWRT's menuconfig: "Utilities") and minicom on the host side. It isn't very fast. The speed in bytes/s is approx. a tenth of the baudrate of the serial line.
