I'm running Openwrt Backfire 10.03.1 on two WNDR-37000v2.

I have two 3G Modems (ZTE-652 (contract) and D-Link DWM-156 (prepaid) as backup).

Both are working properly after I've figure out how to get them running: 
D-Link: http://www.draisberghof.de/usb_modeswit … .php?t=817
ZTE: http://www.draisberghof.de/usb_modeswit … .php?t=810

I've got a very strange behavior now:

The speed of both modems decreased by more than 50% - without any previous signs...

I've reinstalled openwrt - freshly configured them (without using a backup/restore) but the speed still was verrrry slow (1-2 MBit/s)

Honestly I absolutly don't understand why this is/was happening.

Can this cause some sort of "overload" if this command is started every time the usb-device is connected?
