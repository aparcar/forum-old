Hi all,

I'm new to openwrt so please forgive me if this is a stupid question. I'm having difficulty forcing OpenVPN on OpenWRT to NOT pull routes from the server. I've googled and searched for forum to no avail.

I've tried a few options in /etc/config/openvpn but whatever I try it still pulls the routes-

route-nopull
option route-nopull
option route-nopull 1

With either of the above it still pulls the routes, any ideas?

Thanks

Post #2
arokh

The correct option would be

option route_nopull 1

If you are unsure, you can always just look at the /etc/init.d/openvpn, which contains all the mappings from the UCI syntax.
