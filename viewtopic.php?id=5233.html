The AD81F641642 SDRAM memory is cross compatible with Samsung K4S641632, Hitachi HM5264165, NEC 4564163 chips, etc.
Its organization is 1M*16Bit*4Banks, so its capacity is 8Mbytes. You can find the datasheet here:
http://www.samsung.com/Products/Semicon … _rev11.pdf (~300k)
http://www.datasheetarchive.com/AdFreeD … le=1877376 (~1Mb)
http://www.elpida.com/eolpdfs/E0149N10.pdf
