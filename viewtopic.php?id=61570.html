Could you post a photo or paste the entire boot log leading up to the error?

edit: ALSO, about the web GUI. No OpenWRT does not come with a GUI by default. However, you can install a web interface called LuCI through the OpenWRT package manager once you get your install running. https://wiki.openwrt.org/doc/howto/luci.essentials

You can also build your own OpenWRT image that automatically includes luci if that's convenient for you.

Maybe I am wrong, but as far as I can tell that looks like a successful boot. Does the Raspberry Pi have an IP address? Have you tried telneting to it? You should be able to plug directly into it and then telnet into it with `telnet 192.168.1.1`. You may have to configure your computers ethernet interface settings manually as this:

gateway: 192.168.1.1
static IP: 192.168.1.10
subnet: 255.255.255.0
