Just check the ...openwrt.patch file. It contains all the button modification etc. that I have done.

You need to re-create my build environment like I explain in message #2 of this thread. Then change the router in .config.init to the correct one and possible check filenames in the .sh build scripts (as they seek WNDR* files in some cases). And finally, check the ...openwrt.patch file that contains all the button modification etc. that I am doing. You will find all of those there. But none of those modifications should really hurt you with some other router. Merely some button in your other router may not work.
