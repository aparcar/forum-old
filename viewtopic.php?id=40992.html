You basically have to open the SSH (TCP port 22) and HTTP (TCP port 80) on the WAN interface in the firewall config.

You can do that using the LuCI web admin interface by going in "Network > Firewall > Trafic Rules > Open ports on router:".

You can also edit directly the "/etc/config/firewall" file and add something like this:

config rule
        option name 'Allow-HTTP-Admin'
        option target 'ACCEPT'
        option src 'wan'
        option proto 'tcp'
        option dest_port '80'

config rule
        option name 'Allow-SSH-Admin'
        option target 'ACCEPT'
        option src 'wan'
        option proto 'tcp'
        option dest_port '22'
