Hi,

I managed to lock me out of the router by setting the LAN (eth0) to dhcp. Now the router is not accessible via default IP. The reset button appears to have no functionality (pressed it for about 10 sec) . How can I regain access to the router? It does not matter if all settings are lost.

UPDATE:
I've connected the router to my other router which assigned an IP to it via DHCP. I changed the LAN status back to static and set the IP to 192.168.2.1, but now I can't reach the router via this IP...


SOLVED:
I repeatedly pushed the reset button in quick succession after unpluggin and then plugging the router. The "SYS" LED flashed in quick succession and I was able to telnet the router at 192.168.1.1 using a static IP (e.g. 192.168.1.2) via cable!

From here I could do a reset I think this should be added to the wiki!


Thank you!
