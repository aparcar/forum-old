Had immense trouble getting a DLNA server to work when the server was connected behind a relayd bridge.Using a Seagate Central running Twonky DLNA server, streaming to BubbleUPnP on Android.

The trick in this thread worked, however to make the change stick I edited /etc/init.d/relayd to set the igmp_snooping value on startup.

Add         echo "0">/sys/devices/virtual/net/br-lan/bridge/multicast_snooping here:

start_service() {
        include /lib/network
        config_load network
        config_foreach start_relay interface
        echo "0">/sys/devices/virtual/net/br-lan/bridge/multicast_snooping
}
