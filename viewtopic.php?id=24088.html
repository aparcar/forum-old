ok... to tweak windows 7 to proper access samba shares, i had to go to the control panel > administrative tools > Local Security Policy

under Local Security Policy, in the tree select "Local Policies" and then "Security Options"

double click the following two items listed

Network security: Minimum session security for NTLM SSP based (including secure RPC) clients
Network security: Minimum session security for NTLM SSP based (including secure RPC) servers

under both these sections these items will appear

Require NTLMv2 session security
Require 128-bit encryption

deselect both of these items under both sections and then hit apply, then ok

now double click on "Network security: LAN Manager authentication level" and from the dropdown menu select "Send LM & NTLM - use NTLMv2 session security if negotiated"

hit apply, then ok, and youre done, samba shares will now work with windows 7
