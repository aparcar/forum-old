I assume you have read the wiki page, especially about halfway down about installation.
http://wiki.openwrt.org/toh/linksys/wrt54g
The description is not very clear but it appears that you need to start by installing an old brcm-2.4 version to get out of stock.

Trying to run BB on this hardware seems not a good idea.  Stay with older versions, there's really no loss there since development on the WRT54 series has been inactive for a while.
