mohnkhan

I have built kernel modules, but for that i have a full OpenWRT build environment setup.
I would suggest you to have a Linux box
then do the following set of commands to get your very own personalized version

1. svn checkout.. using the info from this page.. https://dev.openwrt.org/wiki/GetSource
svn co svn://svn.openwrt.org/openwrt/branches/backfire
2.scripts /feeds update

make package/symlinks

and get a image up and  running then, will give you the next step of hints to compile kmods

Post #5
Dogge
24 May 2010, 10:28

'make package/symlinks' is deprecated... If you have to install packages you should use also the feeds script like this: ./scripts/feeds install name

Look at feeds/packages/multimedia/linux-uvc/Makefile and feeds/packages/multimedia/qc-usb/Makefile and adapt it for your package.
