I have setup the connection to a VPN at work using the following steps on  my Open WRT enabled router.

Fresh install of OpenWrt Attitude Adjustment 12.09-rc1 / LuCI 0.11 on my TP-Link 1043ND.

(This is all in Luci)

Set my log in password

Set the router to get it's LAN interface IP from the DHCP on my local network.

Then Installed 'ppp-mod-pptp'

Then created a new interface which I called 'VPN', I am quite imaginative you see :-)
Then set the VPN sever, user name and password.

Assign the VPN to LAN Firewall section. (Not sure if that is right, but I thought it can't hurt?)

Then Reboot the router.

Any way I'm now lost as to how to get a PC plugged into the router to use the VPN rather than what I guess is the default route for network access.

(I thought that now all the devices would be connected to the VPN at work).
I setup a static route, I tried using the default gateway and the VPN gateway but nothing changed when I go to WhatIsMyIsIP.com for example.

Is there something I'm missing please?
Any help or advise is much appreciated.
