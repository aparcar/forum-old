The V2 version is not well supported by OpenWrt due to its Broadcom chipset.  The unit is completely different inside versus a V1 or V3 / V4.  The stock firmware may not even be OpenWrt.

Consider trying the latest Netgear firmware if you haven't.  They may have added more options.

Type mount with no options to see how things are mounted now.  Usually the overlay disk (the writeable part of the root filesystem) is already rw.  It will have a device name like /dev/mtdblock3 (assuming the firmware is OpenWrt).

You have a larger problem that the vi editor is apparently not present in the firmware.  The only way to edit a file may be to save it from your web browser, edit on your PC then use the snarf or wget web-page downloader (if it has either of those) to copy it back to the router.  This method would require setting up a web server on your PC.
