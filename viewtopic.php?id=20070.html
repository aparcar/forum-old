Have you restarted your router?

Try telnetting to it before you try anything else.

in cmd prompt type

telnet 192.168.1.1

if it gives you a connection to the router then proceed with setting a password using the command passwd

also if you can connect in telnet and still cant get a web interface then use the firstboot command sequence

Restart router in failsafe mode (differs by router, but for the WRT54GL and other linksys):
1. restart router, when DMZ light comes on, hold reset for 3 seconds until DMZ light starts flashing.
2. at this point connect computer directly to port 1 on router.
3. give computer static ip of 192.168.1.155.
4. restart networking on computer.
5. run: telnet 192.168.1.1 -l root
Run 'firstboot'
Run 'sync'
Run 'passwd' to reset root password.
Fix computer networking and restart the router.
Now log in as normal.

if you cant connect then usually a pwr light means you have a bad flash

get the tftp2 utility from here http://www.shadowsoftware.net/shadowgam â€¦ /tftp2.exe

set your nic card to a static ip address of 192.168.1.2  subnet 255.255.255.0

chose your firmware file to flash and set retries to 15 or more

unplug your router and hit the upgrade button on the program

plug back in your router

if it works you'll see a bar pop up and say upgrading

-if it does this wait 10mins unplug your router and the plug back in your router should be flashed

-if it doesnt do this and just says erasing, go back and unplug the router again and start over
