bramsbox wrote:

Somehow I think I need to do something to have it visible but I can't seem to find out what exactly.

Go to the respective sub-menu and use up/down arrow keys to nagivate to the item. Then, use the left/right arrow keys to navigate through the <Select>    < Exit >    < Help > options. Select the < Help >, option and hit the ENTER key to enter into help page. A screen dump on etables help page is shown below:
