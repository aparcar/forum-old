Hi all,

I installed OpenWrt 10.03.1 on my TP-Link TL-WR1043ND and was pleasently surprised how easy it was. However, I have some problems that are not covered by the wiki or the forum so far.

I live in a student dorm and can only access the internet if I build a VPN-tunnel to the VPN-gateway of my university. Until I bought the router I could easily connect from my PC using the official Cisco AnyConnect client or openconnect. Now, I want the router to build the tunnel and share it with all devices in my LAN. So I downloaded the openconnect package and its dependencies, scp'd them onto the router and installed them. I also acquired a vpnc-script form the openconnect homepage. Finally, I wrote an init script:

#!/bin/sh /etc/rc.common
# Copyright (C) 2006 OpenWrt.org
START=49

start() {
echo "PASS" | openconnect -s /etc/vpnc-script -u USER -i tun0 --no-cert-check --passwd-on-stdin -b SERVER
}

stop() {
        killall openconnect
}

Now I have a connection from the router to the internet, but it is slower then a connection from my PC. Also, it fails to load some pages that are loaded if I connect from my PC (through the router).

Any ideas how I can find out whats going wrong?

Regards,
Schlaxx

P.S. I'm new to OpenWrt, so take into account that I may have made some stupid mistake :-)
