Bartman007
14 Jun 2006, 23:40

There are two things that you could mean by atheros device:
1) a router using just an atheros WLAN card (like the WGT634U which is a broadcom chipset with an atheros miniPCI card.)
2) a router using an atheros SoC processor

In either case the answer can be found in the Table of Hardware.  If the status of the device in question is not Supported or WIP then there are no plans to port OpenWRT to that device at this time.

Post #9
malbon
22 Jun 2006, 19:19

Hi Putin,

The 88E6060 is the Marvell Ethernet switch.
The IC42S16800 is the SDRAM chip (8Mb x 16 bits, so 16Mb in use)
The 25P16V6P is the Flash Chip 2Mb  (ar2315/ar2316/ar2317 use SPI for interfacing with flash which only requires 4 pins)

Thanks

Mike
