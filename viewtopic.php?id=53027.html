Hi hoatienii

Check this items:

1. The port 8080 is fordwarded to the ip 192.168.1.19
2. Your Internet connection is giving you a Public IP address. Some times you have another router and only are using OpenWRT to make some tests or there is a ADSL Router in your place.
3. When you access from Internet you need to use your public IP address and not the Local IP address of your Webcam. You can get your public IP address from http://ipaddress.com/

Please explain a little more for a better help.

Best regards.

gamba47
