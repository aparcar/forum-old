when you compiled the CC image did you change the default serial baud rate?
the router boot on the RB532 uses 115200, an unchanged stock compile would use 38400 i believe (been a while since I've rolled my own.)

you can either re-compile with the change or just switch your terminal program to the correct baud rate at the point it stops displaying output.
