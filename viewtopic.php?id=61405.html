By the way I Just did an experiment and it works!!!
In wrt740n which has mwan3 I create and interface called trunk bind to eth0.400 then in -->network-->switch I SET eth0.400 tagging all vlans eth1=wan eth0.200=wan2 and eth0.300=wan3 plus lan port.
Then Trunk interface set to static Ip 192.168.7.1 netmask 255.255.255.0 broadcast 192.168.7.255 and set dhcp server then add to lan zone.
After that I plug eth0.400 port to buffalo wan port.
Then install babeld and in /etc/config/babeld  set the following interfaces: eth0.2=buffalo wan, eth1, eth0.200 and erh0.300
Reboot
Faster internet, no lag, low ping
