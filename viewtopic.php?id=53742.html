I am using Arokh's build on my TP-Link TL-WDR4900 and with older kernels my extroot works perfectly fine by using this:

First by running this command:

tar -C /overlay -cvf - . | tar -C /mnt/sda1 -xf -

Then adding this to fstab:

config mount
        option target        /overlay
        option device        /dev/sda1
        option fstype        ext4
        option options       rw,sync
        option enabled       1
        option enabled_fsck  0

And reboot, but with kernel 3.16 this at least doesn't work, I don't know if I need to use a different command, different config in fstab or if it is a bug, but I would really love to get this working. Anyone has a working extroot overlay on 3.16 and wants to share how he/she got it working?
