_SOLVED_

What i did:

echo "mrt:*:1000:1000:mrt:/home/mrt:/bin/ash" >> /etc/passwd
echo "mrt:x:1000:" >> /etc/group
chown mrt.mrt /home/mrt
passwd mrt

now it works, didnt use any adduser package.
