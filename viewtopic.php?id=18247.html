I'm trying to build Kamikaze firmware with all available packages but get stuck with an error. Here is what I do:

First download latest svn trunk with: 

svn co https://svn.openwrt.org/openwrt/trunk/

Now I'm synced with 13780 revision

Then go to downloaded directory in subfolder "scripts" and update packages with:

./feeds update -a packages luci

After that I install packages with:

./feeds install -a packages luci

Everything seems OK - packages have been downloaded and symlinks in "package" folder nave been created, but when I try to:

make world V=99

in "trunk" directory I receive an error:
