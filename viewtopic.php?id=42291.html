tremmert wrote:

I am UNABLE to access the web interface though.  Any thoughts?

telnet into the router, run "opkg update", then "opkg install luci". Finally, run "/etc/init.d/uhttpd start" to start the web server. The image you installed probably did not come with a web interface and these commands will install it.

For more details, check out the wiki
