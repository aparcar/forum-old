So I had the same trouble but after some looking around I got mine to work like this.  First I installed the luci-app-qos and luci-app-splash.

I added a 8 hour time ,1024x256 qos and added a lan rule to the interface section.  The wifi was in the lan firewall zone.

That got the installation working with the default splash page.

I added a custom html to the"Edit the complete splash text" under splashtext section under but it wouldn't save.  After looking around in the router through ssh (Midnight Commander) is your friend.  I got the Splashtext section working by adding this directory /usr/lib/luci-splash. Now I have it working with a custom html splash page.

Hope this helps
