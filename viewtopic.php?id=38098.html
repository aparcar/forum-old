Self-answered question.

Fortunately, revision 29638 [10.03.1-RC6] still included support for glibc [https://dev.openwrt.org/changeset/29638 â€¦ re_10.03.1].

~/openwrt/backfire$ svnversion .
~/openwrt/backfire$ svn update -r 29638
~/openwrt/backfire$ make defconfig
~/openwrt/backfire$ make menuconfig

Advanced Configuration Options
Toolchain Option
Toolchain Options > C Library implementation >  glibc

Now, it's time for running into more trouble with glibc down the road.

glibc in Backfire has been marked as "broken" by r32105. For that reason it is not shown in menuconfig any more.
https://dev.openwrt.org/changeset/32105
