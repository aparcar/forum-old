I built an image for my Eee 701 using the generic combined X86 VM image, adding the necessary kernel modules from within the running VM, and then converting the VMDK to a raw img file using the qemu-img.exe program from the QEMU package. Then to install that on the 701 I did the following, which I suppose should also work for you:

1. create a bootable USB linux drive containing System Rescue CD 4.5.2 or later
2. copy the newly-built OpenWRT img file to a folder on that drive
3. boot the 701 using the USB drive until you get a command line
4. dd if=/path/to/my-new-eeepc-openwrt.img of=/dev/sda
5. startx and then run gparted
6. use gparted to expand sda2 into the free space on the disk
7. reboot

I discovered that the same image and procedure will work with an Eee 900 as well.

BTW, you can configure the Eee PC to run multiple VLAN's on the ethernet port. So if you have a switch with proper 802.1Q support (e.g. a Netgear Prosafe Plus), you can hook the Eee to a trunk port on that and have it act as a firewall between additional wired zones.
