Hi,

I am not sure if it 100% relevant, but...
I have much older ASUS WL-500G v1.  Connection to provider is PPPoE (it was PPTP 3 years ago, but than they changed to PPPoE)
Physical interface is Ethernet (10/100).
I subscribed to 15Mbit Unlimited (for traffic), and I fully utilize this bandwidth.
Sustained transfer rate I observe (when running Torrent with appx. 1000 connections)  is 15Mbps.
CPU utilization is usually below 40%, so there is a possibility for upside.

Hope this helps.
