No, they're definetely not the same.

http://wiki.openwrt.org/doc/howto/buildroot.exigence
describes how to install the OpenWRT buildroot

http://wiki.openwrt.org/doc/howto/build
describes how to build the firmware image

You already noted them in the correct order:
1st install the buildroot
2nd build the firmware image

Post #16
tmo26
5 Dec 2014, 11:52

You can speed up the build with

make -j 5 V=s

The factor 5 can be chosen freely. Some say "number of CPU cores +1", some "number of CPU cores -1", some "number of CPU cores x2". I have 4 cores and it it works for me with "-j 5".

Be aware: If you get errors while building with "-j 5", re-run without it.
