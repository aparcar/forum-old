The device-specific configs used for releases are not available anywhere.
But the images have been compiled just with the default settings of the generic platform (like ar71xx) at the build moment. There are no exotic optimizations or settings. Practically the same thing as starting with an empty .config and then selecting just the platform (e.g. ar71xx).

Normally you also select the device, as there is no need to build images for all the possible devices.

But there have been updates to CC15.05 sources since the original release, and some of those changes may be the reason for your difficulties.
