I have recently acquired the same router, lol. So do you want to go back to stock or want to flash OpenWRT? In any case, it is really very simple. Just go to your "Upgrade Firmware" page of the router, select the firmware, and accept (click OK/Update/Upload).

Which is the right firmware? Well, the stock firmware you can get from TP-Link's own site. OpenWRT firmware you can get from http://wiki.openwrt.org/toh/tp-link/tl-wdr3600 , as you know I'm sure. Or if you don't, click either link (depending on which version you want. Barrier Breaker is newer) on the first line below the first table on the page.

How to set OpenWRT up? That depends on what you want, so a dummy guide you need to find yourself. It can be real simple, it can be amazingly complicated...  Or post whatever things you need help with here.

That was info you should really have mentioned the first time, lol. Well, how come there is no GUI? Did the image not come with one? Because then you might have gotten the wrong image, cause mine came with one (Barrier Breaker). You can do 2 things:

Upgrade OpenWRT with a new image (only when you've already got it on your router!), use sysupgrade as described here: http://wiki.openwrt.org/doc/howto/generic.sysupgrade .

If you don't wanna do that, why don't you just install the GUI? Type "opkg install lucy" as described here: http://wiki.openwrt.org/doc/howto/luci.essentials .

Btw, if you have put a password on your router since flashing new firmware, you can't use telnet. Use something like putty instead.
