Hi everyone

Recently stuck my ISP-provided router into bridge mode and got hold of a WNDR3700 to put OpenWRT on. It's working perfectly so far.

One thing I'd quite like to do is record bandwidth usage through the WAN interface. On some old IPCop boxes I still have in service there's a great addon (Extra Graphs) which record overall totals per hour/day (ingress and egress) but also allow you to specify any ports that you're interested in recording. It also generates graphs.

An IPCop box has much more power and storage than my OpenWRT router so I was just wondering what's possible in this respect with the WNDR3700?

From reading a few bits in the wiki http://wiki.openwrt.org/doc/howto/bwmon and here http://wiki.openwrt.org/doc/howto/vnstat it looks like vnstat and bmon could do the job of recording the data quite nicely. If I can get the data off in some sort of machine-readable format then I'm happy to parse this myself and generate the graphs. bandwidthd looks like a bit of a beast to run on the WNDR3700.

A few questions...

1) Do either of these utilities allow me to record the data going through a particular port in addition to the totals? Is there a way to do that with OpenWRT?

2) Both bmon and vnstat appear to be able to output to a directory accessible by HTTP using the uhttp package. Can both of these packages use uhttp at the same time? Does this interfere with th LuCi interface?
I suppose I'm sort of wondering if I would need to setup Apache-style vhosts within uhttpd or whether each runs its own instance of uhttpd?

3) Data storage to WNDR3700's internal memory. I'm usually cautious when writing too much data to NAND/flash of embedded devices in case I gradually wear down the chips. Would this be something I should worry about with the WNDR3700?
I was thinking it might be an idea to dig out an old 512MB USB flash drive and connect that to the WNDR3700 to store the data somewhere persistent. Then I would also ship the latest data to one of my lab servers with rsync every 12-24 hours or so if it needs processing
Does anyone have any examples of the output from bmon and vnstat?

That's it for now but I'm sure there'll be more. Thanks in advance!

Cheers, B
