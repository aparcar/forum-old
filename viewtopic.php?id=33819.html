I looked again and there's indeed no nmbd package for backfire, this means your samba will not appear in the network environment and you must access it by ip, e.g. by entering "\\192.168.1.1" in the explorer address bar.
Alternatively I believe it is possible to announce samba via avahi, see http://en.gentoo-wiki.com/wiki/Avahi#Samba for example.
