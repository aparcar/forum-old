Start with this http://wiki.openwrt.org/doc/howto/firstlogin
If your router has been reset to fail safe then this should work.

Your device should be at 192.168.1.1.  Set the IP of a client to 192.168.1.2 and do not connect the wan., just a lan cable.

In general if you have a valid flash you should be able to login to Luci and set your password with out needing telnet.   If you use telnet, it gets disabled after you set the initial password and you need to ssh after that.

After you get in and set the password, reboot and then all should work.  Connect your WAN.

No clue why this is a bridge, or you think it's a bridge.  If you configured it as a bridge and did not set the LAN IP for the device as a static IP you have no DHCP and thus no IP for the device, and be unable to connect.  Your main router will need to supply an IP or you need to get the device reset.  Not familiar with the devices reset process.
