OpenWrt is a modular distribution that uses opkg to install packages containing additional programs/functionality.

pciutils contains lspci
wireless-tools contains iwconfig
kmod-madwifi contains wlanconfig

Run 

opkg update

to update the package list, then use 

opkg install foo

to install the foo package
