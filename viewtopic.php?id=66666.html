You don't really mention how you set up your "OpenWRT repeater bridge", but if you're using a relayd pseudo-bridge, it's not going to work (no igmpproxy for IPv6, recent attempts to get an equivalent for IPv6 merged into netdev failed; IPv6 relies heavily on working IGMP support, without it, it won't work). MAC80211 based WDS (4addr) however does transparently support IPv6, but effectively requires the same router firmware (well, mainline mac80211 drivers and a firmware supporting 4addr, in contrast to using weird vendor drivers) on both routers.

Homenet is designed to solve these problems without a lot of configuration. OpenWrt has a package - hnet-full - that implements the Homenet protocols.

The http://homewrt.org site provides links to various resources. I just created a howto for installing it and it worked to the limit of my testing (although I did not play with 6in4). The guide is at: http://www.homewrt.org/doku.php?id=hnet-lede - I tested it on LEDE (a fork of OpenWrt), but it should work for OpenWrt as well.
