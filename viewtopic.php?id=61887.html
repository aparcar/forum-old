Some suggestions:

(1) Recreate the config file:

rm -f /etc/config/wireless
wifi detect > /etc/config/wireless

and then edit it. I remember that in some chaos chalmer versions this helped.

(2) Refresh firmware-4.bin

Get a newer ath10k firmware version compared to the normally installed version - the newer version is more stable although this should not solve by itself your problem

https://github.com/kvalo/ath10k-firmwar … _10.2.4.48

and use this file to replace the firmware-4.bin located in
/lib/firmware/ath10k/QCA988X/hw2.0/

reboot afterwards to make the change of the firmware effective
