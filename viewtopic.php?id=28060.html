I have to correct my statements.
Simpy typing "alias bit='transmission-daemon -a A.B.C.* -b -p 5678 -T -w /share -P 56565 -o -y -er -M'" does work, but is gone after a reboot.
To make it work permanently, I had to add
[ -x /usr/bin/transmission-daemon ] && alias bit='transmission-daemon -a A.B.C.* -b -p 5678 -T -w /share -P 56565 -o -y -er -M'

Without the apostrophes only the transmission-daemon command is executed correctly and the parameters are executed as independent commands - therefore the apostrophes are needed, so that the following parameters are considered to be part of the command.
Finally, what I wanted to do, works  - thanks again.
