You over worked yourself.

This is how I would do using a Linux computer as root.

  1. Login as root into your Linux computer.

  2. Plug your USB pen drive to one of its USB ports.

  3. This step is optional. Using any fsck utility to check/fix your USB pen drive partition.

  4. Plug a new USB drive to another USB port. If this USB drive is new, you may want to create/format a new partition on it.

  5. Once both drives are clean and mounted, then you can perform a directory tree copying with (tar | tar) from one USB drive to the other.

That's all there to it.
