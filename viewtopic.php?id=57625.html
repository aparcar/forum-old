If you own/control the AP's, I suggest running a mesh network.  AP-STA systems just don't handle roaming/moving clients well.

If the robot is aware of its location it could run a script to force a connect to different AP's as it moves in and out of pre-defined coverage areas.  Thus it would know it should connect to a "better" AP (and which AP that is) before signal is lost.
