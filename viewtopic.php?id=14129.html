lkleinow
16 Jan 2008, 17:37

I got a spcacat that worked with 7.09 (2.6) from here:
http://roombahacking.com/software/openwrt/spcacat.gz

It's not a package so do this:
wget http://roombahacking.com/software/openwrt/spcacat.gz
gunzip spcacat.gz
mv spcacat /usr/bin

Also see this page, in Polish but mostly code that anyone can read:
http://eko.one.pl/index.php?page=openwrt-webcam
