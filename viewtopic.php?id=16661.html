I successfully used the µOLED-160-GMD1 and the µOLED-32028-PMD3 that you can see in the picture. But in theory any 4D Systems Display Graphics Module with Serial Platform PmmC should work, the resolution is auto-detected.
I connected them both directly via the UART1 soldering pads of my WL-500gP using the power supply of the router and via the µUSB-CE5.
