Dear All:

I would like to do something like this

config 'redirect'
        option 'src' 'wan'
        option '_name' 'SSH to misha-d630'
        option 'proto' 'tcp'
        option 'src_dport' '500'
        option 'dest_ip' '192.168.1.50'
        option 'dest_port' '500'

but instead would like say, port 500 to be forwarded to port 500 on 192.168.1.50, port 501 from wan to be forwarded to port 501 on 192.168.1.50, etc., through say, port 40000. How can I do this?

Thank you
Misha

Post #2
jaro
13 Apr 2010, 12:46

Try something like this:

config redirect
       option src       wan
       option src_dport 500-1000
       option dest      lan
       option dest_ip   192.168.1.50
       option proto     tcp
