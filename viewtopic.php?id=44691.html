I am not. Please use google to check if I was right and also, for the reason.

The Gargoyle WebUI has some really cool features like shunting some CPU load to the host machine for drawing stuff. I especially love the pie graphs for the network scheduler. One could create more based on them, for quota-usage, comparison of wireless vs. wired traffic, etc.

There are some howtos in OpenWrt-Wiki for RRDTool: http://wiki.openwrt.org/doc/howto/statistic.rrdtool which is more versatile and advanced, but it can't do pie charts :-(
