I think I have to add, that I use "Wine" to run LoL on Linux.

This usually isn't a big problem. Worked well with my old Fritz!Box 5050 and now works well with my "new" Fritz!Box 7270.

As soon as I use an OpenWRT based router, login fails *every single time*. It works again immediately as soon as the Fritz!Box is used again.

So there seems to be some problem with Wine, but for some (still unknown reason) this problem doesn't occur if a proprietary AVM router is used.

So as long as I don't know which change on router side could mimic the AVM router behaviour or the Wine bug is fixed, I'll have to stick with my AVM router.
