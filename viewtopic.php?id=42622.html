Hey Guys,

my wan port is broken so I need to switch it to some of the normal ethernet ports but I can´t figure out how to this.

Some guy had a similar problem

but he has vlan enabled and I don´t need vlan and can´t figure out how to change my /etc/config/network accordingly.

here is my config

config 'switch' 'eth0'

config 'interface' 'loopback'
    option 'ifname' 'lo'
    option 'proto' 'static'
    option 'ipaddr' '127.0.0.1'
    option 'netmask' '255.0.0.0'

config 'interface' 'lan'
    option 'type' 'bridge'
    option 'ifname' 'eth0.0'
    option 'proto' 'static'
    option 'ipaddr' '192.168.1.1'
    option 'netmask' '255.255.255.0'

config 'interface' 'wan'
    option 'ifname' 'eth0.1'
    option 'proto' 'dhcp'

config 'switch_vlan' 'eth0_0'
    option 'device' 'eth0'
    option 'vlan' '0'
    option 'ports' '0 1 2 3 5'

config 'switch_vlan' 'eth0_1'
    option 'device' 'eth0'
    option 'vlan' '1'
    option 'ports' '4 5'

please help me out, thanks alot!

In the 'switch_vlan' you'll need to change what ports are assigned where. Ports 0-3 are your normal ports, 4 is wan, and 5 is the connection to the cpu.

If you wanted to change port 4 to wan you would need to do this:

config 'switch_vlan' 'eth0_0'
    option 'device' 'eth0'
    option 'vlan' '0'
    option 'ports' '0 1 2 5'

config 'switch_vlan' 'eth0_1'
    option 'device' 'eth0'
    option 'vlan' '1'
    option 'ports' '3 5'

I've never actually tried this, but I think it should work.
