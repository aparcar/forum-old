eximido wrote:

You can use crontab with simple bash script to check whether your service is running and restart it if necessary.

You can find an example of such a script here: https://forum.openwrt.org/viewtopic.php?pid=127505

This line in script '/etc/hotplug.d/button/05-reset' is monitoring the existence of a process containing word 'blink_wps':
  ps | grep -v grep | grep blink_wps
  if [ $? == 0 ] ; then

If you have a script with similar logic run by crontab every 10 minutes(?), you achieve pretty much what you are looking for.
