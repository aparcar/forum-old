Hi,

an interface receives a DNS server as part of dhcpv6 (handled by odhcp6c)
and it is written to /etc/resolv.conf.

Is there a way to write the DNS server to another file for just one specific interface?

regards,

This script put in /etc/hotplug.d/iface did the trick:

#!/bin/sh

[ "$INTERFACE" = "wan" ] || exit
  
. /lib/functions/network.sh

dnsserver=""
network_get_dnsserver dnsserver public6
echo "nameserver $dnsserver" > /tmp/resolv.conf.wan.auto
