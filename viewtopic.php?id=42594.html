Sorry is the problem getting snort to run or getting it to log to your server? If the latter then the system/logging page of the GUI is where you set it. Or /etc/config/system

config system
        option log_ip '192.168.0.2'
        option conloglevel '8'
        option cronloglevel '8'
         ......

You may or may not need a rule or two for
'-p udp --dport syslog' depending on how things are set up.
