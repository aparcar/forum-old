Hello, i am using OpenWrt/PandoraBox R3 14.09 (r1597) / LuCI

I can´t  ping between devices on the 2.4G / 5G wifi. If i connect 2 devices on the same WiFi SSID (2.4 / 5G) They refused to communicate - not able to ping them. But from these devices connected on Wifi i can ping device connected via cable or on different channel ( 2.4G ---> 5G ---> or on cable )

If i connect ONE device to 2.4G and another one to 5G. It is working fine.

Can please anybody help me what is wrong? Firewall setting are all set on "Accept" and other settings are default.

This is really big problem for me, because i am using 90% devices on 2.4G wifi like printer, notebooks etc.
Ofcourse option ISOLATED in WIFI is OFF.

Thank. SOLVED !!!

Need to run the following:
iwpriv ra0 set NoForwarding=0
iwpriv rai0 set NoForwarding=0

You can add it to /etc/rc.local to persists across reboots
