Hi All!

I was recently searching around for some information on OpenWRT, and generally on some difficult questions about antennas and spatial streams and things, and I found that there are some StackExchange sites where people post about OpenWRT and networking but there is no StackExchange site dedicated to open source routers, community networking, or mesh stuff.

So I also found that there's a way of making proposals about new StackExchange sites that should exist, so I proposed one for Open Source routers (i.e. OpenWRT), Community Networking and Mesh

It will not become a new StackExchange site unless it gets a bunch of supporters, so if you think it's a good idea, please go here and Follow, and ask questions that you would like to see:

area51.stackexchange.com/proposals/111278/open-source-wifi-community-networking-mesh

I think there are some major benefits using a Q&A thing instead of forums:

- Very well indexed and searchable questions and answers
- No repeated questions or answers.
- Community takes care of making sure the best answer goes to the top with the voting system
- Good tagging system
- Reputation system means people rewarded for good answers and can participate in more ways

Just an idea for now - this site is *just the proposal*, so 'Follow' to support it:

area51.stackexchange.com/proposals/111278/open-source-wifi-community-networking-mesh
