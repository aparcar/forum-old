Well my router died today, so I was going to use this as an opportunity to upgrade the hardware on my network.  I am currently hosting a dedicated Minecraft server at a friends house (he has fiber!) for a few friends (10-20) to remotely log in.

My goal is to increase security on my network by separating the server onto either its own subnet or a VLAN, and then create a VPN so that the only way to administer the server (via ssh or web interface) is to first log in to the VPN.  This would allow players to log into the game server easily via port-forward, but separate the server onto its own part of the network, and restrict who can gain any access to it.

OKAY so my question is:  should I invest in a VPN firewall gateway appliance, or should I use OpenWrt/DD-WRT/Tomato flashed router to create a VPN?  Is it possible and would it be worth it?  If I use a consumer-grade router, such as the Netgear r4500 or similar, could I save some money and still gain functionality by changing the firmware to an open-source solution?  Or, does a small-business firewall (like one from ZyXEL) have superior hardware that would make a difference?

What is your recommendation or experience?

I went with Untangle at my work. We have the u150 for the main office and the u10 for the branch office. Connected with an OpenVPN tunnel. They've been rock solid since purchase. That being said, they aren't cheap for the end user. The u10 is $1k plus $250/year. It is also only rated for 10 users, but I don't think that is a hard cap.

I run OpenWRT on my home router a Buffalo WZR-HP-G450H, and find it much faster than DD-WRT. Also DD-WRT that ships with the Buffalo WZR-HP-G450H is broken right now also. While the u10 has 1Gb of RAM, atom CPU, 160 GB SSD. The Buffalo WZR-HP-G450H  has 64MB RAM, 400Mhz cpu, and 32MB flash. Specs wise, the u10 is far more capable, but you would still need a wireless gateway (if you want wireless) to connect to it.

You could build a small box and put the open source untangle on it. It would do everything you're looking for, I think. Then just buy a cheap wireless router, slap OpenWRT on it, and make put it in gateway mode.

For my home use OpenWRT is great.

-Reece
