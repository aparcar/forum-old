http://www.gl-inet.com/mifi/ is available in various configs, which include a battery option. It is delivered with OpenWrt and a custom GL.iNet GUI in addition to LuCi.  The latest SW (2.21) has WDS included, though this GL.iNet firmware is still beta.  Not sure if it's also in the delivered 2.20. 

Its an Atheros 9331 based unit. I think for WDS you want a solution that has a chipset that is the same as the rest of your network, so not sure this is "best"
