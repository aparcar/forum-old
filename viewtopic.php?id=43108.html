The default network configuration in the latest builds won't work on this router. To get it to work, you have to build yourself an image. Change the target/linux/ar7/base-files/etc/config/network file. Change the line:

   option ifname    "eth0 eth1 eth0.1 eth0.2"

to

  option ifname "eth0"
