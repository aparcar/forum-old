You are using an ancient beta version of Openwrt. 10.03.1-RC6 (1 November 2011) was a release candidate for 10.03.1, which was released on 21 December 2011.  Even the final 10.03.1 is currrently already unsupported, as it has been superseded by 12.09 Attitude Adjustment. Backfire has mainly year 2010 code, so it is really old.

You should update to either the final Backfire 10.03.1 or much more preferably to Attitude Adjustment 12.09 and then install the new module. AA12.09 would be from 2012 and is much more modern.

The 10.03.1 packages might be compatible with your firmware, but it is not certain. You may try updating opkg.conf to point to the new directory, but there is risk that the package will brick your router.
