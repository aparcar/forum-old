hi

basically wrt1900acs = wrt1200ac = wrt1900acv2 with some small improvements
see wiki pagest @ wiki.openwrt.org or wikidevi.com

your build code is mvebu Marvell Armada 385 Shelby
http://wiki.openwrt.org/toh/linksys/wrt1900acs

more detailed info can be approximated from wrt1900ac models
http://wiki.openwrt.org/toh/linksys/wrt1900ac

if you are in doubt flash only one partition and if doesnt work execute the 3 restarts procedure from wrt1900ac
also fallow this thread https://forum.openwrt.org/viewtopic.php … 52#p302252  ..its long but it will answer all questions regarding bugs..

acquire a ttl usb serial adapter as it is the only way to debug and trigger failsafe mode

in any case prepare for wifi bugs especially for 2.4G channel; disable ipv6.. its buggy on trunk

if you flash trunk.. it has ssh instead of telnet and you have to install luci
the RCs have luci instaled by default
