Post #13
Bill_MI
11 Feb 2010, 04:15

Greg, all passwords should be the same and AFAIK all user names are "root".  At least that's the intention.  Try root with an empty password - does that work?  Telnet, SSH (ssh root@192.168.1.1) or WebIF will all use the same user=root, pass=(what-you-set).

Just FYI, this is the Whiterussian forum and the filename openwrt-wrt54g-2.4-squashfs.bin indicates a Kamikaze (the current) release.  I mention this only to help these confusing little things so easy to miss.

EDIT: Did you know ssh in all Linux distros attempt to login as the current user?  So that "root" in: ssh root@192.168.1.1 is needed.

HTH,
Bill
