Hi,
I am able to disable console by modified inittab under /etc/
::sysinit:/etc/init.d/rcS S boot
::shutdown:/etc/init.d/rcS K shutdown
#ttyS0::askfirst:/bin/ash --login  #<------------- comment this line to disable console on ttyS0

I want to modify it during the build time so that I don't need to manually modify inittab, then I put inittab in
files/etc/inittab  for build

After build, burn  image on RB411 board. but inittab was modified by system to enable console on ttyS0 again
The inittab looks like
::sysinit:/etc/init.d/rcS S boot
::shutdown:/etc/init.d/rcS K shutdown
#ttyS0::askfirst:/bin/ash --login  #<------------- comment this line to disable console on ttyS0
ttyS0::askfirst:/bin/ash --login   


Who write the ttyS0::askfirst:/bin/ash --login to inittab again, how to disable it during the build time

to inittab file
