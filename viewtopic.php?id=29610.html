The WZR-HP-G300NH is a pretty nice router. Did you happen to buy yours after seeing the PirateBox? As far as stability goes, I've yet to have any problems with mine. I've been running mine for almost a year now. I've used both OpenWRT and the stock 1.76 firmware for months at a time. The stock firmware has the most reliable wireless, but OpenWRT has much more flexiblity in everything else. OpenWRT and DDWRT both seem to have somewhat unstable wireless on this unit but it does work fairly well. My guess is that the stock firmware is using a different wireless driver (maybe the official Atheros driver?) instead of the open source ATH9K driver. I don't understand why ATH9K is still unstable even after I've seen fixes regularly mentioned.

I would be using the stock firmware if it had bittorrent seed support. I couldn't figure out how to seed without downloading first. So back to OpenWRT which has several bittorrent clients that all blow away the Buffalo app.
