Good news for those still interested in this router! It seems the problems lobonse encountered with the chaos calmer build (post #129) have been resolved. And it includes Luci!

Because I just created this account, I am not allowed to post links. If you go to this post of his (#127):

lobonse wrote:

Hey, I sadly don't have the original buildroot, you might try chaos calmer (15.05) since openwrt claims it works on the b1. If not, maybe I could help with compile errors. Maybe. I'm sorta noob myself.

follow his link to the list Devices supported by Chaos Calmer 15.05. Find the DIR-601 B1 row and click the View/Edit Data link. At the bottom, you'll see the Firmware OpenWrt Install URL. Unfortunately, to use the Stock firmware to flash you need the NA build. Copy the URL and paste it into your browser. Replace "factory.bin" with "factory-NA.bin". (NOTE: I had already flashed with a version that did not include Luci, thus I used the "sysupgrade.bin".) I am trying to set it up as a Routed Client now, I may need to install relayd. Haven't done exhaustive tests but it looks great so far. Thanks everyone!
