Beware of the current LEDE snapshot, it bricked my NBG6616. It's now in a boot loop and it doesn't even get far enough for reflashing via TFTP. I don't have a TTL to USB module (yet) so can't really diagnose this further or try and fix it via console...

I have used lots of LEDE snapshot builds on my NBG6616 in the past with no issues whatsoever. I updated via LuCI and checked the sha256sum like I always do, so it seems like there's something wrong with the firmware itself.
