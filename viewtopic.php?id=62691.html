I've bought a new TP-Link Archer C7 to extend my wlan range.
After some time, I've setted up a new wlan which is extending my existing one, but it doesn't works as I wish.

The following I want to do:

[MainRouter, 192.168.1.1] is in my Room. In another Room is a second router [SecondRouter, 192.168.1.2], this router should be in the same network as the main router, connected trought the wlan to the main. On the second router, I've installed OpenWrt. The second router should just extend the wlan from the first router. All request should be done in the same network, that means, I'll have access to all devices, like printers etc.

So my current configuration is:
Interface: Lan

Protocol: Static Address
IPv4 Address: 192.168.1.2
IPv4 Mask: 255.255.255.0
Gateway: 192.168.1.1
Broadcast: 192.168.1.255
DNS: 192.168.1.1
(DHCP Server on second router is deactived)

After this, I've connected to my main router trought wlan with a new wlan interface

Name of the network (standard value) wwan
Assign firewall-zone (standard value) wwan

The second router is connected successfully to the main router.

Now I'm able to access the internet, but no devices in my lokal network.
What can I do, to solve my problem?
