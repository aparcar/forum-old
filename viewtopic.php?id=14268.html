Post #8
BlackNight
27 Jan 2008, 17:03

Use the following commands:

cd /usr/lib
wget http://www.linux-geek.org/openwrt/libjpeg.so.62.0.0
ln -s ./libjpeg.so.62.0.0 ./libjpeg.so.62
ln -s libjpeg.so.62.0.0 ./libjpeg.so

cd /usr/bin
wget http://www.linux-geek.org/openwrt/motion
chmod +x ./motion

cd /etc
wget http://www.linux-geek.org/openwrt/motion-dist.conf

Edit it for your needs and

mv ./motion-dist.conf ./motion.conf

Is it working? How many FPS do you get?
