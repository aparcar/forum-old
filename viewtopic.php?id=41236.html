Hello all,

I'm a new user to OpenWRT. I got it up and running without issue on my Netgear WNDR3800. Everything seems to be working great so far, but I noticed that my internet was slower than usual. I headed over to speedtest.net to test the connection, and it's showing that (after several tests), I'm getting anywhere between 12-15 Mbps down on average. My connection is normally in the 50 meg range.

Upload speeds appear to be fine.

To narrow it down, I connected the modem directly to my computer, and tried another Netgear I had lying around with the stock firmware. In both cases, download speeds returned to normal. To isolate it to the wireless on the router running OpenWRT, I ran a patch cable from the router to my PC. Again, download speeds are normal. So it appears to be a wireless issue.

Any suggestions? I'm basically brand new to OpenWRT so aside from what I've already done, I really don't know where I should be looking and what I should be testing. If it helps, I haven't made any changes to the default OpenWRT config and my ISP is Comcast (I'm in the US).

Any and all help is appreciated.
