F5BJR
22 Feb 2018, 01:58

WINSCP protocol SCP -> 192.168.1.1 or your openwrt IP ( default port 22 ) login : root password : yourpassword

copy original_software.bin in /tmp

Putty -> ssh 192.168.1.1 or your openwrt IP -> login : root password : yourpassword

cd /tmp   

mtd -r write original_software.bin firmware

after reboot you have the original firmware with IP : 192.168.169.1
