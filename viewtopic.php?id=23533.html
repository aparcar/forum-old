Hello. I registered to this forum hoping to get some help with my Bytton GPRS wireless router. So here it goes.

I bought this router from a second-hand dealer. It has OpenWRT Kamikaze installed and it worked fine 'til the moment when I had to move it. I disconnected the adapter, which means that I cut the power to the router (it doesn't have an ON/OFF or Shutdown button) and after reconnecting it it didn't assign an IP address for the computer. So basically I have no way of communicating with the router. The LEDs labeled Power and AIR and LAN 1 to 4 are all fine, meaning that they glow as normal.

I tried setting a manual IP address for the computer, but no connection could be made towards the router. No ping reply, no browser interface, no telnet. I also tried to reset the router to factory settings by pressing and holding the Reset button on the back of it, but nothing happened. Later I learned that by default the Reset button in OpenWRT is disabled so it won't do anything unless it's installed.

The only occasion that the router is responding is when I boot it while I keep the Reset button pushed. Then the Power LED starts blinking and if I set manual IP for my computer, then I get ping reply from 192.168.1.1. But as far as I know this mode is used only if someone wants to reinstall the firmware.

And I don't wish to reinstall the firmware. I just want to reset the router to factory settings if there's a chance for that. Or start the DHCP somehow. I hope I provided all the info needed to understand the situation and solve it. Please, any help or piece of advice would be greatly appreciated. Also, if you have some tips to avoid these kind of situations in the future, I would appreciate those too.
