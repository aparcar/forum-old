Post #2
RangerZ
28 Jan 2017, 03:16

There is no "button" to push to convert between configs.  It can be done with custom scripts, but you can take advantage of the backup and restore function.  Create a "router" config, perform a backup thru Luci.  It will save a file with a generic name to your desktop.  Rename the file "Router".  Reconfigure the device to an ap and again make a backup and call the config "AP".  If you have USB configured on your router you can store the files right on a flash drive.  You can then Restore the config file and reboot.
