We released this month a free SkySpotOS GUI interface based on OpenWRT for RouterStation and RouterStation PRO. This release supports Atheros radios with madWiFi and ath9k drivers.

Hi Dick,

The SkySpotOS GUI is released in three editions:

1. SkySpotOS Standard - free community edition that can be use with SkySpot Community Cloud Controller or without.
2. SkySpotOS Enterprise - free and valid with SkySpot Cloud Controller services.
3. SkySpotOS Professional - paid edition with additional interface for external RadiusAAA server and Web based Captive Portal.

Enterprise and Professional editions have implemented OLSR Mesh mode in 2.4Ghz or 5GHz.

The SkySpotOS GUI for Ubiquiti RouterStation and RouterStationPRO has currently the same features but different binaries from OpenWRT.
The RouterStation PRO has a 1Gigbit Ethernet switch, but RouterStation has 100Mbits Ethernet switch.

We already prepared and tested AR8316 switch configurations for RouterStationPRO but some additional work is required to implement these configurations in the GUI. The next release for RouterStation PRO will have switch features implemented.

Regards,
Pat
