Hey!

What is the correct term for the use case as explained below and - if possible - how can it be achieved using a wrt54gl and OpenWrt or similar?

A wireles network "A" is installed, and I have two valid IPs. "A" is connected to the internet. I can't change Network "A".

Now, I need to set up a second network "B" at a different subnet.
"B" will have wireless and cabel connections.
All devices in "B" should be linked/routed to one of A's IPs and need to be accessible by internet connections.

Any Ideas?


Thank you, guys!!!

This is routed client mode.  There is a wiki page about it.  Very simply, you make a wifi client that connects to A, and use it as your WAN device instead of the fifth ethernet port.  Everything else can be a default configuration.  As you noted, you need to be sure your lan subnet is not the same as the wan (that you get from A).
