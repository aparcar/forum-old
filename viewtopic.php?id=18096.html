By default, all channels are in the scan list.
Here are some examples on how to change that:

# whitelist example
iwpriv ath0 setscanlist -ALL # remove all channels
iwpriv ath0 setscanlist +5250-5300 # add 5.25-5.3 GHz

# blacklist example
iwpriv ath0 setscanlist +ALL
iwpriv ath0 setscanlist -5300
