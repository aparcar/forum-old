Hi Albert,

Another newbie here but been struggling lately with OpenWrt wireless, network, dhcp & wireless config files.

First of all whatever changes you will do in the these config files you need to restart respective service that they are reloaded. As a newbie I mostly restart the whole box (services although can be found from /etc/init.d/).

But what do looks incorrect to me is the network config file for 'lan' as below;

        option ipaddr '192.168.1.11'
        option netmask '255.255.255.0'
        option gateway '192.168.1.1'
        option dns '192.168.1.1'

You have defined you 'lan' static IP to  '192.168.1.11' but both GW and DNS are pointing to different address .1
I'd change the 'lan' IP to '192.168.1.1' and remove totally the GW and DNS definitions as those will be the same as 'lan' IP by default (assumption as that's how it works for me).

And don't forget to restart your box (or all the related services) after the changes.
