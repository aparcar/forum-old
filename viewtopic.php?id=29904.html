rc5 is not yet officially out, but those bytes might quite well prove to be a rather thorough test build for rc5.  I think that I visited that snapshot directory after April 29th and then there were still be earlier test builds from early April, so that April 29th build has surfaced there later. The SVN revision numbers vary between chipsets (at least svn r26774-r26786), so it does not look like to be a uniformly built version. So, does not quite look as a final one.

On the other hand, it looks like a complete set of modules, so you might test it. It is probably the only ready-made complete set of packages compatible for wndr3700v2.

If you install it, then configure your system to fetch additional packages from the same snapshot directory, so that you get uniformly built modules: edit /etc/opkg.conf to point into that location.

(...and even if something goes wrong, the unbricking of wndr3700 is rather easy. Just follow the advice in Wiki.)
