The LEDE project has Pi3 builds.

Configuration is the same as other routers.  Like other single Ethernet port devices, you need to get wifi up first so you can log in on the lan via wifi.  Otherwise you won't be able to log in to the router after changing the Ethernet port to be a wan connection.

If you're making a "dumb AP"*, do not turn on the DHCP client.  Also turn off the DHCP server.  Leave the Ethernet and wifi AP bridged as lan.

* the "dumb AP" is the simplest way to "share your LAN network by wifi".  With little elaboration, it directly converts wireless packets to wired.  The main router on the LAN makes all DHCP serves and routing decisions.
