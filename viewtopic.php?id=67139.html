I'm not sure why you're having that problem, i have an EA3500 and it's working fine, but the reason it can happen is because the EA3500 has two partitions, so for example, if you're installing from the Linksys GUI and it's on partition 1, you're actually installing to partition 2 and the Linksys GUI is still on partition 1.

You can try installing OpenWrt twice, using the factory.bin file if installing from the Linksys GUI and the sysupgrade.tar file for installation from OpenWrt, that will put OpenWrt on both partitions, i don't know if that will solve the problem, but if you don't need the Linksys GUI it might be worth a try. On OpenWrt you can see what partition you're on by running

fw_printenv boot_part

If you're on Linksys, you can check the partition by accessing the sys info page, which i think is at 192.168.1.1/sysinfo.cgi, but i'm not positive.
