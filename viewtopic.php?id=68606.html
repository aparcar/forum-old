You found bad tutorial or misunderstood something.
Its possible to forward GRE proto (IP protocol number 47) to specific IP on LAN.
But it will be only one-to-one mapping. You won't have more than one pptp server behind NAT. GRE does not use ports at all.
When you make outgoing PPTP session from LAN  nat helper is involved for correct NAT traversal. It tracks and separates sessions so multiple devices can connect. Without this helper only one pptp session can occur at the same time.
