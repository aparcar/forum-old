just a little remark for those using gentoo/sabayon. If you want to compile trunk's docs (Adjust Attitude) but get some weirds errors or, worst, you see your openwrt.html keep growing well over 2GB (yes it may happen if you issue 'make docs V=99') just install texlive-langfrench-2011, as it contains the file aeguill.spy which is needed for proper compilation.

See this bug for reference:

https://bugs.gentoo.org/247703?id=247703
