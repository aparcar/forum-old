Thanks for the push in the correct direction. After reading the numinous processes that people were using I did get Openwrt installed on my TL-WDR4300. FYI  this is what worked for me.

Went to ddwrt’s site , Beta area “ http://www.dd-wrt.com/site/support/other-downloads”  opened up the 2016 folder and the current month. Found the files for the TLWDR4300 and downloaded tl-wdr4300_us-webflash.bin . I was able to install DDwrt on the router.
Next I downloaded the web revert from the general release area of ddwrt for the TL-WDR4300 “tl-wdr4300v1-webrevert.rar “. After extracting the bin I was able to go back to an older version of TP-Link firmware that supported Openwrt upgrade.
Next download and install the standard openwrt-ar71xx-generic-tl-wdr4300-v1-squashfs-**factory**.bin .
Than upgraded to the final Openwrt firmware…
