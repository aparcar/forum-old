You should not need to edit the hostapd config file directly.  All OpenWRT configuration is (supposed) to be done through what is in /etc/config.  For a comprehensive set of /etc/config/wireless settings, see http://wiki.openwrt.org/doc/uci/wireless.

Im using revision 44455 on my TP-LINK Archer C7 v2 and tx power is in my /etc/config/wireless set this way

config wifi-device 'radio1'
        option type 'mac80211'
        option channel '11'
        option hwmode '11g'
        option path 'platform/qca955x_wmac'
        option htmode 'HT20'
        option country 'CZ'
        option txpower '10'
