I have started testing the DIR-615 or DIR615.

I have found it very easy.
It has a clearly labeled UART 4pin port 'CON5' [1VCC,TX,RX,GND] (needs MAX232),
a secod port 'CON4' -probably ICE,
and a full unpopulated PCI-bus port. 

I tested the UART port.
It connects at 115200;8N1
It provides the bootup output, -and a full Serial Term using the Linux ASH Shell Script !
I have used commands, including: ls, ps, free, echo, cat, gpio.
The script commands seem to work fine.

... I think I this means that I can do development on the box itself. if I can find or build a compiler !
- A lot easier than compiling the full Filesystem on the PC , then downloading to the router (and praying no bugs).

I posted the Output on this page :
  http://jlc.iwarp.com/DIR615.html

See : Index >> Hardware Hacking >> Hacking the DIR-615

Ralink RT3050/RT3052 support in Trunk is not complete. There are no working Wi-Fi or USB drivers.

For these platforms, I've always recommended using the fonosfera OpenWrt. blogic ported the Ralink SDK to OpenWrt, so it uses the proprietary wireless driver from Ralink and is on kernel 2.6.21, but it works.
