No... as far as I'm aware there isn't support for this yet. It's very similar hardware wise to the TP-Link TL-WR841ND v9 and as such I've got the Ethernet working and the flash is visible, so it's possible to netboot ( but there are no pre-compiled images. ) It needs someone who knows how these devices boot so that OpenWRT can be installed in flash to replace RouterOS.

It may also be possible to use metarouter ( not sure if it's actually supported on those boxes ) but if so it would have to be a small image.
