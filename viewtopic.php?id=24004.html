Post #12
ath
27 Mar 2010, 23:17
regmeplease wrote:

My original question is still unanswered, though.

To the best of my knowledge, there aren't any wireless routers with an integrated ADSL modem supported by OpenWrt and working perfectly (I would be happy to be proved wrong, though )

If you want to run OpenWrt on an ADSL modem your best bet is still an AR7, but you will probably encounter problems with the wireless driver. My suggestion is to buy a good wireless router (preferably Atheros-N based) and a cheap used AR7 modem off eBay.

Another good choice might be a wireless router with USB and a Linux supported USB modem. I haven't tried it yet, but someone here on the forums reported success.
