Hi all,

upgraded my SE 505 V2 from Whiterussian (out of 2006) to Kamikaze 8.09.1.
After installation the router does only respond to a couple of pings.
The first few pings go unanswered, then around 5 pings get replied to,
then the box goes silent.
No broadcast, no replies, nothing, neither on LAN nor on WAN or WLAN.
Only the LED for LAN is on.

I have tried the available images for brcm47xx and brcm-2.4 from Kamikaze 8.09.1 and 8.09.
The same happens with self compiled images (Kamikaze 8.09.1) with reduced image size (around 1.8M).

The latest Whiterussian image (whiterussian/newest/default/openwrt-brcm-2.4-squashfs.trx)
does work with one exemption. Calling the web interface leads to an error.

For verification purpose I tested the original firmware and dd-wrt v2.4 micro images which work
flawlessly.

Any suggestions or ideas?
