Try to telnet into the router and type passwd to set your password.
Then you have to download your firmware into the router (use /tmp directory).

Try:

root@OpenWrt:~# wget http://downloads.x-wrt.org/xwrt/kamikaze/7.09/brcm-2.4/openwrt-brcm-2.4-squashfs.trx

or alternatively scp that file into the /tmp directory.

The firmware image of Kamikaze 7.09 I mentioned comes with the web interface.
The last step to do is:

root@OpenWrt:~# mtd -r write openwrt-brcm-2.4-squashfs.trx linux

This is in my opinion the easier way to reflash your Linksys.
