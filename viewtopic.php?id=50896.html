Are you first time setting it up?  You need figure out what kind of WAN connection your ISP provides.  Some provides direct connect with DHCP support and the others need you to dial up ppoe which requires user id and password.  Look at your old router to figure out how it was connected.  If you are using Verizon FIOS and the old on was using COAX, you have to call them to switch data port to ethernet.

I'm not sure if you have to release the ip address before switching the router.  I'm using Verizon FIOS and WNDR3800 with OpenWRT.  Verizon FIOS memorizes the old mac.  It's a must to either release the ip on the old router before switching or have to call Verizon to release the ip (power off ONT for a half hour also works).  If I don't do it, the new router won't work even cloning the old mac.
