marcin.kasinski wrote:

expected normal login prompt.

Openwrt does not offer login prompt. You press enter on serial console and have root.
It can also not support USB keyboard in default config. Dont expect too much in HDMI.
This system is not for display, its for network.
Better connect to serial console or test if 192.168.1.1 pings from network.
