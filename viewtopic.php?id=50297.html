Hey there.

Try this one:

wget http://mirror2.malwaredomains.com/files/BOOT -q -O - | grep PRIMARY | awk '{ print "127.0.0.1 " $2 }' > /tmp/blacklisthosts

Now open LuCI, go to Network -> DHCP and DNS. Tab "Resolv and Host Files" and add the /tmp/blacklisthosts as "Additional Hosts files".

I don't know anything about terms and conditions to use this data, just typed 30 seconds into my bash.

You might need to reload your dnsmasq whenever you rewrite the file:

killall -HUP dnsmasq
