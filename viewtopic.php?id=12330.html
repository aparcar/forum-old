It's my first time with openwrt. I read documentation, common errors and faqs. I stiil have a problem.
I own a WRT54GL, I installed openwrt-wrt54g-squashfs.bin.
First of all I can telnet to 192.168.1.1 on a LAN port only if press reset to get into failsafe. I presume this is right
Then, with normal booting process It look like openwrt gets  its address by the dhcp server on the lan.
In fact the only address I can ping is 192.168.178.22 on the WAN port.
But on that address telnet and ssh are refused as well as web browser access.
I'm sure I'm missing some point. Please help

I didn't get much help... ;-(

Anyway finally I've been able to access my openwrt!
How?

While in failsafe mode i've connected via telnet an cleared the nvram with "mtd -r erase nvram"
After the reboot the web interface has been reachable at 192.168.1.1 !!!

I've now put its first configuration.

Thanks anyway
