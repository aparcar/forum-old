I'm using Technisat CableStar Combo HD CI with Tvheadend on Openwrt, but I'm afraid you can't buy it any more.

You need the following for this one to work in make kernel_menuconfig:

+CONFIG_USB=y
+CONFIG_MEDIA_SUPPORT=y
+CONFIG_MEDIA_DIGITAL_TV_SUPPORT=y
# CONFIG_DVB_NET is not set
+CONFIG_MEDIA_USB_SUPPORT=y
+CONFIG_DVB_USB_V2=y
+CONFIG_DVB_USB_AZ6007=y
+CONFIG_MEDIA_TUNER_MT2063=y
+CONFIG_DVB_DRXK=y

Also the following firmware files are needed:
root@OpenWrt:~# ls -l /lib/firmware/
-rw-r--r--    1 root     root         15913 Dec  6 02:05 dvb-usb-af9015.fw
-rw-r--r--    1 root     root         50222 Dec  6 02:05 dvb-usb-technisat-cablestar-hdci-drxk.fw
-rw-r--r--    1 root     root          8832 Dec  6 02:05 dvb-usb-terratec-h7-az6007.fw
