Be advised that i am an absolute novice . I did the task out of the need to make it work ;

using a windows machine , i downloaded the sysupgrade image that user borromini uploaded (link in page 2, post 37)

also downloaded PuTTy and WinSCP (both free)

connected using PuTTy via ethernet cable , SSH to ip 192.168.1.1

user root , password empty, assigned new password with passwd command

with winscp , set protocol to SCP, Ip 192.168.1.1, user root, password the password you assigned

with winscp transferred the sysupgrade image from my hard disk to /tmp directory in AP , then run from putty :

sysupgrade -v /tmp/openwrt-ar71xx-generic-tl-wa901nd-v4-squashfs-sysupgrade.bin

wait ...and voila, you have LuCi
