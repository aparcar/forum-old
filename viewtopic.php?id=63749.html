Hello All,

I am using SOM9331 and ported Chaos Calmer 15.05 the module is working fine, I have installed all the required packages for the USB storage. I have also installed VSFTPD for accessing the files from USB drive. The problem is when we connect the USB device, after some times the device is disconnecting automatically and connecting again and each time device number is changing. We have written a script for creating the directory and mounting the USB drive based on the no of partition and an unmount script for deleting the directories and restarting the VSFTPD. We have tried with both VFAT and EXT file system and the behavior is same.

For testing we have disabled the VSFTPD and the scripts for mounting and unmounting, but still the USB device is disconnecting and connecting back again automatically.

In some of the post (Not in openwrt) they told it may be issue with the power, i have checked the power and supply is capable of providing the power required by the USB drive and SOM9331 module.

Some devices act like this.
For example, yota LTE modem first present itself as usb cdrom, after 2 seconds disconnects itself from usb and reconnects as usb network adapter.
Usb device has its own firmware which control how it acts on usb bus.
Anyway I see no problem if all you need is usb storage. /dev/sda? are always the same.
Using blkid is even better because it will work even if it connects as /dev/sdb
