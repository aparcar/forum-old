Try the failsafe mode: Boot up the router and as soon as the WPS button starts blinking press it a few times, it should blink rapidly then (at least my MR3020 does it that way, I've read about some versions requiring flicking the mode switch instead of pressing the WPS button). You can then follow the howto at http://wiki.openwrt.org/doc/howto/generic.failsafe to mount the rootfs and edit your configuration, or worst case issue firstboot and start over from scratch.
