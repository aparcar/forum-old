RangerZ
26 Sep 2015, 18:05

I did discover this am that the LED config is missing.  You might want this

config led
    option name 'LAN'
    option sysfs 'alix:1'
    option default '1'
    option trigger 'netdev'
    option dev 'br-lan'
    option mode 'tx rx'

config led
    option default '0'
    option name 'WiFi'
    option sysfs 'alix:2'
    option trigger 'none'

config led
    option default '0'
    option name 'Heartbeat'
    option sysfs 'alix:3'
    option trigger 'heartbeat'

for the system file.
