On a TL-WR841ND, to set as access point and WAN port as swich/lan we have deleted all interfaces except LAN and then enabled eth0 as LAN. On TL-WR941ND this procedure made the device inaccessible.

How set wan port as switch on a TL-WR941ND working as access point?
