I don't know if it's the best choice, but I think Buffalo WZR-HP-AG300H or WZR-600DHP (they are basically the same), are a pretty good choice. Plenty of flash (32MB) to install packages into it and not use extroot, 128MB RAM, Dual Band (Atheros), Gigabit LAN, USB. This is pretty much all you need from a single device.
