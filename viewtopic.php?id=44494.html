Hi i tried to install openwrt via tftp because the installation of dd-wrt is broken.

The OS on my computer is ?ubuntu with tftp-hpa installed.

I read the wiki but had several problems.

Here the wik articles:

http://wiki.openwrt.org/doc/howto/gener … ernet.port

http://wiki.openwrt.org/doc/howto/gener … p#example1

On the WR703N the bootloader is u-boot and u-boot has "only" a tftp client installed.

My first problem was the example 1 in the wiki article. In the example the image uboot.img is used. Which is very confusing. Because the title is "installing openwrt…" not "installing a bootloader".

What i did so far:

Installed tftp-hpa put openwrt.bin the the root (/var/lib/tftpboot).

Checked the file.

Assigned an ip address:

ip link set eth0 up
ip addr add 192.168.1.2/24 dev eth0

Then i started tcpdump / wireshark and  connected the the cables.

After that i started the router. It takes ~15 sec until the first packet arrives. And its a DHCP broadcast (from 0.0.0.0).

I captured a few other packets (MDNS (ipv6), icmpv6 and dhcp (from 0.0.0.0) but i can't reproduce that. ATM i only get DHCP packets.

I am a bit confused. It would be great if somebody could explain the procedure to me.
