I have set up a VPN server for Android and iOS devices (did not try with Windows or MacOS machines). OpenVPN was not supported on my old (2.3) Android devices, and PPTP is frequently deemed as insecure; so I finally opted for L2TP. I am currently using openswan / ipsecd with xl2tpd, and it works pretty well.

I had to use several HOWTOs, guides, blog posts, forum messages, .... to make it work, and I have not bookmarked most of them; but I can provide my configuration files, if you need them.
