AFAIK, an Asus WL-G520U and a Linksys WRT54GL both are built on a Broadcom platform with a 4/16MB Flash/RAM, except a WL-G520U has a USB2 port. If both devices are sold @ the same price, I will prefer to get the WL-G520U mainly because it has a USB2 port that can be used to attach some USB devices.

Go for Asus WL-520gU. Under $30 in HK. I use it to connect my office openvpn gateway, can fully utilize my 2Mbit vdsl link. USB can be used for additional storage, though 10Mbit/s (USB1.1?) only. Also need to mod the source code and build yourself if a accurate clock is important.
