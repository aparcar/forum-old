I got the same error when I tried to install it through opkg, I couldn't solve it.

Is this a firmware that you built yourself? If yes, simple build your fw with batman. That should have no problem.

in your openwrt repo dir, just type scripts/feed install batman-adv. Then make menuconfig, look for batman adv, select it, then build your fw.
