Hello Guys!
I really need your help with my network configuration.
I have following situation.
-    DSL router
o    Internet Connection
o    3 Devices on switch
o    Wired connection to OpenWRT-Router
-    OpenWRT-Router
o    Wireless Access Point
o    More devise on switch

Since the DSL-Router is from my ISP I can’t change it as easy but I’m facing some really bad problems with it when a few devices are connected and active.
So I bought a second router and flashed OpenWRT 14.07 over it.
Since I also don’t trust the DSL-Router I want to have it in a different Subnet (10.0.0.0/24) than the rest of my Home-Network (192.168.1.0/24). So I thought it would be nice if the DSL-Router has the Address 10.0.0.1/24 and the OpenWRT-Router gets 192.168.1.1/24 and all Devices get the OpenWRT-Adress as Defaultgetway.
This setup would be relatively easy but the problem is, I have devices attached to booth rooters and I want to be able communicate between all of them.
I tried lots of stuff but all failed. Is this setup even possible? If yes, how?

hi

some houghts
-considering you want to get all of them to comunicate with eachother ... get a switch
-considering they all pass trough DSL router.. theres no point in adding another router

regarding OWRT
-if you disable lan side DHCP it works as a switch; also WIFI works; they take dhcp ip from DSL lan side; connect DSL lan side to OWRT lan side
-also you can transform OWRT wan port into a lan port(its easy search forum..make shure you can get failsafe mode before messing with switch config )
