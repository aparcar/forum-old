Hi

In my case I have plugged USB stick, used fdisk to create partition (for swap You have to change the id: 82 is regular Linux and 83 is swap). Then I had to use package called e2fsprogs to format the stick (I used mke2fs as a file system). Command "mkswap partition" will make an partition avaliable for swap. After that You can simply write in console "swapon partition" and  enjoy extended RAM on Your router.

Success.
