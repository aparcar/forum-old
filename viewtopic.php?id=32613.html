1. Help yourself as far as possible:
http://wiki.openwrt.org/doc/howto/wireless.overview

2. use "tcpdump" (yes, you can install and after usage de-install it again!) For package installation see http://wiki.openwrt.org/doc/techref/opkg
Read 1. and use it on the right interface.

3. LuCI has some fancy graphs by default now: http://wiki.openwrt.org/doc/howto/luci.essentials
Use the commands: "logread", "dmesg", "iw" and "iwconfig" and "ifconfig" and "netstat" etc... ;-)
