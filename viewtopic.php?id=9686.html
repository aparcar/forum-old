That depends strongly upon your workload. With the basic OpenWRT setup, you won't notice to much of a difference. First there is enough memory to satisfy all programs, and second the programs running are all so small and minimal, that there is not much fat that could get swapped out without penalty. (Standard Linux boxes often swap out the getty processes, and other unused stuff, but OpenWRT is to lean for that)

If you run heavier workloads on the box, you probably should add swap, but no matter what you do, the Asus only has 32MB RAM. If you really have a workload that needs more memory you can do only one thing, take a fast Flash-USB drive, and use that for swap. (That's similiar how Vista is "stealing" memory to run acceptably). The drawback is, that depending upon usage your flash drive might get used up someday. (Flash USB sticks should manage their chips so that overwriting the same block all the time maps the block to different flash chips over time. OTOH Flash USB sticks are not really a brand-name high-price product nowadays.)
