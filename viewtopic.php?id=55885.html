I think you complicate it for yourself:
The WRT300Nv1.1 seems reasonably supported. It is listed under Work-in-progress:
http://wiki.openwrt.org/toh/wip
But seems ok from 10.03.1.

You can download the firmware here (I doubt it would be in the 14.07 tree if it does not work at all):
http://downloads.openwrt.org/barrier_br â€¦ x/generic/
The usual advice is to flash from OEM firmware (not from DD-WRT).

Obviously the Wiki documentation is not completely up-to-date and consistent. If you want to help the community, post your findings about the router here (or even better, update the Wiki, but posting here is better than nothing). Also, you may want to change the topic name to something like "status of WRT300N 1.1" to get help.

And to answer your question... if you want to build a firmware yourself, download the Image Builder from the same folder as the firmware above. You can read more about the Image Builder (sometimes called Image Generator) on the Wiki.

Finally a disclaimer: I have never used a WRT300N 1.1. The documentation/status is obviously a bit unclear. I doubt you will destroy/brick the router with the 14.07 firmware I directed you to. But I dont know and I guarantee nothing.
