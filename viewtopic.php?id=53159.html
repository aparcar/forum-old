problem solved
i have used this commands and now works



/etc/init.d/uhttpd disable
/etc/init.d/uhttpd stop
opkg remove luci
reboot
opkg update
opkg install luci
/etc/init.d/uhttpd enable
/etc/init.d/uhttpd start
