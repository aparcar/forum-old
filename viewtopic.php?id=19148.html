With the typo corrected it works fine here with 8.09. Below a small example for testing:

/etc/hotplug.d/iface/10-test

#!/bin/sh

if [ "$INTERFACE" = "wan" ] && [ "$ACTION" = "ifup" ]
then
        echo "hotplug test" > /tmp/hotplug-test.txt
fi
root@OpenWrt:/# ACTION=ifup INTERFACE=wan /sbin/hotplug-call iface

/tmp/hotplug-test.txt

hotplug test
