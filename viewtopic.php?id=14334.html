I don't know of anything like that, but you need to give more information.  What protocol is used to talk from the rb532 to the other device over the serial cable?  What protocol do you want to use to talk over the network to the rb532?

From what you have said, you could use SSH over the network to the rb532.  In the SSH session, run minicom or something similar and type commands that get sent over the serial line.  If this is not what you want, you will need to explain better what it is you want
