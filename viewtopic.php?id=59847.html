The oldpackages repo is still there and the old autossh sources are there:
http://git.openwrt.org/?p=packages.git; … sh;hb=HEAD
Oldpackages is not compiled for releases (like C15.05) or trunk snapshots, so you need to compile packages by yourself.

If you want the package imported into Github packages (for trunk), please copy the sources to your tree, update the version to the current (apparently 1.4e), edit Makefile to include you as the maintainer, and create a pull request.
https://github.com/openwrt/packages/blo … IBUTING.md
