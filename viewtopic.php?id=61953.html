If you want to make a custom build you should better use this commands

git clone git://git.openwrt.org/12.09/openwrt.git
git clone git://git.openwrt.org/12.09/packages.git

But this is for downloading source code on your personal computer with a Linux based distro like Ubuntu or Debian. Then you still need to customize and build all the environment and firmware.

The sourcecode at git contains some fixes never released, we could say it is a 12.09.1 version never released.

About the flash size, could be true AA is better, since probably it has better rootfs compression algorithms.

But for devices with 16MB RAM, the Backfire version is better as stated in https://forum.openwrt.org/viewtopic.php?id=43764

Lower end devices with only 16 MiB RAM will easily run out of Memory, for bcm47xx based devices is Backfire with brcm-2.4 recommended
