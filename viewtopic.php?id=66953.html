You might want to contact Asus and see if there USB router will work for your application needs. There are not many Mfrs that produce USB routers that connect that way. Over all USB is not a great medium for networking data traffic and performance is slow compared to LAN wired and wireless, thus some reasons why you don't see much use on the mobile side. Laptops generally come with LAN wire and wifi connections for internet and networking access. Desktops are LAN wired however can be used with wireless adapters.

Yes, I did that at first.
It's really easy to do on Windows via the command: "netsh wlan set hostednetwork XX". And the SD-Card can be configured to connect to it. No problem here.
The problem is for Mac, where there's absolutely no API to share a connection through Wi-Fi. The only way to do that, is to share a connection via the Settings -> Sharing menu, and here, you can only share Ethernet to Wi-Fi.
Not Wi-Fi to Wi-Fi. So:
1. If people are already connected through Wi-Fi, they can't create a network
2. If they're on a Mac, they will have to manually open the sharing menu and configure things here (which is not plug'n'play )
