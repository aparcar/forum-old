Log in wired, go to network -- wifi click the edit button on your AP interface (on a basic setup there is only one), at the bottom of the edit page make sure that the interface is connected only to the lan network.

Can be done through LuCI.  In system -- administration make sure ssh interface is 'unspecified', uncheck password authentication, place public key in text box.  Then go to network -- firewall -- open ports on router, open port 22.
