I am trying to set up OpenWrt on a TP-Link Archer C7 as a replacement for an old NETGEAR WG103 access point.  (I'll be using it as an access point only, so I have no need of the Ethernet switching or router functions.)

One of the nice things about the WG103 is its ability to support multiple wireless networks (SSIDs) and map each one to a different VLAN.

Can OpenWrt do this?  If so, how?

Post #2
mk24
3 May 2017, 01:37

Yes it can.

Go to the switch page and create several new VLANs.  Make them tagged to the Ethernet port you're going to connect to the main router.  Make sure everything on that port even the original LAN is tagged.

Then go to the network interfaces page and make a new interface for each VLAN.  If you're just going to use them as a bridge over to wifi, you can set "unmanaged" as the protocol.  Use eth0.X as the physical setting with X being the VLAN number.

Finally, create your wifi APs and attach them to the respective network interfaces.
