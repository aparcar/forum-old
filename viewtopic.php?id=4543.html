- Can you configure a WDS using the HTTP interface?

Yes, if you download the latest pre-release from nbd it comes with a nice web interface (http://downloads.openwrt.org/people/nbd/whiterussian/)

- Can you configure DNS and static DHCP within the HTTP interface?

DNS, yes.
Static DHCP, yes.

- Is there any marked difference in reliability or speed between HyperWRT and OpenWRT?

I havent tried HyperWRT for a REALLY long time, but OpenWRT is VERY reliable and gives me better speed than any other firmware I tried recently (Olegs firmware for ASUS WL-500G deluxe, and official Linksys for WRT54G)
