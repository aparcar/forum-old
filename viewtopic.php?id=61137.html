The WD My Cloud employs a Mindspeed Comcerto 2000-Series (to be exact, it's a M86261G-12 dual-core ARM-Cortex A9 SoC, rated at 650 MHz) -- which is not an OpenWrt target (yet). It is complemented by 256 MB of RAM and a 512 MB flash chip, which the system uses only as a bootstrap for booting off the hard drive. The stock firmware is running a heavily customized Debian Wheezy, but it's not a problem to run a completely clean, full and proper Debian Jessie on it.
