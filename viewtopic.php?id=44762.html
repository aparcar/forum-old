The questions What wireless modes of operation does driver x on hardware y support?
and what limitations are there when combining modes pop up quite frequently.

Is there some satisfying answer with regards to wireless-testing?

Is there a place in the Internet where this is being documented? E.g. wireless.kernel.org.org does not say anything about the capability of multiple wireless modes of operation at the same time.

If you go to the drivers-page, e.g. ath9k#Features_and_modes_of_operation this information is still missing. Other drivers don't even offer this information...

If you go to OpenWrt "Kamikaze" Documentation (search for "limitations when combining modes") you find some info for "Broadcom" and "Atheros".

I'd like this information for the drivers themselves (i.e. ath9k, ath10k, b43, brcmsmac, etc.) and I'd like it for the current driver version.
