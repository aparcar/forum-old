@aaronac8

http://wiki.openwrt.org/doc/howto/firstlogin

if your rooters IP is 192.168.1.1 you should set your PC to an adress in the same subnet. for this example something like 192.168.1.2 with the same subnetmask.

after that take a program like Putty http://en.wikipedia.org/wiki/PuTTY or any other and connect with telnet to your ip address.
set your password as described in http://wiki.openwrt.org/doc/howto/firstlogin

exit. now telnet is disabled. use from now on ssh. (putty can also ssh connect )

now connect with SSH to your rooter.

and proceed with basic config. set ip, subnetmask, dns and gateway for your demands
http://wiki.openwrt.org/doc/howto/basic.config
