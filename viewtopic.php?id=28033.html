The WL500GPV2 has an integrated Broadcom radio... so you're stuck with that.  I'm not sure how the new 2.6 Broadcom drivers handle multi-SSID, but I think it should work.  Just give it a try... all you have to do is add another wifi-iface with a different SSID for the respective wifi-device.  You can easily do this via LuCI or the /etc/config/wireless file.

Post #11
fyi
9 Jan 2011, 10:36

Don't just wait for the answer. It's east to verify whether it works or not. I don't have BCM5354 router so I can't do it for you. But according to the OpenWrt manual, it should work with Broadcom proprietary driver. Didn't you read the links I provided in #4? Why don't you verify by yourself?

/etc/config/wireless:

config    wifi-device    wl0
    option    type    broadcom
    option    txpower    17    #17:50mW, 20:100mW
    option    channel    11

config    wifi-iface
    option    device    wl0
    option    network    lan
    option    mode    ap
    option    ssid    private
    option    encryption    psk2
    option    key    private_key

config    wifi-iface
    option    device    wl0
    option    network    lan
    option    mode    ap
    option    ssid    public
    option    encryption    psk2
    option    key    public_key
