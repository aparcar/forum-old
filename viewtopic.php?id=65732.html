masimiqbal wrote:

I want to custom build my OpenWRT But problem is when I include the packages which I need, it does not make the squashfactory.bin file in /bin/ARxx directory.

You might have too many packages included, so that the compiled firmware image exceed the available flash size in your router. Compilation will then fail rather silently. Enable verbose logging using "make V=s" and look for too large errors in the final image stage.

You might need to drop a few packages from the build.
