After image update, the IP still keep in 192.168.1.1

Did you check the box 'do not keep settings' when you upgraded the firmware? (or pass the '-n' flag to 'sysupgrade' command)

Besides overriding default config via 'files' dir, you can change the default config itself. Just look at these two files:
<buildroot>/target/linux/ar71xx/base-files/etc/uci-defaults/network (02_network for newer versions of OpenWRT)
<buildroot>/package/base-files/files/lib/functions/uci-defaults.sh

When you open them, you will understand what to do.
