I am quite new to building an openwrt backfire rc6 firmware (for TP-Link 941ND) from scratch (I am aiming at saving some space and adding openvpn).
I currently stumble over a missing ...sysupgrade.bin.

Details:
  

 svn co svn://svn.openwrt.org/openwrt/branches/backfire
 cd backfire
 ./scripts/feeds update packages luci
 ./scripts/feeds install -a -p luci
 make menuconfig
 -> enable luci, enable openvpn, disable usb
 make

Result in bin-directory
  

md5sums                         openwrt-ar71xx-root.jffs2-64k    openwrt-ar71xx-uImage-gzip.bin  openwrt-ar71xx-vmlinux.elf   packages

(I get that file if I leave out the feed update/install/enable luci... etc. part)

This results in two questions:
- how to get an openwrt-ar71xx-sysuprade.bin ?
