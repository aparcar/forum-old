Yep you got the right file. Version 1.x use the same firmware files. It may be worth updating to the latest stock firmware before upgrading to OpenWrt. Some TP-Link routers gained a recovery mode in the boot loader (which is left alone by OpenWrt) - this may come in handy if you have a failed flash now or later. Upgrading the boot loader later will require going back to stock or doing something fancy/complicated.
