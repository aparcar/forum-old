Where to start... that's always a difficult question. As you said yourself you're more a windows guy, you'll find the learning curve to be quite steep. Only advice i can give you is read read and read. Try the setup a build environment for OpenWRT by installing Ubuntu on a virtual machine. Use the wiki and google and don't give up.

As for the opteron part, as it is a x86 architecture, x86 pre-built images should probably work.

Good luck.
