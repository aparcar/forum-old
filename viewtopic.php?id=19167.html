Nope its not PHP, as PHP is far too inefficient for our purposes.
When a file is uploaded through a browser its usually transmitted as multipart/form-data in the Body of the POST-request.

That means the browser encloses the file into a multipart MIME message.
This is described in http://www.ietf.org/rfc/rfc1867.txt.
