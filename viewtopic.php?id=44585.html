Thanks again jow.

I was confused by the description of Atheros WDS on the wiki, which I interpreted as not supporting the simultaneous use of a radio to create a routed client, and an AP repeater.

In fact, only the routed client mode is supported inherently when Atheros WDS is enabled (unlike Broadcom WDS, which inherently provides both roles). However, an additional AP can be manually added to the client, on the same radio (which can have the same SSID and encryption as the master, "repeating" it).

From the diagram on the wiki, it looked like the additional AP would require "relayd" to operate, but that's not the case. In fact, if you follow the relayd recipe, you don't even get an AP on the client, so the diagram is misleading.

Sorted.
