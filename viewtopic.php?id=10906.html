Just got a WMU-9000VPN by OvisLink.

It's a DSL Router with WLAN and 4 USB Ports.

What I have got so far Data wise is the following:

CPU: MIPS 170MHz Brecis 2006 CPU
Flash/Ram: 8MB Flash, 32MB SDRAM
USB Controller: VIA ehci_hdc PCI-Controller / 4 ports
Switch: Fiber Logic 10/100 Mbit Ethernet Controller + 4-port Switch
WAN: Fiber Logic 10/100 Mbit
WLAN: (mini-PCI) Intersil 3880IK 802.11g

Currently runs uClinux.

Found something about the Brecis CPU on the net at http://www.linux-mips.org/wiki/Brecis_SOC produces Linux-Mips.org
Searching the OpenWRT Forum abit I found the CPU mentioned in the Archived Gigabyte section.

Anybody tried this device before? Tried flashing OpenWRT on it?
