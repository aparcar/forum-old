Kamikaze is ancient. Don't use it.
Backfire 10.03.1 is the current stable version. It is ok.
Also the trunk "Attitude Adjustment" development version is ok for most of the time.

Whole source of the web interface is available there, so you can change anything you want (and have skills to do).

Reagrding licenses & "allowed stuff", you better read the license stuff. E.g. https://dev.openwrt.org/browser/trunk/LICENSE
