andromeda wrote:

I'm not able to find in the repositories (trunk) mod_gsmopen. Is there any reason that module can't be used in OpenWRT?

IIRC, OpenWRT doesn't keep a copy of FS source package. The source package will be automatically fetched directly from one of FS GIT repository servers when the package gets compiled the 1st time. Of course, you can manually download FS source from any FS GIT servers, i.e. git clone git://git.freeswitch.org/freeswitch.git. If you just wanna get the mod_gsmopen source, you can do a Google search to find out where to download the gsmlib-1.10 source package.

AFAICT, mod_gsmopen isn't compile able out of the box. However, you are welcome to help the development to port this module to OpenWRT.
