Possibly the 5-year old Backfire imagegenerator does not like your (likely) much more recent Linux in your buildhost. Something has changed in the OS and something needed by the build system is now located in a different place.

You might try installing a Linux from year 2010-2011 to a virtual machine (Virtualbox etc.) and running the image generator there.
