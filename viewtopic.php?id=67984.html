JFTR, OpenVPN will handle all these cases gracefully throgh any sort of network downtime, no intervention by you or a script necessary. An OpenVPN client dialing into your server will retry forever on its own, maybe just set the "keepalive" parameter sensibly (personally, I use "10 60").
