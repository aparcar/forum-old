I want the support of OpenWrt for NETGEAR N300 PR2000 Trek stimulate! We have three these brilliant devices (http://youtu.be/Fyo7l8J5Q7I  /  https://www.youtube.com/watch?v=lA4vwYXqLZ0).
Travel Router, Range Extender, Access Point and Wireless Bridge function quite well but unfortunately the original software is very Basic. The device is small, uses very little power, and thanks to USB completely independent of mains supplies with a power bank.
It works so well to share Internetverbidungen that require authentication page to enter credentials, z. B. Hotspots Telekom, etc in hotels ..
It is the only Travel Router, which can be very easily retrofitted with external antenna connections even.

With the Netgear OpenWrt PR2000 would then really perfect!

Very, very thanks, if you could provide for the great part OpenWrt firmware!

Pauline
