Hello,

     I am trying to setup an OpenWrt router in VirtualBox to connect 3 other virtual machines to each other and to the internet.  I have been following the guide on the wiki but I haven't been able to ping anything.

     I have network adapter 1 setup with the "Internal Network" option to connect to the first of the three VMs and network adapter 2 setup with the "NAT" option to connect to the outside world.

My /etc/config/network looks like this: 

config interface 'lan'
     option ifname 'eth0'
     option type 'bridge'
     option proto 'static'
     option ipaddr '192.168.1.1'
     option netmask '255.255.255.0'
     option ip6assign '60'

config interface 'wan'
     option ifname 'eth1'
     option proto 'dhcp'

Any help will be greatly appreciated.

Just in case anyone else is looking for a solution to a similar problem, I was able to resolve this.  I needed to edit the "/etc/dnsmasq.conf" by adding some dns servers.  I chose to use Google's public servers. The edit I made is as follows:

server=8.8.8.8
server=8.8.4.4

Also, if anybody has any comment/critiques of this solution please let me know. 

Thanks!
