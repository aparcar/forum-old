Post #6
jefbuan
25 Sep 2011, 22:09

add it to ur /etc/firewall.user 

iptables -p all -d ipofblockwebsite -j DROP

or  add this to /etc/config/firewall

config rule
        option src              lan
        option dest             wan
        option dest_ip        ipofblockwebsite
        option target           REJECT
