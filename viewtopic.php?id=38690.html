Post #2
jow
15 Aug 2012, 18:26

mkdir -p files/etc/config/
scp root@box:/etc/config/network files/etc/config/network
(edit as desired)
make V=99

In the root of the directory that you checked out the distribution to, create files/etc/config/, copy an existing network config file you have into it (or the default as a template and edit).

Job done - this file will be written into your firmware image.

http://wiki.openwrt.org/doc/howto/build#custom.files

E.g. if your <buildroot dir> is /openwrt/trunk and you want some files to be copied into firmware image's /etc/config directory, the correct place to put them is /openwrt/trunk/files/etc/config.
