to Jin

I think this problem came from a big package wasnt fit into your flash, right?

what I did is following

I was trying installing samba, then I got the espace error and bla bla bla

then I deleted the lock file, and then, I checked usr/lib/opkg/info
look for the package you failed to install due low space

in my case samba-common.list

open it with any text editor

it will show some files It "installed", but most probably it failes to do, so just delete your last installation package files, and then perform a :

opkg remove "the last app you tried to install"

it should work
