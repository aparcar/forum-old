1. http://bit.ly/MCusMd (you should read openwrt.wiki and google for it
2. Wireless is disabled by defualt (but you can enable it with "vi /etc/config/wireless " remove line where it says.
3. make sure you enabled networking with default options  with
"/etc/init.d/uhttpd enable"
"/etc/init.d/uhttpd start"
(restart router after that)
try to update and install luci package
