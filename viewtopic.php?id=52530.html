Hi,
I picked up a Dlink "DIR-862L Wireless AC1600 Dual Band Gigabit Router" and I wanted to see if openwrt would run on it.
Dlink has a GPL tarball available for download, I did download it and following its README I managed to build the stock firmware for it.
So far its looking promising.

It has 802.11AC wifi, a USB port for file sharing, 4 gigabit ports for LAN, and a gigabit port for the WAN.

I soldered a USB to 232 TTL adaptor to the available holes, and now I can view the boot messages and issues commands to poke around the system.

I don't see too much information about this router yet. Anyone else looking into this one?
Any chance the wireless will work under openwrt without much trouble?

I finally figured out how to get into this bootloader.

You need to hold down the reset bottom on the bottom of the router while powering it up.
This puts the bootloader in httpd recovery mode and you can access it over the network to load a firmware.

While in this mode on the serial console you can also press Ctrl + C and break into the u-boot shell.

hope this helps others
