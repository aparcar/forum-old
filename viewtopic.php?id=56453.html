So if i sum up based on the image: you need an Accesspoint, while dhcp / dns ect. is still handled by your already existing router?

The minimumneeded for that would be:

uci set network.lan.ipaddr=192.168.100.2
uci set network.lan.gateway=192.168.100.1
uci set dhcp.lan.ignore=1
uci commit

will:
- set the openwrt IP to 192.168.100.2
- use gateway 192.168.100.1
- disable dhcpd on your openwrt box

You can still do more (like disable the WAN interface and add it to the internal switch depending on your model).
The link you posted is a bit outdated (for an older openwrt version), maybe take a look here:
http://wiki.openwrt.org/doc/recipes/dumbap

To allow ssh, just set a password. Setting one will  enable your dropbear sshd.
After rebooting you can  ssh to root@192.168.100.2 to access your openwrt box.
