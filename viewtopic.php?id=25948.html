I forgot to mention: the device uses a GCT chip and the source for the modified Linux kernel used by the device is available for download. Per the "Modified Directory List" posted on Infomark's site, the changes to the kernel were made in the following directories:
arch/arm/
arch/arm/boot/compressed/
arch/arm/kernel/
arch/arm/mm/
drivers/char/
drivers/i2c/busses/
drivers/mtd/chips/
drivers/net/
drivers/net/wimax
drivers/net/wireless
drivers/serial/
drivers/usb/gadget/
include/asm-arm/
include/asm-arm/arch-vincent7/
include/linux/
include/net/
mm/
net/core/
net/ipv4/
arch/arm/mach-vincent/

This seems rather promising. Thoughts on next steps?
