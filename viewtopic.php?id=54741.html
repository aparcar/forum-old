I don't see the problem. The webinterface doesn't use virtual hosts, so any domain which resolves to your ip address should do.

If you want to use the public interface you'll have to set a port forwarding from the wan to a lan address. Or set a firewall rule to permit wan access on that port. I'd rather use https than http. In that case you'll have to enable https either.
