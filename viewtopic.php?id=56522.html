I wandered why I managed to break my router configuration several times. The only chance to get it working again was to key in everything again after doing a factory reset.

To get down to the source of the issue I saved my configuration to my PC and diffed it. Et voila, several unexpected changes. One of the most obvious where that the ' (quotes) are gone.

And if you assume the UI does some consistency checks, be also warned. The trailing blank in the DNS server list managed it into configuration. I haven't checked but my experience is that there are no checks at all. They could have done a consistency check IP and if there is a DNS server answering, but...

I really wonder how anybody managed to create a non trivial configuration with this UI. Where is right place to drop some bug reports?
