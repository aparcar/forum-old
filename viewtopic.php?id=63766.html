Have you looked at the docs on bridged client mode?  There is also a thread in Community Documentation on setting up repeaters: https://forum.openwrt.org/viewtopic.php?id=39077

What is the source of the wireless network you are trying to extend?  Is it another device running OpenWRT? What device, and more importantly, what chipset?

The ideal situation for setting up an OpenWRT device as a WiFi extender is when the main router/AP is also running OpenWRT on a device capable of running 4-address mode, which includes Most Qualcomm/Atheros devices, the MediaTek/Ralink devices I've tried, and probably most everything else, EXCLUDING broadcom.

Assuming that is the case, configure the main router so that the access point is configured as "Access Point (WDS), rather than just "Access Point."  Reset the new device. Configure the LAN (br-lan) interface with a static IP in the range of your main network, so you can always connect to it. Disable the DHCP server on the LAN interface. From here, you should be able to set it as a bridged-client using WDS, similarly to the description here (not limited to Atheros devices): https://wiki.openwrt.org/doc/recipes/atheroswds
