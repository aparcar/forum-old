There are a few sendmail like programs available (mini-sendmail, ssmtp) to send email from the shell. There is also a possibility of using full sendmail server. I haven't managed to build a package myself, but unslug wl500g repository does have a sendmail package and it works perfectly with WR 0.9. It's rather big, so I've build myself a squashfs image containing all the software I'd need (including sendmail and bind) - It's over 5MB so you still need 8MB router though, but it has everything
