For each VLAN you need to add an interface (bridge).  I use names such as BR_VL9 for
VLAN 9 VID 9.

You add / setup a DHCP server for each interface

In advanced setting DHCP Options you add option 6
6,8.8.8.8,8.8.4.4 (for Google IPv4)
