https://dev.openwrt.org/browser/trunk/t … twork#L294
That script generates the network config on the first boot to /etc/config/network. It is not stored anywhere.
        ucidef_set_interfaces_lan_wan "eth0" "eth1"
        ucidef_add_switch "switch0" "1" "1"
        ucidef_add_switch_vlan "switch0" "1" "0 1 2 3 4"

eth0 = lan
eth1 = wan
That was BB trunk version.
