Post #2
dziny
24 Apr 2016, 22:32

Most likely you've lost connection because the lan IP of the router is not in the range your network has.
Manually setup IP on your computer to be in the IP range of that router and try to ping it. If this works,  ssh into the router and change its ip to a static one that is in the normal range your network operates on. Now you should be able to access it whenever you want.
