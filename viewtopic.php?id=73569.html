Yes, it's trivial to do, whether through the web interface or the CLI.

By default, the router will issue IP addresses via DHCP in the same subnet as the LAN interface. If the IP address changes, the DHCP scope also changes automatically.

Remember that the IP address will revert back to 192.168.1.1 if the router is ever reset to defaults.
