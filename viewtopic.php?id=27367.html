Post #2
aport
17 Nov 2010, 20:38

The 3G side of the U600 uses the cdc_acm driver, which should create /dev/ttyACM0. The built-in 3G scripts in OpenWrt should bring this online without much effort:

config interface wan
        option device /dev/ttyACM0
        option proto 3g
        option service cdma

You won't be able to use the wimax side without proprietary drivers from the chip manufacturer.
