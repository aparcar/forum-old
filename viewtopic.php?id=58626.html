I don't know anything about switches, but are the connections configured in this router or further up the chain?  By that I mean, are these managed, static interfaces in OpenWrt or are they DHCP Clients?

If the Wi-Fi only VLAN interface is configured using the 'static address' protocol (where you assign an IP address and Gateway to it), you can just put it on a different subnet from all the "networked" interfaces.  Once you do that, it's really easy to connect or isolate it using the firewall zones, but by default, just having it on a separate subnet will make the connected clients isolated from all the other access points on the standard subnet.


Read this: http://wiki.openwrt.org/doc/recipes/guest-wlan


And this forum thread, specifically, which will show you how to configure it for Internet-only access: https://forum.openwrt.org/viewtopic.php?id=28317
