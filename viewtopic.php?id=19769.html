on 4MB there is more than enough space for OpenVPN and all its dependencies.
I guess you had a problem with the latest libopenssl_0.9.8i-3: It produces a false error: "not enough memory". You have to use the slightly older libopenssl_0.9.8i-1 from 8.09 RC2 instead which works perfectly.
