I've been using the Netgear WNDR3800.  I bought a refurb on Newegg for about $70 CDN.

It works like a charm.  So far, no lock ups.  It has speed to spare, for my purposes.  I have a 100Mb/s FibreOp link from Bell Aliant.  It has had no problem operating at full bandwidth with connection tracking and QOS configured.   Both the 2.4GHz and 5GHz wifi have been rock solid.  USB is fine too, though, I only ran it through its paces, with a high load, long enough to make sure the router was fully working.
