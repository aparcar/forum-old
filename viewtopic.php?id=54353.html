Hello,

I've installed OpenWrt Barrier Breaker 14.07 / LuCI Trunk (0.12+svn-r10530) on my Router Linksys by Cisco WRT160NL:
http://wiki.openwrt.org/toh/linksys/wrt160nl

I have successfully added tc and it's dependencies.

What I want to do is to have tc netem rules in my local area network. That means two or more computers are connected on different LAN ports at the router and i want to introduce a deterministic loss between them without configuring the computers.

I set something like:
    

tc qdisc add dev eth0 root netem loss 10%

but that configures loss to the router (192.168.1.1) only and not between my different computers.
It seems the communication is only going through the switch without passing the router.

Is it possible to set up some VLANs to get a behaviour as described above?
Something like this would be useful:

tc qdisc add dev eth0.1 root netem loss 10%
tc qdisc add dev eth0.2 root netem loss 10%
tc qdisc add dev eth0.3 root netem loss 10%
tc qdisc add dev eth0.4 root netem loss 10%

How do I create VLANs so that communication between those computers is still possible?
