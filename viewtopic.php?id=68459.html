Recent source commits bump the kernel to 3.18.44 in Chaos Calmer, which fixes Dirty Cow. For now, if the binaries are old, you are correct that you need to build your own image from source.

One of the less endearing aspects to OpenWrt is the lack of sufficient developer attention to security fixes and updates for the stable version.

Wigyori commited this fix 20 days ago. Given the severity of the bug, there's certainly been enough time to get the buildbots working on a minor version bump to 15.05.2
