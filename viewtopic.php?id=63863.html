meazz1 wrote:

I am using daily build from snapshot download link

what you have is the prebuilt trunk.  update is easy with the luci flash new firmware image function, keep settings if needed and there is also the backup file list where you can list files or scripts you want to keep with the new firmware image.  luci is not included with trunk so you need to install.

there is no real tangible benefit in keeping up with nightly builds unless the change applies to your router.  i usually check here https://dev.openwrt.org/browser/trunk to get an idea on the changes.  others can chime in if there is a better way to check changes.
