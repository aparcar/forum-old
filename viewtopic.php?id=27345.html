I'm using a Ubiquiti RSPro and an SR71-A card with 10.03.1-RC3 installed. The card seems to work fine, but after some time (anywhere from hours to days) the card seems to "reset" and I can no longer associate with it or see it on scans. wlanconfig on the router shows the card, but it does not show the ssid and stuff that it normally would:

# iwconfig wlan0
wlan0     IEEE 802.11abgn  Mode:Master  Frequency:5.805 GHz  Tx-Power=24 dBm   
          RTS thr:off   Fragment thr:off
          Power Management:off

Dmesg | tail just shows this:

device br-lan left promiscuous mode
device wlan0 left promiscuous mode
br-lan: port 2(wlan0) entering disabled state
ADDRCONF(NETDEV_UP): wlan0: link is not ready
device wlan0 entered promiscuous mode
br-lan: port 2(wlan0) entering forwarding state
device wlan0 left promiscuous mode
br-lan: port 2(wlan0) entering disabled state
device wlan0 entered promiscuous mode
br-lan: port 2(wlan0) entering forwarding state

/etc/config/wireless:

config 'wifi-device' 'radio0'
    option 'type' 'mac80211'
    option 'macaddr' '00:15:6d:84:66:1b'
    list 'ht_capab' 'SHORT-GI-40'
    list 'ht_capab' 'DSSS_CCK-40'
    option 'htmode' 'HT40-'
    option 'txpower' '24'
    option 'channel' '161'
    option 'hwmode' '11na'

config 'wifi-iface'
    option 'device' 'radio0'
    option 'network' 'lan'
    option 'mode' 'ap'
    option 'ssid' 'xxxxxxxxx'
    option 'encryption' 'psk2'
    option 'key' 'xxxxxxxxxxx'
    option 'eap_type' 'TLS'
