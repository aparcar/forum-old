I've just built a binary.  You'll find it on

  http://www.pps.jussieu.fr/~jch/software … -mipsel.gz
  http://www.pps.jussieu.fr/~jch/software … sel.gz.asc

It's just a binary, not a package, so you'll need to write your own init script.  You'll
probably want to run it as so:

  polipo diskCacheRoot="" chunkHighMark=4194304 logFile=/var/log/polipo.log daemonise=true

You may also want to set up a new user so that you don't run it as root.
