OpenVPN in combination with Windows 7 has an issue about forwarding UDP broadcasts to all adapters.  No matter what adapter priorities are set in the operating system games like to send broadcast requests for LAN servers on the main LAN adapter, not the VPN one.

For Blur with OpenVPN I first tried to use hanksoft UDPBroadcastForwarder, but that didn't work, however HamachiBroadcastFixGUI did. Make sure to run the forwarder on the client computers.

Another alternative that might work is to use Tunngle instead of OpenVPN. Tunngle is a currently free game centric VPN based on OpenVPN. Since Februrary they allow you to set up password protected private "rooms" with up to 32 players.
