I've been trying to duplicate a Thomson configuration on one draytek router, which has a modified version of OpenWRT Kamikaze. I've been able to duplicate almost everything except the IPTV part because of this:

To make IGMP work on my ISP, the thomson configuration needs to use a so-called "OBC Bridge". I initially thought that this was a Out-Of-Band interface (OOB), but it doesn't seem to be it. So, after researching for its meaning/function, i was able to gather the following bits and pieces of information regarding this special bridge port:


A) OBC: Is short for On Board Controller and indicates the physical bridge port.

B) A special bridge port is the OBC, which is an internal bridge port and can be seen as the port to the network layer.
