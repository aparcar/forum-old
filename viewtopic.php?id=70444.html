I've been following through the wiki trying to put together a OpenWrt router in virtualbox for a small network (3 virtual machines).  I've successfully connected the first machine to the router and the internet but I'm having trouble making a connection to the second computer. 

Just throwing configs at the wall to see what sticks, I copied the config for the first computer in the /etc/config/network file as follows.

config interface 'lan'
     option ifname 'eth0'
     option type 'bridge'
     option proto 'static'
     option ipaddr '192.168.1.1'
     option netmask '255.255.255.0'

config interface 'lan2'
     option ifname 'eth2'
     option type 'bridge'
     option proto 'static'
     option ipaddr '192.168.1.1'
     option netmask '255.255.255.0'

The interfaces eth0 and eth2 are "internal network adapters" in virtualbox.  Any help will be appreciated.  Thanks!
