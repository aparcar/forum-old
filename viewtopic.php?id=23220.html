wavesailor wrote:

Hey cybermcm, did you ever find a Luci frontend for WOL?

The Luci WOL module "luci-app-wol" exists for the current trunk and self-built Backfire, which use Luci 0.10 or even newer Luci trunk. But the module does not exist for older Kamikaze/Whiterussian/8.09.2 versions, nor for Backfire 10.03.1-rc4 from November 2010 (that still used Luci 0.9).

I have not tested the WOL module, but it seems to exist.

You should either self compile OpenWrt, use the OpenWrt trunk snapshots ( http://downloads.openwrt.org/snapshots/trunk/ ) or Backfire 10.03.1-rc5-test snapshots ( http://downloads.openwrt.org/snapshots/ â€¦ 5-testing/ ), and then possibly manually install the luci-app-wol package (is the auto-configured download directory link points wrongly).
