dustyuncle wrote:

I'm not trying to do anything special at the moment, just turn DHCP on so it can take the PPPOE connection from my main router using a regular LAN port from the main router.

The "lan" in OpenWrt is a software bridge that includes all LAN ports. If you really want to use *one* of LAN ports for a different purpose, you need to configure it to behave separately first.
