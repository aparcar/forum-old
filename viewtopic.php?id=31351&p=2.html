after place file image to folder "/tmp" you need rename that... example rename "code.bin"

you copy command  below to putty:

cd /tmp
mtd -e firmware -r write code.bin firmware 
reboot

but if you copy sysupgrade.bin to /tmp, that command:

cd /tmp
sysupgrade code.bin 
reboot
