9 Jan 2009, 21:34

When you start transmission-daemon the file /root/.config/transmission/settings.json is created
It contains the settings including which port the web interface lives on. Default is 9091

You may need to change rpc-whitelist if you want to access it remote.
Mine looks like this:

    "rpc-whitelist": "127.0.0.1,192.168.1.*",
