Hi all,

For those not familiar with NZBGet, it's a usenet/newsgroups downloader which can also run on OpenWrt. It's a nice power saving solution for those that not want to have their desktop running 24/7.

I know NZBGet is outdated in the repositories, but I managed to get it to work on Barrier Breaker and resolving all dependency problems one encounters using this outdated package. Here's the video: https://www.youtube.com/watch?v=HItT02gLZJk

Btw, if someone could update the nzbget packages, or maybe cross compile the latest NZBGet for MIPS big endian, that'd be awesome!
