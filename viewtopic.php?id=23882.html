oleg106 wrote:

What is the difference between these two ..

openwrt-ar71xx-dir-825-b1-squashfs

My understanding is "...-backup-loader" will be accepted by DLink's firmware as an acceptable upgrade.  If you're flashing from any 3rd party firmware, I think they're the same.

For the 5Ghz radio, make sure you've installed crda and set the reg domain.  See this post for more info: https://forum.openwrt.org/viewtopic.php … 82#p101782
