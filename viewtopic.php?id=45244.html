Well, if you wanna get into routers, why not try to un-brick it?
That would be good practice and you would learn quite a bit! Plus: extremely satisfactory to see the router working again after you unbrick it. It's pretty easy, but it would require you to open up the router.

If you want to, step 1 is to buy an DKU-5 (also known as CA-42) Nokia cable. They're really cheap - a few Euro/dollars - online.

Btw: flashing and unbricking can be done from Windows, so no worries about that!


//edit: to answer your question in the topic title: I don't think you did anything wrong. It was just bad luck. Take a look here:
http://downloads.openwrt.org/snapshots/trunk/
This is where the latest revision of Openwrt (known as "trunk") is automatically uploaded. These are not "stable" releases, but rather the latest and I believe they are uploaded once a week.

The folders are named after the different chipsets of routers. The WR703N is in the ar71xx folder. You will see a factory file (to flash from stock firmware, like you) and a sysupgrade file, which is for upgrading if you already have openwrt on the device.

Another tip: read the wiki of your router. It contains so much info.
http://wiki.openwrt.org/toh/tp-link/tl-wr703n

Info on how to install openwrt on the device, how to enter failsafe (which you can still try!!) to debrick, how to get back to stock firmware, etc

Here is how to unbrick using the Nokia cable: http://wiki.openwrt.org/toh/tp-link/tl- … al.console
Here is how to enter failsafe: http://wiki.openwrt.org/toh/tp-link/tl- … lsafe.mode (try this first!!)

read through it it will help!

Wait.. are you sure the router is actually bricked? Openwrt often doesn't come with Luci - the webinterface control panel.
In this case going to http://192.168.1.1 will give you nothing, you would have to configure it using Telnet/SSH (you can download Putty for that on Windows)

Maybe connect you computer to the router and try pinging it? (start CMD on Windows - type "ping 192.168.1.1")
Does it give you anything? In which case the router isn't bricked.

and no, you do not need internet to flash openwrt
