It is pretty straight forward on Raspberry Pi and x86:
http://techfindings.one/archives/2175

I suppose all ARMv7 targets are equally easy, but I have not tried.

When it comes to MIPS... I did build an old 0.10 release for mipsel (ramips/mt7620):
http://techfindings.one/archives/2064

Problem with MIPS is that 0.12/4.0 produce "Illegal Instruction" (both mipsel and mips).
I think it can be solved by building the kernel with FPU emulation (this is default in Linux, but disabled in OpenWRT).
I am currently investigating, but I am not done.

I would expect horrible performance of typical MIPS targets (such as mt7620 and ar71xx), but if I get Node.JS running I will surely test the performance as well.
