I assume you are looking for the Services tab for USB support.  You first need to add the files for USB support.  Start here: https://forum.openwrt.org/viewtopic.php?id=58475

As for the wifi, you need to go into wifi and a second connection under Radio0.  If the device you have says Mode:client it is your WAN side connection.  Click add and follow the instructions to create a wireless AP that you can use to connect your PC\phone etc.

Be aware that when you change the "Mode:client" connection you may also need to recreate the AP connection.  I use a USB wireless device (search EDIMAX 7811) for this reason.  Yes it uses the USB port.  https://forum.openwrt.org/viewtopic.php?id=58548
