Hello,

I can tell you that I use OpenWRT for quite some years even before purchasing my TP-Link  1043ND router I flashed it on the same day, so I can't give much feedback regarding the original setup.

What I can tell you is that OpenWRT works great in TP-1043 ND. I've been using it for 3 years and in January I bought a new TP-1043ND V2 and works great. Both routers have the root FS in a USB pen so I have lots of space to add applications. I couldn't be happier.

I even have a VLAN setup in my house with both TP-Link routers.  https://forum.openwrt.org/viewtopic.php?id=56016

The process of installing openwrt in these routers is pretty straight forward. Never had any problem and I've been doing either initial flashing as well as release upgrades.

I'd definitely recommend OpenWRT in TP-1043ND.

Hope this helps,
Jabss
