Hi nbd

Just want to let you know, that your feb 7th build has some problems with wireless (nas)

After some time, the wlan simply dissapears....it leaves a nas running like this :

/usr/sbin/nas -P /var/run/nas.lan.pid -l br0 -H 34954

Instead of with the entire string like this:

 /usr/sbin/nas -P /var/run/nas.lan.pid -l br0 -H 34954 -i eth1 -A -m 4 -k [PASSWORD] -s [SSID] -w 2 -g 3600

This in effect renders it useless :-)

I have seen it twice over the last 10 hours or so now.
