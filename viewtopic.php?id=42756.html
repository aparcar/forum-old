So you already built your own image? Then just build a new image with the kmod-usb2 included (i.e. select it from the menuconfig) and install that. Kernel modules are always tied to the exact kernel you have built, so downloading one from the repository does not work.

Of course, you could've installed the supported version 12.09-rc1 directly from http://downloads.openwrt.org/attitude_a … 71xx/nand/. In this case you can just download additional modules from the repository.
