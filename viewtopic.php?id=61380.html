Here is an answer to anyone who may run into this same problem.

Keep in mind "sysupgrade" in the filename.

Keep in mind "sysupgrade" in the filename.

If you want to upgrade from an older OpenWrt version, use the sysupgrade image.
If you are coming from factory firmware, and want to install OpenWrt for the first time, use the factory image.
