If it's a DSL modem and the router combo, afaik you will lose the DSL modem functionality if you flash OpenWrt.

If your ISP-supplied router is intercepting DNS requests on port 53, you may check if OpenDNS supports other ports (like 5353) and if you can set laptop to use the non-standard DNS port.

Overall tho, wayne1958 is right, if you want to be in control it's better not to use the ISP-supplied combos. See if you can buy a standalone DSL modem which you can put in a bridge mode and use your own (ideally OpenWrt-compatible) router with.
