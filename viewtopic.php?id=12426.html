You need a development system which requires a Linux box. Since your're on a Windows host the easiest way is to install some Linux distribution in a virtual machine. For the VM on Windows I'd recommend the free VMware Player and for the distribution Ubuntu Feisty Fawn 7.04. Setup and configure this first, then come back and we can talk further.
If you don't like VMs and your Notebook or PC supports to boot from an external USB harddrive get a spare one and install Ubuntu there. This is what I'm doing.

Some useful URLs:
- http://www.vmware.com/products/player/
- http://www.easyvmx.com/
- http://www.ubuntu.com/
