The problem appears if you want to compile extra packages.
Do "svn co https://svn.openwrt.org/openwrt/tags/kamikaze_7.09" after it do
"make menuconfig" then load the packages "svn co https://svn.openwrt.org/openwrt/packages"  then create the symlinks. Now "make menuconfig" doesn't show  the error. If you don't do "make menuconfig" before you load the packages then you get that error.
