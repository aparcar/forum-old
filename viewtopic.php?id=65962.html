Sounds like the original OS of the buildhost did not contain all required libraries/compilation tools, or the tools were targeting a too modern kernel version. Installing another OS helped. Note that the working "new" OS is a quite ancient one, instead of a modern one...

See also similar discussion in https://forum.openwrt.org/viewtopic.php … 71#p295171

That kind of problems may happen if you try to compile either "current Openwrt source with an ancient Linux" or "ancient Openwrt source branch with a modern Linux"
