I see TP-Link WR740N v5.0 listed there and on the last page I see "Revert file for TP-Link WR740Nv5", the file listed there can be downloaded, just make an account on dd-wrt forum to see the download link.

Revert back to TP-Link firmware and flash OpenWRT from TP-Link firmware. Think I read on this forum that dd-wrt -> openwrt end up in bricked routers. Just to be safe do dd-wrt -> webrevert to tp-link firmware from dd-wrt (the webrevert to tp-link firmware provided by dd-wrt, not other one) -> openwrt.
I'd use an UPS when I flash the router just to make sure that a random power cut will not brick it.

If you don't see a login option in dd-wrt forum (I actually don't see such an option, supposed to be a link to the login page http://www.dd-wrt.com/phpBB2/login.php but I see none in dd-wrt forum...) just click on new topic, it will show u the login page...

At least for my 841nd v5 the webrevert from dd-wrt has some bits added to the start of the image compared to the original tp-link firmware. Probably those bits are added to all the webrevert from dd-wrt (there might be an answered somewhere about the particular added bits but I never looked for an answer , some of them are the same for all and the one that are not the same look to be some sort of checksum).
