It might be silly questions. But, ...
- What's the output for swap usage when `free` runs?
- Did you make the swap partition in an USB storage, not in the soon-to-be-full flash memory? (IMHO, the swapping should be done blindly by the OS(linux), not by an application such as transmission or rtorrent.)
- You application still crashes for just seeding, if you remove all downloads, emptying the download queue?
- How about if you limit the peer amounts, the cache-size, ... in settings?
