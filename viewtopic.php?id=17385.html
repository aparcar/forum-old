Hi.

Same subnet on LAN and WAN does not work. It confuses the kernel so that it's unnable to route correctly.
Use a different subnet for lan and wan (like 192.168.1.1/24 and 192.168.2.1/24).

~ JoW

config interface lan

    option ifname    'eth0.0 eth0.2'

    option ipaddr    192.168.1.1
    option netmask    255.255.255.0


#### WAN configuration
config interface    wan
    option ifname    "eth0.1"
