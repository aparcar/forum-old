As I am still playing around with my HAME MiFI F1 (https://forum.openwrt.org/viewtopic.php?id=46549) and looked now through several other boot-logs and information, I got almost sure, that none of those companies are producing their software themselves.  Searching around, I figured out, that RALINK, now Mediatek has an SDK, where the manufactures can configure the firmware menudriven through all parts from u-boot, linux.kernel and all embedded tools.
The user manual describes a lot of hardware aspects, how you can drive certain GPIOs and configure the Soc from the running system.

Unfortunately I found up to now no source, to download the SDK (you need version 4.1.0 that RT5350 is covered).

I was only able to read in the User-Manual, but unable to download it.

When you go to the chinese search-engine baidu.com and enter 
"RALINK SDK 4.1"
as search term, one of the first entries is a link to the PDF. However that is presented in a browser based reader, where you can read it in full, but print only the first 5 pages.

I found no way downloading the PDF and the corresponding SDK, as all explanations are chinese and google-translate is getting problems with a lot of javascript in this page.

Has anybody managed to get hold on those files or has an idea, where you can get it?
