I have openwrt running on my G3010, it was supported by the ifxmips target which is now merged into lantiq. At the time I flashed the device (2 years ago), there was no usb, vlan, dsl-support making the device a dumb switch with openwrt on it.

It would be cool to start with vlan-support for the switch, so the device can function as a router. There's currently a discussion going on the mailinglist: http://article.gmane.org/gmane.comp.emb … devel/9246
