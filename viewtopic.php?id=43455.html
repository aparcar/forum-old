* adults (This has to have full internet)
* kids
* media (this is for mediaplayers)
* servers (and printers)

some of them will have wifi connected as well (mainly main, kids and maybe media)

Adults, and partly servers, should have full internet connectivity
Media and Kids should only be allowed internet based on their mac addresses (a list of specific mac addresses on each network, which have internet connection).

media players : 192.168.0.48 - 192.168.0.63
