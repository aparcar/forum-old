Build you own image, include everything you need, and remove everything you don't need.
you probably don't need ppp and pppoe (+kernel modules), if you don't use it as router you won't need firewall, iptable, etc. and if you include everthing in the image you don't need opkg either.
But maybe you can use something elso to read the serial which doesn't need a big package like python.
