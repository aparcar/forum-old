In a regular setup openwrt comes with dnsmasq enabled which acts as a caching dns proxy for local clients. It automaticly uses the dns servers from wan uplink as forwarders. There should be no need to configure the clients.

If you still want to set the dns servers on the clients you should use dhcp settings on the router.
