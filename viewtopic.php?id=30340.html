Hi,

I don't think this setup will work well, and I don't think you will get the expected bandwith-gain of 3 sector-antennas if they are connected to one (mimo)wificard.

If you really want to use 3 sectorantennas you should get 3 wificards and a board which could hold them (routerstation (pro), mikrotik routerboards,...) with this setup you could run every sector antenna on a different channel which would minimize the interference.

If you need high bandwith in each sector, you could use a mimo-sector-antenna. this antennas can send a vertial-polarized stream and the horizontal-polarized stream in the same direction (2t2r).

the question is what do want to achieve, and whats your budget.

big budget and many clients which connects from different destinations 3 sectorantennas + 3 wificards. If you need high bandwith maybe mimo-sector-antennas and mimo-wificards.
little budget only one wifi and invest your money in a good omidirectional antenna.
