Enter bios and check if it can see the disk.
Ensure VM is configured to use MBR not GPT/EFI.
Check boot order.
Try configuring disk as SATA.
X86 openwrt uses grub. Grub is loaded by bios. If bios can see the disk and grub does not load then vmdk file is bad
