You'll want to double-check that DNS works correctly both in the router and on your pc behind it. Assuming that both systems are able to resovle DNS correctly then the issue is going to be a firewall configuration. Try 

/etc/init.d/firewall stop

If that works then you'll need to take a look at the /etc/firewall.user file to configure the firewall.
