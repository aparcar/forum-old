Hello everybody,  I need your help. I have configured isatap tunnel in my router and my router can access IPv6, but how to assign IPv6 address for my local machines?

My modification:
/etc/config/network

config interface 'wan6'
  option proto '6in4'
  option ipaddr 'my ipv4 address'
  option peeraddr 'isatap ipv4 address'
  option ip6addr 'isatap ipv6 address:my ipv4 address/64'
  option ip6prefix 'fc00:512b:512b::/64'
  option delegate '0'

P.S. I cannot install package radvd.
