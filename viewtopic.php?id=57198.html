You compile a version of collectd for your Debian server with the same additional datatypes that the patches in Openwrt have added to collectd in Openwrt.

there is at least "stations", defined by this patch:
https://github.com/openwrt/packages/blo … ugin.patch
The change to types.db is at the end: https://github.com/openwrt/packages/blo … patch#L270

I don't think that the other patches have impact for you, but you might still check them:
https://github.com/openwrt/packages/tre … td/patches

EDIT:
the file types.db might be editable directly in the Debian system.
Read https://collectd.org/documentation/manp … db.5.shtml
