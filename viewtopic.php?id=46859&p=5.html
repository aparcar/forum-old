Post #110
ofutur
4 Jan 2016, 21:26

For people starting with these devices:

Wiki: https://wiki.openwrt.org/toh/tp-link/tl-wa850re
Official Chaos Calmer v15.05 for the tl-wa750re: https://downloads.openwrt.org/chaos_cal … actory.bin
Initial config: https://forum.openwrt.org/viewtopic.php?id=39077

1) Just use TP-LINK's GUI to upload OpenWRT
2) Connect LAN cable
3) Go to 192.168.1.1
4) Set a password
5) SSH to device
6) Paste and adapt config from forum example
7) Finalise from the GUI

If you mess up the config and cant access the device, just press reset until it reboots and start again.
