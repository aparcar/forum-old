Shockingly Max, you actually offered a useful link, albeit in your normal arrogant and condescending manner.

For those interested in the answer with out the drama, the following registry key on the Windows client needs to be added or modified.  With no key the default is 45 seconds.

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\LanmanWorkstation\Parameters]
"SessTimeout"=dword:00000bb8

You should be able to copy the above directly into a reg file.

I have used the value of 3000, but it's probably overkill.  The value is seconds in 'decimal' for the key.  The setting takes effect immediately (no reboot needed).  The MS support references are here:
