Hi,

I have a 4gb crucial USB stick I'm trying to add to my ASUS WL-500GP v2 router. I'm doing this in other to install more packages. The router isn't detecting the USB. Here is what I have done (I'm only interested using usb version 2):

1) Flash the router with kamikaze 8.09.1 r16278 (openwrt-brcm47xx-squashfs.trx)
2) configure the router and install a few packages

Install USB kernel modules
=====================
opkg install kmod-usb-core
opkg install kmod-usb2
opkg install kmod-usb-storage
opkg install kmod-fs-ext3
opkg install kmod-nls-cp437 kmod-nls-iso8859-1
opkg install usbutils
Rebbot the router

Installing formating tools
===================
opkg install e2fsprogs
opkg install fdisk
Reboot the router

Configure USB
============
Format USB disk using fdisk and mke2fs
Attach USB disk  to router

Testing if router is there
===================
lsusb      =>  It 's not there
fdisk -l    => Nothing is shown
dmesg    => The usb disk isn't detected

I tried other usb disk i have it is still not showing.

Is there something wrong with the version of Kamikaze I'm using that prevent the USB from showing? I'm missing a few steps?

Thanks for your help,
Jude1747
