Hey Guys -

I've been using OpenWRT for a few years now and love it as using it has added the ability to add so many features to my otherwise plain-jane routers not to mention stability.

After getting fiber a few days ago, I decided it was also time for an upgrade to my primary router at home.  It just doesn't seem right to have internet that is faster than your local wifi.   I replaced my old BUffalo WZR-600DHP (which still wasn't that bad) with a new Linksys WRT1900ACS.  While waiting for it to arrive, I downloaded all OpenWRT files I'd need to install it.

After setting the new router up yesterday, I was playing in the OEM firmware but actually found that it seemed to be the best and most feature-rich OEM one I had used yet.  I've been using it for about 24 hours now and actually kindof like it - but - miss OpenWRT.

So... my question is - besides the thousands of packages OpenWRT offers (most of which I do not use), what would you guys consider the main reasons to proceed with installing / using OpenWRT over the OEM Linksys firmware?  Previously, I mostly used it for OpenVPN, guest network, and a few other options that OEM firmwares mostly didn't offer - but now it does. 

The main priority for me is performance, wireless speed, & range so whatever is better at that is probably what I'd go with.  The range of the WRT1900ACS is already about 20% better than the prior router, but that's probably just hardware.  Know I might be able to overclock or add more power to WiFi using OpenWRT, but don't know if it's worth it or safe.  I actually tried a build of OpenWRT on a new Archer C5 router I got about a year ago and AC (never tried overclocking it) never really worked correctly.  To be fair, I believe support for AC was in development at that time; but didn't know how it fared currently.

If it helps, I currently have about 25 devices (~20 are wireless - 5Ghz when supported) on my home network which now uses a 250Mbps fiber for it's source/WAN.  The biggest LAN users are my primary PC and primary Kodi PC.  Kodi is (now) connected via 5Ghz AC (650Mbps) to my primary PC (ethernet) which hosts all of it's content as well as many other things.  Next would be a Roku box (wireless) that my wife uses to play Gilmore Girls literally all night long (whole different topic ) via Plex hosted on primary PC.

So - any thoughts?  Thanks for taking the time to read and comment!!
