You could probably replace the rom with something bigger, maybe even the ram.
But that is a lot of work. Then you will need to create a new firmware (only the Tenda Ralink devices seem to be supported at the moment).
Is there official source code available? If not, then it becomes even more difficult.
J31 in your pic looks like it might be a serial connection. A multimeter might tell you which pad is what. If you can get that, then a boot log might help.
Personally, I would just get a supported router off ebay!
