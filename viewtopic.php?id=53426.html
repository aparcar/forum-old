Short answer: No.

Long answer: What are you trying to use to make calls? How do you plan to route calls from whatever you're using to make them, out to the public telephone service? Real VoIP is an entire industry and skillset of it's own...it's not simply a way to make phone calls over the internet. If you want to make free calls, just use Skype (or similiar), it's relative cost effective, easy to do.

If you want to run a PBX with SIP endpoints, voicemail and applications, etc all running on your own device with uplinks to a carrier using SIP trunking then look into Asterisk, but be prepared to learn a lot and do a lot of work.

No, VoIP is not a magic way for making free calls to PSTN. You will need to connect to some SIP provider for making calls to "normal phones" (but SIP-to-SIP usually free). Or make your own gateway and make calls through it, using your own phone line.

In short: no.
