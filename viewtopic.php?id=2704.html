I too just purchased a WRT54G v4.  After trying a few of the ping hacks and ended up using the Linksys interface to get OpenWrt on.  The first thing I did after a reboot (and as per the OpenWrt's Wiki) was to set boot_wait.  From there I've only fiddle-dinked around and did a test reflash (tftp) to see if boot_wait was set correctly.  It was.  I did reflash the Linksys firmware (tftp) and noticed that Linksys changed boot_wait back as I couln't re-reflash OpenWrt.  So, yes, with my WRT54G V4 I had to (twice) use the Linksys interface to get another firware on.

Hope this helps.
