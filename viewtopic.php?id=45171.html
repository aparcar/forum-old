I think I found the problem.  When I used backfire I had to install the kmod-brcm-wl-mimo module to get wireless to work.  I don't think the upgrade removed the module so AA was getting confused.  I tried a reset to factory defaults but ended up with a brick.  Thanks to the ASUS recovery tools I unbricked the router using ASUS firmware.  I then was able to install AA without issues.  I still see the following errors:
Jul  8 19:16:07 OpenWrt kern.err kernel: [    0.508000] bcm47xx_sflash: error registering platform driver: -19
Jul  8 19:16:07 OpenWrt kern.err kernel: [    0.516000] bcm47xx_nflash: error registering platform driver: -19
but everything seems to be working so far.
