Post #4
hnyman
3 Sep 2012, 16:37

That Backfire 10.03.1 is the last official release, but it is ancient.

You can download the correct new trunk snapshot from http://downloads.openwrt.org/snapshots/trunk/

You can use the LuCI GUI to update. First download the correct firmware and then flash it with LuCI. (Use the "sysupgrade" version of the firmware, not "factory install").
