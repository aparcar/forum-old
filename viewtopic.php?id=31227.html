type ifconfig in the terminal and make sure you have an ip address in the 192.168.1.* range.. if not do 

sudo ifconfig eth0 192.168.1.200

and try to ping it again. Also make sure before you do that, you right click on the NetworkManager Applet and uncheck enable networking, this will prevent ubuntu from trying to change the settings you will be putting in place manually.
