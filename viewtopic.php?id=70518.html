Check the firewall, the network config, and routing.  I find the UCI firewall very difficult to understand, so I roll my own iptables rules, so my examples aren't going to be much use.

I have a number of VLANs, 4 inward facing, 2 outward facing, and it all works but it took a while to work out.

Also, if you're going through a switch, make sure it supports vlans, and that it's correctly configured on the ports you're using for the vlans.
