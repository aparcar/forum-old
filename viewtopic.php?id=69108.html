It would be helpful if you could post your /etc/config/network and /etc/config/wireless files.   The screenshot does show most of the configuration, but it is helpful to see the full config.

Also, it looks like vid 50 is configured as DHCP (are you using it as the management network?) whereas vid 51 is "unmanaged".  Can you give a little more background on what you are trying to accomplish?

From what I can see, it does appear that vid 50 and vid 51 have no way to move packets between them, so you should have full isolation (what I think you are going for)

The only issue I see is that you have access to the management interface from vid 50, which could be a security issue depending on your configuration.
