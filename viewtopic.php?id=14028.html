Try using the Firmware Recovery Utility that comes on the cd that came in the box. You can write any firmware to it with this. Also, if using this utility on Vista, make sure that the lan is set to Private on the Vista side. Or just find an XP box to run Firmware Recovery from.

After recovery, make sure the thing reboots completely before unplugging it, or you'll have to do it again.
