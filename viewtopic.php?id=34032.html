Is kmod-macvlan listed among the installed packages of your firmware build? If not, install it an try again after restarting the network or rebooting the device.
You can do install manually by uploading the opkg file of your build to your ns2's /tmp directory by scp, then install with opkg -i /tmp/kmod-macvlan*.opkg.
A transcript of errors from logread would help, if the advise above does not solve your problem.
