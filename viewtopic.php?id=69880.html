Hi,

I'm trying to get a VTI to come up on a client router with a strongswan roadwarrior setup.  After the VPN comes up a run an updown script and this is what I see in the log:

Thu Feb 16 22:59:27 2017 daemon.info : 12[CHD] updown: add tunnel "ip_vti0" failed: No such device

I tried to add one manually and I get an error that makes no sense to me.  I enter:

ip tunnel add "vti1" local 10.105.0.200 remote 54.175.95.179 mode vti okey 2 ikey 2

And get back:

add tunnel "ip_vti0" failed: No such device

It doesn't make sense to me as this is not the name of the tunnel I entered.  Thoughts? Thanks.

Chris

Post #2

I figured it out.  You have to install the 'vti' package AND the kernel modules 'kmod-ip-vti' and 'kmod-ip6-vti'. I incorrectly assumed the kernel mods would be installed with the 'vti' package.  Silly me...
