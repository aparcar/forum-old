Hi I have a WRT150N and I am wondering if there is something I can do to block an IP range from being accessed from my wireless computers eg   I want to block my wireless computers from accessing the main router IP range.

Here is my configuration    Internet - Main Router(Lan Port 1) - (Wan Port) Linksys router for guest wireless

I need to block 192.168.1.1-254 from accessing the 192.168.50.1-254 but still allow them to access the internet.

Any ideas would be appreciated.  Thanks

Post #2
xsoldier777
20 Oct 2008, 21:33

Solution Found

iptables -I FORWARD -s 192.168.1.0/24 -d 192.168.50.0/24 -j DROP
