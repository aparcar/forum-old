For most routers the "factory" = "sysupgrade" + ID/padding to make the image compatible with the router's OEM flashing routine.

Like nitroshift said, you use "factory" for the initial installation of Openwrt.

You should likely install 15.05.1, not the old 15.05.
See https://downloads.openwrt.org/chaos_cal … x/generic/
