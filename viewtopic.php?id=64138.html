you can also reach gigabit speeds with atom board with at least 2.6ghz clock speed. Had one mine that has 8 cores and had 4 port gigabit nics (Intel based) which supports VLAN and LAN passthrough, so each port can work as a standard switch if you wanted to has ecc ram and the price is no where near buying a 5th gen intel processors.

no addon cards installed since all that I needed is on board.

not to mention that it always stays around 9watts (12wats at full cpu usage whcih is highly unlikely to happen) all the time with 15 watts at boot up.

dont have any wifi on the setup since I got a separate AP for that kind of task which is more reliable to be honest, you never go wrong with ubiquity AP.
