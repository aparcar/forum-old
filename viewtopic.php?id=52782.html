See here: http://wiki.openwrt.org/inbox/dsl-g604t ( G604T is almost the same hardware as DSL-524T ).
I used the sequence of commands listed here and works with my DSL-G604T.

quote "MEDIA FLSH"
binary
debug
hash
put "openwrt-ar7-squashfs.bin" "openwrt-ar7-squashfs.bin mtd4"
quote REBOOT
quit
