Currently, Staples has this NETGEAR AC1200 10/100 Mbps Dual-Band Wi-Fi Router (R6100-100NAS) $24.99 + Free S/H within US. It is an Atheros AR9344 based CPU clocked @560 Mhz with 128/128 MB RAM/FLASH. This router is supported by OpenWRT v15.05. TBH, I don't know if 560 MHz is powerful enough to run FreeSWITCH or YATE, but I think it should do with Asterisk. So, if you had a need for a 100 Mbps WiFi router, this may not be a bad deal.
