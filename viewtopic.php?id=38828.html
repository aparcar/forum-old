You should forget Backfire and take a new trunk.
3800 eats the same images as 3700v2. And there are separate trunk snapshots for 3800 in http://downloads.openwrt.org/snapshots/trunk/ar71xx/

There are also ready community images, tailored for 3700/3800 in forum's community builds section, that fully support all the buttons in 3800 etc.

My build: https://forum.openwrt.org/viewtopic.php?id=28392
(newest build from August 20th: http://koti.welho.com/hnyman1/Openwrt/t … 012-08-20/ )
