Hi, I don't know if you found an answer, but I got a lot of help from reading this: https://forum.openwrt.org/viewtopic.php?id=30810.

Basically to do tftp, you need to have a switch.  I have another router with dd-wrt that I configured as a switch using this http://www.dd-wrt.com/wiki/index.php/Switch.

That is a biggie there.  Without the switch your computer will have to make sure it has an ethernet link first and by that time the tftp server on the router would have been turned off.

Also, make sure you set your arp links and after you successfully tftp, make sure you delete those arp links with sudo arp -da (I was banging my head wondering why I couldn't get to the log in of the router nor ping it, and it's because I left this little step out.

Hope this helps!

sm0ke
