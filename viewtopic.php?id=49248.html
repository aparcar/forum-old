It is probably not for you.

Using Openwrt more extensively requries some (not much) Linux CLI skills.
Especially if you want to use ADSL capabilities, which are not part of the regular Openwrt scope.

Similarly "built-in VPN" is probably related to the Omantel firmware, not actually to the device. I doubt if Openwrt has support for that VPN.

You did not even mention the exact equipment for which you are looking for advice, so no wonder that nodoby has answered before me.

Give more details and edit the subject to be more clear (direct reference to the device), so you may get more answers.
