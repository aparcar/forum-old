Post #28
RoundSparrow
5 Jan 2009, 23:54

In the ar9102/ar9103 thread geminijatt was asking how to flash, via web interface.

geminijatt:  yes - web interface.  After make, in bin directory you should get openwrt-ar71xx-tew-632brp-squashfs.webui file - upload that to the web interface.  once OPENWRT is on, the only way i currently know to re-flash is to use the recovery mode web interface.  HOLD reset button for 30 seconds after power on, the status light should blink - connect via cabled network to 192.168.0.1 in brower and uplaod there.  The same openwrt-ar71xx-tew-632brp-squashfs.webui should work.
