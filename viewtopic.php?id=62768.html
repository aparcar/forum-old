Just use the "option macaddr" in the wifi-iface section.
(The file /etc/config/wireless must be edited).
For example...

config wifi-iface
        option device 'radio0'
        option network 'lan'
        option mode 'ap'
        option ssid 'my_ssid'
        option hidden '0'
        option encryption 'psk2+ccmp'
        option key 'my_password'
        option macaddr '01:02:03:04:05:06'
