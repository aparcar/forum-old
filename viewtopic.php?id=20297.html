Hi,

I'm using the "brcm-2.4" version of Kamikaze with the "kmod-brcm-wl-mimo" driver. I have been using it as a client (sta+wpa2) for a while and it has been working perfectly. Now I would like to add AP (ap+wpa2) to my setup and have configured the necessary information in the "/etc/config/wireless" file. The AP seems to start up as it should but when I try to connect to it the client fails to get an IP address - if I disable the encryption everything works. I have tried both WPA and WPA2 for the AP but have so far only been successful with an unencrypted network (The router's client mode is still using WPA2 though). Have I missed anything in my configuration or is this a known issue?

/etc/config/wireless

config wifi-device wl0
  option type broadcom
  option channel 11
  option disabled 0

config wifi-iface
  option device 'wl0'
  option network 'wan'
  option ssid 'One'
  option mode sta
  option encryption psk2
  option key 'FirstKey'

config wifi-iface
  option device 'wl0'
  option network 'lan'
  option ssid 'Two'
  option mode ap
  option encryption psk2
  option key 'SecondKey'

Kind Regards
Andreas
