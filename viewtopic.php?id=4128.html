None of them are for the WRT54G v3.1. The Image Builder you downloaded creates only images for the Texas Instruments AR7 platform.

For the WRT54G v3.1 use the images and the Image Builder found in .

Mux wrote:

what is the diffrence between openwrt-wrt54g3g-squashfs.bin and openwrt-wrt54g-squashfs.bin

Aren't the filenames self-explanatory?

openwrt-wrt54g-squashfs.bin = WRT54G models (all versions except v5)
openwrt-wrt54g3g-squashfs.bin = WRT54G 3G (that's for the UMTS router with cardbus)
