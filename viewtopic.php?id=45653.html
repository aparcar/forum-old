I set up my linksys wrt54gl v1.1  into 4 Vlans. I also want to ping successfully between Vlans.
So, I set up the /etc/config/network as the following link

https://forum.openwrt.org/viewtopic.php?id=16934

When I restart  /etc/init.d/network restart
It shows an error " route: SIOCADDRT: No such process "

I find the problem at these code.It seems like I can not add the static routing table.
What I should  do, please help...


###### You only need the stuff below if you want to be able to route from vlan0 to your new vlan2 and vice versa! if they are going to be isolated you can leave this out.
config route
    option interface lan
    option target 192.168.2.0
    option gateway 192.168.2.1
config route
    option interface lan2
    option target 192.168.1.0
    option gateway 192.168.1.1
