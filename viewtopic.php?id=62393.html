There really aren't any beyond those for the shell itself. Settings are instead contained in the UCI System and can be accessed with the uci command and some helper functions in some .sh files under  /lib/.

ex

root@OpenWrt:~# uci get network.lan.ipaddr
192.168.1.1
root@OpenWrt:~#
