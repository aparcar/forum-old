For starting bridge everything into the lan and run it with one router as the "master" responsible for DHCP serving and the connection to the Internet (if any). 

In other words it is a simple multipoint "dumb AP" setup except that you have 5 GHz mesh interfaces linking the routers instead of Ethernet cables.

Luci cannot fully configure a mesh interface, you need to do it manually such as by placing the following in /etc/config/wireless: 

config wifi-iface
    option device 'radio1'
    option mode 'mesh'
    option mesh_id 'mymesh'
    option network 'lan'

It is essential that all the 5 GHz radios be on the same channel and that it is a channel allowed in your country.  You can set the 2.4 GHz radios to different channels.

Mesh mode may not be available with all hardware, especially Broadcom.
