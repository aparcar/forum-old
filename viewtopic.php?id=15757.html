Thank you for your answer.

I found the error but i really dont understand the logic behind this.

At the old WRT-box running whiterussian i had a nvram-variable
wan_ipaddr=192.168.1.3

This address was never used, the wan-port on this box is not needed. Nevertheless, when i changed this adress to
wan_ipaddr=192.168.0.3

it suddenly started working. I could ping this box from the new WRT-box.
