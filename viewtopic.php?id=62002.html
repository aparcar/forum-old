Typically, bandwidth is monitored using snmp. You can install snmp by the following command #opkg intall snmpd

The issue is that you need to have a collection station (PC) to poll the router for the stats you want. Then after a month you can create your own reports from the collection station.

Anyhow, collectd is fine and all, but I am not aware of how long those stats are kept.
