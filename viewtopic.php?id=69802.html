Had similar problem afte last "git pull" and rebuilding working image.
Neded to do
cd /source
mv .config .config_keep
#Caution: next will kill a lot ! Check doku !
make distclean
mv .config_keep .config
./script/feeds update -a
./script/feeds install -a
make menuconfig
make
