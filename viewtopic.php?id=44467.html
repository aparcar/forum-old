I actually got the WAN connection to my ISP working pretty easily in the end.
I turned the modem off for the day whilst at work, came back and set the MAC address of the WAN to the LAN MAC +1 and also set it in the client id field. Turned on the modem and it came up straight away.
Not sure if it was setting the WAN MAC, client id or letting the existing DHCP lease expire at my ISP (that's the most likely) but glad it's working.

Hope this is useful to someone.
