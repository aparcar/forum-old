HOWTO
~~~~~

Needed pins of the tplink:
(see the picture http://piie.net/pic/tplink-radio/tplink-radio-open.jpg)
    1) GND - find it on C37, pointing to the memory chip
    2) 3,3V - find it on "LED4 +" pin, now refered as VCC
    3) label TP_IN
    4) label TP_OUT

attiny85 circuit:

               ______

Are you sure about the way of controlling the router? I use the out-of-the-box running web server that Openwrt brings with it and made some ash/cgi scripts that produces links and buttons to control mpd. I then access them through my mobile browser. So I can listen to mp3, flac and aac streams and music shared through Samba (even grep albums in this Samba share).
I also made an extremely simple IR receiver circuit (connected to the Mic input of the soundcard) and use LIRC  and a spare remote control, and this works great.
I dont know about the space required to store all that as I also use Samba to store the mpd package and LIRC.
