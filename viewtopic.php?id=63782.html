Post #5
maartenjd
30 Mar 2016, 14:30

Did you use the right firmware image? Did you install additional packages that filled up your flash memory?

Do 

df -h

to see how much memory is left.

Reflash the firmware (that even runs on my WL500G De Luxe with 4 MB flash memory). For correct firmware look here: https://wiki.openwrt.org/toh/hwdata/asus/asus_wl-330ge.

and only install the software that you really need. What do you want to run on it, else than it being an access point or a repeater?
