jp wrote:

Is there a PDF/PS or some version of the documentation for OpenWRT suitable for viewing offline?

Not for WhiteRussian.

But Kamikaze documentation is in tex format (in /trunk/docs/) and you can generate a PDF auf of the tex files by simply running 'make' in the /trunk/docs/ directory (requires tetex-bin and tetex-extra packages to be installed on Ubuntu).
