Post #8
written_direcon
13 Mar 2014, 21:14

There is your problem. The package kmod-fs-ext4 contains filesystem drivers for EXT2/3/4 but not FAT32.

You should remove kmod-fs-ext4 and install kmod-fs-vfat instead then if your USB pen drive has FAT32 partition on it.

opkg update
opkg remove kmod-fs-ext4
opkg install kmod-fs-vfat

And then it should work.
