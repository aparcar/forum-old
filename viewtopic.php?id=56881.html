As far as I can tell the R7500 is not supported by OpenWrt in any way. The stuff on Myopenrouter is - code-wise - mostly open source, but more than that it is marketing. Netgear releases custom OpenWrt builds for a lot of their routers, but their official firmwares are actually based off old OpenWrt code since a few years (e.g. the official firmware for my WNDR3700 is based on old OpenWrt code).

Your best bet is to flash an official Netgear release - although it probably won't differ much from what you tried, besides some branding and maybe some extra binary blobs.

Hope you get it fixed.

OpenWrt on Netgear R7500 might be WiP with help of users having one unit.
see the thread for the R8000

Reasons:
- CPU/SOC is WiP / was added recently (ipq806x) - atm only a dev board exists - this is probably very similar to the real hardware
- its QCA so a little bit more source code is availble
- from the dual radio at least one should be supported
- Switch should be supported too
- usb3 (45345) should work
source for these is description at: https://wikidevi.com/wiki/Netgear_R7500

a serial header is already available on board so only an adapter is needed for logging boot + other data from the board

first step for support would probably be posting bootlogs, content of nvram and possibly other stuff (cat /proc/mtd)
