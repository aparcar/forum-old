I recently purchased a TPLINK WA901ND AP, expecting a v3 which is supported by openwrt.  I received the new v4 revision, which appears just recently started shipping.

The stock firmware unfortunately does not support operating as a full router w/ NAT, which was how I intended to use the device.

The SoC (CPU/wifi) is a TP9343 (chip says TP9343-AL3A).  The board otherwise looks identical to revision v3.

This chip is also used in the TP-LINK TL-WR941ND v6.x/TL-WR940ND v3.x.  There are some notes about WR941ND v6 support, which seem to suggest it is supported with a custom build only (generated images cause kernel faults) - https://forum.openwrt.org/viewtopic.php?pid=293475 & http://patchwork.ozlabs.org/patch/504451/ .

I have build my own openwrt images before, but never for an unsupported device.  Does anyone else have one of these and is working on support, or should I try to get a serial console working and venture into this myself?

Any tips?
