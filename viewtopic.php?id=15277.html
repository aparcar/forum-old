i dont' have your platform but if xwrt works you can go and see if in etc/rc.d there is a file Sxxdropbear where xx i think it's 50, don't know for sure. if it's not (very weird), then edit the file custom-start-up in /etc/init.d/ and add a line that says:

dropbear -p 22

save, apply and reboot.

hope this helps
