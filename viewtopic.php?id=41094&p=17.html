Post #404
spencer
17 Jan 2015, 20:10

Hey @tadeus, the file that ends in *-factory.img is used when switching from the stock Netgear firmware via their web interface or when installing via the TFTP method (especially useful when switching between builds created by different groups/people and when you may need all the settings reset to defaults). The file that ends in *-sysupgrade.tar is used by the command line sysupgrade utility and also in the LuCI web interface when you're already running modern OpenWrt firmware.
