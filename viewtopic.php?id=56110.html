To access the router's www server from the WAN side, you need to open port 80 in the firewall.

Another option is to make the Encore strictly a LAN device (a "dumb AP").  In that configuration it has no WAN network.

When using the router as a router, you should not have the lan and wan IP's in the same subnet.  Change the lan to 192.168.2.1 for example.
