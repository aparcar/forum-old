But you start with the wrong prerequisites: You can't script and you are not able to do some own work like compiling, ... ("All I read now is that I have to load files and reassamble etc. But this is to hard for me.")

OpenWRT is in most cases not like a restaurant where you order something and you get exactly what you like.
OpenWRT is in most cases like a restaurant with a buffet: Everything is there, but you have to combine and serve the dish by yourself.

If you want something, and there's no easy solution readily available: Do it yourself. If you don't do it, nobody will do it for you.

Last advise: Get over your fear of scripting and compiling. It's not that hard, and in case you encounter errors, there's still the OpenWRT forum to help you.
BUT: Be prepared, it could take a significant amount of time until you reached your goal.

Shairport is included in BB. Firstly you have to set usb audio support http://wiki.openwrt.org/doc/howto/usb.audio and to ensure your sound card is detected.
Then install shairport:

opkg update
opkg install shairport
vi /etc/config/shairport #edit the name of your airplay device and password if needed
/etc/init.d/dbus start
/etc/init.d/dbus enable
/etc/init.d/avahi-daemon start
/etc/init.d/avahi-daemon enable
/etc/init.d/shairport start
/etc/init.d/shairport enable

then if all ok then you should be able to see it in itunes or iphone
but i'm not sure you'll have enough space on 4MB flash
