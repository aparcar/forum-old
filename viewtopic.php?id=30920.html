Another OpenWrt newbie here. Have a WNDR3700v2 too and succesfully installed OpenWrt on it.
My suggestion to you, revert your 3700v2 back to Netgears own firmware. See the manual for that. Then go here: http://enduser.subsignal.org/~trondah/ and download the openwrt-wndr3700v2-r27600-squashfs-factory.img file. Flash the img file from the Netgear webGUI.
For more info see this thread: https://forum.openwrt.org/viewtopic.php … 22&p=1 Good luck!
