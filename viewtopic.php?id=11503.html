It's possible with both OpenWrt Kamikaze and DD-WRT (only v2.4 Beta).
The only one problem using both OpenWrt and DD-WRT, is Broadcom driver. Making a site survey using a Unix system, you will be able to see all your ssids. Making a site survey using Windows, you will be able to see just one ssid per time.
Typing "iwconfig" on OpenWrt, i just could see only one wl0 interface working correctly. About wl0.1 I could read "No wireless extensions". But site surveying I could see that interface and I could correctly use it.
