I got a webcam working with the Philips pwc/pwcx driver and an Asus WL500g: see forum thread http://forum.openwrt.org/viewtopic.php?id=3682

But performance was miserable. Do yourself a favour and buy a cheap ethernet-attached webcam instead. (The Edimax is very cheap - although I've not tried it - or Panasonic have some low-end ones too)

Then you can point it at something like zoneminder running on a PC, and the low-speed MIPS CPU in the Asus only has to forward the IP packets back and forth
