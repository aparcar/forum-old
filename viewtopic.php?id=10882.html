You are using the latest X-Wrt image or the updated the latest webif2. The /etc directory has permissions 0755, both hosts and ethers files have permissions 0644. It should be sufficient for the dnsmasq running under the unpriviledged user nobody.

You can fix it by the command "chmod 0755 /etc".

You could help investigating the issue, if you would be able to find out the reason.
