If you are new to qos I'd go with what openwrt seems to recommend https://wiki.openwrt.org/doc/uci/qos

All that blocking can be done by iptables. Add the required rules to /etc/firewall.user. source-mac and destiantion-port are the relevant options. "man iptables" contains all the glorious details. The luci firewall interface probably can greatly help in setting up rules.
