How did you "reset to default"? It appears you deleted your wireless settings. Just login to the console and issue "firstboot", that should reset the router properly.

Also, generally you don't need to reflash to reset the router. But keep in mind that you lose all installed packages, and you'll probably not be able to install some packages again because the trunk build id has changed. Try to reset first, if you run into these problems, sysupgrade.
