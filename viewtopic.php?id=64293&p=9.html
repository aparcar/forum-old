Post #206
gurangax
10 Jan 2018, 17:16
Ghosty wrote:

When i enterd the modem`s interface there is an option for update the modem but he ask for an firmware in a zip format. Where can i find that zip?

I think it is inside tplink firmware. I have managed to extract it. I'll give example. download old tplink firmware Archer_MR200(EU)_V1_160310.zip from tplink download page. extract and you will find Archer MR200v1_0.9.1_0.0_full_update(160317)_1041_1021_2005.bin. So it is supposed to be the full update version.
open this file in hex editing app and extract from 0x007d0200 until end. save it as zip file. you won't be able to unzip it properly with windows as it uses unix names. you can try with this zip files to update

start point is 0x7d0200 use hex editing app. extract from this address to end. or you can just delete all from 0x0 until 0x7d01ff. the starting data header should be PK for zip file. My net is slow.
