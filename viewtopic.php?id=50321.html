Not unless someone knowledgeable ports OpenWRT/DDwrt to the device and usually fringe brands such as this are not worth a devs time.

if you want added functionality on a soho router, the way to start is to read and research the OpenWRT wiki and forums, identify a supported brand, model AND version with the features you desire and then go buy that. For an ordinary user, there really is no other relatively painless way.
