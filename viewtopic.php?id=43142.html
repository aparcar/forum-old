Hi all,

I have a D-LINK DIR-825 rev B1 flashed with Attitude Adjustment 12.09-rc1. Beside changing IP and setting gateway + DNS there have been no changes after the initial flashing.

Afterwards I have been installing the following packages:
- kmod-sched
- ip
- tc
- nodogsplash

When trying to run /etc/init.d/nodogsplash enable it does complete without any return.

However nodogsplash is not starting. When running ndsctl status I get the following error:

nodogsplash probably not started (Error: No such file or directory)

May I ask you if any of you already experienced this issue in the past and if you have been able to fix it?

Thanks

PS: I had the same error on this router previously with the latest Backfire build

Here: http://kokoro.ucsd.edu/nodogsplash/
it says to start running:

/etc/init.d/S65nodogsplash start

notice the 'S65'
