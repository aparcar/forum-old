This is not a bug you can fix or which needs fixing.  It's an informational message about a non-standard but common USB device implementation.

I proposed removing the message a while ago, but there were some developers who found it useful so I dropped the proposal.  See the discussion here: http://www.spinics.net/lists/linux-usb/msg77499.html
