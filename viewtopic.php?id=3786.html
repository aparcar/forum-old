As far as I can see this is supported by the Prism54 driver.  That is incorporated into the latest 2.6 kernels
(at least the PCI bits are, I am not sure about the USB bits yet - they will surely come if they are not there yet) so you need to build yourself an image with a recent 2.6 kernel and then it might just work.  If not look out for the change logs from Linus to see what gets added to each new release.
