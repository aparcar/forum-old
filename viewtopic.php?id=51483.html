This setup works with DGN2000, never tried it with DGN2200. You can test it though:

1. Access the web-user interface of the modem by typing http://192.168.0.1 in the address field.
2. When prompted, enter “admin” for the username and “password” for the password (if you haven't changed yet the password for the interface).
3. Once logged in change the address URL in the address bar from:
http://192.168.0.1/setup.cgi?next_file=start.htm
to:
http://192.168.0.1/setup.cgi?next_file=mode.htm
4. This opens a hidden configuration page on router. Select 'Modem (Modem only)' and click Apply. This will convert the router into a simple ADSL mode. Router (Modem + Router) sets the device to modem-router mode.
5. After setting up the modem to Bridge mode, Basic Settings option is gone under Setup tab. You could still configure the ADSL settings though.
6. You can now connect the DGN2200 to your router or computer and setup for a PPPoE connection.

I understand - I have a similar Netgear router, and an OpenWRT router.

I suggest (I think I used to do like this):

Make everything work with the Netgear only, in Router Mode, first (it sucks to not have internet access)

When the Netgear works fine, plug in the OpenWRT (as you did) and reconfigure OpenWRT
- Disable NAT (you dont want double NAT as described above)
- Disable DHCP (you dont want two DHCP servers)
- Perhaps set a static IP to your OpenWRT

Now you have

Internet 1.2.3.4 <- Netgear (NAT) -> 192.168.0.1
OpenWRT has only ONE IP=192.168.0.2 (or something else)

All computers (including OpenWRT) should now have
- Default Gateway = 192.168.0.1

You can now use WLAN from OpenWRT and/or Netgear - your choice.
You can also connect clients to Ethernet Ports in either Netgear/OpenWRT.

If you want to use more features on OpenWRT, you can configure your Netgear with DMZ=192.168.0.2. This will send all traffic that Netgear does not know what to do with, directly to OpenWRT.

If you want to, you can also disable DHCP on Netgear and activate it on OpenWRT (then it is probably a good idea to not use WiFi from Netgear.

Please understand there may be some information above that is not 100% accurate... There is a lot to learn about home networking, and OpenWRT is great. But I suggest focus on getting a simple setup that works first (with the Netgear), so you can benefit of the Video that you wanted in the first place. Then you can read, learn and play with OpenWRT as you have time and inspiration.

webmanoffesto wrote:

I want Broadband for streaming video. Is there a +/- $25 modem which would do this for me

You have a DSL line, therefor you already have broadband. The DGN2000 already does ADSL2+, which is the "fastest" technology to work your current DSL connection (up to something like 16000 kbit/s on the receiving end, give or take).

A new modem most probably won't speed up your line, OpenWrt certainly won't, and neither can any help from us. DSL line speeds are almost exclusively decided by your DSL provider and the actual physical quality of your line.

You could of course switch to another technology, VDSL or cable broadband. But this is quite a process involving switching contracts, and we cannot possibly make any recommendations, let alone an informed suggestion, to you about that.
