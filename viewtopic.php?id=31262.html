Anybody have this device?
I need caldata partition backup for testing purposes.

To make backup, you need to login with SSH router and execute commands:

cat /proc/mtd
(this give you partition listing, like 

root@TwilightRouter:~# cat /proc/mtd  
dev:    size   erasesize  name
mtd0: 00040000 00010000 "uboot"
mtd1: 00010000 00010000 "config"
mtd2: 00100000 00010000 "kernel"
mtd3: 00510000 00010000 "rootfs"
mtd4: 00010000 00010000 "caldata"
mtd5: 00190000 00010000 "unknown"
mtd6: 00610000 00010000 "firmware"

then copy caldata to ramdisk: 

if=/dev/mtd4ro of=/tmp/caldata.backup
128+0 records in
128+0 records out

and download /tmp/caldata.backup to your computer.
