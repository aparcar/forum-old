That red color is normal: OpenWrt assigns a different color to each interface, in a random fashion; as "tavish" pointed out, that interface is up and running fine.

Your problem is that you have two 192.168.1.0/24 networks, one on each side of the OpenWrt Router. Easiest solution is to configure your OpenWrt router to use a 192.168.0.0/24 (or any other 192.168.x.0/24 network with "x" different to "1") network for the LAN interface. Or you can configure the FibreCable router to use another network, too.

You could also bridge the LAN and WAN interfaces together, so you effectively have one single network, but it is a bit more complex; it all depends n what you plan to do with your network.
