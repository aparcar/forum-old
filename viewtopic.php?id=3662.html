1) docs
Some documentation

2) experimental-20050525
Latest experimental OpenWrt version (before White Russian RC1) -- Don't use it, it's obsolete.

3) people
Directories which contains builds from the developers

4) reference
....

5) sources
The source tarballs use by the buildroot. Is used as a mirror when all other mirrors in the Makefile are not working.

6) utils
Some more or less useful utilities for OpenWrt

7) whiterussian
White Russian release candidates and the package repository

8) bin/,default/,micro/,pptp/
It's described in the FAQ.

9) OpenWrt-ImageBuilder-Linux-i686.tar.bz2
Contains the OpenWrt Image Builder to build your own images.

10) OpenWrt-SDK-Linux-i686-1.tar.bz2
Contains the OpenWrt Software Development Kit (SDK) to build your own ipkg packages.

11) md5sums
Contains the md5sum from each file. Your downloaded file should have the same md5sum as in the file.
