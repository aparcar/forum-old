Would you mind posting the contents of your /etc/config/p910nd. Mine looks like this:

config p910nd
        option device        /dev/usb/lp0
        option port          0
        option bidirectional 1
        option enabled       1

Then just run the following commands:

/etc/init.d/p910nd start
/etc/init.d/p910nd enable

Verify that your server is running by executing #ps:

/usr/sbin/p9100d -b -f /dev/usb/lp0 0
