From my own notes:

Check php installation

echo "<?php phpinfo(); ?>" > /srv/www/info.php

Access info.php with your browser (in this case, the server is running on port 8092; adapt IP and port as needed)

http://192.168.178.25:8092/info.php

When you run the script, it will output a lot of boxes full of configuration information.

If instead your browser prompts you to download php.info, then something is wrong:

- http://stackoverflow.com/questions/2559 … uting-them
- http://stackoverflow.com/questions/1777 … uting-them
- http://askubuntu.com/questions/460710/n … cessing-it
