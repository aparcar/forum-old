I believe that you can place the single wireless radio into client mode (sta) and then use it your single wan gateway.  However, it can be kinda tricky to do this successfully while not getting locked out of the box. You'll have to edit the network, wireless, and firewall sections of the /etc/config/ Refer to the wiki.


Also if you want use multiple wans youll have to have multiple wireless boxes feeding into multiwan router using vlans to seperate the wans which all require different subnets.  Heres a topology of my setup.


Wireless 1    Wireless 2  Wireless3
x.x.0.1            x.x.1.1        x.x.2.1
  | |                     | |               | |
|------------------------------------------|
| eth1        eth0.2             eth0.3  |
|                                                    |
|3700v2 Multiwan,vlan,vpn,          |
|------------------------------------------|


This affords load balancing ~ 3x the speed and failover and is a stable solution.
It certainly wasn't easy but works pretty well masking the wireless connections instability.   Tell us more about the hardware you have available.  It may be best to pick up some cheep compatible routers to use as wan the gateways. 

Another idea would be to get multiple usb wireless adapters and use those on the single router.  That would be awesome if someone could successfully to that.
