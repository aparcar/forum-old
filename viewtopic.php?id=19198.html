This is my first forum post.  I recently moved from dd-wrt to OpenWRT and I'm really delighted with the flexibility it gives me.  I use it in production as a semi automated test solution for WiMAX links.

Any advice welcome and sorry if it's been posted already (I couldn't find it).  I first started seeing the problem in the last 2 or 3 days, but it could have been like that for up to two weeks.

Hi Bread,

You can temporarily use a workaround and modify your /etc/ipkg.conf file to look like:

src release http://downloads.openwrt.org/kamikaze/7 … 4/packages
#src packages http://downloads.openwrt.org/kamikaze/packages/mipsel
src packages http://downloads.openwrt.org/kamikaze/7 … ges/mipsel
dest root /
dest ram /tmp

But the problem is, what happens if repository structure ischanged back again .... :-(
