i've built this firmware for personal use and would like to share it with anyone who's interested but new to Openwrt. it allows you to plug your webcam into the router and use it as a surveillance camera. so you can monitor your home or office in real time from the internet without the cost of dedicated surveilance tools.i have already made all the necessary configuration so that you need no further configuration to make it work if your webcam supports mjpg format and is UVC-compatible.just plug your webcam into router and you will be able to view live video from a browser by visiting 192.168.1.1:8080/?action=streamer(username:openwrt/password:openwrt). I've also managed to squeeze other features into the tiny box, of those are 3G, luci webGUI, USB storage support(ext4,vfat and read-only ntfs), running openwrt from external storage, DDNS,FTP server.the following webcam is confirmed working with this firmware:

LOGITECH c270
LOGITECH C110
HD5000
AONI D881
Microsoft LifeCam Cinema
