Hi there !

I have the following Setup here:
* Asus Wl500 gP (mipsel)
* Speedport W900V with AVM/TKom kernel but with OpenWrt Root on USB (mipsel)

I wanted to experiment with asterisk. So I compiled asterisk for mipsel using kamikaze bcrm47xx-2.6 svn trunk.
But there was lack of capi support for asterisk14 in kamikaze.

So I created some packages for openwrt.

libcapi (isdnutils) - With libcapi20 and remote CAPI-Client support
capiinfo (isdnutils) - A info programm for CAPI
capifax (isdnutils) - A fax program for CAPI
rcapid (isdnutils) - Remote CAPI Server
asterisk14-chan-capi - A channel module for Asterisk 1.4
memstat - a memory statistics dumper.

The files are source packages for kamikaze trunk (I used rev 10792).
Simply link the contained directories to KAMIKAZE-SVNDIR/packages.
In the config menu you can now select these new things in category Net and Admin.

You can get the files via http://vvv.borkum.net/files/openwrt-ipkg/ .

Not all packages are testet. Capiiinfo, rcapid, libcapi and capifax are working. chan-capi i could not test yet because of lack of time. It just compiles.

But may be some of you are interested.

cya
