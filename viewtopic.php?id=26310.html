Eventhough your router is bricked, but if it is still running with firewall, WAN interfaces do not accept packets coming in.

If I am not mistaken, /etc/config/firewall file states so.

/etc/config/firewall wrote:

config 'zone' 'wan'
    option 'input' 'REJECT'
    option 'output' 'ACCEPT'
    option 'forward' 'REJECT'

What about on the LAN interface? Set a static IP address of 192.168.1.2.

Then try reflashing the router. Had to get the timing right. I assume you didn't set the nvram boot_wait and boot_time parameters. It will be harder.
