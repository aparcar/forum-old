do failsafe.. if the led starts blinking fast:

set your computer ip to something like 192.168.1.100
subnet 255.255.255.0
no gateway

then try to ping 192.168.1.1

if you get a response you can use telnet to go in
then reset the settings
(most often with command: firstboot)
