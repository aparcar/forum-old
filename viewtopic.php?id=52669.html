The only solution I have found to this problem is to copy my saved configuration file to the ".config" file, for instance, "cp ralink3.config .config".  Then "make menuconfig" shows those settings, and "make" works to produce the correct binaries.

Simply saving on exit after making changes does not save the ".config" file with my setup.
