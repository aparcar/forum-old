I believe there's more than one solution:
1) use WDS
2) use routed client mode between fonera and another router and set up a tunnel with TAP device, and bridge it with your eth.

it really depends on whether you can install openwrt on your master router.
I have brcm chipset in my "client" router, but still use #2, because I need multicast routing.

a) Run WDS. Both actiontec 724 and fonera should install OpenWrt.
or
b) Install Gargoyle (OpenWrt) or DD-WRT (r14896/r15704) on fonera.

Edit:
Routed Client can also do the job. Just set the desktop as DMZ.
