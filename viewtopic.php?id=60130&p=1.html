Hey I was able to install OpenWRT on the TP-Link TL-WR841N v10 by using the v9 firmware only and without using Serial Console.
Download : OpenWRT-15.05
Open the router page .[default user : admin , password: admin]
Do a factory reset after taking a look at your current settings .
Go to System tools > Firmware upgrade.
Browse to the downloaded file and upload.

PS: I will upload a DD-WRT too but someone needs to test it as I couldnt find time to test it on my friend's router. So anyone who is interested can pm me.
