regarding vlan.. luci>network>switch and there you should see 2 vlans(virtual lan) the lan phisical ports are grouped(linked) into a vlan and the wan port is the second vlan; to remove one from lan you set it to off and activate it in a newly created vlan...
after you created the 3rd vlan you need to create an interface for it luci>network>interfaces and in this interface @firewall tab to set it as wan

a vlan is the logic behind physical port
basically you will have as many vlans as physical ports.. unless you link them together.. they are completely separated and all can have 192.168.1.1.. as subnet.. they dont communicate with each other unless you link them in which case you might have problems if all have same subnet

for you to define a link between them you have to create an interface.. which will assign an ip and a firewall zone.. the firewall zone is the link between them

so thats the theory of vlan.. and you example is good
regarding failsafe.. i gave you some links and a fair warning.. id read them if i were you.. you do as you please
