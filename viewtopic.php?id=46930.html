I have several TP-LINK 3600. They have a 2,4 GHz and a 5 GHz WAN interface.

The 5 GHz Interface is used to couple the Routers: One is AP, the other ones ar clients. The 2,4 Ghz interface is used to create another WAN with an own SSID and an own ip-subnet. The routers cannot bridge the 5 GHz client WAN interface to the 3,4 GHz AP WAN interface. So I have to route the traffic. I chose different SSIDs f√ºr every subnet so I can force Tablets, Phones and so on to use a specific AP.

If you use UPnP streaming stuff you have to run igmp-proxy. If you use Squeezeboxen (or the Squeezebox player or controller app on Android) you have to run udp-broadcast-relay.
