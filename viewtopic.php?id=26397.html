Custom build succeeds and is installed.  To my surprise the dnsmasq and dhcp options still exist in LUCI menus and throw errors when selected.  This is not a huge problem, because maradns can be configured manually and I can just avoid selecting those menu options.

However, just wondering if its possible to change those menu options through config files or similar?
