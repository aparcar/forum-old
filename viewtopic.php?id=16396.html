I have made a workaround by writing script for cron that checks if Openvpn is running or not every 5 mins. If it finds it down it starts it again:

#!/bin/sh
NAME=[o]penvpn
ps | grep $NAME
if [ $? -eq 0 ]; then
   echo "Name Found"
else
   /etc/init.d/S50openvpn
fi
