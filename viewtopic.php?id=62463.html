why is this suprising?
WDS is the only solution which gives you a fully transparent layer2-bridge.

If you use any pseudobrigde-setup you will get in all sorts of trouble, and you will have to figure out which broadcasts relayd isn't passing.

I don't think this applies to windows 7/8/10 but there was something called WINS-Server back in the days to allow communication between windows machines that aren't in the same broadcastdomain. Active Directory switched to DNS for this purpose, so maybe that's a possible way to go (?) or maybe you should run samba as PDC on your openwrt-box
