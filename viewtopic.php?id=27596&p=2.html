So you tried the trunk firmware for v3 model (not the backfire 10.03.1 as it does not support v3) and it still does not flash. Am I right?
If so, I guess this is a bug. Wiki says that it is supported since r28269. This file, in theory, should be accepted by factory web interface:

http://downloads.openwrt.org/snapshots/ … actory.bin

BTW, you cannot use v1 firmware for v3 router. Well, you may use u-boot tftp flash method in order to overcome the factory
checks.

Check older forum topics for this model and also bug reports about v3. It may have important information.

TL-WR741ND Wiki:

Note: Newer TP-Link retail firmwares (confirmed on v2.4 devices, but definitely not on v2.0) include a crude mechanism to prevent you uploading unsuitable firmware. It does this by checking the filename, so you may need to rename the *-factory.bin file to the name of a TP-Link retail firmware file. e.g. wr741nv1_en_3_12_4_up(100910).bin
