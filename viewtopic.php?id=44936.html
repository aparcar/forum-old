Hi all,

I'm considering moving to a VDSL2 connection and am looking for a supported device. I'd appreciate input on the following questions:

1) Are there any supported VDSL2 modem/routers? The wiki and forum seem to suggest no, but I'd love to be wrong about this. Support would need to include VLANs on the WAN side, as I wish to be able to run a VOIP phone and the providers here do this through VLANs.

2) If there are no supported devices, my next choice would be to bridge a VDSL2 device (with native firmware) to an OpenWRT wired router. Anything obviously wrong with this idea? Subpoints:
   a) Need to maintain VLANs on WAN on bridged device. Is this likely to be possible? (I don't see why not).
   b) Recommended  devices for bridging in this way?
   c) Recommended hardware for the downstream OpenWRT router? Would prefer wired only.

Thanks everyone!

At least Lantiq, Broadcom, Conexant and Ikanos have VDSL2-IP in their product portfolios. Given that companies attitude it seems very unlikely that there ever will be FOSS drivers for the Broadcom DSL-solution. The same seems true for Ikanos.
