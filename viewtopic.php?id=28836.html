Had the same issue some time ago.
I think it has something to do with the Luci-interface that's outdated with the samba-package.
Someone else found a solution for me:

opkg update
opkg list-upgradable

In the list you'll see the packages you need to upgrade: opkg upgrade "package"
After you did this, the interface will normally appear again.

Well this particular error is not due to package upgrades, it is just a syntax error or entirely missing /etc/config/samba.
The command below will tell the exact location of the error.

uci show samba > /dev/null
