Bottom line is stretching to something like the WR1043ND would give you more value for money than an older router like the WGT634U. Just don't expect 'gigabit switch' and 'USB 2' to mean that it can read files from the USB storage at 1000Mbit/s or you'll be very disappointed (I get a maximum of 130Mbit/s doing a test read direct from the WR1043ND's memory using iperf, so reading USB storage at 100Mbit/s would be an absolute maximum, at a guess [compared to 220Mbit/s from my Buffalo Linkstation Live 1TB] - but the gigabit switch can pass traffic at a good speed).
