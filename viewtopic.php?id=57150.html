Hello list!
I've a basic question regarding packages.

In 

make menuconfig

I have a package "FOO" set to be compiled as module, i.e. 

PACKAGE_FOO=m

In my understanding, this package is not included in the kernel image but it is compiled, in fact I see a "FOO.ipk" file in the "bin/ARCH/packages" tree. I know that I can install the "FOO.ipk" file by using "opkg", but I need to copy this file to the board first.

How can I do that? At the moment I'm using "scp", but I would like to have it included in the rootfs tarball that is generated by buildroot, in order to have it available as I flash OpenWrt on the board. Is it possible?

Thank you and kind regards,
Giorgio Buffa
