dsntstu wrote:

when I try to use telnet 192.168.1.1 I keep getting this error "Could not open connecton to the host, on port 23: Connecion failed."

In the current master or 17.01 there is no telnet (in port 23).
There is only ssh (in port 22).
That is also mentioned in the wiki: https://wiki.openwrt.org/doc/howto/firstlogin


(Note: there is still telnet in the ancient 15.05.1 release, but I hope you are not using that with wrt1900acs)
