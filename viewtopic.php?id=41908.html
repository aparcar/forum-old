It failed because the script said file /home/newone/openwrt/trunk/build_dir/target-mipsel_r2_uClibc-0.9.33.2/linux-ramips_rt305x/vmlinux-all0239-3g.bin.lzma is greater than (917504 - 64) - by (932896-(917504 - 64)) bytes.

You seem to be building Barrier Breaker. The 917504 value is present in my 3.3.8 linux makefiles, so it could be as simple as somebody hasn't gotten around to ammending it. You may be able to patch the value but have size issues with your 4MB of flash later on in making the image.
