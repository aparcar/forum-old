Hi, I've been running Tomato on an Asus RT-N16 for years but Tomato is no longer being developed and I'd like to move to a newer open source router firmware that has a more active development community. OpenWrt looks interesting but I'd like to find something that's easy to use like Tomato was it - basically download the firmware image, flash the router, and your done - you can hit it with a browser and use/configure it immediately. Tomato was as easy to use as OEM firmware. I take it from what I've read so far, OpenWrt is more of a build-it-yourself thing where you need to spend a lot of time using ssh doing command line stuff, which is definitely not something I want to deal with. But I read in some posts hints that an easy-to-use web GUI exists and that in some cases a ready-to-install downloadable firmware file can be gotten for a few lucky types of hardware. I've not had any luck tracking down anything more specific though. I have seen the hardware list in the wiki but it doesn't seem to address the availability of read-to-use firmware. So I guess this is leading up to a couple of questions:

1. Is there really some version of OpenWrt firmware that comes with a web GUI in a Tomato-like easy to use, download, flash, and run type package?

2. If the answer to #1 is yes, is such a package available for my existing Asus RT-N16?

3. If the answer to #2 is no, can someone recommend a specific router I could buy for which such a version of OpenWrt is available?

You have two main GUIs for OpenWRT, the default, advenced one called Luci and a simpler version called Gargoyle.

The latest stable version of OpenWRT with Luci for your router can be found here:
http://downloads.openwrt.org/chaos_calm … x/mips74k/

For Gargoyle, you don't have a special build, just a generic one built for the platform Asus uses:
http://www.gargoyle-router.com/download … uashfs.trx

Which might work, but... I would ask at their forums first before trying to install this.

You can flash a new firmware from Tomato, just use a cable, don't do it ower wifi.
