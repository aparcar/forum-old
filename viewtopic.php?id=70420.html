The UCI system parses /etc/config/wireless and generates hostapd.conf and wpa-supplicant.conf files in /var/run (which is a RAM disk).  Then it launches hostapd and/or wpa-supplicant referenced to those configs.  The actual programs should be very similar to those used on big Linux distributions.
