I just flashed my WNCE2001 router with the corresponding squashfs factory firmware* with help of tftp32. I can login to the router via telnet/ssh and with Luci.

"Too few erase blocks (3)" means that the firmware image is too large for the system to create the r/w jffs2 file system. There needs to be at least 4 or 5 free 64kB blocks, if I remember it right.

If you get that error, you need to decrease the firmware size, e.g. be leaving out a few packages.
