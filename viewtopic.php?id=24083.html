you could edit rc.local (base-files/files/etc/rc.local) and put your parameters at the end:

rmmod usbserial
insmod usbserial vendor=0xAAAA product=0xBBBB

Of course if you have any other usbserial drivers installed, you'll have to rmmod them first before you can rmmod usbserial.


OR

you can make a directory in the build system root dir called "files" and make a 60-usb-serial with:

usbserial vendor=0xAAAA product=0xBBBB

This will overwrite the 60-usb-serial that is created when the module is compiled.
