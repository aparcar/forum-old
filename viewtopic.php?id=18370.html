Problem:

        Network map and summary: http://bonmot.ca/~denny/maps/openwrtnet.txt

Symptoms

           First user connects without issue, system works as expect.

           Second user connects and first user gets "bumped" and the first connection gets dropped.

Problem Background

Other Software Installed

            I have installed all of the nat modules and extentions.

What this problem is not

          This is not openwrt to cisco VPN connections

What this problem is

         Cisco VPN Client pass-through
