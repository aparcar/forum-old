I've build the image for my raspberry 3 from the source in the git repo. It worked fine and the only things missing was adding the wifi chip driver from the rapsberry 3 github repository to get wlan0 working.

This is my /etc/opkg/distfeeds.conf:

src/gz designated_driver_base http://downloads.openwrt.org/snapshots/trunk/brcm2708/generic/packages/base
src/gz designated_driver_kernel http://downloads.openwrt.org/snapshots/trunk/brcm2708/generic/packages/kernel
