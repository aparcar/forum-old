erz wrote:

I want to set 2,4Ghz rx and tx on one antenna and 5Ghz rx and tx on the other.

By thorough testing (using one antenna and trying txantenna and rxantenna settings)
2.4ghz: txantenna=1; rxantenna=1 (will set it on left antenna, next to LAN ports)
5ghz: txantenna=1; rxantenna=1 (will set it on right antenna, next to power jack)

Now what I do not understand about the TX/RXantenna:
Is it broken?
On 2.4 and 5ghz txantenna and rxantenna "2" does not do anything. In a meaning that it does not swich anything but previous setting remains.
If txantenna is "3" both antennas work on 2.4 and 5 ghz
With txantenna 1 on 2.4 it is set on left antenna.
With txantenna 2 on 5ghz it is set on right antenna.
Kind of weird...

Maybe somebody could clear it up a bit.
