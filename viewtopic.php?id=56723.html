Not sure this is the best way but
Just one way: System -> Startup -> /etc/rc.local

# Put your custom commands here that should be executed once
# the system init finished. By default this file does nothing.

ip6tables -F
ip6tables -P INPUT DROP
ip6tables -P FORWARD DROP
ip6tables -P OUTPUT DROP

exit 0

Submit and reboot, now it should drop all v6 packets

This would drop all IPv6 Packets but not disable it. (could lead to timeouts if an ipv6 ip is resolved...)

(maybe) better:

- in /etc/config/firewall, you can find the following comment, uncomment it:
# Uncomment this line to disable ipv6 rules
#       option disable_ipv6     1

- and add this to /etc/sysctl.conf
net.ipv6.conf.all.disable_ipv6=1
