The zoneinfo packages have no effect in Luci. And not much effect in the core Openwrt, either...

The timezones used in Luci are built-in and included in the Luci source code. Check the two *.lua files here:
http://luci.subsignal.org/trac/browser/ â€¦ s/zoneinfo

If you want to add something, you could:
- edit the source file (if you build Luci locally for your Openwrt build, as I do), or
- create a patch to add the correct timezone info and send the patch to Luci sources via a Luci ticket (http://luci.subsignal.org/trac/newticket)

EDIT:
In a live Openwrt system, those Luci files are located in /usr/lib/lua/luci/sys/zoneinfo
You could edit them there with a text editor, and then reboot...
