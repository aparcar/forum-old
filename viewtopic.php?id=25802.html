I had similar problem with my Pixart Imaging, Inc. PAC7311 Trust WB-3300p (ID 093a:2608) gspca webcam.

Mine use PJPG palette.
On top of that gspca_pac7311 kernel driver uses v4l only (not v4l2).

So  my webcam worked after

- installing libv4l package (probably you do not need this step),
- adding export LD_PRELOAD=/usr/lib/libv4l/v4l2convert.so (probably you do not need this step),
- fswebcam worked (from fswebcam package)

fswebcam can handle PJPG palettes.

I hope is helped.

BR,
Mark
