I found the solution on a polish site:
http://eko.one.pl/?p=openwrt-mr3420
Google translate will help.

In short you have to logon the router with Putty.

Then use wget to load the firmware into the tmp directory from the web.
After that you can use the command above to flash.

Code sample after login:
cd tmp
wget http://.../code.bin
mtd -e firmware -r write code.bin firmware

That's it!

Hope I could help.
