Folks,

I am looking for a way to use dynamic URL filtering on behalf of DNS queries. I lalready use Opendns, but the filtering is static for the complete network.

OpenDNS offers together with Netgear 'Live Parental Control' which seems to support some kind of dynamic filtering with individual settings for hosts within the same network (I don't know if it is on basis of MAC, IP or maybe individual USer credentials).

As OpenDNS use the public IP as reference for their filter settings, it would be easy to adjust with multiple static IPs (which you will not get as a 'standard' consumer from your ISP).

As Netgear offers this feature together with their routers, even for those 'standard' customers, I assume they use some kind of tagging for the DNS requests. Does anybody know how to use this with OpenWRT and what needs to be ordered at OpenDNS to support it?

Regards

P.S.: just found the Netgear manual. It seems there are just two options: using the OpenDNS URL filtering or just 'bypass' it. So again assumption: either the bypass acount is just forwarded to a different DNS or Netgear uses a kind of IP anonymizer which changes the public IP address of the requesting bypass account...building both should not be an issue with OpenWRT
