Hey there.

Setting wifi via command line is pretty easy.
Add a little curl or wget to it to fetch the desired configuration from a remote HTTP server, maybe use any digital signature and you're done.

Currently I push configuration via SSH from one instance to a couple of others, just by calling this:

ssh openwrthost ash < wifi-setup-script.sh

But of course you can inverse control by not pushing data but pulling. Use a cron job and you're done.

The actually nice part comes when you write your wifi setup script.
