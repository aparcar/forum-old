That's why i recommended to have serial...

Anyway, did you try also reset button once it's booted up? You do not see any wifi network

the good news is that the gui accepted the fw, so there is some hope..

The problem was that all the original english fw have also uboot included, an that header needs a different checksum, and then after the usual 0x20200 there is another header...

So you need to first trim uboot, then patch the header.. Then i also removed the "boot" word from filename but maybe that's not needed...
