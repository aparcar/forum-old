Post #5
written_direcon
14 Aug 2011, 13:43 
opkg update
opkg install kmod-ath9k wpad-mini 
rm -f /etc/config/wireless
wifi detect > /etc/config/wireless
uci set wireless.@wifi-device[0].disabled=0
wifi up

This should make your wireless working.

GOOD NEWS!!!
I have downloaded the file wr941nv2_en_3_11_8_up(100609).bin from tp-link and did the following command:
mtd -e firmware -r write /tmp/wr941.bin firmware

Boom!
The firmware was back to the tp-link and the radio is now working!!
