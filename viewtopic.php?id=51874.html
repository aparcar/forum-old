One more thing.
After successful flashed openwrt firmware to G450H router.
I cannot flash any other firmware again.

my steps are(as this wiki said http://wiki.openwrt.org/toh/buffalo/wzr … recovery):
1.set ip to 192.168.11.2 using command "ifconfig eth0 192.168.11.2"
2.set an static arp using command "arp -s 192.168.11.1 02:aa:bb:cc:dd:23 -i eth0"
3.tftp 192.168.11.1, set modes verbose/binary/trace, then put xxx.bin
4.power on the router and hit enter in the tftp command window.

the red led started up, then a while later turned to off, then wireless led light up, openwrt running ....
it seems the router ignored my tftp request in boot stage
