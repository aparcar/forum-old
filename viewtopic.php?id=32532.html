I need to make a few configuration changes though, and I would love to get the communities' opinion -- as they know it best!

This is my proposed configuration;

Router One
- Set-up with my ADSL2 modem, as my main router.
- Two SSID's -- one a hotspot, the other a private network.
- Proposed third SSID to connect the two routers (too much for the chipset to handle?)

Router Two
- Using a USB Hub; connected to a printer and a scanner
- Doesn't need to repeat the private network, nor hotspot network
- But, cannot be connected to Router One via Ethernet

The Question

I assume if I just made router two a client, that it would purely host the printer and scanner? Should I create a new SSID just for bridging the two routers?
Additionally, if I were to (for any reason) decide to connect a laptop (for example) to Router two, would that be usable on the private LAN on router one using a specific AP method? How would this affect performance?

I'm happy to clarify anything, because I may have missed something! Any assistance would be appreciated!
