I've got a Verizon/Novatel 5510L tethered to a Hootoo TM02 running OpenWrt? The Hootoo is running an OpenVPN client that connects to my VPN server through the 5510L. My devices connect to the Hootoo and get a direct connection to the VPN server.

The problem is the Hootoo is bottlenecking the VPN connection. If I connect to the VPN directly through the Verizon hotspot and use the openvpn connect app on my phone I'll get around 12Mbit down and 3Mbit up, but I can't use openvpn connect on my iPad for reasons beyond my control, hence this setup. When I connect through the Hootoo I get about 3Mbit/0.8Mbit which is way too slow for my taste.

So is there a faster portable router out there that has a USB port to tether and can run OpenWrt?

milankocvara

Do you use nat or so on on hootoo? If not, try to unload all modules, you do not need. (Simply move the files from /etc/modules.d to for example /etc/modules.bak. In /etc/modules.d should stay all files which are in /etc/modules.boot/) and reboot router.
For example nf_conntrack decreases the troughput in about 30% depending on the count of packets. Does not matter, if NAT is used.
