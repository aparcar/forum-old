FWIW, that build was before 10.03.1 was released....
If you read Simon-Notes.txt there it will tell you that actually that driver build never seems to work/load .

I believe, the problem with the switch was fixed in the 10.03.1 release version -- i.e. all 4 Ethernet ports work.  AFAIK there has never been working VLAN configuration support for that marvell chipset ethernet.

Also, to have WPA2 Wireless working (TNETW1130 ONLY -- e.g. DG834Gv2), NOT on TNETW1350A (e.g. DG834Gv3) you need a later wireless driver in e.g. a 'trunk' build which I may be able to find you at least a binary of...

The build linked above is probably not so useful any more .
