Sounds like your router is not bricked if you can get to luci, so that's good. You should be able to flash the OEM firmware (with uboot removed) using standard Luci flashing functionality. Not sure why it didn't work, although it's unclear to me if the first attempt was with uboot removed or not?

The other thing is that you need to get into the webadmin.  The whole "30 30 30" is DD-WRT voodoo magic that doesn't really do anything. What you need is to get into OpenWrt failsafe mode, which is described in the wiki.

Another option is the TFTP flash, which some TP-Link routers have. Those instructions are also in the wiki. Pay close attention to the IP addresses you need to set - some TP-Link routers use 192.168.1.x and some are 192.168.0.x.

Last resort is a serial flash, but I don't think you're anywhere near that at this point.
