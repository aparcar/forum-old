Two options:
WinSCP SCP 192.168.1.1 tplink.bin -> /root/tmp
PuTTY SSH 192.168.1.1
cd /tmp
mtd -e firmware -r write tplink.bin firmware

Failsafe mode & ftp (or http) server 192.168.1.2
telnet 192.168.1.1
cd /tmp
wget ftp://192.168.1.2/tplink.bin
mtd -e firmware -r write tplink.bin firmware

<!> Use firmware without boot, otherwise you will brick your router and then you need serial console to recover the router.

There is no boot in firmware for WR941ND_v3, so it`s ok: http://www.tp-link.com/Resources/softwa â¦ 120201.zip

Start WinSCP-> Choose File protocol: SCP-> Enter for host name 192.168.1.1-> Then Login-> Enter User Name and Password ("root" and "password" or what you set after "mount_root" and "passwd")-> Then upload (drag&drop) binary file (tplink.bin) to /tmp folder in the right side from WinSCP-> Start PuTTY and select SSH-> Enter for Host 192.168.1.1 -> Open -> Enter username and password -> cd /tmp -> mtd -r write tplink.bin firmware -> After restart try 192.168.0.1.

Use Windows.
