http://downloads.openwrt.org/backports/, choose your OpenWrt version and then find it on the "package" directory. You have manually to install the package downloading it on the /tmp directory of the AP and then typing: 

ipkg install /tmp/iperf_2.0.2-1_mipsel.ipk

I installed it 3-4 months ago on WRT54GL using WhiteRussian 0.9, and worked pretty well.
