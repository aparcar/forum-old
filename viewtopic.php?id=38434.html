GoWest

i tried lots of different solutions and made lots of tests ... including download/upload speeds link reliability encryption level etc ... the best solution i finally settled with is AWS S3 :
- on linux/openwrt you can use for ex the "s3cmd" client in python incl to sync files
- you get client apps on Android MSWin etc
- i found the net link to be always reliable and stable even when connecting from a different country
- if you need extra security on AWS you can even activate MFA and you can use for ex GoogleAuth client on Android

btw when you analyze network traffic of many famous cloud backup solutions providers ... often you will find that your files are in fact redirected to AWS !

for those interested you can also compile s3fs to work on openwrt

and another thing ... it's a VERY cheap solution bec you pay only for what you use ... and AWS is so rich of features much more than just S3 backups ... just type on youtube you can find lots of easy tutorials ... and whenever i needed to contact customer service i always got a full tech answer the same day
