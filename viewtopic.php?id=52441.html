Post #6
farrukh
28 Aug 2014, 11:18

So I assume your service is DSL and your ISP has provided a modem? It is important to know how that modem is configured i.e. As a DHCP router or a bridge.

If your scenario is like this: ISP > modem (set to bridge mode) > openwrt router. Then you need to configure your openwrt device as a router using following commands:

uci set network.wan.proto=pppoe
uci set network.wan.username=xxxxxxxxx
uci set network.wan.password=xxxxxxx
uci commit network
ifup wan

replace xxxxxx with your own credentials.

Post #8
farrukh
28 Aug 2014, 11:41

uci set network.lan.proto=static
uci set network.lan.ipaddr=192.168.1.2
uci set network.lan.netmask=255.255.255.0
uci set network.lan.gateway=192.168.1.1
uci set network.lan.dns=8.8.8.8

/etc/init.d/network restart
----------------
replace the numbers to fit your network.
