For the easy, minimal path, here's what I've done on a couple of our APs to run the latest final release build successfully:

  • Get TFTP server and serial connection working, of course

  • Download kernel.bin and rootfs-squashfs.bin for the correct device from the latest build (as of today at https://downloads.openwrt.org/chaos_cal … /generic/)

  • Put those two files in your TFTP root

  • Boot the AP with serial connection active, keep pressing a key quickly until it drops to <ar71xx> prompt

  • Copy/paste the flashing commands into your serial session https://wiki.openwrt.org/toh/meraki/mr16

  • After it finishes and reboots into OpenWRT, change root password with 'passwd'

  • Done.  LuCi web interface is available to configure device.

This information is available in a few different places but I thought I would put it all right in this thread since it shows up high in the Google search results for anyone else looking to get OpenWRT running on their Meraki MR16.  This is the USB adapter I used http://www.readytoflyquads.com/cp2102-usb-adapter
