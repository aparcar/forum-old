I know this is an old post but I came across it yesterday :-)

This is more in case someone also land here through Google...

You are welcome to look at the RADIUSdesk project which include two sub projects

  • MESHdesk

  • APdesk

http://www.radiusdesk.com/

You can now source those $10 Mediatek devices from China and have them centrally controlled either as APs or as mesh networks (Much like Cloudtrax but now under your own control and with some added features not included with Cloudtrax)

And since it is OpenSource the code is available to tweak.

We also added support for 802.11AC based Atheros devices recently and so far the meshes been rock solid using Ath10K. (These devices goes for just over 50USD for decent quality ones)
