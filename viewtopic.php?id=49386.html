Trunk doesnt come with Luci (the web interface).

Follow the link nebbia posted

1) log in with telnet, set password
2) log in with SSH using the password you set
3) install luci:
opkg update
opkg install luci
/etc/init.d/uhttpd start
/etc/init.d/uhttpd enable
