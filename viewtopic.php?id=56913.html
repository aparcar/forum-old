drawz
20 Apr 2015, 16:18

Snapshots are pretty stable right now, but keep in mind that they don't include a web GUI by default. You'll have to install that manually (see Luci entry in the wiki). Also keep in mind that any packages you want to install need to come from the same nightly build (more or less). So download firmware, flash firmware, and install any packages in one sitting.

Just install the kernel module:

opkg update
opkg install kmod-ath10k

At least thats required for an Archer C7 v2 which is quite similar to your model. Maybe delete your /etc/config/wireless and reboot your router.
You should have both interfaces mentioned in that file after the reboot (i think the second wifi interface would be added if you dont delete it but im not sure).
