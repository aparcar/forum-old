Two options to return factory firmware from OpenWRT:
Renamed ArcherC7v1_en_3_13_34_up_boot(140402).bin (i renamed it to tplink_boot.bin) upload to /tmp folder of rooter through SCP protocol (WinSCP). Connect with PuTTY SSH and type these commands:

cd /tmp
dd if=tplink_boot.bin of=tplink.bin skip=257 bs=512
mtd -e firmware -r write tplink.bin firmware

or failsafe mode & ftp server 192.168.1.2
Run "telnet 192.168.1.1" :

cd /tmp
wget ftp://192.168.1.2/tplink_boot.bin
dd if=tplink_boot.bin of=tplink.bin skip=257 bs=512
cat /proc/mtd
mtd -e firmware -r write tplink.bin firmware

or Use Hex editor to cut first 20200 bytes. Download edited bin here: http://www28.zippyshare.com/v/6473793/file.html
This file is ready to flash with web interface.
