Hi all
    I've recently acquired a wireless broadband router box which I'm hoping to put OpenWRT on. The model number is not listed as being supported directly, and it has a number of peculiarities, so I'm looking for help in identifying whether I stand a chance with existing OpenWRT firmwares.

The product label says 'WGV614-A3", which would make it a CyberTan Broadband Router. And indeed the CD-ROM PDF manual describes it as that. As you'd expect, it has four LAN Ethernet ports and a WAN RJ-45. I can connect to the web admin page and this is how it's described.

However, I've seen elsewhere on the OpenWRT wiki that the unit with this model number (WGV614) is supposedly a a clone/OEM of the Linksys WRTP54G. And an archive of the cybertan page here:

    http://web.archive.org/web/200605140020 … wgv614.asp

describes it as a 'VOIP Security Gateway', with two VOIP ports. So there is something very odd here, in that in theory this unit has *no* VOIP ports.

But, looking inside the unit... as well as the WAN RJ-45 I found two RJ-11 telephone sockets, blanked off by the rear panel! Also it is populated with two telephony chips (see below). So maybe this was destined to be one sort of box, and ended up as another after some OEM deal or another.

The hardware inside is as follows:
- CPU: IXP425 processor. A nearby Crystal says '33.330'
- RAM: 2 off CSI IC42S16800-7T (4Mb each?)
- Flash: 1 off Intel TE28F640, paper label says '1.00.14'
- 1 off Infineon ADM6996L chip (ethernet switch)
- 2 off telephony chips - si3210-KT
etc.

So again, I presume it can't really be a clone of the WRTP54G, as that is listed as being AR7-based?

There is space for one 20-pin header (JTAG?) and two 5-pin headers which I think are 3V3 serial ports ;-)

The mini-PCI radio says "FCCID: WRV54GP2M' and the MAC addresses OUI is that of CyberTan.

nmap -P0 -O 192.168.15.100 returns 'linux 2.6', but I'm not holding out hope for a CyberTan SDK...

I've put a couple of pictures of the inside here:

http://images.jkn.webfactional.com/cybe … CF0148.JPG
http://images.jkn.webfactional.com/cybe … CF0150.JPG

So - soes this sound similar to any other better-documented unit, and given this HW, what would be the best thing to try booting from?

Thanks for any pointers and/or suggestions.

    J^n
