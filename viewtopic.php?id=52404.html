Long story:
First, I updated from Attitude Adjustment, installed packages for mounting the hfs usb drive, and found that '/dev/sda2' always mounts in read-only. Always. Then I re-plugged the drive into other usb port, and found that '/dev/sdb2' (though it's the same disk!) mounts read-write without a fuss.
But then, when I tried to set up a network share, Opkg said that there is no neither 'netatalk' nor 'samba36-server' available (yes,  I did 'opkg update')

I thought that my sysupgrade got corrupted, and repeated it.
Et voila - I have all the packages available, but now my usb drive becomes 'sda' no matter what port I plug it in!!
And if I force the drive to be 'sdb' by pluggin in usb stick first, I still can't mout it in read-write anymore.

How on Earth?! It's still the same Barrier Breaker 14.07-rc3!!
