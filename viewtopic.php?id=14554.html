You could download the .trx version of the right image (say, be lazy and get the openwrt+x-wrt off http://www.x-wrt.org)

cd /tmp/
http://downloads.x-wrt.org/xwrt/firmware_images/whiterussian/0.9/milestone-3-rc2/default_extra/openwrt-brcm-2.4-squashfs.trx
mtd write /tmp/openwrt-brcm-2.4-squashfs.trx linux && reboot
