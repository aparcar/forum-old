Martin's right.

You can do *anything* with OpenWrt ! Except bash.org top #2... but we've got people (probably [mbm] in secret) working on that.

The wiki has a GREAT WEALTH of information. Howto's, docs, and almost anything you wanted to know. The forums and IRC logs supplement that, and fill in gaps in the wiki.

However, you should have basic knowledge of Linux  before using OpenWrt. If you've never heard of "vi", then it's probably not for you.

nbd's web interface is really very good, but if that's the only part of OpenWrt you use, you're losing a lot. nbd's qosfw script (traffic shaping) has to be manually configured, I believe. That's vi. No nano/pico for WhiteRussian!

So, if you're willing to learn, gopher it! But, know what you're doing.

Cheers,

- Yasha
