Hi,

I tried to use HT40 with my TL-WR841N.
The relevant entries in /etc/config/wireless look like this:

config wifi-device 'radio0'                                                                                                                                           
        option type 'mac80211'
        option hwmode '11g'
        option path 'platform/qca953x_wmac'
        option txpower '19'
        option country 'DE'
        option noscan '1'
        option disabled '0'
        option channel '6'
        option htmode 'HT40+'

The TL-WR841N connects nicely to my main router (an AVM Fritz!Box) but this main router shows a connection with 20 MHz and a max rate of 130 Mbit/s only.

Any ideas?

The AP must also be configured for HT40.  And the sub channel needs to be clear if its 'noscan' is not set.  Setting 'noscan' is a bad idea.  If you have neighbors, stay with HT20.

@mk24 - thanks, works nicely with your hints.

Apparently I had to uncheck "WLAN-Koexistenz aktiv (2,4-GHz-Frequenzband)" in my FritzBox main AP to get 40MHz.
