You can still install LuCI by running the following line via PuTTY (just copy-paste it):
opkg update && opkg install luci-app-firewall luci-mod-admin-full luci-proto-ppp

Package dependencies will install everything else automatically.
Note: I'm not sure if this includes everything the default LuCI has. So if you miss something in LuCI you have to install the missing packages by yourself.

Have fun!
