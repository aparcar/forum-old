Post #2
giri@nwrk.biz
28 Jun 2016, 16:45

Are you building your image completely from source? If yes configure your build with make menuconfig.
opkg is not needed. The Cryptographically signed package lists thing is used to authenticate downloaded packages.If you drop opkg you will not need this. uclient-fetch is a replacement for busybox wget. If you do not need it, you can remove it.

I wuldn't touch the first time init scripts. I think they are used to set up the filesystem etc. but I am not sure about that.

giri@nwrk.biz
30 Jun 2016, 01:52

It should be enabled by default.
You can configure linux kernel dependent stuff via 

$ make kernel_menuconfig.

But be careful, the kernel is the "layer" between hard and software. I something stops working you may have disabled a driver.

Another thing you can do to shrink the image size is dropping ipv6 support. (Only if you do not need it of course )
