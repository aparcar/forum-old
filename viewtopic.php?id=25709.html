The Option GE301 actually exposes 3 different USB ports.

/dev/usb/tts/0 - used for data connectivity (i.e. you chat script)
/dev/usb/tts/1 - used  for?
/dev/usb/tts/2 - used to query the modem about the signal strength and accessing the SMS buffer, etc, etc.

Make sure you have "/dev/usb/tts/0" configured in your "/etc/config/network" file for the 'device'
