(serial and WiFi working)

I can install OpenWrt by connecting to the router web interface through WiFi,
but cannot use Ethernet to login and continue with the setup afterwards.
1. How can I preconfig the OpenWrt image to work right out of the box (with serial and WiFi enabled)?

Hello,

flash the firmware via wifi - webinterface and finish the configuration via serial port (enable wifi and if you like install luci).

The wiki has a lot of instructions on how to get serial working and how to flash firmware via serial if bricked.

But to be clear, this is on your own risk! Flashing via wifi is not perfect but it should be possible imho.

Regards
Sebastian
