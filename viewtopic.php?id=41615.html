There are several ways to do it.

One way is to use scp, which is a separate program which uses the ssh-protocol.

Copy to ssh server: scp -P <port> <local file> <user>@<server>:<remote file>

Copy from ssh server: scp -P <port> <user>@<server>:<remote file> <local file>

Depending on the operating system on your computer, you may be able to mount a remote file system using the ssh protocole. This is possible in for example Ubuntu (and other Linux dists with Gnome). (Use connect to server and select SSH as type in that case.)
