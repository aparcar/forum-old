Greetings all, my first post here. I'm trying to install OpenWrt on a Linksys RE1000 v2. It looks like DD-WRT can work with v1 (RE1000 v1) of this wireless repeater as described here: http://www.dd-wrt.com/phpBB2/viewtopic.php?t=154741

The procedure used required changing the first 4 bytes of the DD-WRT firmware to "RE1K", tricking the device to install it. However this trick does not work with the v2 version. Looking at the stock firmware for both versions, i can see that v1 has the 'RE1K' in it's first 4 bytes but the v2 doesn't. Matter of fact, the v2 firmware doesn't seem to start with any readable text in it's first 4 bytes. How would one get around this? so i can install another 3rd party firmware.
