I made a few routes to be able to route between my 3 places using route command. I wrote the command to rc.local, so that they could be set again after restart. But it seems they are erased after /etc/init.d/network reload. What is the correct way to make them persitant?

Post #2
neryba
18 Jan 2014, 22:44

create script in /etc/hotplug.d/iface/ directory where add :

[ "$ACTION" = ifup ] && [ "$INTERFACE" = "WAN" ] && {
#here add yours routes
}
or add route to network config with http://wiki.openwrt.org/doc/uci/network#ipv4.routes
