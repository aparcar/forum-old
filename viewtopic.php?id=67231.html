I think it's a combination of both size of ram and flash size (only 4MB). Most openwrt images have sizes near 8MB these days unless you trim a lot of "fat" from them. If the router had USB you can go them the extroot way and have fs on USB flash but that is not possible here.
Also you've mentioned brcm-24 - that's another reason as no-none really wants to support 2.4 kernel any more. I believe this router actually can run 2.6 kernel as well which I believe tomato does.

Don't get me wrong, I like this router at one point I've bought and ran 3 of WHR-G125 initially with dd-wrt and later switching to tomato But it's finally showing it's age so I have gradually replaced them with newer hardware. Still have one sitting on my shelf and I'm sure I'll find good use for it. It's definitely great as a spare or as a travel router given its size.
