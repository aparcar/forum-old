Hello forum, I am playing with my router and I sometimes put it on Monitor mode to scan some bytes. Them I want to put on Managed mode (client), how could I restart interface without reboot?

I use to reboot the router to apply changes.

Post #2
fyi
3 Aug 2011, 23:13 
wifi down
uci set wireless.@wifi-iface[0].mode=monitor
wifi up
wifi down
uci revert wireless
wifi up
