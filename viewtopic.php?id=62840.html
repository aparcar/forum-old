I'm running CC 15.05(this, to be exact: image) on the WRT1900ACS and have some huge problems with wifi performance.

What happens is that I only get ~15 Mbps download speed and ~50 Mbps upload speed using the 5 GHz radio, with the following settings:

config wifi-device 'radio0'                                                     
        option type 'mac80211'                                                  
        option hwmode '11a'
        option path 'soc/soc:pcie-controller/pci0000:00/0000:00:01.0/0000:01:00.
        option country 'NO'  
        option disabled '0'   
        option htmode 'VHT80'     
        option txpower '20'          
        option channel '40'       
                                     
config wifi-iface                    
        option device 'radio0'    
        option network 'lan'         
        option mode 'ap'                                                        
        option ssid 'Network 5GHz'
        option encryption 'psk2+ccmp'                                           
        option key 'password'

But if I reboot the router the performance increases to above 300 Mbps in both directions. So it works really well until it has been running for a couple of hours, and then it becomes insanely slow.

I have testet both between WLAN <-> WAN and  WLAN <-> LAN, and the bottleneck is clearly the wireless connection.

After some searching I see that others has the same problem. Is there a way to fix it?

Is this one of the bugs the wiki speaks of that I will get rid of by downgrading to RC3? Is there any bugs in RC3 that is fixed in Final that I should be aware of?

I'm just trying to figure out what to use to get an stable experience. So far everything seems to work, with the exception for the strange wireless behavior.

I really appreciate help and information on this, and form what I've seen by searching others do too.
