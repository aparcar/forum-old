and update my Airlink101 ar525w router using web interface. After the route rebooted, my PC could not get any ip address. Post #2
Wodin
11 Feb 2008, 00:59

See if you can get into failsafe mode

Run netcat as specified in that page and see if you get the message at about the same time as the DMZ light comes on.  If you do you can probably get into failsafe mode.

By the way, if you changed the IP address of the router before flashing it, the failsafe IP address will most likely be the IP address you set it to, and not 192.168.1.1.
