You don't install anything on the device. It's a USB wireless adapter. You install Openwrt on a router and then plug the device in to it.

If you want reliable antenna switching you might want to stick to a standard Atheros or older Broadcom router. The drivers for USB Atheros devices are quite limited in comparison.

Openwrt is an operating system. While technically the WNDA3200 does have an operating system of sorts, it's hardware capabilities prevent it from running a general purpose operating system (e.g. Linux/Openwrt), it can only connect to another device and cannot run independently.

Nonetheless I'm still not sure why you'd want to. You'd have to connect the adapter to something and the cheapest thing you could connect it to is a router - anything else would be even more expensive.

An external wireless switch would imply you already have another wireless radio to connect the switch to - and chances are that already has an internal switch so why the need for a USB adapter?
