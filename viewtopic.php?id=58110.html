Yes, you can configure it as a client/repeater for a WPA2 Enterprise network using wpad.

There are two versions of wpad. "wpad-mini" and "wpad".
The "wpad mini" doesn't include support for WPA2 Enterprise.
The "wpad" is the full package and supports WPA2 Enterprise.

One of them is usually pre-installed.
If you have the mini version installed, just remove it and install the full "wpad".
