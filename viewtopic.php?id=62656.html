Hi there,

I've written an unbrick utility for Netgear routers using the NMRP protocol. So far I've tested it on a Netgear EX2700, but I'd be happy if others could test it on their Netgear routers as well.

Note that the image file must be in the same format as the stock image files, so you cannot directly flash openwrt sysupgrade images, (openwrt -factory images work though)

Instructions, source code and binaries for Linux, OS X and Windows can be found at https://github.com/jclehner/nmrpflash.
