The package responsible for active ftp support over nat is kmod-ipt-nathelper. To allow incoming ftp data connections you will probably need to match ftp and allow connections on firewall, quick google gave me http://slacksite.com/other/ftp-appendix2.html but how to alter /etc/config/firewall exactly I have no idea.

Much easier way is probably to use pasv command before dir or data transfer ;-) If you use some ftp gui, look for "use passive ftp" checkbox.
