In “make menuconfig”, set “/Target System” to “Atheros AR71xx/AR7240/AR913x”. You can either leave “/Target Profile” set to “Default Profile (all drivers)” or you can change it to “Buffalo WZR-HP-G300NH”. Both choices will build an image for your board; the default profile also builds for many other ar71xx boards.

You might also want to choose packages and make other changes while you’re in “make menuconfig”.
