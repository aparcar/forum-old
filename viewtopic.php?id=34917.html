Hi, I was hoping you guys could help me by recommending hardware for a project I'm creating.

I'm looking to place a router in a remote location, powered by solar and battery. It will be off most of the time, but a passerby will be able to turn it on. Router will boot and provide a wifi network that clients will be able to join. Router will provide a captive portal where users can sign a virtual guest book for the remote location. Data is POSTed to a small DB server on the router, and users can view previous guestbook entries that have been left. There will be no internet connectivity.

So the requirements as I see them:
Wifi router
small http server
captive portal to redirect all traffic to http server
small db server
some kind of scripting language to post data from http server to db
low-ish power due to battery/solar setup

I've never worked with OpenWRT previously, but I've done some sysadmin work so I'm not a total beginner. Happy to spend a few extra bucks to buy the hardware that will give me as few headaches as possible.

Thanks!
John

The TP-Link TL-MR11U is supposed to work with a battery.

In case the usual 4 or 8MiB of flash memory won't be enough, you could use a USB stick. Then you also require an USB port.
