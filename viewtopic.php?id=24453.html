jow

kshbu0: for OpenWrt it does not matter on which media it runs, x86 uses the same codebase as the builds for wrt54gl and others. There are virtually zero writes to flash/harddisk etc. when the system runs. Stuff like pid files are stored in /tmp which is a ramdisk and the systemlog is a ring buffer in ram. Writes only occur if you edit config files or change stuff in the web interface.

Also use squashfs images on your CF card, ext2 is not really suitable as flash filesystem and it is easily damaged by sudden powerloss and such.
