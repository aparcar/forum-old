You need to install two packages, php5 and php5-cgi.
Installation can be done through the web interface (System -> Software) or through command line (telnet or ssh) with the command "opkg install <package_name>".

Then you use a text editor like vi or nano (nano is not installed by default) in command line, and you uncomment (that means remove the "#") the line that refers to the php interpreter, in /etc/config/uhttpd file

Restart the web server (/etc/init.d/uhttpd restart) or reboot the device and php should work now.

I just hope your router has enough free space for package installation...
