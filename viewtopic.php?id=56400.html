Adze
22 Mar 2015, 12:14

This is perfectly normal. As you are on PON, this is a point-to-multipoint connection. If you wish to communicate with other users on the same pon connection, frames should first be send to the ISP gateway (OLT) and then back again to the subscriber (ONT). All the ip addresses you see on eth1 are the ip addresses from your neighbors, which are on the same pon connection and are only reachable via the mac addresses from the pon gateway (proxy arp).

http://en.wikipedia.org/wiki/Passive_optical_network
