Post #7
makro
7 Aug 2014, 16:11
nozombian wrote:

So if client has iv6 only, it is invisible to arp?

Yes, because ARP is specific to IPv4. Neighbor Discovery Protocol serves the same purpose in IPv6. 

ip neigh show

shows both IPv4 and IPv6 entries because neigh doesn't refer to NDP specifically, only to the term (network) neighbor in general. You can use ip -4 neigh show and ip -6 neigh show to list them separately, if desired.
