If OpenWrt is open and it times out (/etc/luci file variable for sauth is exceeded) and you open a new tab to login, OpenWrt will return to the login screen.

Make sure all your OpenWrt tabs are closed and try again. 

You are not specific on Browsers and OS, which you should include in general for diagnostics.
