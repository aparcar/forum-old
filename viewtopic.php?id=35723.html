Hi all,

I am building an image for my HP thin client 5340 and willing to have router up and running right after dd-ing image into device.
Here i can raead following:

Custom files
In many cases, you may want to have a custom image that has been pre-configured to your link. If so, place your custom files in

<buildroot dir>/files/
For example, let's say that you want an image with a pre-configured /etc/config/firewall, then place your modified firewall config in here:

<buildroot dir>/files/etc/config/

But the question is that some programs need specific permissions to run. For example lcd4linux will not start if permissions of file /etc/lcd4linux.conf will be other than 600.
If i'll put permissions 600 to the original <buildroot dir>/files/etc/lcd4linux.conf, build process will fail complaining that the file is not readable.

So I am asking help/guide to set correct permissions to the specific files after build has been finished.

Thanks in advance.
