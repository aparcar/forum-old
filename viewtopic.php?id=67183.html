It depends of your wifi chipset.

e.g. in my ath9k htc usb wifi dongle (tplink) encryption is perfomed in the firmware of wifi dongle, that can be disable in order to perfom a software encryption in the router. To allow this you must specify the option "nohwcrypt=1" when load the ath9k kernel module.

Once enabled software encryption, the modules are wpad for station and hostapd for ap mode.
