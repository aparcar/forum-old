Now i'm Openwrt Backfire (10.3) on a D-link DIR-320 router.
I already added a usb thumb to have some extra storage space for additional packages.

In opkg.conf i added:

It's any way to avoid it telling LUCI gui to install extra packages to usb thumb instead of root?

opkg seems to pick the first entry from opkg.conf if none are specified.

I added my mounted storage as the first dest entry in /etc/opkg.conf and now new packages are being installed there by default. free space status must be pulled from a different conf, so it does not reflect the new destination.

Now you can un-install & re-install packages to free up a bit a space too (as things work better with more than zero bytes free).
