Since yesterday I set up OpenWRT Backfire on my TP-link wr1043nd and I am really enjoying it so far!

I configured a PPPoE connection on the WAN interface, which connects through a modem that's set up in front of it, which works perfectly fine and stable (so far). The only issue I have is in configuring OpenWRT to use custom DNS servers instead of the ones provided by my ISP. My goal is to be able to watch Netflix from abroad, by using the DNS servers provided by Unotelly.com.

I have tried this in the Luci interface by going to Network > LAN and by filling out the IP address in the 'use custom DNS servers' field. This doesn't really seems to work, though. I am still able to surf the internet, so I presume I'm still connecting through my ISP's DNS's.

Can anyone help me resolve this issue?
I have a feeling I'm missing something really simple ...

You need to configure the custom servers on the wan interface.

Thank you for your response, jow!

The option was a little bit hidden. Had to uncheck the 'Use DNS servers advertised by peer' option first. I think this'll solve the issue. I'm going to test right away ...

EDIT: in the meanwhile I've tested the new config and it seems to work now. Thanks again!
