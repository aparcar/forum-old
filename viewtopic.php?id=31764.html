If you mean to "update the package feed" with your patch for all users, then you should create a patch against the SVN repository and use the bug tracker to create a ticket with explanation and the patch included. https://dev.openwrt.org/timeline

Developers may notice it there and include it to the main source, althgouh formally you should mail the patch to a mailing list:
https://dev.openwrt.org/wiki/SubmittingPatches
