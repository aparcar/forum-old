Hmm, again answering myself ;-) as there is coming more and more light in this and at least
some people are looking at this thread and might be happy to get some infos.

It is now more or less clear that there was a buggy release from 29-May-2012 preventing access to the ethernetports of the box.
This faulty firmware is now removed from http://downloads.openwrt.org/snapshots/trunk/ar71xx/

Problem is, that at the moment there is no firmware at all for almost all devices in this trunk.

So if someone can provide a working firmware for tl-wr740n version 4 please let me/us know. I'm willing to test a debrick procedure
as soon as I have a working firmware (and ok course having soldered this damn serial connection).
