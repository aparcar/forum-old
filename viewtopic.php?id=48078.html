I've noticed that after I define a Static Lease for some device, it doesn't always seem to take effect - ever.
I have one IP Cam that came up at 192.168.1.207.  Since I wanted it in a cluster with my other IP cams, I defined a static lease for it at .120.  That was 2 weeks ago.  The camera has been powered off/on multiple times, but it's still at 207.

Why isn't OpenWRT giving it the .120 address?
