Default arp is shell function defined in /etc/profile :

[ -x /usr/bin/arp ] || arp() { cat /proc/net/arp; }

use ip neigh del <x> dev <y>
My script to flush all neigbors :

#!/bin/sh

for v in 4 6 ; do
 ip -$v n | sed -nr "s/^(.*) dev ([^ ]*) .*$/ip -${v} n del \1 dev \2/p" |
  while read c ; do
   ($c)
  done
done

If "ip" curses you need ip-full
