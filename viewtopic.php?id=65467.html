1 - I do not know what is causing this problem but you can avoid it by working directly with the aria2 configuration file. The Luci in this case is totally unnecessary.

After doing so just add to rc.local the command to load the settings (if the setting is wrong it does not load), example:

aria2c -c -D --conf-path=/etc/aria.conf


2 - I believe that this limitation of servers is the package itself available for calmer chaos, when tested in this version also noticed few resources.
