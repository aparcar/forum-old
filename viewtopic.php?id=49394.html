So,  have you seen the router at 192.168.1.1 since you flashed it with OpenWrt?  Is it still there?  If it was and you have edited the network settings and now lost it, then you have two options. 
Plan A is to search for it using netscan, remembering that you must have a network card on the same subnet as netscan is searching.  It might take a long time, especially if you have screwed up the subnet rather than the IP address.
Plan B is to treat as a 'bricked' router and solder a serial to USB adapter on, get the instructions by googling 'bricked wr703n'.  This might take even longer if your soldering skills are not up to it and you might finish up with a dead router.
Best of luck,
John
