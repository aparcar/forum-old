The other way around, since we are in an OpenWrt forum:

Yes, it is possible to run OpenWrt on that router, the Wiki page shows several revisions/variants, so you might want to keep an eye on that.

And if you run OpenWrt on the router then yes, it is possible to run OpenVPN. However, the router only has 4 MB of flash. If memory serves, there is not enough space left to simply install OpenVPN, you may have to build your own image (to get OpenVPN compressed) or use extroot (to extend the flash space).

Post #8
metai
28 Jul 2014, 00:22

You cannot uninstall packages that come with a build. They are baked into the compressed SquashFS image. "Uninstalling" only marks them as deleted but does not free up the actual space. The only way to not have them take up space on the flash is by creating a custom image that doesn't include them in the first place.

/tmp/ is a "ramdisk" and exists only in memory.
