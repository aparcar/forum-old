@ jow

This Lantiq/Danube WIFI ADSL router (WiFi+adsl+ 4lan ports) is brand new target (SIEMENS SX763) supported by OpenWrt could it be that the developers forgot to include switch config option for this router?

After I added: 

config 'switch' 'eth0'
        option 'enable' '0'
        option 'enable_vlan' '0'
        option 'reset' '1'

I got Switch tab under Network config section!

IMHO this switch config option should be integrated for this target.
