Dioptimizer
28 Sep 2012, 10:57

Likely JEDEC ID of flash memory - some other. If not difficult: download then follow the instructions in this thread
load and run test uboot:

tftp 80000000 8Muboot_RAM_version.bin

go 80000000

fast put in console tt (not tpl)
and run the command:

flinfo

(print FLASH memory information)

If the value JEDEC ID is not as in MTD driver source:

{ "en25q32b", INFO(0x1c3016, 0, 64 * 1024,  64, 0) }

- it will speed up the supporting your hardware version by OpenWrt
