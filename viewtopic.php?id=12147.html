You can pickup your script in /etc/hotplug.d/iface/
Script will be executed each ifup or ifdown (wan down ...)

Here is ntpclient script wich has a part for ifup state and ifdown state :

#!/bin/sh

case "${ACTION:-ifup}" in
        ifup)
               code for ifup
        ;;
        ifdown)
                code for ifdown
        ;;
esac
