Post #6
hnyman
4 Apr 2016, 16:54

You should try both ssh and telnet with
- empty password
- the old root password that you used to enter the Luci GUI in order to flash the new firmware (when you downgraded from DD to CC). The default option is to keep settings in sysupgrade, so the old password might still be valid.

(You might also reboot the device once more, if you have not yet done that.)

If none of those work, then you are heading toward reseting your device.
DIR-825 has many variations, but I guess that the default "generic failsafe" mode should work.
https://wiki.openwrt.org/doc/howto/generic.failsafe

Second observation:
Are you sure that you have flashed the correct version of DIR-825 firmwares? Your link (that you already deleted) showed that you had applied the firmware for "DIR-825 B1" version. Is that correct (and matches the label the bottom of the router)? There are many versions of DIR-825 with totally different hardware, so you need to be careful. (there should be a version check, so likely you have the correct version, but just asking...)
https://wiki.openwrt.org/toh/d-link/dir-825
