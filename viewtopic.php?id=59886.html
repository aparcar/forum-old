Hi Folks! ...

This device can successful booting with openwrt bleeding edge (r47494)
I created an image for the ZBT-WG2626 using the latest ImageBuilder-ramips-MT7621 from trunk.

make -j4 PROFILE=SAP-G3200U3 image PACKAGES="kmod-cfg80211 kmod-i2c-mt7621 kmod-mt7601u kmod-mt76 kmod-mwifiex-pcie kmod-rtc-pcf8563 base-files odhcpd -odhcpd6 kmod-mt76 uci2dat 8021xd mtd pciutils usbutils wireless-tools"

This board (zbt-wg2626) is compatible with PBR-M1 (PandoraBox model1), then you can build a new image from Chaos Calmer, using the imagebuilder for ramips-mt7621

make -j4 image PROFILE="PBR-M1" PACKAGES="kmod-mt76 kmod-i2c-mt7621 kmod-mii kmod-sdhci-mt7620"
