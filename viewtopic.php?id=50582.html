Dhcp should work, but if your router somehow fails to connect, you won't ever reach it, because it does not get ip. Strange is that it responds to ping, but you should use static ips for devices reachable only via network. What I'd recommend: now get your v8 router in failsafe, firstboot, set by "dumb ap" recipe and set wifi again.
