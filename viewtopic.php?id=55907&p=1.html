lc
23 Dec 2015, 15:51
ichzocke wrote:

Hi i bought a new cpe210 v1.1. Will there be a openwrt firmware for v1.1? thx

To make TP-Link Stock Firmware load OpenWRT on CPE210 v1.1 hardware you have to edit
tools/firmware-utils/src/tplink-safeloader.c

Search for 

"CPE210(TP-LINK|UN|N300-5):1.0\r\n"

Add a new line 

"CPE210(TP-LINK|UN|N300-5):1.1\r\n"

For CPE510 v1.1 it would be 

"CPE510(TP-LINK|UN|N300-5):1.1\r\n"

Recompile --> done

I am not a dev. Maybe a dev could apply this change to the CC trunk.
