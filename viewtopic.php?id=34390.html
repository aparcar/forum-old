Assuming your script works (we'll get to that in a second), after you've run "/etc/init.d/tmount enable" do you see a symlink in /etc/rc.d named S59tmount or something similar, with whatever number you set in the script?

Second - does your script work? In a running system, any script in /etc/init.d should work, such as "/etc/init.d/telnet status" - does your script do what you want if you type "/etc/init.d/tmount start" and then "/etc/init.d/tmount status"?
