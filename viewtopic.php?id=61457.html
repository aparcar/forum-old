stas2z
19 Dec 2015, 13:00

if no one here have the hacked art, the only way is build it by yourself

you need your art partition and xxd tool (linux, but have a windows port) which can dump binary to text and also reverse build binary from existing dump

here is my post about hacked art for wdr3600 https://forum.openwrt.org/viewtopic.php … 89#p300089
you can easily found unmodified wdr3600 art and comprare it with the hacked one to determine where the changes are made and do the same with your art partition
you can check your changes made with pepe2k's ar9300_eeprom tool https://github.com/pepe2k/ar9300_eeprom (linux only, but probably can be compiled under cygwin) and also this repo contains original wdr3600 art dump

a note, wdr3600 art contains regulation data both for 2.4 and 5ghz, but for your router you need to find only 2.4ghz data, 5ghz can be empty or garbage

khmtambi wrote:

Guyz I am using wdr3500 board... but when I try to run the command ./ar9300_eeprom -d art.bin it gives me output:
"Board not found"... please help

./ar9300_eeprom -y0 art.backup -u art.modified

art.backup is the original dump from your art partition
art.modified is the output of the ar9300_eeprom script. You use this file to write back on the art partition
