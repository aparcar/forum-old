johnisaacson wrote:

I'm curious how you would even know you had OpenWrt correctly installed?

In the same way you know that OpenWrt is installed on a router: Connect the Pi to your ethernet network and use another computer to connect to the Pi at the usual address.

johnisaacson wrote:

to attach a usb serial port adapter to use a standard pc or laptop. From there as said above
you can use a telnet session to first login.

Then you have a serial connection, no Telnet. Telnet is a protocol used on a TCP connection.

You can do everything you need with a serial terminal including setting up network and ssh. If you have any kind of a problem and it doesn't boot up perfectly then you are unable to see the problem. Lastly it is cheap in that all you need is a rs232 device to usb (carefull of the voltages ie 3.3 versus 5v) and a terminal program on your networked computer. I work the "peter principle..if something can go wrong...it will."

For telnet if you want try this link http://logmett.com/index.php?/products/teraterm.html

Good luck.
