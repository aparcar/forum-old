Post #2
CyrusFF
1 Jun 2015, 21:22

IPv6 configuration works differently. Router Advertisements are used to announce routes, DHCPv6 is mainly optional and doesn't do anything with routing.

For Router Advertisements there is also no "router address", instead clients use the source-address of the  RA as gateway for all the routes advertised by a router.

In general the default setting should be fine.
