Took me some time, but after using a spare linux system, I found out what driver to use.
The TP-Link is a fairly new USB3.0 Gigabit ethernet dongle, which I connected to my Raspberry Pi B+ running OpenWRT.
The package to install is kmod-usb-net-cdc-ether. It'l pick up on the TP-Link.
