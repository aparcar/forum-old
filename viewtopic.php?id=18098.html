While I don't have an answer to my first question, I found the radio settings for the AC881:

AT!SELRAT=XX

XX can be one of the following:

00 = Autmatic
01 = UMTS G3 Only
02 = GSM G2 Only
03 = UMTS G3 Preferred
04 = GSM G2 Preferred
