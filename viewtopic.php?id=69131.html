When you use snapshot builds, you'll have to get all the kmods you're going to need on the same day you downloaded the image.  If you need another one later you'll have to upgrade your entire firmware to that day's build and reinstall all your kmods and packages.

You could also download the entire packages tree (especially the kmods) along with the image and keep a local copy that matches your build.  Then you set up a small web server on your PC and point the /etc/opkg/sources configuration to it, so when you add a package later it gets it from your collection rather than the ones on the server which may no longer match.
