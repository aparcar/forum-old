It's option rxantenna and option txantenna. Example:

config 'wifi-device' 'wl0'
        option 'type' 'broadcom'
        option 'channel' '5'
        option 'rxantenna' '1'
        option 'txantenna' '0'

Seems to work on 8.09.1, but it can also be done with the web interface.
