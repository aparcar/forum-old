1+2) After flashing the overlay partition is empty, use extroot recipe [1] to use an usb stick instead.

3) Never put a swap file on nand, most likely you don't need swap anyway.

4) You can still use the router though it would be a pain.

5) They would be hidden away.

[1] https://wiki.openwrt.org/doc/howto/extroot
