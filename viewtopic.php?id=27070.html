WAN port - disabled, WAN connectivity is provided via another router.

Than I added via LuCI:
  collectd-mod-cpu
  collectd-mod-interface
  collectd-mod-network
and

Rebooted router. I see "Realtime Network Traffic" and "Realtime Wireless Status" in Status menu.

And ... there is no traffic in "Realtime Network Traffic" - just the same 0.6 kiB/s.

What is going wrong?
