I have a Linksys WRT54GL v1.1 that is currently running Victecs modified Tomato 1.23 firmware. It works well but has tempted me with what can be done with this router. The firmware is *almost* capable of everything I want it to do but getting help doing that is really hard on the forums.

Anyways after thinking about it openWRT's solution sounds alot better, install the packages you want instead of waiting for the author to add the functionality. Here is what I'd like to do with my router:

-QoS functionality - for magicjack VOIP and my gaming.
-Per-IP bandwidth monitoring - so I can figure out who is using the net and killing my ping in game.
-Have an easy way to temporarily kick/throttle users bandwidth so I can resume gaming.

-I Dont need wireless access since I have another router handling that.

I am not familiar with the various WRT's (DD, Open etc), can anyone advise me which would work best for me?

I know victecs tomato has QoS (very easy to set up too). Per ip you have to enable imq's tel-net into the router, sometimes they dont stick to assigned hosts and when the router reboots they are all lost. It's a mess and hard to get support on as its not a very important feature to victec. The kicking/throttling thing I've seen on tomato with some advanced scripting but that was for more permanent throttling and the configuration generator script site went down a long time ago. It also wouldn't be convenient for quick logging in and enabling throttles that only need to last a few hours.
