I think it's not!

Because every device is worked as a FAT AP,the 802.11 MAC layer is done in every individual device.
The solution is AC+FIT AP,every device(FIT AP)just work as a radio,many MAC layer thing go the AC.

I hope one day some cool guys can start a AC+AP project based on openwrt and openwrt supported device,that the roaming everyone expect!
