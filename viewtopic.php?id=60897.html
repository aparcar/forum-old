/etc is not read-only.

You can easily create config files on a PC, if the PC runs Linux ;-)

As for Windows, you can edit files using WinSCP, if you edit them in the right side of WinSCP. This way you edit files directly in the router. If you edit files in the left side, you will have problems with Windows end-of-line character.
