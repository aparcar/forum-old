Are there any implementations for Multi-link PPP on the open-wrt? I would like to try the following configuration as well (apart from the multi-radio multi-channel config that I just posted):

                         \        / <----------MLPPPa------>\        / <-- Radio2a
                          Node1a                                   Node2a
                             |                                              |
                         \   |   /  <----------MLPPPb------>\   |   / <-- Radio2b
                          Node1b                                   Node2b

We thus effectively get a theoretical 54+54 = 108Mbps between the two node-groups.
