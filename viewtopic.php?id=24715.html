Hello,

Here is the solution for the boot problem with a Soekris 4801. I found it on http://wiki.soekris.info/Pxelinux_hangs â€¦ _connected

The solution is to modify the menu.lst in /boot/grub :

Before :
serial --unit=0 --speed=38400 --word=8 --parity=no --stop=1
terminal --timeout=5 console serial

default 0
timeout 5
...

After :
serial --unit=0 --speed=38400 --word=8 --parity=no --stop=1
terminal --timeout=0 --dumb serial
hiddenmenu

default 0
timeout 5
...
