STA interfaces will not work as part of a bridge.  This is because the AP expects all traffic to a STA to go to only one MAC address.  If the AP is compatible, you could use AP-WDS and STA-WDS mode.  This transmits the destination MAC as part of the packet so you can bridge.

Not sure why such would brick though.
