It will probably work, but expect it to be pretty slow... it has a 200MHz processor after all

Beware of the slow USB transfer rate, mine has worked for years as an IMAP server (with Dovecot) but I've finally moved it to a real server as it was getting slower and slower.

Mysql can definitely work - I've used mysql-4 for some tests, remember to disable InnoDB and follow some online tutorials to configure it to save as much memory as possible.

2.4 or 2.6? Mine is working perfectly with a 2.6 trunk from some time ago (r15160)
