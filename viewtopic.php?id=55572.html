I've got my Huawei dongle working on BB by installing a few extra packages:

kmod-usb2
kmod-usb-net
kmod-usb-net-rndis
kmod-usb-net-cdc-ether
kmod-usb-net-cdc-ncm
kmod-usb-serial
kmod-usb-serial-option
kmod-usb-serial-wwan
usbutils
udev

I've no idea which is actually required, and don't have the time to go through one by one.

On CC you have to install in addition:

kmod-usb-net-huawei-cdc-ncm

Also, the command for usb_modeswitch is usbmode
