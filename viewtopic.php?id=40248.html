After configuring the build (selecting packages in make menuconfig etc.), you can do "make download" that will force the buildroot to download all needed packages. Then later, during the actual compilation make, there is no need to download.

The downloaded packages are stored on 'dl' subdirectory of your buildroot, so you need to download each package only once (unless you destroy them with make distclean or manual delete).

An naturally you can also download manually each lackge, if you just look into the download path stored in the package Makefile, and use that to download the corect version with your browser, wgeet or whatever.
