rizla3066 wrote:

could you recommend a similar spec router that would have full use of all features under openwrt? i can do without dual core and maybe even some flash memory but i would need dual wan, adsl2+ and cable and dual 2.4Ghz and 5Ghz wireless tho

I'm not aware of any dual band adsl2+ router supported by OpenWrt. All supported adsl2+ ones are single band only. All supported dual band routers have no adsl2+. So your nearest thing would be combining an adsl2+ modem and a dual band router, changing one of the lan ports into an additional wan port (should be doable on all supported dual band routers).

Download and transfer (e.g. using SCP) the packages listed below to your OpenWrt router onto the RAM disk in /tmp/luci-offline-packages and enable "uci set wireless.@wifi-device[0].disabled=0; uci commit wireless; wifi"
