Hi WFU03,


1. Choose a decent OpenWRT supported router with enough horsepower for heavy streaming. I would highly recommend you WNDR3700v2 (http://www.komplett.no/k/ki.aspx?sku=492607). It is not clear from the description on the website if the router is v1 or v2. But even if it is v1, it comes with 8MB flash which is OK (v2 has 16MB flash). The other important features are the same - 64MB RAM, 680 MHz CPU, dual band WiFi radio.

2. OpenWRT has matured great in last years and you can use its LuCI web interface to do most of the settings you might ever need. But even if you need to tweak something from the command interface, that is not rocket science

3. You don't have to be a programmer to configure and use OpenWRT Basic knowledge of Linux will be of great help, but even if you don't feel like having strong Linux skills, it is not that hard at all. There are a lot of howtos here in the forum to get you up and running quickly.

I hope this helps.


Cheers,
zn

WFU03 wrote:

I live in Norway, but want to use a VPN to watch country-restricted TV shows.

If you wanna an encrypted VPN connection, make sure your router has a built-in cryptography chip set. If you are using any 3G / 4G connection, perhaps you may be interested in some PogoPlug Linux embedded system that runs on a Kirkwood Marvell chip set that has a built-in cryptography chip. In this case, you use one of the USB to connect to the 3G / 4G USB dongle as your WAN and connect an n-port gIGE switch to its single gIGE port.
