Here I am replying to myself again .  I actually figured it all out.  The environment variable that sets the path to the libs is LD_LIBRARY_PATH.  The reason that it was difficult to find is that it is not defined by default (at least I couldn't find it).  My guess is that it is assumed to be /lib and /usr/lib if it is not defined.  Anyway, I will be writing a wiki page detailing how to install and use packages on a remote drive (in my case an SD card).
