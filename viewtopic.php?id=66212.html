Hello,
I bought a E1700 and installed OpenWRT and I can confirm that the switch ports are only working at 100Mb/s. It seems that there is a misconfiguration of the MT7530 chip in the drivers.
I tried official chaos calmer for the device, tried latest trunk version too but still the ports are limited to 100Mb/s. You can configure VLANs though.

With the original official firmware the port are working at 1Gb/s with no issue. I think you can configure VLANs too, I can check if someone is interested.

I'm currently trying to figure out what is wrong with the drivers but having never worked with drivers, I'm not sure I'll be able to solve the problem.

I have serial access to the device (J2 header) and can post boot logs, maybe someone more knowledgeable can help.
I also extracted the original firmware and it seems that it is possible to get the original initialization process from it.
