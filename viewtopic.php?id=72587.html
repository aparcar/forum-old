I have recompiled "master" for BS-U25AWF-H1

add 99_enable_wifi in: . ../etc/uci-defaults for start wifi with this command :
uci set wireless.@wifi-device[0].disabled=0; uci commit wireless; wifi
That is for compatibility with BS-U25CWF because this product is without LAN
