What type is your Internet connection? xDSL, Cable... ?

Configure Internet access in /etc/config/network and connect to the Internet then do:

opkg update
opkg install luci
/etc/init.d/uhttpd start
/etc/init.d/uhttpd enable

Finally point your browser to LuCI Web UI.
