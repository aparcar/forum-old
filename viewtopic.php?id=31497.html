is EoIP a special protocoll, or is it just a term for layer2-bridging over an IP-connection?
You could realize layer2-bridging very easily with openvpn in ethernet bridge-mode

If you install the luci-openvpn-package you can configure it in the webinterface. There are Templates for "Server Configuration for an ethernet bridge vpn" and "Client configuration for an ethernet bridge vpn" which should make it easy. I never used the webinterface for that, and I run openvpn in routing-mode with an manual configuration, but I guess the stuff in the webinterface works just fine.

Or you look at  http://wiki.openwrt.org/doc/howto/vpn.openvpn
the serverconfig should work just fine, the client-config file need to be changed to the syntax used in openwrt (UCI)
