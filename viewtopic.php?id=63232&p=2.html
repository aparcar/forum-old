disregard his procedure as there's no need to download big data and waste time. pull the source go to menuconfig and before you start customizing your build or saving changes uncheck ipv6 support from global build settings and busybox. all other ipv6 related packages will be automatically unchecked. if you can't stand ipv6 menus in luci pages you can remove those too, but this takes a bit more time to do.
