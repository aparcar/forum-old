Have you tried the failsafe mode?
When the router starts - pick the reset button untl the led blinks rapidly.
Then give yourself a static IP address of 192.168.1.2 and telnet to your device (now 192.168.1.1)
to repair your configuration. If your OpenWrt installation uses the squashfs file system - then use
the "mount_root" command to be able to make permament changes.
