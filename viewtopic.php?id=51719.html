Depends on your driver.

The command 'iw list' will tell you what it is capable of, you should get a line somewhere that says something like this:

 
        valid interface combinations:
                 * #{ managed, WDS, P2P-client } <= 2048, #{ IBSS, AP, mesh point, P2P-GO } <= 8,
                   total <= 2048, #channels <= 1

With 'IBSS' being ad-hoc. In the above example my driver supports up to 2048 WDS or client connections or 8 AP or Ad-hoc connections.
