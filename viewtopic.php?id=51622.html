danymarc
16 Jul 2014, 23:51

Upload the file using Winscp in the /tmp folder of your router.

Connect with Putty and give this commands:

cd /tmp

mtd -r write /tmp/tplink.bin firmware (use a cutted FW )

Every command at once hitting Enter.


If it is not the last version just upgrade normally after flashing this one.

ok, RESOLVED!
i'm a real noob, i have to admit it....
the solution was really simple: get through ssh into the device (usb stick unplugged) and give:
cd /tmp
wget http://www.whatever.com/whatever/firmware.bin
sysupgrade firmware.bin firmware

i was missing the wget part and just copying the firmware through winscp, but in the wrong folder
