I'm trying to get it working now.  I was also getting the "unsupported dictionary type: hash" error and was able to fix it by editing /etc/postfix/main.cf and replacing "hash:/etc/postfix/sasl_passwd" with "txthash:/etc/postfix/sasl_passwd". 

This tells postfix to read the sasl_passwd file as plain txt and with no file extension.  If you generated a sasl_passwd.* file earlier, you can get rid of it now.
