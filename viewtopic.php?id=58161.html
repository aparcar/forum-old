Can you telnet in?  On a non-LuCI install, you have to telnet in and set a password before you can use ssh.  ssh will act like it wants a password but there is actually no way to log in.

Also, if that doesn't help and you actually set a password you don't remember anymore:
- activate failsafe (however that's done on the WNDR3400, I'm not familiar with that device)
- telnet in
- mount_root
- passwd
- reboot into regular operation
