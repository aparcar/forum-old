Why would you not install the backfire? There is already in the firmware luci ...
or pick up an ftp server on the PC, and copy all of the packages there, and from there is set

1. We set a server on PC of FTP, for example Golden FTP Serve.
2. We feed to him a folder with packages(packages).
3. We correct the file of /etc/opkg.conf, prescribing a way to our server:

src/gz snapshots ftp://192.168.1.27/packages
dest root /mnt
dest ram /tmp
lists_dir ext /var/opkg-lists
option overlay_root /overlay

PS opkg.conf  You can edit the Web of muzzle. System-Software-Configuration.

for you OpenWRT is already set sewing, then you must load character of the new sewing on router in the folder /of tmp(for example by means of scp or WinSCP) under the name code.bin, whereupon connected to router by means of cantilever and will execute a command:

in putty:

cd /tmp

then

mtd -e firmware -r write code.bin firmware


PS code.bin it`s name firmware
