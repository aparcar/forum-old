You need to install all kernel-related modules from the same snapshot of a build run, as there is an overzealously strict kernel options checksum. Trunk snapshots get refreshed every day, so you need to be quick.

In practice, flash a new firmware and immediately install all needed packages.

Or if you are feeling adventurous, you can override the check with opkg option "--force-depends".
http://wiki.openwrt.org/doc/techref/opkg#options
