I used to use ddns-scripts to keep a legacy dyndns account alive, the default settings worked fine for me without any flooding. I too had a cable isp connection with an IP that never changes.

The check interval will just check and compare the current wan ip at the given interval but do not trigger an update if the IP didn't change. The force update will send an update even if the address didn't change. It should be fine to set that to 15 days or something.
