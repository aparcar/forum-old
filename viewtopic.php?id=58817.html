from picture you provided your router has only 1 ethernet port
id say openwrt is functional inside but you cant access it becouse the ethernet port is configured as wan and wifi is disabled
you could get a USB TTL serial adapter and test the 4 pin connector on top of the soc cip
if you manage to connect (usual parameters are 115200 8N1, no flow control, use putty, use only rx, tx and GND pins) you could try to configure wifi and wan properly

you could have wifi as wan and ethernet as lan or viceversa
or you could try ADSL FW recovery(you need a M@xnet 630i FW; couldnt find any detail on gogle...)

http://wiki.openwrt.org/doc/howto/generic.failsafe
http://wiki.openwrt.org/doc/howto/generic.sysupgrade
