HI,

uci show firewall gives me a lot of rule entries like:

  firewall.@rule[7].src=lan
  firewall.@rule[7].dest_port=22
  firewall.@rule[7].target=ACCEPT
  firewall.@rule[7].proto=tcp
  firewall.@rule[8]=rule

Now is there an elegant way to get the "target" value for the entry that has src set to "lan"?
I've  made a script, but is there a more elegant / build in function to do so?

Here is my not so nice version:

get()
  {
  name=$1
  local i=0
  while uci -q get firewall.@zone[$i].name > /dev/null; do
    new_name=`uci -q get firewall.@zone[$i].name` 
    [ "$new_name"  = "$name" ] && {
      uci -q get firewall.@zone[$i].input
      return
    }
    i=`expr $i + 1` 
  done
  }
