The answer is none. Backfire uses 2.6.x, Barrier Breaker uses 3.10.x and Chaos Calmer uses 3.18. You can't simply upgrade the kernel version (in a supported way) in OpenWrt as it's baked in the squashfs and the build system caters to a specific version.
