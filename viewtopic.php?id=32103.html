Anyway, I have a 3.3V, Tx, Rx, and GRD connectors soldered on and I'm trying to access the serial console so that I can erase the nvram and load up the default firmware. Funny thing is that when I opened the router one of the radios was actually not plugged in. I'm pretty disappointed with Linksys for that...
