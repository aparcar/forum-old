How to send DHCP option suboption in openwrt?

config dhcp 'lan'
        option interface 'lan'
        option start '100'
        option limit '150'
        option leasetime '12h'
        list dhcp_option '42,194.126.115.242' #works, no prob
        list dhcp_option 'vendor:43,16,"Europe/Tallinn"' #no works, need to send option 43 suboption 16 as ascii "Europe/Tallinn"
