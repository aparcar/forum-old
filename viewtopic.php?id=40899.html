I can login with root without password. But I am asked for password every time I try to login with the user.

Is there any bug in dropbear?

Try to add -v option (verbose) when u try to login. Should give you some more info.

logread -f   show you realtime what's happening on server side.

Make sure permission of ~/.ssh and files are owned by user.

As last option you can install sshd from repo, it's the full version and I'm not having problem at all with it.

You don't need to remove dropbear, they can live together, just make sure they listen on different ports/ips.
