Hi everyone

I had OpenWRT designated driver (trunk) running on an Zyxel NGB6616. I tried to update to a new trunk version, but unfortunately it bricked the router. Only the ethernet port was flashing and the router not functioning/responding.

I wasn't able to activate failsafe mode, so I copied a new firmware to the router by tftp, which worked. The  power LED, Internet  and ethernet LED are lighted again (WLAN LED not lighted, but the key symbol is lighted), but router isn't working. Also no DHCP. After setting PC to 192.168.1.2 and 255.255.255.0 I can ping the router successfully, but no access by telnet/shh (connection refused). I tried this after tftp'ing the original firmware and also the open wrt trunk, no change.

Does anyone have suggestions how I can get the router running again?

thanks for any help

p4p4
