small form factor router and openvpn tunneling are good for our trips, thus uses some old devices but functional properly to get a trial,

1) TP-LINK 710N (TW) version was not working, it is not be able to flash openwrt, did not yet understand why.

2) TP-LINK 702N, failed.

3) TP-LINK MR3020, 32M RAM, good to go, RAM disk is working for openvpn

4) ASUS WL-330ge, Luci dead once WIFI is enabled, serial log shows out of memory (?), https://forum.openwrt.org/viewtopic.php?id=63782

5) ASUS WL-500G delux, same response as 330ge did

6) ASUS RT-13U B1, big footprint, good to go


The memory requirement and footprint is getting hungry on 16M RAM device, to run Openwrt as a minimal WIFI router, it may be an issue but seems no addressed yet or caught attention.
