I have written a gen_mc replacement utility (gen-mc).

It is know to compile on QNX4 and Ian MacLaren has ported it to LINUX.

We were able to generate the SSL mini-certificates to secure the Linksys|Sipura-to-Linksys|Sipura VoIP communications. Probably the same mini-certs should work with the eyeBeam softphones too.

I would like to make this utility public, feel free to improve it!

The "pap2-rt31p2-adminGuide v2.pdf" file describes how to install and use those mini-certificates in great detail.
