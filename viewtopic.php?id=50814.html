I think that it should be quite simple, 1. you just need to flash openwrt firmware, 2. enable wifi and test if you can connect via wifi, 3. remove corresponding eth in /etc/config/network from lan, assign it to wan and enable pppoe. I don't have the device, so I don't know if it is eth0, eth1, eth0.1, you will have to figure out with ifconfig.

PPPOE howto: http://wiki.openwrt.org/doc/howto/internet.connection
