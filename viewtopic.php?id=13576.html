hi!

there is some information on macsat.com [0],  although  there is only the wmu-6000 from airlive listed. i'm working at the moment with the connectec MGB100 and openwrt on it, got all informations from macsat's wiki and forum. 
fast howto: installed a newer firmware (b7), then schuftis image with autoboot enabled, and then paganos autobootimage  first on a usb stick and now on the harddisk. i'm compiling kernel modules with schuftis toolchain/source and other packages via openwrt.
