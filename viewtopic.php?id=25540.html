Well, sending firmware image over tftp never worked with TL-WR1043ND. Tftp method mainly works with broadcom devices (CFE bootloader).  To unbrick you'll need serial console, read the wiki http://wiki.openwrt.org/toh/tp-link/tl-wr1043nd#serial

Your options are,

a) Access OpenWrt FailSafe mode.
b) Do a 30/30/30 reset if DD-WRT is running.
c) Access U-Boot from serial console.
d) SPI Flash Programmer. OpenWrt / Report from failed attempt with TL-WR1043ND
e) JTAG. OpenWrt / WRT160NL flashing and kernel panic
