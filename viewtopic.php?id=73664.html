Using LEDE Reboot 17.01.4 and each time I update the 'Remote host name or ip address' for an OpenVPN entry I have to reboot the router for the changes to take effect. Save & Apply doesn't actually change the IP address so I'm guessing the router is still connected to the old host. Any idea how can I connect to the new host without rebooting the router?
