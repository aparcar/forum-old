I would see 3 options:
1. use one of the existing buttons (like reset, wds or whatever you've got on that device) and change some config files.
2. connect something to your usb that has a button (numeric keypad, usb-gpio, usb-i2c with i2c-gpio-expander) and edit some config files (maybe even build new image).
3. locate some unused gpio on the board and hook up a switch and resistor, then edit some config files and build a new image. You will void your warranty.
And if you find the time, please improve the wiki page of that router :-)
