You install gphoto2 like every other package in OpenWRT, either via LuCi (you sure know how to do this by now)

or via console ->

opkg update
opkg install gphoto2

That's it!

CamRanger is in fact based on OpenWrt + some proprietary software:
/etc/openwrt_release has this:

DISTRIB_ID="OpenWrt"
DISTRIB_RELEASE="Bleeding Edge"
DISTRIB_REVISION="r38820"
DISTRIB_CODENAME="barrier_breaker"
DISTRIB_TARGET="ar71xx/generic"
DISTRIB_DESCRIPTION="OpenWrt Barrier Breaker r38820"

As far as I can tell from a quick glimpse it's not using gphoto.
Also I cannot find the GPL source code for at least the OpenWrt Buildroot on the vendor's website.

But then again I couldn't care less as DSLRdashboard pretty much does the same for free with standard OpenWrt hardware.
