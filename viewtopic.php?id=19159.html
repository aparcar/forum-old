Following RoundSparrow's Slashdot submission "Contest For a Better Open-WRT Wireless Router GUI" and follow-up comment, I'm quite interested in the Planex (PCI) MZK-W04NU.

I've been away from OpenWrt for a number of months now after moving to a desktop-based router to support my EVDO / cellular Internet connection and a few other needs (details here.)  Things may have improved substantially since I last looked at this, but I always had a difficult time finding an OpenWrt-supported device that met what I think is a basic list of requirements, and is something I blogged about 2 years ago.

I've been quite happy with the Linksys WRT54G / WRT54GL.  The RF transceiver appears to have decent power and quality, at least for the price.  It has dual-diversity antennas, and are removable by R-TNC connections.  It includes a 4-port 10/100 Ethernet switch, and is somewhat easily powered by POE.  However, now that I have a laptop with 802.11n built-in, it'd be nice to upgrade to a wireless router/AP that supports it.  The support for the Broadcom chipset using the b43 driver under kernel 2.6 is still listed as "not production ready".

Particularly, it seems that the OpenWrt devs have long favored the Atheros chipsets over Broadcom - and it seems for good reason, especially considering how Atheros seems to be much more supportive of open source than Broadcom.  However, the last time I looked, I had a difficult time finding any Atheros-based device that was supported by OpenWrt and had any features left to offer.

The Planex (PCI) MZK-W04NU looks like a rather promising preferred replacement.  It has an ample 8 MB of flash, 32 MB RAM, and a USB port.  It also has usable RP-SMA antenna connectors.  However, according to http://wiki.openwrt.org/Hardware/Planex, it is still not supported by a "stable" OpenWrt release.  While I wouldn't mind trying the "trunk" version, are there any details about the "LAN Ethernet switch not working"?

I'd also be curious about mounting options.  There are wall-mount brackets available for the Linksys WRT54G / GL available from Linksys and others.  I looked at PCI's web site and read through the user guide, and don't see anything about mounting options.

There is already an existing forum thread on this router:

I examined the case, I see no screw head style wall mounts.  You could easily adapt the upright foot that slides to wing for screw mounting if you stuck it out from the wall in T fashion.  It is not a particularly heavy device, so hopefully some easy options can come up.

The LAN switch is now working.  Primarily the ath9k wireless driver is the main focus on the AP81 router platforms... a lot of good activity going on this week!
