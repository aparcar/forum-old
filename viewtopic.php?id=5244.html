Hi Yan,

Actually I can help a little with the wgt624v3. (well maybe you'll be helping us more but ...).

The situation is this, madwifi hals are not available at present for any of the SOC chips that use AHB to connect the radios, which is all of them. However there is some work being done on getting them ready if a compatible hal does become available.
I am specifically working on the RedBoot bootloader for atheros based devices. If you can supply some information about the internals of this device specifically the flash chip (it's a SPI flash probably), the ram, and the switch type. I can then make a redboot image that runs in ram for testing.

As to what device you should use today, a wgt634u would be my suggestion.

Mike
