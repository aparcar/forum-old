OpenWrt (and a lot of other *nix based devices) don't do a dhcp request with there hostname by default.

In Luci you can set the hostname under:
Network -> Interfaces -> [interface] , and fill it out behind 'Hostname to send when requesting DHCP'

under cli:
uci set network.[interface].hostname="example"

or just edit the file: /etc/config/network and add under the interface the following:
option hostname 'example'

p.s. Replace [interface] with the interface you want (E.g. wan) and of course change example to something more meaningfull
