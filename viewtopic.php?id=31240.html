philchetcuti wrote:

So then, I assume I'm to use opkg then?

Yes.
And you might need to edit /etc/opkg.conf to include the directory where you might place those installable add-on modules.
E.g. 

src/gz USBmnt file:/mnt/packages
dest root /                     
dest ram /tmp                   
lists_dir ext /var/opkg-lists   
option overlay_root /overlay
