Hello everyone,

I am trying to briefly turn off the link on one of the Ethernet ports (the LAN port) on an MT7620A-based board. On an older (RT305X-based) board, I would simply do:

swconfig dev rt305x port 0 set disable 1                                       
swconfig dev rt305x set apply

but this no longer works with the MT7620A driver. There is no "disable" command on the ports on that switch.

Are there any other alternatives? The reason I need to turn off the link is to propagate a "network down" event to a computer behind the router and have it re-request an IP via DHCP when I turn the link back on (it's a 3G router and the computer behind it needs to know when the 3G link goes down).

Thank you!
Razvan
