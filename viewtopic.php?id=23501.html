Hi,

I'm running a WGT634U with 8.09.2. Samba3 is installed and works just fine. I want to make it appear in my Mac Finder. So I installed avahi-daemon. I remember running this setup in an earlier version just fine. My smb.service looks like this:

<?xml version="1.0" standalone='no'?><!--*-nxml-*-->
<!DOCTYPE service-group SYSTEM "avahi-service.dtd">
<service-group>
<name replace-wildcards="yes">%h</name>
<service>
    <type>_smb._tcp</type>
    <port>139</port>
</service>

<service>
<type>_device-info._tcp</type>
<port>0</port>
<txt-record>model=Xserve</txt-record>
</service>
</service-group>

After starting avahi-daemon it works well. It shows up in Finder and I can connect instantly. Wonderful.
But after a few minutes, I just can't connect any longer. After some time the icon just disappears. Connecting via "smb://IP" works fine, so SMB is ok.
When I restart avahi, the icon comes back instantly.

I also started avahi in debug- mode, but it gives me absolutely no information when this error occurs. Nor does it crash.

Please, could someone help me?
