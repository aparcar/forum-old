Post #9
hamvil
23 Oct 2008, 12:13

Only the testing version was removed from the repository. The 0.9.4 is still available and I'm currently using it. Please note the following part of the Config.in for madwifi:

config MADWIFI_UPSTREAM
            depends !LINUX_2_6_26
            depends !TARGET_atheros
            bool "Use the upstream release version 0.9.4"
endchoice

As you can see it is not supported on atheros platform and kernel 2.6.26. I think that is because the driver itself is not supported on the kernel. Maybe some of the core developers can clear this out.
