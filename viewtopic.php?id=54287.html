RangerZ
5 Feb 2015, 18:19

Network => DHCP and DNS
Go to the bottom section "Static Lease"
Click "Add"

Enter a User Friendly Name for you HDHR  (HDHR3-US)
Enter the MAC address (use small alpha)
Enter an IP address outside of your Dynamic Host Range (Default is 100-250) 192.168.1.99
Click save and apply
Reboot the SD device

Alternatively if your using windows, download WinSCP and use this to connect to your router (192.168.1.1, SCP, 22 would be default settings)

On the right go to the path \etc\config and find the DHCP file.
Double click and it will open in the WinSCP text Editor
Add the following to the bottom updating for your mac address.

config host
    option name 'HDHR3-US'
    option mac 'insert.mac.here'
    option ip '192.168.1.99'
   
Reference: http://wiki.openwrt.org/doc/uci/dhcp
The parameter lease time is not in the above, and I expect that this will leave you with a non-expiring lease that is always the same IP.  This way you can use http://192.168.1.99 to access your tuna.
