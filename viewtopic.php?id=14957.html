I too did not find support for the rb1xx in the 7.09 release of Kamikaze but it is in the latest SVN and works as advertised. I followed the directions at http://wiki.openwrt.org/OpenWrtDocs/Har … otik/RB100 and it "just worked". The only glitch I saw is that if I try to compile all of the packages some of them blow chunks. I finally gave up and just compiled the stuff that looked useful to me. I used the procedure at http://forum.openwrt.org/viewtopic.php?id=9180 to build KamiKaze, except I built it on a Fedora box and I did not dedicate a machine to it. I simply downloaded the svn into its own directory and followed the instructions to build. I must say I am impressed with the ease with which I was able to get this going.
