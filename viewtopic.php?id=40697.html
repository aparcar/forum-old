EDIT: Okay, after a little more investigation it appears that AAb1 is based on trunk 33624.

Attitude Adjustment is the "release branch" of the next Openwrt release. It has separate code base from the main trunk. AA was split from trunk two moths ago.  AA beta 1 is not "based on trunk x", but is based on AA version x.

Looks like in the latest trunk snapshot round, the buildbot failed to build the luci packages for ar71xx and for that reason they are missing until the next time the ar71xx target platform is built. The binaries are built again roughly every second day. See http://buildbot.openwrt.org/builders/ar71xx

EDIT: I just finished compiling my own build for ar71xx/WNDR3700 and there is nothing wrong with Luci, so it should be there again on the next round.
