You need to install openvpn-openssl.  There is no Luci interface in BB but there is in CC (luci-app-openvpn)

It does not appear your provider offers any config help for this device, but does for linux in general.  Not sure how easy it would be to modify the script.  https://vpnuk.net/setup-linux-openvpn.php

You also need to have WINSCP (good tool for managing config files) to load\edit the OpenVPN config itself, as there is no GUI.

In any event you may want to find a provider that can give you a preconfigured script for OpenWRT.

RangerZ
30 Sep 2015, 17:23

Yes, where it says click here to download the configuration files.  You will need to reformat the content to the OpenWRT format in the OpenVPN Config file.  That's as good as it will get. Then look at your system logs and see what it complains about.  Set your verb to 4 or 5 to see more messages.  Generally this is set at 3 for general use (ie once it's working)

This post and others in the wiki will show how to addd some lines to push the OpenVPN logs to a text file, but you will need to use WinSCP to read these.  http://wiki.openwrt.org/doc/howto/openv … ver-setup.  This post will let you add the logs as a tab in Luci, which I found helpful in trouble shooting.  https://forum.openwrt.org/viewtopic.php?id=58715
