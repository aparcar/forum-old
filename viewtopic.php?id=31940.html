I'm not familiar with the nth packet matching option in iptables, but I know OpenWRT has a lot of netfilter extensions that you can select when you build your own firmware image. So, my suggested route would be the following:

1. Follow http://wiki.openwrt.org/doc/howto/buildroot.exigence and build your own OpenWRT image
2. When running "make menuconfig", head to the Kernel Modules > Netfilter section and see if what you want is listed in there
3. If not, run "make download world" and you should find a new directory called dl/ which contains the iptables sources
4. Extract and patch these sources to include what you like
5. Build OpenWRT with "make" (or "make V=99" to get debug output)

Hope this helps,

Sam
