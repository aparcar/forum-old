legendbb wrote:

Couldn't find luci http configuration interface

How to get it installed and use, do I have to build it in kernel?

If it wasn't included in your firmware image, you should still be able to telnet into the router, then use "opkg install luci".  If you are building your own image, make sure the luci feed has been added, and select the luci packages you want in "make menuconfig".
