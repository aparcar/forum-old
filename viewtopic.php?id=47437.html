Hello,

If your goal is to extend you wireless lan over larger area, maybe you should use a wds
http://wiki.openwrt.org/doc/howto/clientmode - basic info
http://wiki.openwrt.org/doc/recipes/atheroswds - how to config router with atheros wifi module (your case)

If your goal is to configure mesh network and nothing else, maybe you should try to use quagga
http://martybugs.net/wireless/openwrt/quagga.cgi
http://www.nongnu.org/quagga/docs/quagga.pdf
http://www.nongnu.org/quagga/ - quagga structure
just choose routing protocol, download related packages with opkg (see first link). I've found that it's pretty to set basic configuration using quagga documentation (second link)
You can also try OLSR routing
And about batman - I couldn't find any guide how to configure this routing and just gave up

Actually there is many other routing protocols that work with mesh, just google them and see, if they are available for openwrt
