I want to create a server list in dnsmasq section.

root@iptivity:/tmp# cat /etc/config/dhcp

config dnsmasq
    list server '8.8.8.8'
    list server '8.8.4.4'

I want add to add the server list using the uci command.

The following command does not work:

uci add_list system.dnsmasq.server='8.8.8.8'

How can i add a list to the dnsmasq section ?

The "config dnsmasq" section is anonymous so you have to address it relatively by type:

uci add_list system.@dnsmasq[0].server='8.8.8.8'
