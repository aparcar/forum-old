Did you checkout the packages and xwrt feeds?

./scripts/feeds update -a           # Checkout all feeds configured in feeds.conf (list feeds: ./scripts/feeds list -s)
./scripts/feeds install webif       # Creates the symlinks (in package/feeds/packages/) for the webif package and the haserl dependency

Then it should be listed in menuconfig > Adminitration > webif
