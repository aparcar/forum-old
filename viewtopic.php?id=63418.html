I haven't found a "stock" router firmware that is anywhere near as nice as OpenWRT. Plus they seem to all have undocumented backdoors.

The performance issues with the Archer as I understand it is attributable to the Ath10k driver, which is still fairly new, relatively speaking. So there is much hope that future refinement will address connection drops etc.

The bigger concern is the lockdown that TP-Link is adminstering. I have been unable to flash anything to TP-Link gear that was manufactured after Nov 2015. So I wouldn't buy anything from TP-Link that was manufactured in Dec 2015 or later.

WDR4300 has been discontinued. If you can still find it somewhere, this probably is you best option. If you do go Archer C7, return anything (before opening) that doesn't meet the manufacture date I mentioned.
