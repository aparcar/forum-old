Smallest?  Does it have to be smaller than this one?

http://www.amazon.com/Gl-iNet-Router-Op … B00JKFE0FW

It has OpenWRT built in; there is a proprietary GUI on top of it, which help in quickly configuring it for travel-router duty (where conditions change from hotel to hotel).  Further, it can easily be upgraded to the latest trunk following directions here:

http://wiki.openwrt.org/toh/gl-inet/gl-inet

Also, it is very forgiving.  Screw things up, and it is easy to reflash the original firmware. (Ask if you have trouble.)

There is an active thread

https://forum.openwrt.org/viewtopic.php?id=59101

There is also a refreshed version of this hardware (AR150), but it does not provide much added functionality for most users (e.g an additional LED light, an additional programmable switch), it is only available direct from Hong Kong, and it is not backward compatible, meaning you would need wait for the OpenWRT Gurus to incorporate the patches for this new hardware before you could hope to jump to the trunk (latest version).

I like mine just fine!

Post #16
RangerZ
2 Oct 2015, 03:50

There are 2 different devices, and soon a third. 

The 6416 is the original, and can be upgraded with trunk, but again you will loose all the GL.inet GUI and it's functions.  I believe that you can or will be able to get the gl.inet version of CC soon.  I want to make sure that you are asking the right questions.  You still can add packages to add functionality like VPN.  If you want this for travel, I think you want a stable version, and not really trunk.

For travel, then I will suggest you want to keep the gl.inet firmware, as the GUI is easier to use on a small screen.  You need access to the GUI to connect to hot spots.

This is my single biggest problem with OpenWRT.  The GUI is not mobile device friendly, and is not easy to use to connect with.
