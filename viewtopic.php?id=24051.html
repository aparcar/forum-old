Take a look at Gargoyle:
www.gargoyle-router.com

It is a web interface for OpenWrt (Kamikaze).  And it already has a very good QoS completely configured.  You just have to set parameters that fit your needs.

If you don't need LUCI to tune every single option of your router, Gargoyle is a very nice alternative.  And if necessary you can always tune all stuff by ssh.
