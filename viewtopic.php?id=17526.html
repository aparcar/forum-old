Install the cfdisk, e2fsprogs and kmod-fs-ext3 packages. Then you can partition and format the USB harddisk connected to the router.

You never want to use NTFS with Linux. Yes it has gotten alot more stable, but it not close enough to perfect to be a daily use standard. Even on a desktop running Ubuntu with a 2ghz CPU, you see CPU load spike up when reading and writing NTFS. Given the ability of these devices processing units and it's ability to become corrupt, it's surprising that you never had data lose even in read mode.
