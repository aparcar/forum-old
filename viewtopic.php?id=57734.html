Post #2
mk24
2 Jun 2015, 17:38

I have a WDR3500-- which is very similar to the 4300-- and the new version works well on it.  I have not tested long term uptime though.

Use the 'sysupgrade' image, not 'factory', if you already have OpenWRT and want to change the version.

Also some standard advice is to revert to factory firmware and test with it to rule out a hardware failure.  However, going back to factory is tricky on some TP-Link models.  You can configure the factory firmware as a dumb AP if that is all you need the unit to do.
