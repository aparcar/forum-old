One thig that changes with BNG and DTAG ADSL connection is that now all traffic uses VLAN7, while old style BRAS uses no VLAN tags at all, so maybe just using VLAN7 on the wan interface might solve your problem?
       I have similar issues with a VDSL2 connection where openwrt terminated the pppoe-link, but since the switch to BNG I can not get a working ppp connection anymore. One thing I noticed is that my DTAG speedport w723v Type A in modem mode sends its MAC address as Host-Uniq pppoe-tag, but even using that by adding:
option host_uniq '346bd3454939'
to the wan section of /etc/config/network did not solve the issue for me... That said for ADSL certainly the VLAN tag needs to be used...

Best Regards
        M.
