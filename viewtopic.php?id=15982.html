bigbrett

What distro are you using to complie on?

On Ubuntu 8.04 the packages you need are:

libncurses5-dev ncurses-term zlib1g-dev gawk bison flex autoconf

You can install these from the command line with:

sudo apt-get install libncurses5-dev ncurses-term zlib1g-dev gawk bison flex autoconf

Then you should be able to complie

Post #12
jow
6 Apr 2009, 04:19

Looks like the buildroot still belongs to the root user. Change the owner to the user you created:

chown -R nomeofthenewuser:nameofthenewuser /path/to/openwrt/buildroot

Note that the above command must be executed as root. Change to the normal user *after* you executed the comand above.

~ JoW
