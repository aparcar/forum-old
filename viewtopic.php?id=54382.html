SSH into your router, so you can run Linux commands on the router.  You will need an "SSH client" for your PC, it is included with MacOS just open a Terminal and type ssh root@<router IP address>.  On Windows you will need to install a third-party software.

You will then see a nify 1980's style text "OpenWRT" logo and a root command prompt.  Issue these commands:

cd /tmp     --> this changes the directory so you are working in a RAMdisk, which can store large files, but only until the next reboot.
wget <tp-link URL>  --> this accesses the Internet and downloads the file direct to your router's RAMdisk.
ls -l  --> check the directory, you should see several small files and the one big one you just downloaded.
Step 2 -- Use the resulting file name instead of orig.bin in the dd command.  if= stands for Input File.
You can ls -l again here and confirm you have made a somewhat smaller tplink.bin file.
Step 3 is the same.  Writing the Flash may take a minute.  Then the router will reboot, so no more SSH, you will then set it up like it came new from TP-Link.
