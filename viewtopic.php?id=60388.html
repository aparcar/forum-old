You just didn't understand how opkg works. It never fetches list of available packages because in 99% of time it would waste your RAM.

Just read:
http://wiki.openwrt.org/doc/techref/opkg

"stores it on your RAM partition under /tmp/opkg-lists/snapshots. Currently this occupies about 1,3 MiB of space. OPKG needs the content of this file in order to install or upgrade packages or to print info about them."

"List of available packages is lost upon reboot, so make sure to update the list before trying to install a package"
