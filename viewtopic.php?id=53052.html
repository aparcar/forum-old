A number of people have been working toward the goal of making the entire OpenWrt site more friendly to newcomers (as well as being up to date, correct, and more usable). You can see the discussion of the individual tasks at our Github site:

Github list of issues: https://github.com/prplfoundation/openw … ive/issues

There is a parallel effort to make the Table of Hardware (also called the "Supported Devices" page) better. This is under active development. Read the forum at:

Improving the ToH: https://forum.openwrt.org/viewtopic.php?id=56521

Finally, we have some concrete proposals for the the OpenWrt.org home page and the top page of http://wiki.openwrt.org. Although these pages can only be edited by core developers, we have drafted replacement text for those pages. You can read the drafts at:

DRAFT OpenWrt Home Page: http://test.richb-hanover.com/openwrt-redesign
DRAFT Replacement for OpenWrt wiki home page: http://wiki.openwrt.org/doc/playground

Comments are welcome - each page has a link for posting your thoughts. For the three Guides (Newcomer, Experienced User, and Developer) I would like to hear thoughts for organizing the content for those three important jumping off points.
