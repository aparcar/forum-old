with all my  experience,  IPv6 does not give us speed benefit.. especially when we are using tunneling methods etc., and then again less a single digit percentage of web was IPv6 enabled last when I checked for stats, Its just so that we are ready for the near tomorrow which is bound to come, unavoidable.

Then again so many websites are in accessible depending upon your service provider. so all in all IPv6 world is not mature/ enough (not by technology but by no of routers and other ). Lets say like ipv6.xyz.com is not accessible when using abc provider, while its accessible by using efg  provider.  Most of the old proprietary  tools dont know whats IPv6 which irritates you further, but all this will slowly heal with time

now about accessing IPv6, you need to check in your area if your ISP provides IPv6 if it does not go for a tunnel broker.
start with wikipedia for IPV6, read about 6rd,ds-lite, softwire, 6to4, 4to6, bla bla bla......
Yes openwrt would benefit you on using IPv6 over OEM firmware as it just opens up all the possibilities

Source of information: I earn my bread by building firmwares for routers.
Some may not agree with what I said but its simply true for me.
