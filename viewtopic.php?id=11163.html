Have you already done your homework and read up on these topics?  No offence, but you get better results if you have specific questions and show you have already done some study and serious thinking on the problem. See here:

http://www.catb.org/~esr/faqs/smart-questions.html

Your question makes no sense. A client associates with a single Access Point. Whichever one it associates with, will the one that replies.  If it associates with the WDS "repeater" node, that AP will be responsible for backhaul to it's partner which will imply repeating the packets.  If it associates with the "master" AP unit, then there would be no need for it to repeat anything on WiFi, it will be doing that via the bridge to the wire.
