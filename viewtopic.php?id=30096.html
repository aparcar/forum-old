I am quite new to openwrt.
It is not so easy to me to find up-to-date information in configuring the router the way I like to have it. I am using Backfire (10.03.1-rc4, r24045).
I have successfully installed an ext3-usb-disk and samba.
Since I realised that there is a limitation on writing the flash many times I installed block-extroot (with block-mount and block-hotplug). Now all changes go to disk and not to flash. I was happy!

BUT...

How can I revert this configuration?
Changes in /etc/config/fstab seem to be useless because after first installation of block-extroot I configured the flash:/etc/config/fstab in the way the disk:/etc/config/fstab is in unchanged state...
I can boot in failsave mode. But how to make changes in the flash:/etc/config/fstab. How to access the old flash:/overlay?

19 May 2011, 14:55 
mkdir /mnt/jffs2
mount -t jffs2 /dev/mtdblock$(sed -ne '/rootfs_data/ { s/^mtd//; s/:.*//p }' /proc/mtd) /mnt/jffs2
vi /mnt/jffs2/etc/config/fstab

At first sed -ne '/rootfs_data/ { s/^mtd//; s/:.*//p }' /proc/mtd looked strange to me but it gives just the device number of jffs2 in flash.
So I just typed what I understood:

mkdir /mnt/jffs2
mount -t jffs2 /dev/mtdblock3 /mnt/jffs2
vi /mnt/jffs2/etc/config/fstab

This works fine. Thank you!
