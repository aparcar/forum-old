Post #5
vplessky
27 Oct 2010, 13:31
Klamber wrote:

isn't there really any way i could change the firmware on it ?
What makes it so different from the others ?

The fact that it is not supported in OpenWrt 10.03.1.

You can read FAQ what is required to get support for new (unsupported) device.
In short:
You need to identify chipset, size of flash, memory size, and some other info.
For Atheros chipset minimum size of flash memory is 4MB.
If flash is less than 4MB, your router would not be supported.

Than you need to connect via serial interface to your router, and send output to developers (attach to message on forum)
