ntpclient is a single sync client. 

In some situations where I need the ntp server also I have run both.  The reason is that openntpd tends to be a bit to picky on syncronizing and will fail on busy ISP network.  (Charter cable around here).  ntpclient can be set to be less tolarant with the -g "goodness" flag and still get a syncronization within a second.
