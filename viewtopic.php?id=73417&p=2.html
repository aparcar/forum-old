balanga wrote:

Unfortunately it is locked.

Then your choices are:

  • Persuade the network operator to let you have the unlock code

  • Get a SIM from the network operator which has locked the device

  • Find some information on the Internet which might or might not help you to unlock the device yourself

  • Pay someone for an unlocking service / unlocking code

  • Buy an unlocked device

I took the last option - both my E3372h and E3372s were unlocked from the start, so I didn't have any problems with them.
