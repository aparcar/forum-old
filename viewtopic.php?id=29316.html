Connections between evices coupled to the managed switch and devices coupled to the remaining ethernet ports of the router should be configured via OpenWRT VLAN capabilities ?

Sure it's possible, you need to either use the managed switch port as a normal switch and not do any VLAN configuration on the OpenWRT device (since it will all be in one directed VLAN based on the port), or set the port to which your OpenWRT device is connected to be a "trunk" port, so that is recognizes and uses the 802.1q tagging that the OpenWRT device puts on the port.

See http://wiki.openwrt.org/doc/networking/ … g.concepts (specifically the VLAN trunking section).
