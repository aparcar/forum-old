Howdy.  I don't think anyone has flashed this device with OpenWRT.  I flashed it a few times but had checksum issues.  Right now if you want to use OpenWRT you need to solder in a rs232 serial port and tftpboot using u-boot.  Works, I've done it.

Here is the thread on this router:  http://forum.openwrt.org/viewtopic.php?id=18279
