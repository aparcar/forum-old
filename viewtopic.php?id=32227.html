I'm not sure if all the stuff you listed is supported, but your requirements are all software-requirements.
So if it's supported by openwrt, any router (with wifi) should be able to run this. If it's not supported than no router will be able to handle it.
You have to look if all needed modules and libraries (mostly iptables/netfilter) are available.
Which hardware depends only on your budget and performance-needs (throughput, encryption,...), but first you should check if openwrt can meet your software-requirements.

I'd second eleon216's suggestion. Your requirements are nearly all software-based and any Ethernet-based OpenWrt device (nearly all of them) will suit your needs. I'd suggest looking at:

- Netgear WNDR3700
- Buffalo WZR-HP-G300NH
- TP-LINK TL-WR1043ND

These are all fully supported by OpenWrt, are relatively inexpensive, have five gigabit Ethernet ports, and a relatively fast CPU.
