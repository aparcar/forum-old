I have the Attitude Adjustment (ar71xx) release uImage and jffs2-64k rootfs flashed onto a new Cisco Meraki MR12. While the unit boots properly, the ethernet ports don't show up. I've done some research and it seems like it's an easy fix (ethernet PHY mask patch), but I'm not sure which files to modify in the 12.09 source or what values for the mask will work.

The frustrating thing is that the Kamikaze initramfs image (2.6/ar71xx/21397) recognizes the ports, but won't mount the jffs2 rootfs on boot, although mounting the rootfs from the commandline works. My thought is that the mask values from the Kamikaze AG71xx driver might be able to be ported over. Any insight would be greatly appreciated!

I've also tried using the latest snapshot which gives the same result as the 12.09 release.

If we can get this working, I'll be more than happy to write the wiki page for it.
