- with Kernel 2.4 (brcm-2.4)
  - USB 2.0 Webcam => doesn't work
  - Wifi                    => works

  - USB 2.0 Webcam => doesn't work

Linksys/Cisco wrt54gl:
- with Kernel 2.4 (brcm-2.4)
  - USB 2.0 => doesn't have one
  - Wifi       => works

- with Kernel 2.6  (brcm47xx)
  - USB 2.0 => doesn't have one
  - Wifi       => works

Dusty80 wrote:

The Best of my Tests is the Asus 500gp V1,
but the Router is no longer avialable! :-(

I know it's even older, but in the Broadcom ASUS family the WL-500g Deluxe (with an external USB flash memory) is a very good machine - 2.6 with reliable wireless and USB 2.0. You might be able to find one used on eBay for a low price nowadays.

But if you need something newer, an Atheros-based router is a better choice. Ubiquiti has a lot of good "embedded" models supported by ath5k/ath9k, some of them have USB 2.0. Two of them (RouterStation / Routerstation Pro) are preloaded with OpenWrt from the factory.
