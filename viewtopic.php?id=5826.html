Post #4
Marek
24 May 2006, 13:02

disable dllimit and test it!
One thing is sure - interactive traffic like http will slow down beacuse shaping algorithm has finite impulse response and it take him a while to allocate bandwidth for this new traffic. When you use dl limit there is always some reserve available immediately so web browsing is considerably faster.
