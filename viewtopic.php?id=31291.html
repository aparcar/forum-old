Hi,
I'm a HAM radio operator and priviledged to be able to help with the construction of the HAMNET in South Tyrol - Italy.
For those who not know, HAMNET is a WiFi based network for HAM radio. We use it as a backbone service for packet radio, APRS, and much more.

Now to my problem:
We have received for free about 40 Alvarion Subscriber Units that are no longer in use. But they don't use standart WiFi (even if they use a standart WiFi chipset).
And without a Router Unit we can't use these devices. My colleagues had the idea to remove the Alvarion hardware and use the enclosure (with 5,6GHz patch antenna) for Mikrotik hardware.
BUT the Alvarion hardware is really high quality and, in my opinion it would be a shame not to use it (it would be cheaper too).
So i thought about using OpenWRT, wich brings me to you.

I myself am not a Linux Guru and search some advice and help from you.

But first a bit technical data one the Device:
Model: SU-A-5.4-3-1D-VL
Processor: PowerPC MPC603e (Chip = Processor + PCIBridge = MPC8245)
SDRAM: 2x MT48LC2M32B2 (2x 64MBit)
Flash: JS28F640J3D (64Mbit)
Ethernet MAC: DP83816
and a CPLD
WiFi Chipset: Atheros

Now to my questions:
I have found binaries of OpenWRT for PowerPC processors, but none for the PowerPC 6xx series. Can I use another one or do i need to compile my own?
