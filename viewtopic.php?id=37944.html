ordinary openwrt image download from openwrt are not pre-installed webui and usb-modem support, need to install separately.

plug your mr3220 to your router then config the network by using dhcp then you will have internet connection.
or, if your router has wifi, config your mr3220 as wifi client (STA mode) it will have internet connection too.
