Post #53
jow
21 Nov 2008, 23:58

Hi.

Try this:

opkg update
opkg install kmod-madwifi
wifi detect > /etc/config/wireless
uci set wireless.@wifi-device[0].disabled=0
uci set wireless.@wifi-device[0].channel=5
wifi

If that works, wireless is started on channel 5, ssid is OpenWrt and no encryption is configured.

~ JoW

22 Nov 2008, 00:38

@DeusExMachina:

You'll need kmod-madwifi for basic Wifi functionality + either hostapd if you want the WGT634U to act as an WPA/WPA2 AP or wpa-supplicant if you want it to use as an WPA/WPA2 client.

Regards,
Patrick

PS:
There's a kernel module for the Broadcom encryption acceleration chip on the WGT, (kmod-ocf-ubsec-ssb) but according to http://www.danm.de/files/src/bcm5365p/ it does not work.
