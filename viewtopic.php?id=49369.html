I ran into this issue with my TP-Link TL-MR3020 on Chaos Calmer 15.05, but it turns out it was a browser cache issue I think... I cleared cache and was able to set the password via LuCI.

One oddity though... after being unable to log in or set the password after a reset, I used telnet to set the passwd. After that, LuCI still said no passwd was set. Either way, a browser cache clear fixed my issue.
