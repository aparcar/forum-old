nand read 0x9f6f0000 kernel

NAND read: device 0 offset 212992, size 2097152 ...  2097152 bytes read: OK

bootm 0x9f6f0000

if you dint find nand read then give command

nand read 0x47f4850b 0x47f4850b.
nand read 0x9f310000 kernel
