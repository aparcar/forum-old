I just updated to 15.05, and everything went smoothly except for WiFi.  My router isn't broadcasting an ssid, and on the WiFi luci page I get the message "Wireless is disabled or not associated" under my ssid.  I tried disabling it and re-enabling it, as well as a full power cycle, and neither fixed the issue.

If I hit the scan button I do see some local access points show up, so it is obviously talking to the radio.

I don't mess with openwrt much beyond getting it working, so it is entirely possible I'm missing something obvious, but I didn't find anything using Google that looked applicable.

Any suggestions?

Well, looks like running wifi detect as suggested on irc by borromini did the job.

Comparing my final configs, I think the issue was that I had an option wps_pushbutton in my original config.  I never did get that to work, but it might be causing issues in the new version, or perhaps it causes issues if a module isn't installed.

In any case, I'm back up and running, and simplifying my config was one of my goals anyway.
