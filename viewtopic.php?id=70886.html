If this is DSL, and they provide a two-line modem, the modem should handle the bonding internally in a way that is transparent to the router.  The router would still see one Internet connection the same way it does now, but twice as fast.  The only change you may need to do is to improve the house wiring so that two telephone line twisted pairs go from the phone company to the modem instead of one.
