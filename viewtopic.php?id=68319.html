This should be no problem. If I remember correctly, there were just problems with flash sizes above 16 MB in regular uboot (at least that it would not be able to access areas above the 16 MB position). The 1043ND V1 comes with a ST 25P64V6P, providing 8 MB. But I would expect any similar chip to work properly.
