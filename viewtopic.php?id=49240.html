My situation is the following: I live in a flat-sharing community and my inmate thinks WLAN is bad for him. So he's turning it off as often as he can.

So I set up my wireless options to a hidden ssid since he isn't that into this stuff. However, his Windows 7 still discovers the hidden wlan as "Other Network" - after pulling out my LAN cable, which made the WLAN disappear, all excuses for my part got redundant.

Is there any way to make the access point completly invisible, at least to Windows?

I'm using latest openwrt on my TP-Link TL-WA801ND v2.1.

Using an (maybe illegal) US channel in Germany was the solution.

This hack (http://luci.subsignal.org/~jow/reghack/README.txt) allows the usage of channels 12 and 13 - windows won't scan those channels in Germany, with or without hidden ssid - it won't find the access point.

An android system for example still requires the hidden ssid. Not sure about iOS.
