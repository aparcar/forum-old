Trying to find a new WRT router myself since days.

What about this:
http://wiki.openwrt.org/toh/d-link/dir-825
http://www.dlink.com/us/en/home-solutio … bit-router

New C1 version seems to have 16MB flash, 128MB RAM (Trunk / Barrier Breaker) - not sure about dual on C1 and if you can still get B2.

DLink DIR-835 (Dual):
http://www.dlink.com/us/en/home-solutio … and-router
https://forum.openwrt.org/viewtopic.php?id=42058


Currently I am finding myself just getting headache from all this tweaks on each router hardware / wiki noted firmware issues, serial adapter resistor workarounds, bootloader downgrades for Backfire depending on router, Usb 3G listed power problems and WLAN EU/DE channel mode.

Seems to be you are kinda already on the right track....

Got told this today by KanjiMonster on IRC:
"jtag; with the added fun that he can't directly flash a new bootloader but must boot a uboot over jtag first, since afaik there's no support in accessing the flash over jtag for ar71xx yet"

You might want to decide first what firmware (Backfire, AA, Trunk) you want to run the router on.

According to an IRC tip at least Atheros may be the active platform to go with.

Have fun and good luck!
