Make sure your WWAN interface is not a bridge.  It should only have the wifi client attached to it.
Set the WWAN as DHCP client, so it gets an address from the wifi network.
-- make sure this address is in a different subnet than the LAN.
Set the WWAN firewall zone to be WAN.
