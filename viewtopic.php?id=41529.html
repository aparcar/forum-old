Reset router to default setting.
http://wiki.openwrt.org/doc/howto/generic.failsafe
Login to router via SSH,
mount_root
mtd -r erase rootfs_data


reinstall whole package again
opkg update
opkg install luci
/etc/init.d/uhttpd enable
/etc/init.d/uhttpd start

if you have ever backup router configuration file, you can restore it directly to router by using winscp.
