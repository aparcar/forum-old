The bootloader will probably use the 192.168.2.1 IP (I think) but Kamikaze will use 192.168.1.1 by default.

Try getting into failsafe mode if you can.  If you can't get into failsafe mode, see if you can reflash the router from the bootloader.  I am not familiar with your router and haven't looked up what bootloader it uses, so I don't know if it supports TFTP or FTP etc., but if it does, give it a try.

You might need to get a serial cable hooked up to see what's going on.
