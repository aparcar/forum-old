If you want to build yourself, Supermicro has some nice 4-port boards with intel gigabit on all 4 ports.  The PDSMi-LN4+ is a little older, but I've used the PDSMI+ line for a couple years with Gentoo/XEN and OpenBSD pf (not to be confused with OpenWRT) with great results.  They have a nice line of 1u cases with power supply, including a tiny 14" deep one - that this fits nicely.

One you go main style desktop:  As for compact flash, I'd say go with USB 2.0 FLASH, USB 2.0 SDHC adapters, or SSD via SATA.  Most modern systems can boot file from USB flash or USB 2.0 SDHC if you get a decent USB to SDHC adapter.

EDIT:  Forgot that the http://www.supermicro.com/products/moth … i-LN4+.cfm has built-in Compact Flash support.

Although I haven't frequented it for over a year, the Web Hosting Talk (WHT) forum is a good place to compare notes.  been a year since I've purchased Supermicro, but eWiz used to be best prices in the USA.
