in trunk 43400 with MTK7620N chip, i need to make wireless ssid unseen in Beacon Frame, but i need to make the wifi interface(that is wlan0) unchanged. how could i achieve this?

as far as i know, there is a option item in /etc/config/wireless, that controls the ssid to be hidden or not. this item is:
config wifi-device 'radio0'
    option type 'mac80211'
    option hwmode '11g'
    option path '10180000.wmac'
    option channel 'auto'
    option txpower '17'
    option country 'CN'
    option htmode 'HT40'
    option noscan '1'

config wifi-iface
    option device 'radio0'
    option network 'lan'
    option mode 'ap'
    option ssid ''
    option encryption 'none'
    option key '12345678'
    option maxassoc '32'
    option wps_pushbutton '1'
    option hidden '1'

the last option item:  hidden, is responsible for this

And, hostapd just use this config file to control the wifi behavior.

but if i want to make the ssid hidden, i need to restart the AP; that is not what i want

i want to make the ssid hidden or not,while keeping the AP still there.  how can i do this?
