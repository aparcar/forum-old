You should include:
*your router make, model and revision.
*firmware version you have loaded, if trunk exactly which one
*packages relevant to OpenVPN
*config files including Network, firewall and openvpn
*log files (it's easiest to direct your logs to a file)
*remove any personally identifiable info from the logs and configs like IP, MAC email, keys etc
*include the links to exactly which wikis you are using.  there are a number of them.  I personally like the one that says streamlined" in the title.
The config files and logs are essential.  The rest is good practice.

Here is WORKING configuration that I just made. I was not sleeping 3 days for this... It has lan from both server and client.
Server conf  192.168.6.0
--------
config openvpn 'server'
    option server '10.0.100.0 255.255.255.0'
    option keepalive '10 60'
    option port '1196'
    option verb '3'
    option dev 'tun'
    option mssfix '1420'
    option ca '/etc/openvpn/ca.crt'
    option key '/etc/openvpn/server.key'
    option dh '/etc/openvpn/dh1024.pem'
    option cert '/etc/openvpn/server.crt'
    option route '192.168.5.0 255.255.255.0'
    option enabled '1'
    option client_config_dir '/etc/openvpn/ccd'
    option log '/var/log/openvpn.log'
    option proto 'udp'

                # and in ccd/client_name

iroute 192.168.5.0 255.255.255.0
--------





Client Conf 192.168.5.0
-------
config openvpn 'client_tun'
        option nobind '1'
    option port 1194
        option float '1'
        option client '1'
    option tls-client '1'
    option proto 'udp'
    option resolv-retry 'infinite'
        option reneg_sec '0'
        option dev 'tun'
        option verb '3'
        option persist_tun '1'
        option persist_key '1'
        option remote_cert_tls 'server'
        option remote 'ip.of.vpn'
        option ca '/etc/openvpn/ca.crt'
        option cert '/etc/openvpn/client1.crt'
        option key '/etc/openvpn/client1.key'
        option enabled '1'
    option log '/var/log/openvpn.log'
    option route '192.168.6.0 255.255.255.0'
