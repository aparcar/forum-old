And also another thing:

Does the heatsink on your units also get really hot when running OpenWRT? Like to hot to hold your finger on it for more than 5 seconds?

I'm running CC 15.05.1 SMP. Both cores are enabled.

I managed from another site (cannot put link networkingathome.blogspot.ro/2015/09/setting-up-3g4g-modem-with-openwrt-on.html), but steps are next:
- connect with wireless to a wireless network;
- install (from router menu System - Software) packages: kmod-usb-serial-option, usb-modeswitch and luci-proto-3g.
- reboot router;
- Network/Interfaces and click Add new interface - name it UMTS or else and choose from menu "UMTS/GPRS/EV-DO"
- subimt;
- choose at "modem device" option "/dev/ttyUSB0 or 1 or 2" (test which works);
- write apn name - save and apply;
- go to firewall settings and set to wan.

If connection not work you should unplug USB modem and plug it again.

For more and better info you should search original blog.

Good luck!
