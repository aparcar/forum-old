1.Get a USB usart and connect serially to ERX
Yes connecting only GND TX and RX https://www.amazon.com/UART-Module-Seri … B006JKNWLE
2 Connect to console on ERX with PuTTY.
Yes
3.Setup TFTP server to get the initial image to boot and power on ERX and choose option 1.
Yes you have to build the openwrt-ramips-mt7621-ubnt-erx-initramfs-kernel.bin image and load it to ram via the tftp server
4.Then copy over bin file to run sysupgrade
Yes but it's a tar file openwrt-ramips-mt7621-ubnt-erx-squashfs-sysupgrade.tar
