I would create one SSID on the 2.4GHz radio (let's call it WIFI), and two on the 5GHz radio (let's call them WIFI and WIFI_5G), all of them with the same password. This way, you can configure devices to choose by themselves (use the WIFI network), or force them to the 5GHz network (use the WIFI_5G network); and if you happen to own a device that can do 2.4GHz + 5GHz at the same time, it will also work (use the WIFI network).
