Mine has:
1.Via Vectro VT6212L 4port USB host controller
2.Storlink SL3516 ARM CPU
3.Marvel 88e1111-bab1 10/100/1000 Ultra Gigabit Ethernet Transceiver
4.Vitesse vsc7395xyv Enhanced 5 + 1-Port Integrated Gigabit Ethernet Switch with Transceivers
5.Jmicron JM20339 Hi-Speed USB to SATA Bridge(connects to external ESATA and internal?)
6.Ralink RT2561 based Wifi mPCI card from Billionton Sytems(yes the card can be exchanged, standard mPCI module with small UFL connectors for anntennas ).
7.Spanshion (AMD) 16mb Flash Mem
8.Nanaya ddrII 128mb RAM in two chips.

Some of the chips drivers are allready in the kernel sources, so compiling kernel should be fairly easy,but I dont have the toolchain,and cross-compiller setup for now.

The sad thing is the manufactirer of this box has not provided any updates for a year now.
