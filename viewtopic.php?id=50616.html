A simple approach would be:

# opkg list-installed | cut -f 1 -d ' ' > /etc/config/installed_packages.txt

To save all the installed packages into /etc/config/packages_list.txt, /etc/config/* is by default retained through sysupgrades.

# cat /etc/config/installed_packages.txt | opkg install

After sysupgrade, to install all packages.

A more sophisticated approach can be found here in the forums. I did, however, never use that one myself.

Keep in mind that even while packages can declare that config files are retained through updates, not all packages do so, or may not even know about files you created yourself in other places. Check /lib/upgrade/keep.d/ for descriptions of files and directories that are retained, and amend if necessary.
