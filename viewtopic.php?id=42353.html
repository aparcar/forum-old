What I did was added this firewall rule to stop the flood. You need ip6tables to do this, and it may require additional rules to enable regular ipv6 traffic, if you install it, so be careful.

config rule
        option src              lan
        option proto            udp
        option dest_port        1900
        option target           DROP
        option family           ipv6

Nicer solution would be some kind of miniupnpd config option, but I don't know any.
