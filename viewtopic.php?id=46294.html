From my personal experience with flashing the 692GR router I can not recommend you installing OpenWRT on this router yet.

1. Compiling the release version 12.09, attitude_adjustment, gave me a working openwrt-ramips-rt3883-tew-692gr-squashfs-factory.bin. With this file I was able to flash the router. But after installation it turned out that WIFI is no longer working.

2. Compiling the trunk 38812 with the hope of fixed WIFI support. Unfortunately no sysupgrade file for the 692GR router was created by make.

CU,
Stephan
