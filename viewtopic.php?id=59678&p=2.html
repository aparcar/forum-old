I use a GL-iNet router coupled with a rather bulky Aukey 20,000 mAh USB battery device and don't find it to be too much of an inconvenience. Software wise it is setup with an OpenVPN connection and with a little finessing using my phones hotspot I can connect it to any wifi I want. It does have the potential for a secondary wifi adapter via the USB port assuming you can find the right openwrt drivers for the wifi dongle to use. Considering it uses openwrt the availability of things it could be made to do (samba or minidlna) are certainly expanded. However personally I wouldn't want to use my "travel router" as a media or file server of any sort.

I plan to use a script I found for AAP to see how well that works allowing the router to default to my home or hotspot networks when other available networks aren't around. Ideally it would be nice to get the AP setup regardless of client status but I haven't found information on this quite yet.
The OpenVPN requires a manual setup through ssh, but once I ironed out the problems it was quite easy to have it brought up and it can be stopped/started/reset from the openwrt menu as necessary.

Well guys we are working on the finishing touches. Lots of C, lua, bash, and html5/javascript added with touch screen support. No Luci. Saves wifi hotspot passwords just like ur cell. Disables station mode if no known station is detected within timeout, so you can access the router using wifi. Integrated VPN interface where one screen configures all supported VPN protocols i.e  IPSecure, OpenVPN, IKEv2, L2TP, PPTP. Supports both incoming and outgoing VPN. The travel version of the router does the same stuff minus incoming VPN. We use our own C code to handle high speed multipart/form-data for efficient file upload handling with http upload speeds reaching same  as FTP/SAMBA.  No temporary files generated by upload code. No large memory foot print like in php scripts. No file upload size limits over http. HTML uploads can be a pain if com is lost, so we added auto retry and continue upload feature that works behind the scenes to ensure file uploads are successful. The file sharing feature can convert your travel router into a http server over the net for the files you chose, even when your travel router does not have a public IP address.

Essentially VPN, full control of USB storage using any modern browser,  EZ file sharing with efficient wifi hotspot handling. Most of the stuff that you will need when you travel. I will provide the links soon.
