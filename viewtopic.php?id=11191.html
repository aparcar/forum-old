Hi, i'd like to run an IPv6 to IPv4 conversion deamon.


This is to be able to access machines of the local network from a range of public IPv6 addresses.


This will suppress the need to configure NAT rules each time we need to access a different service on a machine from the local network from outside.


This is a very nice feature of IPv6 to be able to get a /48 subnetwork with as many address as we want.


Today, it's not easy to implement Ipv6 on all machines, specially with hardware devices (like IP phones, camera,... ) , most of them are IPv4 and will stay IPv4 for a long time because their firmware is not opensource.


That's why i'd like to know if there is a package with such a conversion deamon.
