Hello am i missing something as trying to mount a 5TB Seagate Expansion drive NTFS on a 1900ACS V1 using Lede Reboot SNAPSHOT r4576-9fe9175 but it doesn't show up in Mount Points so do i need to enable something to get it to work ? drive works fine under DD-WRT

Same issue, it does not show up in Mount Points. The ntfs packages are installed like David said but for some reason it does not show up.


I mount my ntfs drive by putting the mount command in /etc/rc.local (System>Startup>Local Startup in Luci), here's mine for example -

ntfs-3g /dev/sda1 /mnt/storage -o rw,async

that way it mounts at boot, or you can just run your mount command manually, ntfs will never show up in mount points as far as I'm aware.

edit - just wanted to clarify that while ntfs won't show up in Mount Points, it will show up in Mounted File Systems once mounted (in Luci>Mount Points).
