Hello! I've been using RC4 for sometime now and I must say I extremely happy with it! Here is my problem: I just bought a brand new Palm T|X. This beauty has a builtin wlan card that allows it to connect to a wireless router. That is all fine, and it works on many access points in my work, but I cannot make it work with router. It is a Lynksys WRT54G v3.1 with OpenWRT RC4 installed. I've tried many things already:

logging through IP tables:

Although I can see DHCP requests for all the other computers in my network (3 wireless devices besides the PDA), I get NO packet in from the PALM.

using kismet to passively monitor the wireless network, when I click "connect" at the palm, I can see this flushing:

10:06:22 - 00:0B:6C:4E:05:8A MANAGEMENT (Probe Request a2)

This is the correct mac address from the palm. So, I can see the poor guy is sending "probes" to the network, but the router never answers/detects that. It is like it doesn't sense the packets in the network.

I've tried changing wireless channels and other little stuff, but I'm running out of options here to understand why OpenWRT/my router cannot talk to the Palm.

Is there some magick I can try?

Thanks anyway and keep up the good work!
