RangerZ wrote:

Hi, Yes, More info please!  How do I set this up and what are the issues.

There a lot of routing problems when you are using the same IP address in both places. When your are inside the lan is the same.

In OpenVPN you can push a route. This go to the routing table of the client host. If this route is in the same subnet you will have problems always.
There is a lot of information about networking but always you will learn when the problem is there!
