Don't bother with the reset button. It doesn't work on WDR4900. Well not on my v1.3 green light version at-least.

Try to get hold of this serial software http://www.protonbasic.co.uk/content.php/935-Proton-IDE

It allows you to type "tpl" ahead of router booting process (press and hold enter button after typing tpl). I managed to unbrick a router blindly without even soldering the RX pin (no serial display)

Make sure to run it as admin mode otherwise it won't be able to recognize the COM port.

Optional: For TFTP I always use this one: http://www.solarwinds.com/products/free â€¦ erver.aspx

And don't forget to turn off windows or any other 3rd party firewall software.
