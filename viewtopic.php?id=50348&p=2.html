Post #27
nlh
8 May 2014, 20:37

Do not "mtd" flash the "bin" file to firmware.  In fact, it's not clear why you are using "mtd" at all.  Why not use "sysupgrade -n openwrt-brcm47xx-squashfs.trx" (or change -n to -c if you want to keep your existing configuration)?

The ".bin" files are made by prepending a device specific header to the ".trx" file .  The bin files are needed for flashing via the stock web gui, or for flashing via tftp.  This contains some good info.
