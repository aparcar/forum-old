I before posted in general discussion, should be here maybe:

Latest build for two inexpensive TP-Link routers, the mr 3420 and the wr 703n.

These builds are tested and run 3G i.e. mf 688
or full speed unlimited 4G
with enabled modems such as both the Sierra 320u and the ZTE mf 821.

Latest kernels, some QMI and some rewrites with lua of scripts. Direct IP, Lucl and more built in.

The addition is that for Windows (7) users at least so far, a GUI called ModemGUI has been written that allows interrogation and commands to be made of/to modem while in router, a rare and useful commodity!

This includes signal measuring and monitoring, cell IDs, channel fixing, AT commands and more. very handy when hunting signal.
