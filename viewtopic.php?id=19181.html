I updated the wiki yesterday. Hopefully, I've cleared a few things up. You should consider switching to the latest redboot image (see wiki for details).In addition to the renaming of the file, I'm now tracking the source in a personal repository and have a version number associated with the image (see ver cmd).

The jungo-image essentially issues a load command. However, it uses http instead of the default tftp and lauches another thread that services those requests (i.e. http server).
