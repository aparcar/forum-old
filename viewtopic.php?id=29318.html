Post #6
Borromini
29 Mar 2011, 11:10

You don't seem to get what he's talking about. He's talking about creating a new filesystem (which you can do through a DMG) so that you can compile OpenWrt on that case-sensitive filesystem.

OpenWrt, by default, sets up a cross-compilation environment. It will use the host's tools to build the toolchain for your router's architecture, then build the firmware. The only step you need to do, being on OS X, is installing the Mac-specific tools - be there Xcode and Macports.
