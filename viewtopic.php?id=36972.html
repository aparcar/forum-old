In Backfire, usb-storage is kmod-usb-storage_2.6.32.27-1_ar71xx.ipk
OF COZ it is for kernel 2.6.x

In snapshot, it is kmod-usb-storage_3.3.6-1_ar71xx.ipk
then OF COZ it is for kernel 3.3.6

Then your firmware using kernel 3.3.5
OF COZ it doesn't match.

So, UPDATE your firmware in your router if you wanna install snapshot packages.
Or downgrade the firmware if you gonna install packages from backfire
