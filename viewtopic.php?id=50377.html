Not sure, but you can try SSH to backup and download backup:
https://forum.openwrt.org/viewtopic.php … 30#p179830
https://forum.openwrt.org/viewtopic.php?id=26348

cat /proc/mtd
mtd0: 00040000 00020000 "u-boot"
mtd1: 00020000 00020000 "u-boot-env"
mtd2: 00100000 00020000 "kernel"
mtd3: 01e60000 00020000 "rootfs"
mtd4: 00020000 00020000 "user_property"
mtd5: 00020000 00020000 "art"
mtd6: 01f60000 00020000 "firmware"

cat /dev/mtd2 > /tmp/backup_kernel.bin
cat /dev/mtd3 > /tmp/backup_rootfs.bin

Then download with WinSCP using SCP protocol.
