Hello again.
A user on Gargoyle forums was kind enough to point me to dl.eko.one.pl where i found a build of Gargoyle for the WR710N v2.1 that can be installed directly from factory.
So, i think this can be marked as solved.
BTW: if someone has this router model and needs a webrevert to factory there is a file built by a user on the DD-WRT forum. It works well, since it was the one i used to do just that.
Thank you all for your interest.
