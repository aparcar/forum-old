My understanding is that there are very few modem/router combinations and most are unsupported.

To confirm this yourself, I would go to a site like Amazon or Newegg and enter in the search terms Wireless router and ADSL.  Then check the router database in openwrt/dd-wrt for that particular model.

I ended up getting a Netis DL-4201 ADSL modem + wired router (single port).  It was cheap ~ $20USD and seems to handled a noisy line well.  The documentation was poor but I found a review that was able to get it to work with my ISP in bridge mode and via pppOe.  I set up the modems ethernet port to subnet my SOHO wireless router.

I was not interested in spending much as my ISP still uses pppoe via atm and has no plans to upgrade to ADSL2+ or VDSL.

MY SOHO wireless router presently is an old LInksys wrt54g version 8 running opensource firmware.
