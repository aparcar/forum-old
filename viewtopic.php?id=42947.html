I believe the attitude adjustment rc1  images have LuCI built in.    However, you'll have to download and install openvpn (probably best to install luci-app-vpn  which will pull openvpn in with it as a dependency)

All builds require that you telnet it to set the password first anyway.

Make sure you read the wiki page about how to install this... http://wiki.openwrt.org/toh/tp-link/tl-wdr3600  (really you have to link the linked wdr4300)

Also note that Gargoyle is OpenWRT underneath  (though w/ a different gui and some other mods)

If you are averse to complexity... i think giving Gargoyle another try might be the best way to go.  (in your thread on the airvpn forum you weren't at all clear about WHAT didn't work.   Just saying that "it wouldn't accept the files" is not going to allow people to help you.)
