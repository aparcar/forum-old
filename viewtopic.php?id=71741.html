From the "cat /sys/kernel/debug/usb/devices" it can be discerned that no driver is loaded for the device. The ZTE MF833 is using the Qualcomm MDM9225 modem chipset. Maybe you can take a look at https://wiki.openwrt.org/doc/recipes/ltedongle which discusses using the QMI protocol to talk to the modem chipset.

Check the packages required for QMI on the page I linked above and see if the device gets recognized when the new kernel modules are available.
