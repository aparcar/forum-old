If you compile yourself, you will surely save some space because the compiled packages are better compressed inside image. I remember some success story with luci, openvpn + polarssl (but some people say polarssl is untrustworthy) inside v8 thread and v8 has 4MB flash too, so it should be possible. With openssl, but without luci it is surely possible, luci eats a lot of flash, you may want to try having some fun on the command line?

How to compile yourself:
1. install ubuntu on your pc (or inside virtualbox)
2. read http://wiki.openwrt.org/doc/howto/build
