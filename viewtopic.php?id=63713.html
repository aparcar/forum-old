If its drm free and delivered with multicast then use udpxy

Also gretap may help.
Its faster than openvpn and easy to configure. No user mode processes involved - only kernel.
But it has limitations. Requires external ip addresses on endpoints (may be only one if NAT has gre helper - havent tested). If ext addresses are problematic than grev6tap can help if both ends have ipv6.
This kind of tunnel does not offer any encryption or protection against traffic modification.
