Hi Maf.

Seems like you did an ipkg upgrade with an old firmware. It results with a very beautiful Kernel Panic Ya I know, it's not only the kernel who panics but you too (I was into the same position about 2 weeks ago).

If you have the chance to obtain an official Siemens C55 datacable, cut it off. There is like 7 wires inside it.

Go to the link below to find the pinouts :

http://pinouts.ru/data/siemens_c55_pinout.shtml


Here are the interesting ones :

Vcc = Red
Tx = Black
Rx = Orange
Gnd = Brown

Hope this will help you.
