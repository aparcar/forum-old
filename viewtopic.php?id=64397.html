hnyman

Openwrt has rather strong kernel version checksums to ensure that the kmods are compatible with the kernel. In practice you need to use kmods from the same build run that produced the firmware's kernel itself, so you can't easily use kmods from the release download repo with a private build.

You have basically two options:
* use the the released 15.05.1 firmware and install packages from the 15.05.1 download repo
* build your own firmware from the 15.05 branch and build/include the needed kmods & other packages in the build.
