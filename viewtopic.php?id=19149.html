I have an Netgear WGT634U (http://wiki.openwrt.org/OpenWrtDocs/Har â€¦ ar/WGT634U). The router has 1 WAN, 4 LAN ports. Since I have 4 static IP adresses assigned from my WAN (university network) I don't want to use masquerading but put all ports including WAN and wireless in one VLAN. Reconfigure the switch

root@OpenWrt:~# uci del network.wan
root@OpenWrt:~# uci del network.eth0.vlan1
root@OpenWrt:~# uci set network.eth0.vlan0='0 1 2 3 4 5*'
root@OpenWrt:~# uci commit network
root@OpenWrt:~# reboot

For disable masquerading I'm not sure. Maybe just disable the firewall: /etc/init.d/firewall stop; /etc/init.d/firewall disable
