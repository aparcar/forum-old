Hi,
I have successfully built openWRT firmware for my TP-Link Router and I would like to point out a few issues with the buildsystem.
Firstly
usb-modeswitch-data is broken in the latest backfire branch.
When I include it as a static package it builds fail with error 2
Secondly
For 3g wan the dependency is not satisfied.
The wiki mentioned 1 usb library when it fact 2 usb library is required.
They are libusb - 0.1.12-2 and libusb-1.0 - 1.0.8-1
