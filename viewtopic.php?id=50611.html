Have you tried telnet? Until a root password is set it should accept telnet connections, you can then type "passwd" to set a root password, which will let you log in via SSH. Did you install a trunk image or a stable one? If it's a trunk image, it doesn't install a web interface. See LuCI essentials for detailed installation instructions.
