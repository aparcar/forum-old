For the boot into failsafe mode I do the following:

- Unplug the router's power
- Connect the router's LAN1 port directly to your PC
- Configure your PC with a static IP address e.g. 192.168.1.10 (gateway and DNS is not required).
- Plug the power on and wait for the DMZ LED to light up.
- While the DMZ LED is on immediately press any button (Reset and Secure Easy Setup will work) a few times .
- If done right the DMZ LED will quickly flash 3 times every second. (in normal operation is the DMZ LED off)

Did you get router in Failsafe mode
Power off the router
Unplug the WAN port  if you do not plug out wan port, you cannot enter failsafe mode

Stage 1: Computer IP settings
Set your computer's IP to 192.168.1.2, subnet 255.255.255.0. The router will be reached at 192.168.1.1 when failsafe mode is running.

Stage 2 option 1: Trigger failsafe mode when the SYS LED begins to blink
Router OFF The switch is behind a hole iand require a paper clip or similar tool to, press it with router OFF, now power ON keep pressed until the SYS LED lights with the rapid-flash pattern.
And then got the OPENWRT logo
