yes Midnight Commander is here and working in Openwrt RC5

ipk package mc and ncurses used /opt dir for install , you must write command ipkg -d root ............., no ipkg -d opt !!!

ipkg -d root install http://ipkg.nslu2-linux.org/feeds/unslu … mipsel.ipk
ipkg -d root install http://ipkg.nslu2-linux.org/feeds/unslu … mipsel.ipk

and set environment in /etc/profile

export TERMINFO="/opt/lib/terminfo"
export TERM="linux"
