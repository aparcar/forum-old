build your own image (http://wiki.openwrt.org/doc/howto/build) and include everything you need into the image. the squashfs of the image has a better compressionrate than the jffs2-filesytem of the /overlay-partition. the package is to big to install it into the overlay, but it might fit into the image. and only select packages you really need (e.g. a webinterface isn't mandatory to run openwrt)
