Post #5
eduperez
5 Jun 2015, 11:35

The file "/usr/bin/irssi" is not a shell script but an ELF executable, you cannot do "sh /usr/bin/irssi". Just "irssi" should work, unless "/usr/bin" is not in your path, but then you would have lots of issues; try to execute "/usr/bin/irssi".
