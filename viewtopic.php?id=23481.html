Have you noticed the OpenWrt logo has the motto: WIRELESS FREEDOM ? Perhaps if you have no use for wireless; you haven't fully realized the potential of your router.

I don't use any wireless but if I really wanted to remove it, I guess I'd build it myself. But since I use the trunks; I don't complain about what's in them; I'm actually thankful this $40 router works much better with OpenWrt than the factory firmware!

If you wanting to remove it for memory space, you may consider learning to build or configure without Luci. I found through testing many flashes with various packages installed that I really don't need a lot of flash memory, the more features and dynamics that I add, the more the connection speed will slow. Maybe that's why Tomato's motto is light and mean. So the trunk with Luci and a few small packages is fine for my small router.
