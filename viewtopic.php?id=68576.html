dl12345
17 Nov 2016, 21:39

iw phy <phyname> set name <new name>

Not sure that this will help you though. The phy index is auto-selected by the kernel code and just allocates the next available index. There's no function parameter to specify the name.

Relying on the phy name can be a problem. With a USB wifi interface, loading and unloading the module will cause the phy index to keep incrementing.
