Last night I decided to install OpenWRT 12.09 (openwrt-wrt54gs-squashfs.bin) from the original Linksys firmware. Flashing the router and the initial setup was problem free. However, I have noticed a strange behavior when I max out my connection (ie downloading Usenet binaries). Under heavy load, the router will suddenly drop the connection and not respond to web or SSH connections. Only a reboot will return the router to functional status.

I noticed that there was a ticket reported for wireless stability for brcm47xx, but I am running a wired connection. Do I have a configuration problem possibly, or should I downgrade to a previous release of OpenWRT?
