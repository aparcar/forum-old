I think the multicast is enebled by default,  although I compiled my own firmware I didn't modify that option in the kernel, being it enabled by defualt, if you have the complete toolchain of openwrt, you can type

~/openwrt$ make kernel_menuconfig

and then go to Networking support>>networking option >> IP: multicast routing, if that last option is marked, it has an asterick, then your kernel is configured to support multicast.
