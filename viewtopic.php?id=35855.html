i find a way by using subnet mask "::ffff:ffff:ffff:ffff"

it seems ok, maybe there is better method

config rule
    option src        wan
    option src_ip        !fc00::/7   #ULA should not appear outside
    option dest        lan
    option dest_ip    ::aaaa:bbbb:cccc:dddd/::ffff:ffff:ffff:ffff
    option proto    all
    option family    ipv6
    option target    ACCEPT
