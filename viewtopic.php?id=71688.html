Note that you are using deprecated BB14.07, so it does not have all the later fixes.

I suspect that you will need this fix:
https://dev.openwrt.org/ticket/15209
https://dev.openwrt.org/changeset/42250
https://dev.openwrt.org/changeset/42255/

It has been backported to BB14.07 branch, but only after the release.

So, you will need to re-compile dropbear if you want to continue using that BB14.07
