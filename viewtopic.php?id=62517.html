Googling for "uclient-fetch" ftp yielded the following result: https://dev.openwrt.org/ticket/21757

Check if your busybox still supports wget and if it does I'd try:

rm /bin/wget
ln -s /bin/busybox /bin/wget

If not, you can still install full wget if the router's storage allows AFAIK.
