Not sure how to do the config files for this. I think it makes most sense to have your changes being done in /etc/init.d using a script that calls the appropriate wlanconfig commands. For dhcp you need to have a dhcp server running on one of the nodes, but this does not really make sense in a mesh network. Try using static IP addresses first and set them using ifconfig after you configured the wireless card.

Here's an example config from a NanoStation2:

root@104-61-234-42:~# cat /etc/config/wireless 

config 'wifi-device' 'wifi0'
        option 'type' 'atheros'
    option 'diversity' '0'
    option 'channel' '1'
    option 'antenna' '1'

config 'wifi-iface'
    option 'network' 'wifi0'
    option 'device' 'wifi0'
    option 'bssid' '02:CA:FF:EE:BA:BE'
    option 'rts' '2346'
    option 'txpower' '10'
    option 'ssid' 'leipzig.freifunk.net'
    option 'mode' 'adhoc'
    option 'sw_merge' '1'

~ JoW
