Firstly, I don't think you need a static route for 10.0.0.0/24 because your router is already connecting to it.

Secondly, if you are using 6in4, I think you are having static IPs, so I think point a prefix to your router and assign IP by your router is a better idea.

If you have 6in4 you usually should be able to get a /48 from your provider, if you add that to the first router, the second router will automatically get a part of that prefix just using the default configuration if you are in routed mode (i.e. wan of 2nd connected to lan of 1st).
