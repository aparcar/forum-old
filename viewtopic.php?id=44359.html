TBH I have no idea what your problem actually is. I have zero issues using utime() in most recent trunk.

jow@jell $ cat test.c 
#include <stdio.h>
#include <utime.h>

int main(int argc, char **argv)
{
    const struct utimbuf b = {
        .actime = 1000,
        .modtime = 2000,
    };

    return utime("/tmp", &b);
}
