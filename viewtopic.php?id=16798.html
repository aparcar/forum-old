I've got a Belkin F5D8230-4 v1000, which has the broadcom 4704 processor and airgo card. I've got openwrt-brcm-2.4-squashfs.trx on it and ssh is running. I have not tested the wireless ... and am not sure I will, as I want to use this router as a tiny server on my work LAN.

There is an amazing lack of information about these identical pre-N routers, so I thought I'd share some of what I've learned...

How I tftp'd:
http://forum.openwrt.org/viewtopic.php?id=16613
I literally did exactly like as quoted (as not working) at the top. I scammed my gf's XP laptop and set up the wired NIC, did the 10msec ping and had the other window with tfpt. It took me probably 20 tries and then it took openWrt.


Useful links:
