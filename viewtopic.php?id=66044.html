Ummm ok. Since nobody was apparently interested, I ended up doing it myself.

Build available at http://mionica.ro/openwrt/lantiq/xrx200/, should anyone be interested. Pretty much what you'd expect; LuCI is built-in, as is openvpn-client.

Note: if using IPoE, set your wan/wan6 to the ptm0.X interface, where X is the VLAN number you're assigned. No need to mess with the switch settings, as I found out (the hard way); there's also no need to declare a ptm0.X interface anywhere, it just works - apparently.
