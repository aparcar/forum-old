I think it's long since I'm having this issue, maybe back from kamikaze days.
My wl500gp is based on r31151 brcm47xx and is only able to reach a mere ~10Mbits on LAN ports.

Here are my tests:

~31MiB tgz from router to pc directly connected to eth0.0 @100Mbps:

root@router:/mnt/sdb1# cat extroot-image.tgz | pv -a | nc 192.168.9.1 10000
[1.41MB/s]

same file sent to /dev/null, just to show SD's reading speed:

root@router:/mnt/sdb1# cat extroot-image.tgz | pv -a > /dev/null 
[6.35MB/s]

I'd have expected 1.41MB/s if it only was a 10Mbits ethernet connection.
So I really don't know where the problem stems, considering that qos, wifi, wan, syn protection are off.
