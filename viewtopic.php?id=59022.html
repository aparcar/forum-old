sindhus wrote:

I am running Attitude Adjustment, transmission-daemon and transmission-web. I do not have UPnP. I want to forward transmission's port 51413.

If transmission is running on the same router you are trying to configure, you don't want to forward but you want to open the port:

config rule
        option target 'ACCEPT'
        option src 'wan'
        option proto 'tcpudp'
        option dest_port '51413'
        option name 'Allow-transmission'

My router page is at 192.168.1.1, I have limited the DHCP server to give out addresses from 192.168.1.0 to 192.168.1.24 only.

That is not a very good idea. 192.168.1.0 is a reserved address (referring to the whole subnet), and 192.168.1.1 is obviously your router itself. You should adjust that range.
