Post #2
rmacielalves
20 Jan 2016, 20:19

Edit your /etc/config/wireless and add:

config wifi-device 'radio0'
...
    option noscan '1'

Or from a ssh shell/terminal window:

root@OpenWrt:~# uci set wireless.radio0.noscan=1
root@OpenWrt:~# uci commit

It worked for my WR842ND.
