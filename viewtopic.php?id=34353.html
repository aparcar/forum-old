Does anyone know an easy way to switch DNS from ISP to Google DNS? Looks like there is a DNS forwarding entry in the interfaces tab for my br-lan but don't want to change it for fear of messing up. Thanks.

Post #2
fyi
28 Jan 2012, 05:21 
uci set network.lan.dns="8.8.8.8 8.8.4.4"
ifdown lan ; /etc/init.d/dnsmasq restart ; sleep 1 ; ifup lan
uci revert network.lan.dns

Untested.
