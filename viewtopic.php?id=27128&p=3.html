Bus width must be 16bit. So  you must use a chip that has 32M*16bit (1byte=8bits,    16bit=2byte,     2byte*32M=64MByte)

Some time ago I did a post where I comment a pretty cheap memory module on Ebay  that you can use to mod 8 routers to 64MB:
https://forum.openwrt.org/viewtopic.php?id=42081
