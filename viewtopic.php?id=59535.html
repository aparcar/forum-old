Hi All

I have a TP Link Archer C7 AC1750 v1.1 and whilst flashing OpenWrt, the power cable was accidentally pulled out...

As a result, my router now seems to be stuck in an endless boot cycle. The power and "snowflake" icons are always lit. All four LAN icons flash on and off (with nothing connected). The WAN led and update LEDS also flash slowly.

I have tried to De-brick using TFTPd32 and the following firmware

-OpenWRT
-TPlink (with boot section removed - http://www.friedzombie.com/tplink-stripped-firmware/)
-TPlink with boot section removed by myself

TFTP server set to 192.168.0.66. There is LAN activity when the reset button is held down after power on. I can connect to any of the LAN ports and the firmware file is pulled from the TFTP server. Wireshark shows that all blocks of the firmware are transferred during the TFTP session, however, this does not fix the router. It is still stuck with all lights flashing.

I have also tried to access the router using RS232. However, the router does not respond to any commands. On power on it simply returns [00]. I used an FTDI chip USB to serial adaptor plus termite.

Any ideas on how to proceed?

Any help would be great. Thanks!
