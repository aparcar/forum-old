Just in case... I just found out that my config is wrong...
I updated the /etc/config/system file entries like this

config timeserver 'ntp'
        option enabled '1'
        option enable_server '1'
        list server 'cronos.cenam.mx'
        list server '0.north-america.pool.ntp.org'
        list server '1.north-america.pool.ntp.org'

I reviewed http://wiki.openwrt.org/doc/uci/system and then I noticed I was lacking 'ntp' after config timeserver... This worked and seems to work jsut fine with the WebUI. However it is odd that the WebUI also worked without the 'ntp' parameter (May be it is a bug in the WebUI??)

Anyhow this worked like a charm

As a remark, I upgraded from 10.03 and the config was the one left after the upgrade, so it may be an upgrade issue...
