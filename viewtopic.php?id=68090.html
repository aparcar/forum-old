Well... I made it myself. Now everything works fine, except LEDs.
1. Modify tl-wr841n-v10 factory firmware according to https://forum.openwrt.org/viewtopic.php … 67#p206667
2. Rename firmware to wr843nv2_tp_recovery.bin
3. Start TFTP recovery via bootloader according to https://wiki.openwrt.org/toh/tp-link/tl … v9_v10_v11
4. [OPTIONAL] It might be a good idea then to sysupgrade router via SSH with 841n V10 firmware (e.g. sysupgrade -F openwrt-ar71xx-generic-tl-wr841-v10-squashfs-sysupgrade.bin) - for future updates if no support from OpenWrt.
Have a nice day everybody
