I've been working on an AG241v2 for the last few weeks.  It works fine using 8.09.1 or 8.09.RC2.  I've not managed to get an image built from trunk to work - the networking appears broken.

Flashing new firmware is done via tftp (look in the wiki).  This is a plus because if you brick the device all you've got to do is re-flash firmware at power-on time.

The only problem I have is with the ethernet switch.  It's not possible to configure it in any way.  It works fine so long as you are happy with the power-on configuration (a single VLAN).  I need to configure a couple of port-based VLANs on it, but the driver isn't able to do it.
