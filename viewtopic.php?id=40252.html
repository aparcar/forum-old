A few questions:
1. is the router actually running openwrt at this moment?
1a. if it is, did you telnet into it and set a password (passwd)
1b. if it isn't, then it isn't an openwrt issue, and you may get better answers from a dedicated WRT54G board.
2. if it still has lynksys FW, then have you tried via the web interface: http://wiki.openwrt.org/toh/linksys/wrt54g
3. there is an issue with macs (maybe  others): see troubleshooting here: http://wiki.openwrt.org/doc/howto/generic.flashing.tftp
