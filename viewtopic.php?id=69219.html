Hi,


thehofl wrote:

Which would make more sense for adblocking on a home network? OpenDNS or the methods described on the github tutorial "OpenWRT adblock implementation"?

As far as I know OpenDNS does not provide adblocking.
Apart from that, the main difference would be that using OpenWRT you can do it yourself, so there is no need to send all your DNS traffic to a third party DNS resolver. ( and i assume most people using OpenWRT do care about 'being in control' themselves').

Alex.
