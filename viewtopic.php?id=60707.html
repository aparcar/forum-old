You shouldn't create another thread for the same thing, this way spreads the information and makes it harder to find.
Please put this on the original thread..
Too long wires everywhere. The power regulator 7805 should be placed close to the power application, and it will heat a lot if you plug anything else than a low power USB device, like a hard disk of various usb sticks.
500mA, the standard for high speed USB 2.0 devices, will dissipate 3.5W of power on that little regulator , will heat up and burn.
And if they are two or more devices, even more problems.You need a proper cooling solution, like a heatsink.
Also you need a capacitor in the input and output to avoid voltage oscillations than can make strange things.

So don't tell that it's my design problem when you've done a crappy job!

By the way, nice idea on putting a usb hub.

Good News for everyone, No more compilation required
recently i discovered TL-MR3220 v2 & TL-WR741 v4 Both uses AR9331 CPU
I have successfully tried these firmwares in my TL-WR741ND V4
15.05 Chaos Calmer https://downloads.openwrt.org/chaos_cal … pgrade.bin
14.07 Barrier Breaker https://downloads.openwrt.org/barrier_b … pgrade.bin
Install using mtd and force it

Everything just works fine
USB is working great with no errors at all
Install these packages if you want to mount usb storage
kmod-usb-storage-extras kmod-fs-vfat kmod-fs-ext4

Install these packages to enable tether your 3G and 4G USB dongles and Android tethering
usb-modeswitch kmod-usb-net-rndis kmod-usb-net-cdc-ether kmod-usb-net

For mounting your USB as rootfs or  extroot http://wiki.openwrt.org/doc/howto/extroot
