My plan of action:

* Use the SD-mod so as to gain access to the extra storage.
* Build kernel with mmc.o before flashing.  Cull unneeded extras from kernel if running on limited RAM.
* Install /boot/ to firmware (/dev/hda1).
* All other branches are on /dev/sda1.  Install to flash card directly from computer.

Questions/Issues:

* Where/how to obtain non-/boot/ branch files?  I already have a GS that will work for pulling the branches directly off of it if needed.  Better route available?
* Is there enough room for the base utilities on /dev/hda2?
