It's probably closely related to EasyBox 802, which is based on the Arcadyan ARV752DPW. There are pictures at http://ftp.maddes.net/hardware/DSL_EasyBox_802/. You could start by trying the image compiled for that. The details on how to do that depend on

- the bootloader
- whether the current web interface allows you to flash a foreign firmware (unlikely)

If you can get a bootlog via the serial connection I might be able to help you further.

You might also find this thread helpful:
https://forum.openwrt.org/viewtopic.php … 34&p=1
