How are the Fritzbox and WR703N connected? By Ethernet or Wi-Fi? If Ethernet, you want a bridged AP. Shortly summarized, the FritzBox and WR703N should have IP addresses on the same subnet (e.g. 192.168.1.1 and 192.168.1.2), and only one of them should run a DHCP server. Either of them can do it, but only one at a time.

If they are connected by wireless, you want a routed client with relayd. The WR703N will connect to the FritzBox as a client, and that client interface will be the "wwan" interface in the guide. To be able to access the WR703N in this configuration, you have to set a static IP address for the wwan interface (guide uses DHCP), and configure the same IP address on the relay interface ("stabridge" in the guide), see the bottom of the guide for this.
