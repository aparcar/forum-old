From the web site:

Haserl is a small cgi wrapper that enables shell scripts to be embedded into html documents. It is intended for environments where something like PHP is too big.

This is a young, but promising project, that might interest the web interface team (Hi Sam, Plec & wdl). In this release, shell variables now survive code blocks and uploads are disabled by default.

In order to test it with busybox's httpd, you must create a /www/cgi-bin if missing and put your scripts in it.

The package can be found in my testing repository (and its sources are available too).
