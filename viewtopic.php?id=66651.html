Ok,


This seems to work:

edit the /etc/sysupgrade.conf and put only the folder /etc/example/ in it.

Then:

    sysupgrade -b /tmp/test.tar.gz
    sysupgrade -f /tmp/test.tar.gz last-image.bin;
