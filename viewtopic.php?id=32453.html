This firmware was tested on the om1p wifi device (Atheros 23xx chipset)


The TOR mode is configured as a transparent proxy on ath2. this includes DNS requests - testing shows that tor uses its own DNS servers and communicates to them via SSL connections.  tcpdump is a great tool :0)

TOR v0.2.3.5-alpha was the latest version found at https://www.torproject.org at the time of firmware compilation. opkg update seems to be several versions behind, so the latest package was used in the build process.

note: added tor mode switch to the .drone auto config command...

"Tor 0.2.3.6-alpha includes the fix from 0.2.2.34 for a critical

The new firmware can be downloaded from:

https://github.com/downloads/guinness1/ … ct2011.tgz
