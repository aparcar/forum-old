There are only 1+ Routers that support OpenWrt and have 802.11ac. There are some issues with these still.

None of these units are anyway near "decently" priced except the Archer C7 where you have to open the case and check hardware revision.

TP-Link Archer C7 : 107 to 88EUR special currently runnning at amazon.de
- possibly not capable of running OpenWrt because of hardware revision 1 built in
- hw v1 was only available a limited time and not sold/produced in "large" numbers (~1/2 year ago)
- chances are higher to get a working hw v2 chip now (rumored to be built into Archer C5)

TP-Link Archer C5 : unknown
- possibly hw v2 built in , but unknown, not available yet

Sitecom WLR-8100: ~130EUR
- hw v2 built in 100%
- 16mb flash
- internal antennas only
- already soldered pin headers for serial
- USB3 has some problems

Sitecom WLR-7100: 115€
- unknown hw internally, same OEM like 8100 so maybe same chip like bigger model?

Due to some hints in the Firmware (left by the OEM?) there might be 2 additional units with OpenWrt in
Japan (from IO-Data) or sold by Zyxel (NBG6716) - see http://wiki.openwrt.org/toh/sitecom/wlr-8100
