Configure the cable modem to act as a router and server addresses in the 192.168.1.x range.
Configure the OpenWRT device as a "dumb access port" (google for it, there is a guide).
WDS is not easy, it tends to work only on devices using the same chipset and running the same OS; have a look at "https://wiki.openwrt.org/doc/howto/clientmode".
