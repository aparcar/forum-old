raz123 wrote:

Link?

I can't really give you a specific link since I don't know what device you own... if you had a TP-Link TL-WR1043ND you'd find the SANE binaries for Chaos Calmer here: https://downloads.openwrt.org/latest/ar … /packages/

It might be worth noting that package names have changed; the former "sane-libs" is now "libsane" while "sane-backends" is now "sane-backends-all" – but in most cases you wanna install one (or more) of the new individual backend packages like "sane-apple", "sane-canon", "sane-epson" etc instead of the full sane-backends-all (which is actually a meta package that is dependent on all individual backends).
