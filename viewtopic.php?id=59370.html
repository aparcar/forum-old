For those of you who asked themselves: What is multiweb?

MultiWeb is a completely rewritten set of Rooter scripts that are more stable and provide more modem support. This is in the early stages of development but has been extensively tested and works very well. Some features from Huntsman and FunnelWeb are missing in this version, however.

MultiWeb is designed around providing support for multiple modems running at the same time and Load Balancing is supported to manage these modems. Single modem support is also improved.

http://ofmodemsandmen.com/downloadsm.html
