I think you need to make clear what you want to do with this VPN.  OpenVPN is an application that uses SSL VPN tunneling capabilities to remotely connect computers and resources.  You can use it for free.  For example I use OpenVPN on my router and laptop to allow me to connect to my home network and resources whenever I have an internet connection.  for free.  It just takes time to setup the configuration and get it how you want it, which sometimes is a great amount of time.

There is a web interface if you install and utilize LuCI.
