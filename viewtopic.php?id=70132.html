OpenWrt wifi and other settings should be configured through the uci system.  uci reads files in /etc/config and generates "low level" configuration files (such as hostapd-phy0.conf) dynamically into /var/run.  /var/run is on a RAM disk so it must be regenerated every boot up.

So install kmod-mac80211-hwsim then set up /etc/config/wireless as if you had a real wifi device and see what that does.
