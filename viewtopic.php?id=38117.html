point it at the other router for gateway.

cable into a lan port rather than wan port.

decide how you want your subnet numbering to work and set up as appropriate.

If you are going to use your isp for dns then I assume your old router is picking up dns as part of dhcp from the net.   Point the backfire at the old router for these services.

I'd run the ne router on a static IP although some might set a reserved address in the old dhcp range.
