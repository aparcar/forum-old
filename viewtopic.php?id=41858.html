On attitude_adjustment 12.09rc1 I'm attempting to configure radvd.

I have a simple config in my source tree in ./files/etc/config/radvd

config interface
        option 'interface'        'eth0'

config prefix
        option 'interface'        'eth0'
        option 'prefix'           'aaaa::/64'

config rdnss
        option 'interface'        'eth0'

Having built and booted the target, the configuration is present in /etc/config/radvd as expected but radvd is not running.

If I start it manually with  '/etc/init.d/radvd start' I can see a file has appeared in  /tmp/etc/radvd.conf but it's empty.

'logread | grep radv' shows no error messages.

Any pointers on how to get a handle on what's failing would be very welcome.
