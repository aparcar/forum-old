Looks like network doesn't work. Hopefully you can access its rescue mode. Here's how you do it.

Give your computer static IP address 192.168.1.2 and subnet mask 255.255.255.0

If you're using windows, open the command line console and type

ping -t 192.168.1.1

Unplug the power cable to your wrt54g and plug it back in. Does the ping command get a response back at all. If it does, good news, because you can rescue by tftp flashing.
