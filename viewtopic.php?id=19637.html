Active leases are stored in /tmp/dhcp.leases, so you can show them with "cat /tmp/dhcp.leases" on the command line.
If you want to have the lease time displayed in human readable form, you could type:
awk '{ print strftime("%F %T", $1) substr($0,length($1)+1)}' </tmp/dhcp.leases
