Hi all!
I've been struggling with successfully using this USB wireless card. I've got a V1, with RTL8188EU chipset. As I found out, this driver is only incuded in the Chaos Calmer release. I managed to get it working, by installing kmod-net-rtl8188eu. The driver loads:

[  899.870101] usb 1-1: new high-speed USB device number 3 using ehci-pci
[  900.468760] Chip Version Info: CHIP_8188E_Normal_Chip_TSMC_A_CUT_1T1R_RomVer(0)

The problem is, that this driver does not seem to be nl80211 compatible. So the "wifi detect" and iw commands do not work, only iwconfig. Writing

config wifi-device 'wlan0'
        option type 'wext'

manually to wireless config does not help, I get the "wlan0(wext): Interface type not supported" message...

What can I do to make OpenWRT UCI support the old WEXT driver interface?
