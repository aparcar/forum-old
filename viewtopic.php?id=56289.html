Now I can install Python package on openwrt.
Thank you for  this link : http://rnd.rajven.net/openwrt/mikrotik/ … /packages/
Please down load all these package before install python otherwise you will receive the error.
1.libpthread*.*
2.zlib*.*
3.libffi*.*
4.python-mini*.*
5.python_2.7.3*.*
type command on your prompt : opkg install " those packages"
Your screen look like this in the end:
*****************************
Installing python-mini (2.7.3-2) to root...
Installing python (2.7.3-2) to root...
Configuring python-mini.
Configuring python.
*****************************
