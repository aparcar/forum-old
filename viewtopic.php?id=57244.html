/etc/config/wireless:

(empty line)
config 'wifi-device' 'wl0'
        option 'type' 'broadcom'
        option 'channel' '11'
        option 'txpower' '18'
        option 'hwmode' '11bg'

config 'wifi-iface'
        option 'device' 'wl0'
        option 'network' 'lan'
        option 'mode' 'ap'
        option 'ssid' 'ur2slow'

        option 'encryption' 'psk2+ccmp'

Clearly, I set encryption to WPA2-PSK-CCMP.
