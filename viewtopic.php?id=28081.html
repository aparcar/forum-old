Hallo Guys,

since OpenWrt is more suitable for Hackers and "PowerUsers" I should ask you. This should be helpfull for Geeks and whatever searching for the right router. Today this is not a big deal, since OpenWrt only supports a few routers which are current (which you can buy right now), but this could change. You never know. So what tags are relevant?

! Speed: GigabitEthernet / FastEthernet
! Whether the antennas/antennae are detachable: DetachableAntenna / NonDetachableAntenna
! Whether the WAN-Port is distinct: NonDistinctWANPort / DistinctWANPort

This http://wiki.openwrt.org/toh/mikrotik/rb … .for.vlans seems to be a special one, since the Port is distinct (a distinct PHY) but it is still attached to the Switch. This is different to  http://wiki.openwrt.org/toh/mikrotik/rb … .for.vlans

! USB
! JTAG
! Serial
! 32RAM: Amount of RAM
! 8Flash: Amount of Flash
? Number of Ports: =<3 / >=4
? Driver: ath9k
? VLAN-Switch This is superfluos since almost all Switches are capable of Tagging, or?
? CPU: we have the ISA+Extensions (sic!), then the actual implementation of it (the Core) but these are too many I think, the raw MHz (this could be the least relevant),

{{tag>GigabitEthernet NonDistinctWANPort ath9k DetachableAntenna USB 32RAM 8Flash VLAN-Switch }}

Since OpenWrt is more suitable for developers and nerds and people with special needs, let's try to make this helpful for them. Ideas/Comments?

I think the following tags are important:
USB
JTAG
Serial
RAM(16,32,64,more)
Flash (4,8,more)
Ports (1,5,more)
Wifi Chipset/Driver
Hardware-VLAN (yes, not all can, the fonera 2.0 for example has a hw switch but no vlan cap)
MHz
PoE
Detachable Antenna (if applicable)

I created this ugly wiki: http://wiki.openwrt.org/meta/tags

crypto is important, here you can see the benefits:  http://smorgasbord.gavagai.nl/2010/02/s … re-crypto/
I could imagine there are more crypto units, with different capabilities. So far, my 400MHz MIPS offers 1,9MiB/s read/write over scp without any crypto unit. Exactly enough for ADSL2+ with 16000MBit. So today I do not need one, but later this could be nice.

I added hwVLAN and PoE. Just keep in mind, that there are certain proprietary implementations of PoE.

And what about the SoC? So far, we have only tags for the CPU (Core), but none for the SoC.
