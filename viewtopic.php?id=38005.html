What you're looking for is called load balancing, but it isn't going to work like you're describing.  Each of your local web servers will still have their own unique IP, and if you are pinging them from that particular LAN, or accessing those IPs directly, you will always connect to that exact system.

For load balancing to work, you need the load balancing device to be between the user and the destination machines.  It will work its magic, and pass traffic to the target system transparently to the user.  If you really want this to work on your local network, you would at minimum need to get up a second LAN on your router, or possibly even add a second router if yours doesn't support this capability.

I've never set this up myself, but you can find some info to get you started here:  http://wiki.openwrt.org/doc/uci/multiwan
