8 Sep 2016, 21:50

Username should default to "root". Password (which sounds like you already set one) is the same one you use to log into the gui. If you didn't set a password, then there is no password, since OpenWrt doesn't come with a default password. In this case, just press enter on the password entry (and I would set a password after successful login).
