Post #6
arokh
20 Dec 2011, 15:21

Extroot is a mess, and I would not recommend it to new users.

- Unsolved bugs in overlayfs: https://dev.openwrt.org/ticket/9221
- Complicates firmware upgrade
- Poor documentation, just look at that wiki page

Also, some times it just doesn't make sense to use extroot. In this case, you are recommending extroot to install netatalk. What if the user wants to plug in a different USB drive and have it shared through AFP? Running services are better to place in ROM.
