sc2mpd and mpd2sc work with upmpdcli. upmpdcli is a upnp/dlna player package that already exists for openWRT.

sc2mpd and mpd2sc allow to send or receive Linn's songcast audio stream from/to a computer. sc2mpd is a receiver (or player) and mpd2sc is a sender.

Songcast protocol is very interesting because it is an easy way to send sound from a computer to openWRT devices with low latency.  See details about sc2mpd here.

There's already some softwares that acts as senders (upplay, songcast)

Sources are here (same sources repo than upmpdcli).
https://github.com/medoc92/sc2mpd

I've created this request on github too :
https://github.com/openwrt/packages/issues/2810

If someone is able to create a package for it, it can be very interesting
It's already possible to use it on raspberry.
I can help for testing the new package.
