Hey Guys -

First of all, 'm a huge fan of OpenWRT.  Yesterday, I had fiber installed at home so my internet went from 20Mbps to 250Mbps instantly.  The only downside of that is that my wireless network (which was already old) cannot keep up when downloading - a good problem to have.   I was looking up equipment to replace my current network, but every time I find a router I like, I find that it (or key parts of it) are not supported by OpenWRT doesn't support most of the ones I am interested in.

I'm looking to replace both devices with a single, fairly powerful Router.  If it had ~20% more range that the WZR-HP-AG300H I currently use, I wouldn't need a repeater.  I am not wanting to break the bank, but also don't want the cheapest thing; either.  I'd much prefer it be OpenWRT compatible, but if there's one you would suggest whose stock firmware is still solid and would be a good deal; that would be fine, too.  Below is an overview of what I'm asking followed by a list of how the network I'm replacing is currently configured if it offers any details.

Overall, my questions are...
1. What router (preferably OpenWRT compatible) would you suggest for my setup?

2. If said router may not offer ~20% more range than current primary, what would you suggest for compatible repeater? - OR - if the suggested router is capable of swapping antennas, which would you suggest?

3. Is there a specific (AC) wireless card I should get for best performance / range if needing to upgrade any?

Current Home Wireless Network
Primary Router: Buffalo WZR-HP-AG300H (OpenWrt Chaos Calmer 15.05)
Pretty much a typical setup...

  • Primary firewall

  • AP for 2.4Ghz & 5Ghz A/B/G/N

  • Forwards 42 ports to static ip'd devices

  • OpenVPN host

The current speed when connected to primary's AP via 5Ghz N from a frequent stop (couch), I can copy a single large file at ~4.1MB/s

Media Center Repeater: Netgear WN2500RP (Stock firmware (1.0.0.30_1.0.58))

  • Provides ethernet connections to a few devices

  • Repeats WiFi on SSID that differs from primary's

  • Uses NetGear "Fastlane" to connect to primary router.  5Ghz is dedicated to Repeater <-> Router traffic and 2.4Ghz dedicated to repeating WiFi

What Wireless Network is Used For

  • Providing internet to ~20 devices of all types (Our house is ~2200 sq feet.  Would prefer not having to use Repeater, but currently hardly can connect to primary router's AP from one end of the house)

  • Streaming video files via smb share from server connected to primary via ethernet & dedicated Kodi box (Windows) connected to Repeater via ethernet (Slowest link is the "Fastlane" 5Ghz connection between the two routers)

  • Copy of various files between laptop & server


So... what do you think?  Thanks for any suggestions!!
