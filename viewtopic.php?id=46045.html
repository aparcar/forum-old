I see the ath10k patch has been released . However it is only for trunk. I compared the makefile of mac80211 betweeen 12.09 and trunk. It is totally different. Does anyone know how to port the patch to 12.09? Thanks

They are not that different.

While official AA12.09 has wireless drivers of 2012-09-07 (kmod-80211 in packages download dir)
AA in trunk uses the same wireless drivers like Barrier Breaker : 2013-06-27 already
see:
https://dev.openwrt.org/browser/branche … 1/Makefile
https://dev.openwrt.org/browser/trunk/p … 1/Makefile

These wireless drivers from the backport project (=mac80211 package) should run with older kernels like that one used in AA.

Therefore the patch should apply / you can easily add ath10k related stuff in AA trunk and see if that works. However why would you try AA insteard of BB ? Did you put a new ath10k minipcie in an older router ?
