the easiest way is simple 

patch -p1 < yours-patch-file-here.patch

if you need to add to some folder 

patch -p1 -d folder/ < yours-patch-file-here.patch

But if you wan't understand how OpenWrt working, I recommend to make all changes manually. 

Index: target/linux/brcm63xx/image/Makefile
===================================================================
--- target/linux/brcm63xx/image/Makefile    (revision 33663)
+++ target/linux/brcm63xx/image/Makefile    (working copy)
@@ -218,6 +218,10 @@
     $(call Image/Build/CFE,$(1),AW4339U,6328,DSL274XB-F1-AU,,--signature2="4.06.01.AUF1" --pad 4)
     # D-Link DVA-G3810BN/TL
     $(call Image/Build/CFEFIXUP,$(1),96358VW,DVAG3810BN,6358,DVAG3810BN)
+    # Pirelli Alice Gate W2+ (Vela)
+    $(call Image/Build/CFE,$(1),"Alice Gate W2+",6348,"AliceGateW2+")
+    # Pirelli AGW2+ (AGA)
+    $(call Image/Build/CFE,$(1),AGW2+,6348,AGW2+)
 
     # TP-Link TD-8810A, TD-8810B, TD-8811A, TD-8811B
     $(call Image/Build/CFE,$(1),8L-2M-8M,6338,TP8810_8811)

+++ target/linux/brcm63xx/image/Makefile is file what need to change.
-218 is line number there located in original
+    # Pirelli Alice Gate W2+ (Vela) plus means need to add this line
-                                                      means to delete line
                                                       means do nothing

after you made all changes from patch, select right image and packages in "make menuconfig" and run "make"
As you can see, life is simple:)
