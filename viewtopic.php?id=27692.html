SSH requires you to have a password associated with a username. If you are able to login to LuCi without a password, it means that you have not set a password yet, change or create new password in either case:

Below steps should hold good for both cases:
System > Admin Password
Now change the password
System > Reboot
Login with new password

If the above is successful, connect to router from putty, enter root when prompted for username and also password when prompted for.
