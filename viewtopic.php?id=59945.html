RangerZ
29 Sep 2015, 01:40

There are lots of devices out there that have firmware optimized for their application, but run a hardware set that supports Openwrt.  Any device, like a Pogoplug, that can run OpenWRT becomes a "router" by installing the software.  It may, and probably will, loose some functionality.  Virtually all the "Travel Routers" forgoe their IOS/Android Apps once OpenWRT is installed.

Don't use base your solution on OpenWRT, base it on functionality.  If you do not need a router, don't build one.

Regarding the ZBT, I was just trying to get the info into a post with a title that was appropriate.  I know that are not small, but the car units are kind of cool!
