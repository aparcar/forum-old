anestef10 wrote:

Failed to establish connection

There is nothing special to opkg.
It just downloads the package data files via normal wget (or uclient-fetch).

Sounds like you have network connectivity problems.  Possibly you have defined the internet connectivity wrongly. E.g. you are behind another router/modem and you have not defined the gateway and DNS correctly. Just test downloading anything with wget from command line.

Or possibly your ISP blocks downloads.lede-project.org? But that would be rare.
