Merry Christmas...

I've built something that might interest a few people here. It's called "pinapl" (pronounced "pineapple") and it's a library to make good-looking applications for a small and cheap 320x240 touch screen made by 4D-systems. All you need is a device with a 3.3 or 5 volt serial port than can run Lua.

Please check out github.com/ropg/pinapl

I'd be happy to hear what you think about it.
