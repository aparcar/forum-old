Kamikaze does NOT use NVRAM. Kamikaze stores the changes in the JFFS2 partition on the flash. To reset your password you have to boot into failsafe mode and Telnet to 192.168.1.1. Then you run mount_root; passwd to change the password. Now you can fix your mistakes. If you just like to start over like from a fresh flash just do: mtd -r erase rootfs_data
