The new table(s) of hardware are shaping up, and now we need your help to fill the tables with up to date data.

The first task is: Fill the "Availability" column, minimize the '多'

In the following table, you will see a lot of '多' in the Availability column:

http://wiki.openwrt.org/toh/views/toh_admin_available?dataflt[0]=availability=多

- Please check the table, and if you know that either a device is available or discontinued, please edit the data accordingly (click "View/Edit data"). IMPORTANT: On the Techdata page that opens, edit the data via the LEFT edit button below the box. The editor that appears provides dropdowns for easy setting of the availability status (and other information).
- Only count as available, if it's a new product that's being sold, and only if there's no volume restriction mentioned ("only 1 left").
- If a device is ONLY available in certain countries, e.g. China, please mention this in the "Comments" fields.
- In the summary of your edit (down below, next to where you click "Save"), please provide a source for your information, e.g. "available at amazon.com, amazon.de, ...".

Feel invited to checkout and comment on
Main ToH: http://wiki.openwrt.org/toh/start
Different views: http://wiki.openwrt.org/toh/views/start

Thanks for your support and enjoy the new Tables of Hardware!
