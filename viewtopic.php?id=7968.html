Authentication code for external pstn calls
Extention for sip gateway connection (ip 192.168.1.3)
Italian sound installed

note:
For download firmware backup open ssh console (with putty or sshcommand) and give this command:
# dd if=/dev/mtdblock/1 of=/tmp/firmware.trx

after go http://192.168.1.2/firmware.trx download and save it.

Upgrading / Restoring:
To reflash from within openwrt you will need to use a trx file:
# mtd -r write firmware.trx linux.
