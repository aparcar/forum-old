If you do not have a serial cable, do not try Kamikaze on a WGT634U.  It is highly recommended that you have a serial cable with any router, but it is critical with the WGT because there is no other recovery method in the event that you misconfigure the network settings or put the router into a state that OpenWRT can't fully boot.
