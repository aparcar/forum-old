I have a Linksys WRT54GL 1.1 and installed the newest Kamikaze version without any problem. But unfortunately I set all ports to bridge mode and deactivated the management IP, so the router isn't responding to pings any more.

I tried it through the password recovery mode and I can connect to the box with telnet, but when I want to change someting it tells me "Read-only file system".

All I need is the possibility to set the configuration to the OpenWRT default settings.

So how can I restore the default configuration?

Thank you

Post #2
LinkZ
21 May 2010, 19:56

I think you can erase the jffs2 overlay to restore the defaults back to your flashed squashfs defaults:

mtd -r erase rootfs_data

However, you could also just reflash the router.

Post #3

1) 'failsafe mode' is almost always working (telnet@192.168.1.1)

2) run 'mtd -r erase rootfs_date' to wipe all changes you made after flashing OpenWrt and start over.
