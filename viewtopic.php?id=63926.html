Greetings fellow OpenWRT users, I have recently acquired the linksys e8350 router to later notice the lack of a port to this platform. The SoC used in this router is the exact same one as in the D-Link DIR-885L A1, which is already supported by openwrt (https://wiki.openwrt.org/toh/hwdata/d-l … ir-885l_a1). According to this table http://www.tomshardware.co.uk/router-so … 33464.html, a few more routers (ASRock G10, Compex AP148, Linksys EA8500, Netgear D7800, 7500, R7500v2, and TP-Link Archer C2600 v1.x) could also benefit from this port.
If anyone else has this router and would be interested in helping port OpenWRT to this router, respond back!
