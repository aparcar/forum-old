Openwrt makes what you tell it to do. If you configure it to be wireless client to network ¨wifi¨, it will try to be the client until you configure it as ap, or configure to be client to some other network. If both connections are active, you can control the data flow by changing default route.

I still don know what you are trying to do Take a look at http://wiki.openwrt.org/doc/uci/multiwan, maybe this is what you want.
