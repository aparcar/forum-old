Greetings gents. I'm looking to port OpenWRT to iiNet's "Budii" (the 2nd largest ISP in Australia's latest modem)

I've written up a ticket here https://dev.openwrt.org/ticket/13345

I've also copied copied the GPL code & binary files here http://intelminer.com/Budii (as well as cell phone pictures of the chips and pin headers on the device)

I'm slowly trying to piece together what does what, as this device has a LOT of pin headers on it

I think it would make an excellent addition to the OpenWRT ecosystem, especially as it comes with things like a built in (8GB) USB drive (expandable, is a standard USB drive behind the back panel)


I'd love to get feedback on what people think of this device, as well as helping clean up the patches (they date from kernel 2.6.30) to the latest bcm63xx kernel sources (3.6.11 I believe?)
