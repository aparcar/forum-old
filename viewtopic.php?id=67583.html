The original approach works for me, e.g:
      Name = HTTPS; Protocol = TCP; External Zone = wan; External Port = 443
      Intenral zone = lan; Internal IP address = 192.168.1.1; interal port = 443

BTW, i'd recommend you to work with https to protect your login password.
To switch to https, in menuconfig select
LuCI -> Collections -> luci-ssl
