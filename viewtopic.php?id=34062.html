First: read all you need in openwrt documentation: http://wiki.openwrt.org/doc/start

Basic configuration:  http://wiki.openwrt.org/doc/howto/basic.config

Enter in openwrt GUI:  http://192.168.1.1

click in system ---> Software (here click Update lists)

there exist a line named: "Filter" now type into that line: "luci-app-qos" without quotes and click "find package"

click available packages and click "Install" wait a seconds the router will be install the package.

find package: L7-protocols and install if you want have more service protocols list under QoS configuration

If all is done, you have qos installed and exist under Network ---> QoS

read here:  http://wiki.openwrt.org/doc/uci/qos

enable and type all your configuration,

Tips: for download and upload set 90-95% of total connection to avoid network overhead or you can enable "Calculate overhead"

later set your Classification Rules and click Save & Apply.
go to: System ---> Startup (look for by qos and see if this enable if do not click disable)

reboot your router.

in Wireless configurate your router with Transmit Power:

Max TX Power (2.4 GHz): 22dBm w/HT40, 27dBm w/HT20
Max TX Power (5 GHz): 24dBm

TiP: with HT mode 40 is more Throughput but less wireless transmit power, you choose only test!
security: WPA2 PSK (CCMP) or WPA-PSK/WPA2-PSK mixed mode

click Save & Apply

and you are done! with QoS enable and better performance with network.
