Some of what you wrote suggests that you did not UNCHECK the flag "Keep Setting" and thus the path to the packages is wrong.  It needs to be the path to the final release.  The message about kernel is because there was a kernel version change and the strong swan package you are trying to install is for the old kernel.

You probably should reflash the system with the sysupgrade, uncheck the keep settings, and then reinstall all your packages.

I will suggest you keep a test file with the packages you install and save for future reference and upgrade.  You can also, for the same version of SW (ie CC) backup and then restore your config.

https://downloads.openwrt.org/chaos_cal … x/generic/

The factory is to convert factory firmware to OpenWRT.  If you have OpenWRT running use the sysupgrade. 

You can backup your config and reuse it, though take a good look.  I can not say that nothing was changed during the upgrade, though expect nothing was.

You could get lucky, change the downloads directory, and just reload packages and may be ok.  I just think it's safer in the long run to do it right, especially if you have not done much config to it.
