In my first setting I made many mistakes, and I'm not anymore able to connect to my router - router gets an ip address from WAN, but I do not know its LAN address so I'm not able to connect, as far as it doesn't accept connections from WAN or from LAN with WAN address (which is more strange.........)

I need to restore defaults to enter my router and repeat configuration tests and experiences.

In order to restore defaults I entered in failsafe mode and executed

mtd -r erase nvram,

which didn't give result, the configuration being the same (I can see the wi-fi net I tried to create).

Is this instruction deprecated in kamikaze? How can I get it to work?

Hi.

Since Kamikaze totally ignores nvram, erasing it won't help. However, you want to kill the jffs2 overlay to revert all changes since flash.

Use this command:

mtd -r erase rootfs_data

~ JoW
