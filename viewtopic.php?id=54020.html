Hi also had problems using tftp on windows 10. I finally managed by Mac OS X on my Macbook Pro:

Download PumpKIN (tftp client for Mac OS X). Under PumpKIN preferences set your path for the ras.bin file. And set "read request behavior" to "give all files"

On the Mac: On the network settings (TCP/IP) configure your IP to 192.168.1.33 and netmask 255.255.255.0. Gateway leave blank.
Attach to one of the LAN ports
Copy your firmware file into the tftp directory and name it ras.bin
Start PumpKIN
Turn on NBG6616 holding down WPS key
Keep holding WPS key until the firmware is downloaded and router is flashing <--- this was were I first failed, I didn't wait long enough until I power cycled, takes really long until the LEDs flag, hold WPS key until they flash!

Power cycle device when Power and USB2 LEDs flash. (It will take more than 5 minutes)

This is how it worked for me. Let me know if you succeeded!
