Post #2
MBS
26 Jul 2014, 01:22

Find 2 unused GPIOs (or free used ones, like LEDs, Buttons), connect a 2-10 kOhm resister between each of them and Vcc, install i2c-gpio-custom. Or, get a supported USB-I2C adaptor - depends, if you can live with the downsides of hotplug interfaces.
