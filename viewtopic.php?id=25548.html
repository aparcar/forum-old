Not sure how to do this with luci, but if you edit the file /etc/config/network, do the following:


    1) Find the "config interface wan" section.  Comment the entire section out (or delete it).


    2) Find the "config interface lan" section.  Change the line that says:

        option ifname eth0

    To:

        option ifname "eth0 eth1"


Then reboot.  This will add eth1 (which is usually the wan port) to the br-lan bridge so it's bridged with the other 4 lan ports (and with the wireless).
