So a year latter i got back to my project. I managed to find all files needed. Although i dont speak Chinese, the following site has more than enough info http://d.hatena.ne.jp/k-kuro/20080516/p1
Another site that i found helpful was http://d.hatena.ne.jp/k-kuro/20080516/p1
So lets start!!!
1) Go to http://downloads.openwrt.org/kamikaze/7 … 6/packages and download the following packages
kmod-nls-base_2.6.21.5-atheros-1_mips.ipk
kmod-nls-cp437_2.6.21.5-atheros-1_mips.ipk
kmod-nls-iso8859-1_2.6.21.5-atheros-1_mips.ipk
kmod-fs-ext2_2.6.21.5-atheros-1_mips.ipk
fdisk_2.12r-2_mips.ipk
e2fsprogs_1.39-1_mips.ipk
libuuid_1.39-1_mips.ipk
You will also need this driver: http://fon.testbox.dk/packages/mmc/phro … er7143.ipk
2) Transfer them to fon2100 \tmp and install
ipkg install kmod-nls-base_2.6.21.5-atheros-1_mips.ipk

ipkg install kmod-nls-cp437_2.6.21.5-atheros-1_mips.ipk

ipkg install kmod-nls-iso8859-1_2.6.21.5-atheros-1_mips.ipk

ipkg install 26215-4pinfon2100Driver7143.ipk

insmod /lib/modules/mmc_drv.ko

ipkg install kmod-fs-ext2_2.6.21.5-atheros-1_mips.ipk

ipkg install fdisk_2.12r-2_mips.ipk

ipkg install e2fsprogs_1.39-1_mips.ipk

ipkg install libuuid_1.39-1_mips.ipk

mkdir /mnt/mmc
mv /etc/rc.d/S99mmc /etc/rc.d/S39mmc
vi /etc/rc.d/S39mmc

You will have to format your card

fdisk /dev/mmc
o [ENTER]
n [ENTER] p [ENTER] 1 [ENTER] [ENTER] [ENTER]
t [ENTER] 83 [ENTER]
w [ENTER]
mkfs.ext2 /dev/mmc0   

And mount it

mount /dev/mmc0 /mnt/mmc


THE END
