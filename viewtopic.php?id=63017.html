I'm running Chaos Calmer 15.05 on an Asus RT-N16.

I have DHCP disabled on the router, and I'm running it on another system on the network.

The DHCP broadcast messages are not being forwarded across LAN interfaces, so the external system never sees them. (I've confirmed with Wireshark.)

From /etc/config/dhcp:

config dhcp 'lan'
        option interface 'lan'
        option dhcpv6 'server'
        option ra 'server'
        option ra_management '1'
        option ignore '1'

From /etc/config/network:

config switch_vlan
        option device 'eth0'
        option vlan '1'
        option ports '1 2 3 4 8t'

config interface 'lan'
        option ifname 'eth0.1'
        option force_link '1'
        option type 'bridge'
        option proto 'static'
        option netmask '255.255.255.0'
        option ip6assign '60'
        option ipaddr '192.168.20.1'

Is there something I'm missing that's needed to forwarded the DHCP requests between LAN interfaces?
