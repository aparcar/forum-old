15 Sep 2014, 00:45

Barrier breaker starting with rc3 has luci built in. Any future updates to BB can be downloaded from openwrt.org. Use the "sysupgrade" file. Luci will be included in that upgrade and you'll only need to reinstall any other packages you previously installed. The version will match with the version of BB you download.
