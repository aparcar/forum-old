Hello all,

I have two routers (both Siemens Gigaset SX763).

My goal is to create two networks, one for local wired PC's which are connected to the Internet via first router (ISP's firmware), and the second network (2nd router with OpenWRT) which is dedicated for WiFi clients.
Also I would like to separate these two networks into two different subnets. So far the first network is running fine, but I can't manage the second network (separated) to run on a different subnet. I only managed to make the second network working by setting it on same subnet as first router's network.

What can I do to make the second network isolated and on a different subnet (which would be fully controllable by OpenWRT)?

P.S. - keep in mind that the first network is working on PPPoE, and that the first router doesn't have the capability to make PPPoE pass through in settings (ISP's fault, they have disabled that)!

Any help would be appreciated..

Hi, you can make the 2nd router as a client, say your first router is 192.168.1.1, set you second lan as 192.168.0.1, then scan your wireless, just click the tab"join network", then the 2nd can manage all your wireless if name a essid,"openwrt" is by default not enabled.

Lots of info in Wiki.

Good luck.
