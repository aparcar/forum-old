Hi mate a 30/30/30 reset is a old ddwrt thing. You are verry brave to test such new firmware.lol  You are lucky not to end up with a bricked router. You cant get to the routers userinterface because there is not one installed. The build you have installed is from trunk and it does not come with luci installed. To install it you need to down load putty and Telnet or SSH to your router at 192.168.1.1 username is root and I don't know what your password will be. Then when you get logged in at the command line tipe.
opkg update
opkg install luci-ssl
/etc/init.d/uhttpd start
/etc/init.d/uhttpd enable
reboot

let the router reboot.
Now you should be able to connect to the web server serving LuCI at http://192.168.1.1.
More info here:
https://wiki.openwrt.org/doc/howto/luci.essentials
