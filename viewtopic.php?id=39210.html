Hello. I'm running OpenWRT 12.09-beta on TP-Link TL-WR741ND v1.9 and want to setup a static IPv6 lease for my laptop.

For now, I have the following in my /etc/config/dhcp:

config host
        option name 'Shevchuk-laptop'
        option ip '192.168.1.111'

dnsmasq man says it can be done also for IPv6:

A single dhcp-host may contain an IPv4 address or an IPv6 address, or both. IPv6 addresses must be bracketed by square brackets thus: --dhcp-host=laptop,[1234::56] Note that in IPv6 DHCP, the hardware address is not normally available, so a client must be identified by client-id (called client DUID in IPv6-land) or hostname.

So, how do I do that in OpenWRT?


Thanks in advance.
