Nous travaillons sur un projet de communauté VPN.

Le sujet est de créer un Site web servant d'outils pour la mise en relation de personnes ayant un serveur VPN et de personnes souhaitants devenir clients VPN.

La première étape concernanit la création d'un code heartbeat pour des routeurs DD-WRT hébergeant le service serveur PPTP  afin de récupérer une liste de clients VPN a jours en temps réel pour tout les routeurs.
La deuxieme étape consiste  a créer la meme chose pour des routeurs tournant sur OpenWRT.
La troisième étape consiste a faire une intégration de Openvpn dans le système.

La première étape est maintenant terminée, et nous cherchons donc une personne avec de bonne connaissance VPN sous OpenWRT, afin de nous aider dans cette migration et de réaliser les béta tests.

Si vous pensez etre ce genre de personne, nous serons heureux de vous accuillir a nos cotés(Merci de me contacter par PM).

Bien entendu je répondrais a toute vos questions.
