You can set a custom mac address without luci.
uci set network.wan.macaddr=11:22:33:44:55:66
uci commit network
/etc/init.d/network restart

In case anyone that is very new at this like me is still having problems, I have gone into "System" - "Software" - "Edit Package lists and Installation Targets" and updated the first line to
