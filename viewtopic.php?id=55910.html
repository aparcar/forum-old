For details see spi-gpio driver:
http://git.kernel.org/cgit/linux/kernel … spi-gpio.c

As you should see this driver implements two interfaces: DT and platform data.

For DT documentation see:
http://git.kernel.org/cgit/linux/kernel … i-gpio.txt

For platform data see "spi_gpio_platform_data" defined in spi_gpio.h:
http://git.kernel.org/cgit/linux/kernel … spi_gpio.h

I believe there is no way to probe "spi-gpio" from the command line (using sysfs).
