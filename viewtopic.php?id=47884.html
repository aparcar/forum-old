Assuming you have ssh enabled, ie you set that password.  Using putty from within the folder (= directory) holding the .bin file:

scp openwrt.bin root@192.168.1.1:~/openwrt.bin

This will copy the bin file to the home of the root user on the router, from where you can issue the mtd command.  For that read the wiki.
