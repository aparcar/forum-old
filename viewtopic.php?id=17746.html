jlazkano wrote:

1. Which is the diference between 1.2 and 1.4?

Active development is in 1.4.x and 1.6.x now, so consider 1.2.x depreciated.

jlazkano wrote:

2. Which packages I need?

Is there other software for that???

I run the asterisk14 and asterisk14-res-crypto packages (and their dependencies) which suits my simple needs.  You can go with as many modules/packages as you want, but the base package should get you working with SIP and the asterisk14-res-crypto package is required to use the IAX2 protocol, IIRC.

Keep in mind there are problems using Asterisk 1.4.x within a SquashFS image, so you'll probably want to install it on top of a base OpenWrt installation.
https://dev.openwrt.org/ticket/4210
https://dev.openwrt.org/ticket/3530
https://dev.openwrt.org/ticket/2773
