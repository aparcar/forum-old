kaksi wrote:

I installed nano like this:

ipkg install http://www.lillesansfil.org/ipkg/nano_1 … mipsel.ipk
...
But when I try to use nano it  just is not there....

 find / | grep nano

shows nano installed to /opt/bin/nano, but

echo $PATH

doesn't show /opt/bin in the path.

PATH=$PATH:/opt/bin

fixes this.

The other package is better (newer version, installs to /usr/bin as is standard, etc), but this is how you could troubleshoot similar occurances in the future.
