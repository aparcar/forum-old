Hello everyone!

I have a Netgear CG3100 which someone has thrown away. It seems to work fine, but since it's a cable modem it won't be useful to me unless I manage to install custom firmware on it.

Apparently, firmware upgrades are normally done via the cable by the ISP, so there is no firmware upload page in the web-gui. Also telnet access has been disabled (nmap reports port 23 as closed, not filtered). So I suppose my only options left are serial and jtag (I have located the ports).

Is it plausable to be able to run custom firmware on it? I haven't found any source code for the current firmware, but maybe the device is similar to something else where source code is available. I do not need the modem part to work.
