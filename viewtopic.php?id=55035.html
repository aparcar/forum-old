If you have the option, check that its PSU still provides its nominal voltage under load. These cheap PSUs easily break over time (can't hold the voltage in use), making that a common error condition - and pretty much the only one which might still be worth fixing considering the age and slowness of the device involved (but only if you can be sure that it's really 'just' the PSU).
