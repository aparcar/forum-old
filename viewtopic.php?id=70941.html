Folks,
I have a wireless device running OpenWRT Chaos Calmer 15.05 version. The wireless chipset is Atheros 9000 series. The mac80211 and cfg80211 subsystems are used internally. I was able to successfully enable and run station
mode using iw, ip and wpa_supplicant commands. Background scanning was enabled by using bgscan in the wpa_supplicant.conf.

Now, I'm trying the same using uci commands instead. But bgscan is not supported in UCI. I tried a hack: booted the device up, enabled station mode using UCI. Then manually edited the config file under /var/run/wpa_supplicant-<dev>.conf, added the line 'bgscan="simple:40:-75:120" in the network section, then went to wpa_cli and 'reconfigure', which re-reads the wpa_supplicant conf file. Now background scanning works.

I'm wondering if there is a better way to do this using UCI instead of this hack approach.
