go to Network >> Interface
click edit on LAN interface
disable DHCP
assign a IP address of that router under the same subnet of your main router, eg, your main router is 192.168.1.1, assign your hg556a to 192.168.1.250 or something
save setting


go to Network >> WiFi
click add
add a new
on interface configuration, enter your main router's wifi SSID
set mode to Access Point WDS
check the network LAN
under wireless securty tab
set the encryption that your main router's wifi uses
cipher set it to auto
in key , enter your main router's wifi passphrase/password
click save and apply, wait a few moments
click enable on that new interface your created.
done

tip: to know the ESSID of your main router and its encryption, click the scan button on wifi page in openwrt.
