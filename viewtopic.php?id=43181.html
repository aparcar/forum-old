I'm running 10.03.1, and I'm having this issue:
https://dev.openwrt.org/ticket/10059

Essentially, even though my WAN interface is set with a static IP, it will only hold that IP address for an undetermined period of time and then it will receive an IP address from the DHCP server of the network it's connected to.  This happens with every instance I've deployed.  Typically, it's not an issue because there usually is no DHCP server where I've been assigned a static IP, or if I'm using a static IP on the WAN port, I can disable any DHCP servers on the network to which the WAN port connects.  But in some cases, I don't have that control, so I lose my static WAN IP.  Bringing the interface down and back up resolves the problem for some period of time again, but not permanently.

Any help would be appreciated!
