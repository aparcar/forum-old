Hi,
don't know why, but here the way to install from console.

Download the ddns-scripts for you router's hardware from http://downloads.openwrt.org/snapshots/trunk to your router.
On router's console change to the directory where you downloaded the file to.
> rm -R /tmp/ddns*
> rm -R /var/log/ddns*
> rm -R /var/opkg-lists/*
> opkg install ./ddns-scripts_1.0.0-22_all.ipk
