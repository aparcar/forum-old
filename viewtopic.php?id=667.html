The fact it has a diag light an an amd flash chip means it's an original v1.0. The flashing diag led means that the firmware is not booting or invalid and you need to tftp a working firmware, in tftp's binary mode, to 192.168.1.1.

If you read anything this should have been the first thing that you tried, so I'm a little confused as to why you asked. Make sure to use a standard tftp client, not one from linksys which uses a password, and make sure there's no networking issues trying to send to 192.168.1.1.
