Most of these routers do not have a FPU. Floating point calculations are done via software.  They can't even decode an mp3 in realtime without using an integer based decoder.  Any kind of distributed processing on these devices is wasted effort.  These threads (BOINC/SETI/Folding@Home) pop up every so often. I highly doubt most of these projects can do anything useful on a machine with 16MB of ram anyway.
