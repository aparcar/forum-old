My question is, how should I configure the network on the access point so that I can download and install LuCI?

for example:

uci set network.lan.ipaddr=192.168.1.249
uci set network.lan.gateway=192.168.1.1
uci set network.lan.dns=192.168.1.1
uci commit
/etc/init.d/network restart

note: happened to me that dns is not working when used in LAN network, if it happen fix manually with:

echo "nameserver 192.168.1.1">/etc/resolv.conf
