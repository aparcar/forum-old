The RedBoot is only available for a few seconds after connecting the power cord.
Try opening two terminal sessions and ping (with no count limit / ping forever) both 192.168.1.1 and 192.168.1.254.
Unplug the power cord and plug it again. Wait a few seconds to see if any of those two addresses reply.
If one of them reply, telnet to that address on port 9000.

Hope it helps.
