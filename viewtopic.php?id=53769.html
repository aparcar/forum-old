The simplest way is to have the wired network and the wireless sharing the same address range.
Instead of a static address I now configure the router to always assign a fixed address to a certain interface MAC.
Unless there is a reason for your current settings the easiest thing to do would be to reset the router to factory defaults. You would then need to access the LUCI from the wired LAN  with the PC until you setup the wireless.
The factory defaults for my router are 192.168.1.1 and with DHCP serving in the 192.168.1.x range for both the wired and wireless clients. If you allow your PC to accept dhcp addresses from the router then you should be able to access the LUCI and then setup the wireless. Setting a static address for you PC in the 192.168.1.x range might also work except I haven't tried that for years.
Good luck.
