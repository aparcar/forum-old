Ok, problem was my wifi client, not the router.

First i updated my router to trunk and checked wifi with lucy. The new
view revealed, that my client always connects with 20Mhz, no matter what i do.

So i checked my modules and found out that "brcmsmac" module was loaded
despite the fact, that i installed the closed source broadcom sta driver quite a
while ago, because "brcmsmac" does not support dual band mode.

Seems like a broken dependency in ubuntu 11.10. By default apt should deinstall
the open source driver, when someone installs the broadcom closed source driver.

After blacklisting the driver, rx/tx = 270/180 and iperf = between 8 and 11 MB/s!

Well, stupid me.
should have checked that first
