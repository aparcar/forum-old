My scenario is like this:

[Client] --wifi--> [OpenWRT]  --wan--> [Server 1 (IP 1.2.3.4)]

Now what I'd like to achieve is that all traffic from my client to Server 1 with IP 1.2.3.4 is redirected to another server, lets call it Server 2 with IP 5.6.7.8.

On my Linux box I would usually achieve that with two iptables rules:

iptables -t nat -A PREROUTING -d 1.2.3.4 -j DNAT --to-destination 5.6.7.8
iptables -t nat -A POSTROUTING -s 5.6.7.8 -j SNAT --to-source 1.2.3.4

When using these rules on OpenWRT however, the clients can't establish any connection to Server 2.

Anybody can point me in the right direction?
