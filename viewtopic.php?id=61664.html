Best? I'm not sure. But certainly the most straightforward would be a (bash) script that goes through that file file line by line with

traceroute -n -m 11 $ADDRESS | wc -l

and echo the address into a file if the result is anything less/other than '12'. (The count includes the line that introduces the output, so '11' would mean 10 hops.)
