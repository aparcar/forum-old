Hi
As part of and R&D project (based on OpenWRT) I'm designing a one-off daughter board for Alix 6.x and (probably will fit) 2.x boards.  So far board has on it:

1. Power conditioning - Allows Alix to take 7vdc ->48vdc  (requires bodging a couple of additional wires to the logic board)
2. Keep Alive Power Ride-Through Circuit (Super cap based) Can allow for nice shutdown in case of power failure
3. ADC & I2C I/O GPIO Expansion - 4 channel 12-bit ADC and 16 GPIO lines plus analog input
4. I2C Bus Passthrough with voltage level conversion option
5. Current use / voltage monitoring (requires bodging a couple of additional wires to the logic board)
6. 4 channel thermistor inputs
7. eeprom w. unique ID (addressable on I2C bus)
8. Possible (I can (almost) make it fit) internal USB2 connector

Do a lot of you use Alix boards for OpenWRT?  This would make a pretty awesome addition to a home firewall.  e.g. want to have it turn on lights, open garage doors, sense temperature. If anyone would find this board useful I could do a batch, start a project, blah blah etc etc.  Any other suggested features?  Note there really isn't a lot of room in my PCB design, which sits on the 3 rows of connectors on the board. I'm also designing it so that the assembly still fits into the case with one additional ribbon/breakout protruding.
