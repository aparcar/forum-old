You should use a WDS connection between Linksys and Asus router, bridged to the LAN network on both sides. If you are using a normal client to master wireless connection, you should not bridge the wireless interface on the clients side, and use a different subnet there and routing between the subnets. Beware that this will separate the broadcast domain.
Bridging client wireless interfaces does not work/is not supported on all HW platforms, because of problems using more than one MAC address on the same wireless interface.
