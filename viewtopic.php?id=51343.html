What are "Associated Stations" under WiFi tab? Suddenly there are two (with the same ssid,different macs and different tx,rx rates)

Post #2
Laethnes
29 Jun 2014, 21:26

It's list of Wi-Fi devices connected to your router. The SSID is displayed because OpenWrt is capable of creating multiple wireless networks with single Wi-Fi device and because single router can contain multiple Wi-Fi devices with separate settings - so you can know into which network is given device connected. Remaining info (mac address, ip address, signal, noice, tx, rx) concerns given - connected - device.
