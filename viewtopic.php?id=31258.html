As the tunnel may need to be refreshed / upped for several reason, I am not sure if there is any magic way to say that "this was the final time, now it is permanently up".

Maybe having your hotplug script to check the ipv6 reachbility / ping against some site? If IPv6 traffic works, then you can run the remaing part of the script. Name it to something like 95-6in4 hotplug, so that it runs last with an ifup process.
