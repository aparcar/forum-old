Have you checked out the files using svn yet? If not, see here for instructions.

https://dev.openwrt.org/wiki/GetSource

Once you've checked out the files cd to the directory containing those files. In there you should be able to find a directory called scripts.

#run feeds to see the available options
./scripts/feeds

#have an up to date list of packages (check syntax)
./scripts/feeds

#if I want to lighttpd to appear when I do make menuconfig I'll do this
./scripts/feeds install lighttpd

#now lighttpd should appear in menuconfig
make menuconfig

chibister wrote:

yea, im using openwrt kamikaze 8.09 ...
i tried install 'opkg install lighttpd' on the sd card...
when i issue opkg list_installed, the lighttpd wasn't there ..

i type /etc/init.d/lighttpd start

That's because Openwrt can't find the path to lighttpd. You need to edit the variable PATH in /etc/profile to fix this. You need to do that for LD_LIBRARY_PATH too.
