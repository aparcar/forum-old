17 Jan 2017, 21:55

Super excited, I got to win one! I got xen half working, and already moved towards a docker type solution on my 1200AC. The added 500 mhz will actually make a big difference for me with the 3200. I was trying to get GlusterFS to work and the CPU would peg out on the 1200AC when I did small file copies, large files were fine. I ended up using the USB 3.0 slot with an SSD to function as additional flash memory and the USB 2 slot for the connected raid 1 enclosure. I also found that EXT4 is much less CPU intense than HFS+, but HFS+ had the fastest data transfer speed but due to the CPU bottlenecking it was negligible. If anyone wants tips or could suggest where to post instructions no how ti got GlusterFS to compile and work, let me know.
