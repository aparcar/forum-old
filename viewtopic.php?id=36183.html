regarding ports on Openwrt.

For experiments and simple setup do

    /etc/init.d firewall stop
   /etc/init.d/firewall disable

to disable firewall permanent.

wifi / lan bridging is done in the default setup on /config/wireless

for testing services on ports "netcat" is a very potential tool.
Beware though the buildin nc of busybox is quite stripped version; install "netcat" instead.

regards
3zl
