If your router has 2 independent network adapters (such as eth0, eth1).
For example : eth1 for WAN, eth0 connected to the switch.
You need to create 2 VLANs on the switch.
switch has ports 1 2 3 4.  1 - eth0.0 (for wan), 2 3 4 - eth0.1 (for lan)
Bridge eth0.0 and eth1. This bridge will act as external 2-port switch.

If no :
Then separation between WAN and LAN is already done by the switch and VLANs.
You only have to remove one port from LAN part and add this port to WAN part.

Ensure you know what to do if you cut yourself from the router.
Any mistake can lead to this

In switch config

VLAN_ID 1 : port 1 - off, ports 2,3,4 - untagged

Add second VLAN with ID=2

VLAN_ID 2 : port 1 - untagged, ports 2,3,4 - off

you must be connected thru ports 2,3,4 or lose access

Interfaces -> WAN.
[v] creates a bridge over specified interface(s)
[v] Ethernet Adapter: "eth1" (wan)
[v] VLAN interface: "eth0.2"

something like that
haven't tested. need try & debug
