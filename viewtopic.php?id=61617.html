It has been removed, because the package has no maintainer (AFAIK).
If yoh'd like to use it on Barrier Breaker or Chaos Calmer, you can find it in BB's oldpackages.
In CC you can download it and install from file (there is no dependant package need).

Other way (not really suggested) :
I've created a new opkg config file: /etc/oldpackages.conf which refers only to BB's oldpackages repo, and...
opkg -f /etc/oldpackages.conf update
opkg -f /etc/oldpackages.conf install netstat-nat
