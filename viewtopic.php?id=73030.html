There is no Openwrt/LEDE 17.10 release, nor has there ever been.

Sounds like some private build by github.com/MarvellEmbeddedProcessors

Your link is downloads.openwrt.org/snapshots/trunk/mvebu64/generic/packages/base/Packages.gz
To my knowledge there has never been an official target platform "mvebu64" on the main site. There seems to be one such target on the MarvellEmbeddedProcessors source tree in Github, but then you likely need to use also their download server (if they have one...)
