Yes that is the correct location.  Get the TL-WR740n "sysupgrade" for your hardware version.  Hardware version is on the sticker on the bottom of the router.

That device has only a 4 MB flash chip.  If you flash the release build, there is only space to install a few small packages at run-time.  You may need to use the Image Builder to make a custom image removing some packages you don't need and adding the ones that you want.
