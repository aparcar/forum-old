I installed backfire (10.03.1) and it seems to be much better.
I connected 6 clients to the WLAN (and one over LAN) and run P2P file exchange (over internet, using popular p2p software) on all of them for about an hour or two.
Everything worked fine.

Although later (when I stopped P2P and disconnected most of the clients) I noticed one client having problems.
Windows 8 would report "limited connectivity". The IP settings were normal, but pinging the router failed. After reconnecting it was OK for some minutes and then the same happened. During that time other WLAN clients had no problems.
This happened two maybe 3 times, then I connected the PC with cable.



PS: All the time I used the 2.4GHz radio and WPA2-PSK security.
