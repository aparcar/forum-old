Hi.

There is no OpenWrt-specific stuff for sysctl values. It just uses a vanilla Linux Kernel.

One option I always set myself is:

net.ipv4.tcp_ecn=0

Otherwise strange connection problems might occur on the router itself (opkg update etc.)

~ JoW
