Gabor is rocking on the AP81's lately!  Despite this router having a difficult u-boot, OpenWRT trunk 14674 is now flashable and all devices should be in working order.  TP-Link also released the source code for their u-boot in response to requests.

I'd also like to point out that I suspect the earlier TP-Link TL-WR841ND model should be pretty easily adapted to OpenWRT now this work is done.  Those models also use u-boot and are Atheros based.

There is a thread on the hardware forum where I wanted to discuss trying to solder in USB port on the unpopulated pads.  it also has details on how to solder in serial port on this router, which is more difficult than most.  See here: http://forum.openwrt.org/viewtopic.php?pid=82724
