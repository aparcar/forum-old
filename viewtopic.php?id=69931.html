I think that LuCI uses the shell command iwinfo <device> assoclist to get that information.

You can get more detailed info about connected stations with iw dev <device> station dump.

iwinfo is an OpenWrt specific program written mostly to support LuCI, but iw exists in all mainstream Linux.
