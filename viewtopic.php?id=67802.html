Hi guys!

I made a small portable wireless music system with webradio, dlna/upnp server and dlna/upnp renderer. For this project I used the A5-V11 mini 3G/4G router.

All in all this project costs about 25€ and thanks to the A5-V11 router the setup is pretty small. The installation should be quite easy because everything is preconfigured on an extroot filesystem. I also included ympd a lightweight but powerful web based mpd client! (ympd is a standalone binary, php and uhttpd are not needed.)
