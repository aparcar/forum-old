Try and go to:
Network -> Firewall -> Trafic Rules: Open ports on router:

Give the rule a name and say port 80 under port. It should open the port on the firewall, without having to edit uhttpd and so on.

Post #3

28 Oct 2014, 19:12

Also check if your modem might be blocking the request somehow. You can fix that by setting up DMZ on your modem, so it points to your OpenWRT's IP (remember to use a static IP or tell DHCP on your modem to give the same IP every time to the MAC address of the OpenWRT Router)
