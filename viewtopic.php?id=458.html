What is a shared authentication for WEP ?

From the Linksys documentation:

Shared Secret Authentication:
The Linksys Wireless Access Point (WAP) supports three authentication setting; open systems authentication,shared-key association, and both. Open systems authentication permits any client to use the WAP as long as they know the SSID. Shared-key authentication uses the WEP key as a shared-key to be exchanged between the client and the WAP as a simple form of authentication. WEP must be enabled to use shared-key authentication. The both setting permits either or these methods to work. The default method in the Linksys WAP is â€œBothâ€. The authentication type is changed by going to the Advanced tab and then the wireless tab and selecting â€œShared Keyâ€ from the â€œAuthentication Typeâ€ drop down menu.

Although the out-of-the-box WRT54G(S) have the configuration option for this, it seems there is no configuration option to set the authentication method with OpenWRT.
