Post #11
eduperez
10 Feb 2015, 00:37

OpenWrt is built to work on flash memory, and writing logs lo flash will kill it, so everything is kept in RAM; use "logread" to read the system logs, and "dmesg" for the kernel messages (or use the LuCi interface).
