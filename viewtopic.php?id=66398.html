It's very easy to get a serial port connection on that hardware. Open up the case and plug in.

That being said, you should just be able to recover with the reset button (wipe overlay). That's assuming the button was set up correctly.

https://wiki.openwrt.org/toh/trendnet/tew-823dru

See the four little holes on the right side? The set of three + 1 with the white silkscreen rectangular outline (NOT the one with the two large holes).

If they are like mine, and they probably are:
    JP1 = Rx
    JP2 = Tx
    JP3 = Ground

baud = 115200

Don't plug into the one lone port with the square around it: that's probably +3.3v or +5v.

If you've never connected to a TTL level serial port, say so for more advice.
