well virtualbox will not see your WiFi card not unless you passthrough that WiFi card to your virtual environment, well this requires VT-D support on processor it self, and it only exist on fairly newer processors. If you have a USB WiFi then you can just passthrough that USB WiFi and openwrt will detect it, with that the only thing you need to do is install the required packages, mainly the wifi drivers.
