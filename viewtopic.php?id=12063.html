I just played with this yesterday and it was pretty easy. if you have the X-wrt  web interface installed.

Under Networks->wireless  you can set the mode to 'client'  and enter the ssid of the access point.
Also select 'wan' (I forget the field name)
My test system was open so I didn't need any passcodes but there is a spot for selecting that as well.
Save and apply the changes.   

You may need to reboot the openwrt system to get it to give your host pc the default route and dns
entries (I did).
