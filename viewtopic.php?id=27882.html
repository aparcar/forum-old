I don't think you have to enable ntpclient.  There is no entry in /etc/init.d for ntpclient.  It is tied to the hotplug interface event.  See /etc/hotplug.d/iface/20-ntpclient.  Therefore, it automatically starts up when an interface starts up.  You can configure it via /etc/config/ntpclient.  Confirm it's running by "ps | grep ntp".
