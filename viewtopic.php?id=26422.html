ok ... found
i2c-gpio and i2c-gpio-custom must be loaded both ... some instruction wasn't clear about this
moreover the insmod sequence is important !

compiled all as module
then:
insmod i2c-core
insmod-i2c.algo-bit
insmod i2c-dev
insmod i2c-gpio-custom bus0=0,14,15
insmod i2c-gpio

then it works great !
