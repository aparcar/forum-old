Again,

I have found a solution. I added 2 lines in /etc/firewall.user and I restarted the firewall (/etc/init.d/firewall restart) :

#
iptables -A input_rule -j LOG --log-level info --log-prefix "OpenWRT:DROP: "
iptables -A input_rule -j DROP
#

Now when I look at the log with "logread" I see what I drop.
I don't know if it the best solution but it works. Perhaps writing this log in an other file should better (like /var/log/logdrop).
