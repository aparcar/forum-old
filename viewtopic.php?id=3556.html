I am currently using the U.S Robotics 5461 Printer Server WiFi Router connected to my cable modem. I got this beast US$30 (after some US$30 rebates). According to this Table of Hardware on the OpenWRT web, this beast comes with a 200MHz Broadcom 5350 CPU chipset (WRT54GS v3 uses Broadcom 5352 CPU chipset), only 2/8 MB flash/RAM, a 2.0 USB port, and the OpenWRT firmware has never been tested on this beast. Never mind about only the 2MB flash mainly because perhaps one can attach an old 32MB USB flash card, i.e. SD, CF, etc., to boost the size of flash RAM to hold more software package. The question I have is if this U.S. Robotics 5461 capable to run OpenWRT + asterisk with only 8MB RAM?
