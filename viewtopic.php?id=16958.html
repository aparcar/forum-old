It will be available in stable when 8.09 is released.  Until that day, you must compile the source code from trunk.

If you want separate WAN and LAN ports, you must include a kernel patch here: http://dev.openwrt.org/ticket/4050

See the wiki pages here:
http://wiki.openwrt.org/OpenWrtDocs/Har … nk/DIR-300
(the Airlink 101 AR430W is equivalent, so this is another version of the same thing:)
http://wiki.openwrt.org/OpenWrtDocs/Har … ink/ARxxxW
