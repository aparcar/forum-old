Hi,

I'm tring to setup DNSMasq to assign via DHCP specific DNS servers to certain hosts.
I've done this easily with ISC's dhcp server in the past, but I guess I'm not as familar with DNSMasq.

For example, my kid's PC and tablet should get OpenDNS's servers assigned via DHCP while all others
should still get the default IP of the OpenWrt router.

I've tried editing the host entry in /etc/config/dhcp, but I'm not sure what the options should be.

I know I can install ISC's dhcp server, but if possible, I'd like to stick with DNSMasq

Thanks for any help you can provide.

For those who are interested, I figured it out.

I added the following to /etc/dnsmasq.conf

# Point Kid's devices to OpenDNS
dhcp-mac=set:kidspc,AA:BB:CC:DD:EE:FF
dhcp-option=tag:kidspc,option:dns-server,208.67.222.222,208.67.220.220

Note: Replace AA:BB:CC:DD:EE:FF with your MAC.

Now when kidspc does a DHCP request it gets OpenDNS's DNS servers.
All other devices still get default DNSMasq server.
