Hey everyone,

I am trying to write C code that uses stat and strftime to see the last modified time of a file.

 struct stat attrib;
    stat("file.txt", &attrib);
    char time[50];
    strftime(time, 50, "%Y-%m-%d %H:%M:%S", localtime(&attrib.st_mtime));
    printf ("%s\n", time);

The code is pretty straight forward, and it works fine on Ubuntu, but for whatever reason, running this code in OpenWrt is causing the incorrect year and date to show up. Instead of 2017-05-07, I get 2024-05-01. Considering this code works fine in Ubuntu makes me think its OS related, but running "ls -l" shows the correct date too. Any ideas as to why this is happening?
