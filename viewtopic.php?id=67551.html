1) I'm thinking I need to setup an Ad-Hoc multicasting network. Does that sound right?

To start, I want to setup an Ad-Hoc network between two computers. Unfortunately, I'm having trouble. The systems use the ath9k driver and a recent version of OpenWRT. I initially tried setting up the ad-hoc network using Luci, but I couldn't get the two computers two connect. In the Luci web interface, PC2 showed up as an associated station to PC1, but PC1 did not show up as an associated station to PC2. After that I tried the following commands:

iw wlan0 set type ibss
ip link set wlan0 up
iw wlan0 ibss join adhocTest 2437

At that point, the two PCs appear to be associated with each other, and a random BSSID was generated. I also added ap_scan=2 to the wireless config. However, I tried checking the connection using iperf as I usually do in AP/Client mode, and the connection timed out. So either I'm not testing this correctly, or I haven't got the network setup correctly. Any help would be greatly appreciated.
