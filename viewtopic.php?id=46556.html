Post #6
Squonk
4 Oct 2013, 18:28

The first time login is "admin/admin".

The password warning will stay until you set a password for root.

The "admin/admin" login is then disabled until you revert the router to safe mode.

You should then be able to use the "root" user with your own password to log in.
