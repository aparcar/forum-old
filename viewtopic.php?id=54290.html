A dumb AP has no WAN.  To duplicate what the simple stock firmware typically does, bridge all the ethernets (not sure what you're trying to do with VLANs) and the wifi into one network (lan) and assign a static IP so you know where to find it.   Conventionally AP's are set with static IPs in the 192.168.x.250 range, e.g. 251.
