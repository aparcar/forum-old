Hello,

I am trying to find a wireless router that can run openwrt and has an usb port.

I found the following two models, both around $40:

WL-520gU : has 4MB flash / 16MB ram, supported according to the wiki. However, USB 2.0 is broken, it only supports 1.1
USR5465 : It is not mentioned on the wiki, however the usr5464 is. US Robotics uses a linux-based firmware and offers the source (+build tools) on their homepage, has USB 2.0

Which one should I buy? The USR has USB 2.0 and seems to be open source by design, however I cannot find out how much flash/ram it has. Does anyone know where to find this information?
Btw: According to the source supplied by USR, it is BCM47xx based.
