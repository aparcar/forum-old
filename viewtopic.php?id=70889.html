The instructions depend on the hardware version of your device. To determine which version you are using, open up the device's casing and check which DSL chip it has.

If it has a Lantiq DSL chip, follow these instructions to setup the DSL modem and the network interface https://wiki.openwrt.org/doc/uci/network#dslvdsl

If it has a Broadcom DSL chip, follow these instructions instead https://wiki.openwrt.org/doc/uci/networ â€¦ connection.

Note that this only configures the DSL physical modem and associates a network interface to it. You will still need to configure the LAN / WLAN interfaces appropriately, configure the firewall zones & forwarding and associate the network interfaces with the zones. These settings depend on what kind of a network you want to achieve.

For details on how to configure rest of the network, see https://wiki.openwrt.org/doc/uci/network
For details on how to configure the UCI Firewall, see https://wiki.openwrt.org/doc/uci/firewall
