You can't change "factory defaults" outside of the "factory". You can only set them within the firmware during the build process. Once the device has been flashed with the firmware, those defaults cannot be changed - which is kind of the point.

If you build your own version of the firmware, you can include whatever files you want.
See http://wiki.openwrt.org/doc/howto/build#custom.files

For example, you can include SSH certificates, wifi settings etc.
The included "custom files" will overwrite the defaults ones generated earlier i the build process.

I build a community firmware, where I include certain settings in the binary. Those are then "factory defaults" for the users.
