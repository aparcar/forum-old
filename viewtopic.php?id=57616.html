A default configuration is designed to work much like typical stock firmware and route all internet requests from the LAN to WAN.  Thus you should reset the router to default configuration first.  Then configure your WAN. 

If your WAN is coming in over wifi you need to create a wifi client mode interface with the SSID and password matching the service, then switch the WAN "physical settings" off of the ethernet port and over to the wifi.  Make sure the wan is not a bridge (you should have radio buttons on the setup page that allow only choosing one interface-- not check boxes).

WAN IP address should not be in the same subnet as the LAN.  In other words, if your ISP is giving you 192.168.1.x, change the LAN to 192.168.2.1 or something else other than the 1.1 default.
