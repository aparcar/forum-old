Post #2
Wodin
12 Feb 2008, 22:08

It sounds like your VLANs are not set up when the router is booting, so all the ports are on the same subnet.

Maybe you can put some sort of MAC address filtering on your ADSL modem, or whatever the router is plugged into so that it only allows requests from the router.  Otherwise, give the router a fixed IP address and turn off DHCP on the upstream device.
