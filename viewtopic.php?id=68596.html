It was not bad choice with openssh. I run openssh everywhere. On 6 openwrt systems and several servers/workstations. I have unified config that restricts weak algos and perfectly works with keys.
All this works fine. But openssh has some builtin default restrictions. For example it refuses login if permissions on $HOME/.ssh a wrong.  Just read syslog rather than stating "its not working". It complains there
