@dror88
I am no expert on openwrt but I set up my router with such vpn.
I am using Aztech Hw550-3g router. In this router I set two wifi network 1-n/w connected to my dsl router with client config.
2- n/w works as Master mode (AP). then set up PPTP as vpn.
Now this router only need power onand no other wiring. It will connect to my dsl router using 1-wifi ssid as client then to PPTP VPN (commercial server in US or UK) and wait for clients to connect through its second ssid.

When any client device connect to 2nd ssid it will route all the traffic from client device via PPTP connection i.e. all my device connected to hw550 will always get routed through VPN and get US or UK IP!

Now, I am trying to get team-viewer on this router so that I can manage it from remote location.
