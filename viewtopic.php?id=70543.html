Just wanted to warn the community about a version of great router that has locked firmware update, and that is not listed on the wiki.

I've recently bought LNIB TP-LINK TL-WR710N and planned to update it with OpenWRT. It turned out that even though according to all signs it should be possible, it returns error 18005 no matter what I try.

Some characteristic signs of this version:
- has USB port
- version 1.3 according to label on the router
- blue LED
- foldable North American plug
- most writings on the box, on the device and the user manual are Chinese

The whole web UI is Chinese and there is no way to change it.

There is a firmware update option, but...
- Chinese TP-LINK site has no firmware upgrades for it
- official US firmware doesn't work (error 18005 - wrong version)
- all versions of OpenWRT and DD-WRT don't work as well (error 18005 - wrong version)

Firmware upgrade is not done in an easy way like in TL-WR703N, it requires setting up an TFTP server locally and then initiating firmware upgrade using web UI

Probably it could be updated using serial cable, but I don't have required equipment and I'm not even sure how much ROM and RAM it has available (don't want to open the case because it would make impossible further sale).
