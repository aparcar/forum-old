Use LUA and load socket.http and then it's a simple couple of lines of code. Google 'socket.http lua'

http://w3.impa.br/~diego/software/luasocket/


local http = require("socket.http")
local url = "http://whatever/script.page"
local data = "post string"
local response = http.request( url, data )
