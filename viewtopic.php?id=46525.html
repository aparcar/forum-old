Is the IPv6 ULA prefix randomly generated for each "install" or is it unique only to OPENWRT or build?

I assume that generating a new random value will not break anything.

Post #2
CyrusFF
30 Sep 2013, 18:04

It is randomly generated at a fresh install (without keeping old config information) and after a factory reset. Changing it to another random value or removing it will not break anything.

The randomness should only ensure that if you connect multiple router's together via tunnels or VPNs that there is a very low chance that 2 routers connected will have colliding ULA prefixes.
