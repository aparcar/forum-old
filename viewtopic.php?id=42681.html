Post #10
skm
20 Sep 2013, 15:37

Ok solved.
Airdump dont see what is in  /mnt/usb/usr/sbin so I have created symbolic link in /usr/bin for wireless-tools:

ln -s /mnt/usb/usr/sbin/iwconfig /usr/bin/iwconfig
ln -s /mnt/usb/usr/sbin/iwpriv /usr/bin/iwpriv
ln -s /mnt/usb/usr/sbin/iwlist /usr/bin/iwlist

now it works.
