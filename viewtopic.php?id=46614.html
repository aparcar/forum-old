This question it too basic, you should seach first, the answer is already there.Anyway,

If it's version 1, you can download the stable AA version, firt flash factory, then flashsys upgrade.
http://downloads.openwrt.org/attitude_a … actory.bin
http://downloads.openwrt.org/attitude_a … pgrade.bin

after changing the password, using the web gui interface to configure the router is quite easy.

1. make the lan different for avoidance of conflict with your repeated network, you can make it say 192.168.10.1
2 in wifi, click scan, and click join the network .you can make it simply as a repeater.

now it can do as a repeater. you can change other settings, like power,country,etc.

Unlike the factory firmware, OpenWRT has wifi disabled by default on a fresh install.  The wifi lights will not come on and no wifi signals will be emitted until you log in by Ethernet and configure it.
