There's a really possibility this is a bug with Windows 10.

Firstly have you upgraded your laptop wireless driver to the latest version?     You should be able to use a Windows 8.1 driver.

Have you got another wireless client you could test this with to be sure that it is an Openwrt bug?

Is your laptop downloading anything overnight?

If not, you could run "ping routerIP -t" overnight and see if it stays connected.

Maybe this is a DHCP related bug - perhaps the lease is expiring overnight and it is unable to get a new IP address from the router?
