Is the password not enough protection for you?

The default configuration for OpenWRT bridges the wired and wireless interfaces into a sigle LAN network; it is not
possible to segregate wired and wireless access to the router configuration, unless you undo the bridge and configure separate networks. This can be a problem in some circumstances.
