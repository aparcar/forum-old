Usage: sysupgrade [options] <image file or URL>

Options:
    -d <delay>   add a delay before rebooting
    -f <config>  restore configuration from .tar.gz (file or url)
    -g           gziped image
    -i           interactive mode
    -n         do not save configuration over reflash
    -q           less verbose
    -v           more verbose
