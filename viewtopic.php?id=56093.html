Hi, below attach pictures from my modded wr710n (US 1.0) running Barrier Breaker 14.07 with smb and ftp server, support for ntfs drive, add micro usb connector for external power supply, repeater and add usb adapter wn722n working as wireless lan (useful when repeated network is not available).

Some pics:

USB Hub 4 Ports:

Small USB Hub 2.0 self powered.

External antenna mod:

For this mod I had remove J1 (0 ohm resistor) and cut ANT1, I let untouched capacitors around original antenna.

Micro USB:

Micro USB pins connected to C206 pins, reference: http://wiki.openwrt.org/toh/tp-link/tl-wr710n

Internal setup:

I wanted original USB port working as if unmodified. To do so, I had to insert hub between original USB data lines on motherboard and original USB socket. For this i had lift d- and d+ in original USB port, GND connected to GND below original USB port, while +5VDC was connected to C206 positive pin because original port is limited to 1A.

Final setup:

Working with internal (AC) power supply or this setup can work too with external power supply and at least 5V @ 2A is needed, I used for example iPad 2.1A adapter.
