Well some of the shops/markets do not offer the hardware revision, but I've did a very simple search (http://www.amazon.com/s/ref=nb_sb_noss_ … tp-link+v3) that includes that.  The difference can be quite significant as manufacturers tend to change the hardware (for financial and other reasons) and while v2 of some hardware can be supported , v3 may not be (chipset change or something like that)

Regarding your edit - the biggest difference between fw versions is the kernel that's used in them. What can be a security problem is for example openssl library which is updated for both version. And another note - the 14.07 that you've read is probably the first final version it was introduced into openwrt. 15.05 (Chaos Calmer) most probably supports it too!

Still to receive a concrete answer you need to give us your hardware requirements.
