I am looking for a way to pass assigned DNS servers onto the hosts on my network.  I have attempted to do it by going into the gui and adding the DHCP option: 

lan,6,8.8.8.8,8.8.4.4

which in the /etc/config/dhcp file it looks like it adds:

config 'dhcp' 'lan'
        option 'interface' 'lan'
        option 'limit' '150'
        option 'leasetime' '24h'
        option 'start' '10'
        list 'dhcp_option' 'lan,6,8.8.8.8,8.8.4.4'

The above is correct, something else was wrong with my config that was causing it to fail to start up again. 

On a related note, you cannot have a space " " in a computer's static hostname in the DHCP configuration.
