Hey,

just figured something out....: Its the bootloader mentioned in the wiki whicht causes the Problems.
THAT Bootloader was only able to start the included openwrt firmware - any other firmware which should work, did not.

Now, after I loaded (with the bootloader mentioned in the wiki) the gpl-vioated "breed" version mentioned here https://forum.openwrt.org/viewtopic.php?id=55180   and flashing the latest 15.02 RC2 for this device, it boots without any problem.

But be aware... the "new" breed bootlader works.... but the webinterface is in some hieroglyphic like characters.... my firefox did only display some "Dice-like Blocks" - but since I new the menu structure from the other bootloader i got along....
