Sorry to bother you, but I couldn't access imap.gmail.com using Mozilla Thunderbird behind my router, a WRT54GL running Kamikaze 8.09.

My /etc/config/firewall looks all the same, except for some traffic redirection (6881-6889, the Torrent ports). My /etc/firewall.user looks like that:

#!/bin/sh
MYSERV="10.0.0.3"
for port in 20 21 22 80; do
  iptables -t nat -A prerouting_wan -p tcp --dport $port -j DNAT --to $MYSERV
  iptables -A forwarding_wan -p tcp --dport $port -d $MYSERV -j ACCEPT
done

All tries keeps me telling that a timeout was reached. I tried miska680's solution, but I hadn't reach my goal (http://forum.openwrt.org/viewtopic.php?id=18862). Any ideas?
