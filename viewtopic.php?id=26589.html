If you are running the stock firmware (The TP-link firmware), then you have to use the "...factory.bin" image.
If you are running Openwrt, then use the "...sysupgrade.bin" image.

Other than that, there is no advantage to using the sysupgrade firmware. When I said to use "systemupgrade" in my original post, it was only because you said you had already flashed.

Since you're back to running stock, use the "factory.bin" firmware.

~Jonathan
