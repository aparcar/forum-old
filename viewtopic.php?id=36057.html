Post #10
jow
9 Apr 2012, 23:06

You would first build an initramfs image of the OpenWrt AR7 target, then attach/solder a serial cable onto the PCB to be able to communicate with the bootloader (probably ADAM2).
From there you would attempt to load the initramfs image with TFTP and see if it boots at all and if yes, how far and what peripherals are detected or not. Afterwards you might need to add board specific initalization code to the kernel to setup LEDs, GPIOs etc.
