Hi!

We wanted to let our OpenWrt users know that yaSSL will be exhibiting at Embedded Live in London on October 20th and 21st.  We'll be talking about several topics including how to use CyaSSL to secure your firmware update systems, the yaSSL Embedded Web Server, and the CyaSSL Embedded SSL Library.  If you are a CyaSSL user or customer or would like to learn more about our products, please stop by and visit!

CyaSSL is optimal for use on embedded devices.  Some features include:
- Minimal build size of 30-100kb depending on build options.
- Runtime memory usage of 5-50kb
- OpenSSL Compatibility layer
- DTLS support
- Support for many ciphers, including the RABBIT and HC-128 stream ciphers

If you have any questions, please contact us at info@yassl.com, or visit our website for more information: http://www.yassl.com.

Thanks,
Team yaSSL
