The EX2700 should be supported.
https://wiki.openwrt.org/toh/netgear/netgear_ex2700

With OpenWrt the configuration of a "dumb AP" goes as follows:

Devices such as the EX2700 with only one Ethernet port start off configured with just a LAN network, and the port accessible to this network at 192.168.1.1.  This is mostly what you want, though you will almost certainly need to change the address.
Give device an IP address that is in the range of the main router, but doesn't conflict with any of the other devices in the network.  Set gateway and DNS to the IP of your main router.
Make sure the "Turn off DHCP" box for the LAN is checked.
Configure wifi and enable it.  If you use the same SSID and encryption as the main router, users will generally connect to whichever one is initially stronger without needing to reconfigure.
Save and apply all these changes, then disconnect the cable you were using to configure direct from your PC, and plug into the main router instead.
