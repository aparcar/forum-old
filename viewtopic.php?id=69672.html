First, Barrier Breaker is obsolete, you should use Chaos Calmer 15.05.1

Second, do you have any modem module fitted?  3G functions come from the optional modem module.  The SIM card slot is connected to the modem, not the router CPU.  The module you need depends on which cellular network you are subscribing to.

Once you have a modem, you will then need to install drivers for it.
