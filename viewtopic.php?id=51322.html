If you're talking about these instructions, the instructions are incomplete. With the first command, wget is trying to save the downloaded image into the current directory. After telneting/SSHing in, this would be /root which is neither sufficiently sized nor, at failsafe time, writeable. Simply change into the /tmp directory before you wget the image (cd /tmp).

(JFTR, if the router is working you could have skipped setting up a HTTP server and just copied the file over via SCP in normal operation mode. I get it, you're not that comfortable with OpenWrt and/or its command line yet, which is perfectly understandable. I'm merely pointing out that in all your questions you jump straight to the respective worst case scenario.)
