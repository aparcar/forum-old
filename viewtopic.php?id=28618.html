Restoring the default firmware (or putting the latest Backfire on it) would go a long way, you can clear nvram variables as well afaik (I don't know if WhiteRussian still uses nvram though).

WhiteRussian, even with a web interface, is horribly outdated, and chances are people looking for a second hand router won't know how to get it working.
