It's ok

I follow explaination from this site (in French) : http://petaramesh.org/post/2006/02/10/5 â€¦ -un-wrt54g

download firmware from linksys support site : http://www.linksys.com For my router it's "WRT54G_4.20.8_ETSI_code.bin"

I'm in windows XP.

Setup network card to 10Mbps half duplex with TCP/IP 192.168.1.2/255.255.255.0

open two CMD line windows

the firts with  "ping -t 192.168.1.1" command , the second ready to start with "tftp -i 192.168.1.1 PUT WRT54G_4.20.8_ETSI_code.bin"

When the router response to ping, lauch the tftp command and wait

router restart with linksys firmware.

It's great hardware this router.

I will re-install openwrt quickly.
