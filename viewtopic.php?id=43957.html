Dear *,

I'm running Barrier Braker (r36485) and LuCI (svn-r9812) on an WDR4900 and works perfectly so far. Now, as I understand, the uci firewall is just a "nicer" front end that generates "code" for iptables, which in turn configures some netfilter modules that actually perform the filtering. To that end, the UCI firewall only supports a predefined statements like "rule"s and stuff.

My LuCI interface looks like this: http://tinypic.com/r/33e5r1i/5

Where on earth is the "add rule" button for such custom rules in LuCI? My custom rules show up in the interface under network -> firewall and I can edit, save and delete them. But the only way to _add_ them is through vi'ing /etc/config/firewall. Am I missing something?

Thanks.

Use one of the both "Open port on router" or "New forward rule" buttons, they're basically shortcuts to create a new rule with a few predefined values - you can still fully edit them afterwards
