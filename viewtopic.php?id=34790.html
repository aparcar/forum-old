if u can telnet in safe mode try this command
        mtd -r erase rootfs_data
credits (http://wiki.openwrt.org/doc/howto/generic.failsafe)

if after reboot it doesnt work u have 2 make ftp server on your pc and transfer the firmware image on the router

       wget (url firmware.bin) /tmp
       mtd -r write /tmp/firmware.bin firmware

credits(was in 1043 help... now its is no more)

if those dont work prepare urself 2 open the box and make a serial connection

Wireless is off by default. Since you say lan and wan are working, then all seems to be ok. Can you ping your router?

Run putty, insert host IP, probably 192.168.1.1, select connection type "Telent" and click "Open".
Then follow "login with telnet" here http://wiki.openwrt.org/doc/howto/first … ith.telnet
I think you can can install luci with telnet http://wiki.openwrt.org/doc/howto/luci. … stallation
Then exit telnet and try Web GUI.
