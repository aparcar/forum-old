I'm not usual with these processors, but it seems that it's Eden ESP 533 processor.

We managed to work storage USB key. Now the next turn USB WiFi.

root@OpenWrt:~# lsusb

Bus 001 Device 003: ID 1737:0077 Linksys WUSB54GC v3 802.11g Adapter [Ralink RT2070L]
