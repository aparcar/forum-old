If you have kamikaze 8.09 rc1 running, you should have the luci web interface installed on port 80.
Since the basic fonera has only on ethernet port, you have to decide, how to use it (lan or wan). The default configuration is lan!
Using it as a router would work, but since it has only two network interfaces, and most internet connections are wired (adsl modems, ...), this will not permit wired clients in your home.
If you already have a adsl router (or equivalent), you can bridge lan & wireless, and use the fonera as a simple access point for your wireless clients.

By the way, flashing a newer fonera is very simple with the tools provided bei freifunk.net (for Windows and Linux), even if you want to use your own firmware images. No serial console cable needed, since newer versions have redboot telnet access enabled by default.
See http://download.berlin.freifunk.net/fonera/

Install and configure ntpclient to get date & time from a time server.
If you are using the fonera as a plain access piont, you can set the lan interface/bridge to dhcp and disable dnsmasq (dns & dhcp server, with the command "/etc/init.d/dnsmasq disable").
