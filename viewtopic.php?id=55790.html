I need to make a simple wired router and I thought any router with several ethernet ports would do the job but a line in the Buyer's guide is bugging me, it says "Do not confuse the number of Ports with the number of NICs.".

If I get a router with an integrated ethernet 4 ports switch, will I be able to configure OpenWRT so that port 1 is "on the internet side" while the 3 others are "on the LAN side" ? Or do I have to find a device with a separate "wan" port ?

Almost all routers have a managed switch-- which means you can set up VLAN's and have wan and lan networks share the single CPU NIC port in a multiplexed manner.  This is how the factory firmware works on those models that only have one CPU port but offer a "WAN" jack on the back.  Usually those have a 5 port switch for the common setup of one WAN and 4 LAN.
