Hi,

I think I see the problem: There are actually 2 MAC addresses in the WRT54G:

- One for eth0
- One for eth1

For WDS to work you need to use the MAC address of the eth1 (wifi) interface. This is _NOT_ the same MAC address as printed on the box. So run an 'ifconfig' to find the MAC addresses for eth0 and eth1, and use the eth1 MAC addresses for WDS. You are currently using the eth0 address.
