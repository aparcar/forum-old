The time configuration must be correct on all parties involved in the authentication process:

  • Client (computer, phone, router in client (STA) mode ... (supplicant in 802.1x terminology)

  • Access point you are connecting to (authenticator in 802.1x terminology)

  • RADIUS server (authentication server in 802.1x terminology)

All must have the correct time set and have the correct time zone set. Note that "correct time" varies between systems, Arch wiki is a good resource about time configuration in various operating systems.
