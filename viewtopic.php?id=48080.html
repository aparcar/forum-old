On the Status/Overview page,
1) DHCP Leases table - it sure would be nice to be able to sort the list by any column.
2) In the Associated Stations table - it would be nice to have the Hostname and/or IP address as well as the MAC addr.

On the Network/DHCP and DNS page
1) Sorting Active leases table would be useful
2) Active leases - being able to click on the IP addr or MAC addr any copy to clipboard would be useful. (it's difficult to select just the correct text to copy.)
3) Would be nice if the MAC list that appears when you create a new Static route included ALL the MAC's in the active leases.  Not just a random subset.
