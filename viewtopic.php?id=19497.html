#!/bin/sh

iptables -t nat -F prerouting_wan

### Open port to WAN
# Decomentada por sergio

### This prevent port 80 to be answered by lan IP

### This force transparent proxy except UP traffic

### This prevent port 80 to be acced from WAN
iptables -A INPUT -i eth0.1 -p tcp --dport 80 -j DROP

If all you want is a transparent proxy, why not use tinyproxy?  tinyproxy is small enough that you can fit it in the router. 

If you need other proxy features such as caching, etc., by all means, use squid.
