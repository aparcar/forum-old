Post #9
AraKs
28 Nov 2009, 20:16

I understand...

Your router should run @200Mhz. dd-wrt wiki says that WRT54GS v1.1 works stable up to 250Mhz, without hardware mods:

http://www.dd-wrt.com/wiki/index.php/Linksys_WRT54G/GL/GS/GX#Clock_Speed_and_Overclocking wrote:

WRT54GL v1.1 is stable when overclocked at 250MHz, with no extra cooling. Your mileage may vary.

To modify the CPU clock on this broadcom unit you can use the nvram command to set the clkfreq bootloader variable to the desired value. You can't put random values in there. You should check wich frequences BCM4712 supports.
Remember that it's very dangerous, you can damage the unit or prevent it from booting.
You might be able to change the clock frequency from the web interface, but i'm not really sure.

Give it a try, and let us know if OpenVPN performances increase!
Mattia.

For decent VPN performances, you need at least 600Mhz MIPS CPU. WRT54GS is very old.
If you want to easily recognise high clocked routers, look for the one with gigabit ethernet, wireless N and USB ports for HDD sharing or network printing!

PS:
WiP means Work in Progress
