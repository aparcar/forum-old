Hi!

I built OpenWrt for this router. It has the following features working/tested:

- Built from Attitude Adjustment (trunk)
- 4 switched LAN ports
- Wireless
- WPA2+AES (Preshared key)
- Outgoing NAT
- DHCP server on LAN
- DHCP client on WAN
- DNS relay

The following features are not working/are untested:

- The Wireless LED never lights up
- QoS built but untested
- UPnP built but untested
- It has no web management interface (LuCi failed to build)
- IPv6

It might have something else that I have forgotten to mention (this is my first OpenWrt build, and I have only tested it with my router). If you are interested (these routers are not that bad, but the original 3Com/HP firmware is crap) give my build a shot:

http://gapf.ninth.su/openwrt-atheros-root.squashfs
http://gapf.ninth.su/openwrt-atheros-vmlinux.lzma

Regards and please post your comments/findings in this thread.
