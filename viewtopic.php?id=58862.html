I just upgraded to Chaos Calmer from an earlier OpenWRT version. Now I'm no longer able to establish a PPTP connection to a remote VPN from a VPN client on my lan.

Using tcpdump, it seems like the GRE packets from wan is not being let through to my lan computer.

I did:

  iptables -I INPUT -p gre -j ACCEPT

but this does not help.

I see that people have mentioned that the packages kmod-gre and kmod-nf-nathelper-extra should be installed, but I don't find them in Chaos Calmer.

What should I do?

Edit:

I managed to install the packages. PPTP now works again, and it turns out that the iptables rule above is unnecessary.  Installing the packages was a bit tricky, though:

I had previously done an update of the opkg package files through luci. However, when I did:

  opkg update

on the command line, I suddenly could install kmod-gre.

To install kmod-nf-nathelper-extra, I had to manually download that package and kmod-lib-textsearch from:

  https://downloads.openwrt.org/chaos_cal â€¦ ages/base/

I guess something is weird with the package lists...
