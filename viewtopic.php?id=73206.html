Hi. Yes I was able to recover with fail safe mode

It turned out I had bridged the ethernet adapter and set a router IP on a different subnet that I didn't recall, thus making it nearly impossible to log in.

First I was able to access via Telnet after sniffing out the router subnet and guessing the IP. High excitement to see the CLI interface come up (meaning no brick)!

From there I factory reset the router, and after that I used fail-safe mode too for a second soft brick that I achieved.

A good learning experience.

Happy to help DonJuane if you need it still.
