I'm a complete router and linux newb, so bear with me.  I recently bought a Netgear WNDR3700v2 and flashed it with OpenWRT.  Everything seems to be working really well so far, so I decided to try and add OpenVPN and use a VPN for my home network.  The VPN company I decided to go with (HideMyAss) has an auto-installer for OpenVPN on OpenWRT that both installs the OpenVPN software and configures it correctly for their servers.  The one flaw in the plan, they only included screenshots with a DD-WRT router.  I figured that, in spite of my newbishness, I could probably follow along and figure out what to do in Luci.  Wrong.

I got stumped at one of the first steps.  I'm supposed to paste some code in the "Commands" section of the DD-WRT interface, which appears to let you run a command line through the web interface.  I searched and searched in OpenWRT, but I didn't see anything similar.

So, is it possible to run commands through the web interface?  If so, how?

If it matters, the code begins with: "wget -O /tmp/inst.sh 'http://vpn.hidemyass.com/".

There is no direct way to do that - an indirect hack to execute arbritary commands is the following:
Navigate to the firewall configuration; open the custom rules tab. Paste your shell commands in the window and hit Save & Apply. Afterwards remove the commands and hit Save.
