Post #4
jow
27 Apr 2012, 15:41

The "-d" option in "touch" is protected by "CONFIG_DESKTOP" in busybox.

make menuconfig -> Base System -> busybox -> Busybox Settings -> General Configuration -> [*] Enable options for full-blown desktop systems
