There is wget as a standalone (and complete) program, and there is wget as a busybox command (and reduced). Use "which wget" to determine which executable you are using in each machine; I bet in one of them "wget" is just a symlink to busybox.

Normally, when you install the full wget package, the complete binary replaces the symbolic link; I guess there was something wrong with your installation.
