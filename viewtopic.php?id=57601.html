i mean, arp is alias for cat /proc/net/arp

try run /sbin/arp -h

there is in /etc/profile those magic line


[ -x /usr/bin/arp ] || arp() { cat /proc/net/arp; }

but arp is in /sbin dir

or you can install package net-tools-arp from AA repository. There is no more in BB
