I am using backfire trunk r27943, and after trying to make extroot work but i'm getting a very strange behaviour.

See this.

1) Boot using internal rom
2) mount /dev/sda1 /mnt
3) cp /rom/* /mnt -Ra  (and cp /rom/lib/modules /mnt/lib/modules.BAK -Ra [i will need this to fix the problem])
4) cp /overlay/* /mnt -Ra
5) edit /etc/config/fstab to enable extroot
6) find /lib/modules | grep sfq -> /lib/modules/2.6.39.2/sch_esfq.ko
6) umount /mnt
6) reboot
