What nvram? Openwrt stores settings as files in normal flash.

Go to Openwrt failsafe mode and reset the file system from there.
http://wiki.openwrt.org/doc/howto/generic.failsafe
(see your router's wiki page & forum, if you find more exact info about the moment when to push a button during the boot.)
