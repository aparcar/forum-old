I got this modem working on a laptop running Ubuntu.  There is a long pdf telling you how to do this at Sprint:

http://www4.sprint.com/pcsbusiness/down â€¦ _Guide.pdf

The basic things you need are:

1) The usbserial module loaded;

2) Some sort of dialing software; I was able to do it first with a simple chat script and pppd; then I used KPPP on Ubuntu

3) The user and password are not checked, the dialing number is #777

4) The modem I was using had already been activated; if this is not true, you have to activate it, sigh, under Winders.

My feeling is that it wouldn't be too hard to get this working with a router, but I haven't actually done it.

Hope this helps.
