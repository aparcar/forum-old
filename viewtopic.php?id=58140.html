You should make sure you set the DDNS script to logging and check the log in /var/log/ddns/$name_of_service.

The log is pretty verbose, and very helpful.

Post #3

DDNS scripts now use SSL everywhere, specially in no-ip.com. You need more packages:

CONFIG_PACKAGE_ca-certificates=y
CONFIG_PACKAGE_curl=y
CONFIG_PACKAGE_ddns-scripts_no-ip_com=y

Still the DNS propagation takes 5 minutes, keep calm.
