Inside every router I made for my relatives, I created 2 wlan, both as 'ap':

config wifi-iface
        option device 'radio0'
        option network 'lan'
        option mode 'ap'
        option ssid 'WiFi-for-GrownUps'

config wifi-iface
        option device 'radio0'
        option network 'lan'
        option mode 'ap'
        option ssid 'WiFi-for-Kids'

My problem is: When I, or one of the relatives, try to join a wifi network using LuCI function (Scan -> Join Network), all above 'AP' get removed and replaced with a 'Client' network. And that render their routers Inaccessible.

I don't get it? Why is it necessary to remove all previous wlans before joining a wlan network.

Shouldn't OpenWrt just add 'Client' network using wlan[-1] instead of removing all wlans and creating a new one?
