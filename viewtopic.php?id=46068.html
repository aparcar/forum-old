I'd like to know if it's possible to have one physical interface split in two virtual, by using a standard 8-wire twisted-pair cable, and assigning 4 wires to each interface. In other words, I'd like to connect two end devices to a router having one physical interface, with each device utilizing only 4 wires of an 8-wire cable. Obviously the cable should be split somewhere on the other end, with two 8p8c connectors going to both end devices.

Now I know I could just add a cheap switch or just use a regular router/switch/ap combo in one device like any sane person would do.

The reason I'm asking is this: I've seen many tiny, USB-powered routers with a wireless interface, but only one physical interface. I'd like to connect two end devices (like TV and STB, both having no wireless interfaces) to such router, without any additional devices connected to mains.

Now I know that splitting an interface in such fashion sounds bizarre:) I'm curious if anybody has attempted it (or maybe it's a regular OpenWRT's feature that I have somehow missed)?
