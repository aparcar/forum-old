there are two ways you can do this:

/etc/init.d/network restart

and

ifdown wan
ifup wan

that's all the argument is either wan or lan. For wifi you have to use:

wifi down
wifi up

difference between ifconfig devicename down and ifdown / their respective "up" commands is that ifconfig simply works on the raw interface, while ifup/ifdown are scripts that set additional parameters, like dialing up when talking of the wan interface or setting the correct ip when lan/wifi is the target. if you simply want to get a new ip, use ifdown wan, then wait for a couple seconds and the use ifup wan. that should do the trick
