put in /etc/modules.d/60-usb-serial 

usbserial vendor=0x05c6 product=0x6000

and restart router.

If this not help, try load this device by load it to option driver 

echo '05c6 6000' > /sys/bus/usb-serial/drivers/option1/new_id
