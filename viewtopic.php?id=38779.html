Post #6
written_direcon
21 Aug 2012, 18:18

Your OpenWrt router and the main router are connected over LAN ports on both routers, right?

You have to make sure, that you have only one DHCP server running in your network. Connect to the OpenWrt router via a LAN port and disconnect it from the main router. If you have one on your main router what I guess, then you have to disable the DHCP server on your OpenWrt router with:

/etc/init.d/dhsnasq stop
/etc/init.d/dnsmasq disable

You can also disable the firewall on the OpenWrt router:

/etc/init.d/firewall stop
/etc/init.d/firewall disable

Then on the OpenWrt router set the main router as gateway, DHCP and DNS server:

uci set network.lan.dns=192.168.178.1
uci set network.lan.gateway=192.168.178.1
uci commit network
ifup lan

Then this part should work :-)
