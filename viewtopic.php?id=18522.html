eth0.0 -wan
eth0.1 - lan1

The physical port number to the switch port number mapping is device dependent.

in /etc/config/network:

config switch eth0
        option vlan0    "0 5*"
        option vlan1    "1 5"
        option vlan2    "2 5"
        option vlan3    "3 5"
        option vlan4    "4 5"
