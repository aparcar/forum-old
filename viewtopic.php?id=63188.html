The default configuration of CC15.05 refuses to get an ip address from my modem.   After poking around various forums I have uncovered this as a problem in Openwrt and DDWRT.   I found this solution here in the forums:

1. Disconnect wan cable from router
2. Run: ifconfig eth1 down
3. Run: ethtool -s eth1 speed 100 duplex full
Requires that the package is installed, I used scp to copy the package to the router
4. Connect wan cable
5. Run: ifconfig eth1 up
6. Profit! smile

Which works great, but requires that I manually unplug the cable and then replug it in.  This won't cut it during a power outage.    Can anyone suggest an alternative?   Maybe there is a setting somewhere I can tweak that will force 100Mbps on eth1?

Credit for the fix above goes to enrique, in this thread:
https://forum.openwrt.org/viewtopic.php?id=38164

SORT OF SOLUTION:
Paste this command into the startup command list:
