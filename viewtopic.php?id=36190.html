By default the serial port is occupied by the console. You canâ€™t use the port for other uses then.

Look in /etc/inittab. The last line send the console to serial. Remove that line

root@plink:~# cat /etc/inittab 
::sysinit:/etc/init.d/rcS S boot
::shutdown:/etc/init.d/rcS K shutdown
ttyS0::askfirst:/bin/ash --login
