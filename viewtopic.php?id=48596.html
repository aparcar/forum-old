How to search for the source code of an OpenWrt package:

1. Find where the package is defined

 grep -R "Package/wpad" package/

Result: package/network/services/hostapd/Makefile

2. Look inside the found file

PKG_SOURCE_URL:=git://w1.fi/srv/git/hostap.git

where to download source code

Other information taken from the Makefile:
wpad is a hostapd build variant.
There are different build configs of hostapd for some variants.

Hope it helps
