It used to bei this call:

wifi detect > /etc/config/wireless

Currently it should be this one (without directing the output to a given file):

wifi config

But of course you need a wifi adapter connected to your VM, which I suspect is a USB? Are your sure your USB wifi adapter is compatible?

Since an x86 machine may host any sort of wifi hardware, or none at all, the x86 builds don't include any wifi drivers in the initial image.   You need to identify the chipset in your wireless adapter and install the appropriate driver.
