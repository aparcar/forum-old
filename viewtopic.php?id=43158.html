nebbia88
9 Apr 2013, 13:57

as with mr11u v1 and v2, i only changed the filename and header. it's really the same bin you get for v1.
i DO NOT KNOW if it works on 3040 v2, i only know that it can be flashed from 3040v2 tplink gui, while bin for v1 cannot!
the model name in luci simply read the firmware header so it cannot know that real model was wr703n...

serial console is needed only in case of brick (that is not impossible to happen with untested firmware!)..

I understand your frustration mate lol

The best way to download stuff from openwrt on new install, is to first creat a 'client' i.e. under wifi, scan and select a router that you have access to and then once that set up, dont do anything else, as in creating a 'master' etc, simply connect your laptop or pc to your mr3040 via ethernet, this way at this stage you should  then have internet access from the router you have connected. You can now download package via the software tab with openwrt using LuCI.

Once you have downloaded the packages you need, you can then go on and configure openwrt further via LuCI.

Hope this helps.
