Here is another help which you may already found in the Sveasoft's Forums. They use this config with Alchemy:


defaultroute
noipdefault
nomppe
lock
noauth
name (username here)
debug

Then in /tmp/ppp/pap-secrets:
username * password

I also copied that file to chap-secrets in the same dir.

Then type pptp 10.0.0.138, and let it connect.

Now:
route add 10.0.0.138 vlan1

Now you have Internet. To share it:
iptables -t nat -A POSTROUTING -o ppp0 -j MASQUERADE

After disconnect:
route del default
route add default ppp0
