Ok i made a new check and to go directly from DD-WRT to OpenWrt for  TP-Link WR741ND V1 is possible.

Assumption you have a local webserver or Ftp server:
-My PC was directly wired to one of the lan port of the device without internet connexion (my ip was 192.168.1.129)
-I place the backfire squasfs FW in my web directory and rename it as 1.bin in order to not make a mismach typing name.
-I load a Telnet/ SSH session to the router .
then type the following:

cd /tmp ; wget http://192.168.1.29/1.bin ; mtd -r write 1.bin linux ;

Now i'm able to run Coova chilli on it !...Very cheap 802.11n hotspot device !
