Note ar9331's usb stability issue: https://forum.openwrt.org/viewtopic.php?id=39956

You need a "high speed" usb hub (as reported by dmesg) in order to use a usb/serial dongle with the WR703N unless you have not enabled wifi.  Otherwise there will be random loss of usb connectivity.

With the hub, pl2303, ch341, and cp210x usb serial dongles should work, and probably ftdi if you can find them.
