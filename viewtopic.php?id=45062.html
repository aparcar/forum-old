The wan interface needs to be
   set to dhcp client
   NOT a bridge
   connected ONLY to your wifi client interface that will connect to the AP.  Move any ethernets to the lan side.
The lan needs to be a different IP subent than the one the AP will give you.

You need to see a wan IP on the main status page before you can access any web pages on their login server.

If all that is OK, you should be able to wget or curl the login page.  Since they are using https you need to install the full version of wget or curl.  The little built in Busybox wget does not handle secure connections.
