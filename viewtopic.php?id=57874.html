It's using https and there's no CA on the router.  You can run the following command to disable ssl certificate check.  Also, make sure curl is installed.

sed -i 's/\(retrieve_prog="\/usr\/bin\/curl \)"/\1-k "/' /usr/lib/ddns/dynamic_dns_updater.sh
