1.  In the network page, interfaces, lan.
2.  All incoming connections from the WAN are blocked by default. 
You could block access to the web page by wireless by making two LANs (e.g. a guest network), but then your wired and wireless devices wouldn't be in the same network for sharing printers etc.
You can also disable the web page entirely and administer by ssh.  If you use public-private keys on ssh it is almost unhackable.  Only your machines that have a copy of the private key could get in.
3.  Check the wiki.
