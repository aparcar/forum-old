The development branch (codename: BARRIER BREAKER) does not come with a preinstalled web interface, the idea being that if you're comfortable installing an unstable version you should also be ok with using the cmdline interface.

You need to add the LUCI web interface if you're using BB:
http://wiki.openwrt.org/doc/howto/luci.essentials

AA 12.09 has wifi troubles on 1043, for sure. I installed  BB r38226 and it works fine.
