What version are you using?

There is currently, and has been for some time,  a bug in the current "bleeding edge" that prevents the ethernet interface on the 524t from working!!!
https://dev.openwrt.org/ticket/3124

If you plan to use the dsl-524t as a modem/router with OpenWrt I suggest that you compile and use the r9234 version, it's still the most stable and complete version of openwrt for the dsl-524t. I'm running different versions of OpenWrt on dsl-524t and it's dsl-g624t and as long as I get the partion-values right I have had no problem installing OpenWrt using ftp (exept for the current ethernet bug that is).

I really recommend that you get  a serial interface so you can use the console on the 524 though, that will show you any and all installation or boot problems right away.
