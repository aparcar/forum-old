Is there any chance you did not run

# opkg update

before installing kmod-nf-nathelper-extra?

To see that it is there:

# opkg list | grep kmod-nf-nathelper-extra
kmod-nf-nathelper-extra - 3.18.20-1 - Extra Netfilter (IPv4) Conntrack and NAT helpers Includes: - amanda - h323 - mms - pptp - proto_gre - sip - snmp_basic - broadcast

To install it

# opkg install kmod-nf-nathelper-extra

If you use the Web Gui (Luci), there is a button to update package list before you can install anything.
