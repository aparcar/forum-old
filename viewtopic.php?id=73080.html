I am a well versed Linux user, looking for a way to sneakernet some packages onto my raspberry pi openwrt. I have my openwrt up and running under a fixed IP.

The problem I have with installing packages onto openwrt the normal way is because openwrt does not come with USB wifi drivers, and the only connection I have to the Internet is via wifi. I do not have access to Ethernet based Internet.

So my idea is to download the packages to my Linux computer (via wifi) then copy them over to the SD card.

You could simply mount the ext4 root filesystem on the card on your computer and copy ipk files to a directory e.g. /root, then boot the card in the Pi and run opkg to install them.

Or use the Image Builder.  This makes a custom image with the packages already installed and usable at boot up.
