After some struggle, I got it to work with the phidget-service and connecting directly to the ip and port using C# sample application from a remote PC.

But I had to downgrade to a older version of the Windows Driver.
Both 12.09 and 14.07 uses an older version (1.0.3) of the communication protocol, so you need Phidget Windows Driver from 2_1_6_20100126 to get anything to work.

I am about to download the buildroot and what version of Phidgets library it contains, so maybe I can compile a dist that uses a newer protocol.
