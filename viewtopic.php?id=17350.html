I don't use x-wrt but for flashing from command line I think you need the raw trx image and not the .bin images. Which are you using?
Also, TFTP only works for a few seconds during bootup on most devices, it can be a PITA to get it to work, the timing needs to be dead on. I think most routers want the .bin type firmware images here in order for this to work.
If you can't get access via SSH try using telnet. Openwrt needs you to access it once via telnet and set a password to get ssh running.
Those are just the obvious suggestions, sorry if you already tried these.
