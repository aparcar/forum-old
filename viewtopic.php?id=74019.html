The AR9341 chip has two eth ports.  So you need to software bridge them not hardware switch.

In the GUI, go to network--interfaces and delete the WAN network.  Then edit the LAN--physical settings and click the check boxes for both eth0 and eth1 in the lan.

The LED settings only controls how the LEDs blink, it does not affect any network functions.
