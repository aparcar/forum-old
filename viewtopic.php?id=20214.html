With OpenWRT firmware for RouterStation (r15572 and older), from time to time, I get these error messages:

SQUASHFS error: zlib_fs returned unexpected result 0xfffffffd
SQUASHFS error: Unable to read page, block 5b6lf7, size 80f2
SQUASHFS error: zlib_fs returned unexpected result 0xfffffffd
SQUASHFS error: Unable to read page, block 5b6lf7, size 80f2
SQUASHFS error: zlib_fs returned unexpected result 0xfffffffd
SQUASHFS error: Unable to read page, block 5b6lf7, size 80f2

after this message some binaries in squashfs stop working (dropbear, sometimes vim, etc ...). After a couple of restarts, error has gone away, and all was working OK, but after the time (hours, days) error appears again, random files are broken.   I'm think that is hardware (flash) problem, but I see this error on more RouterStation boards.  Maybe SquashFS bug ?

Tested firmware:
r15572
r13658 (original)
