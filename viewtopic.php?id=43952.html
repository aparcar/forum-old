  • your init script must be able to run on the host system (means put no target specific actions outside of the start() or stop() procedures)

  • the environment might be uninitialized during boot, so you might need to source /etc/profile or set GNUPGHOME yourself

  • the network is not guaranteed to be initialized at init time, so consider a hotplug action instead
