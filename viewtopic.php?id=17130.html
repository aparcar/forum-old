Hi.

1) You can consider the symlink method deprecated. Use the following commands instead:

./scripts/feeds update
./scripts/feeds install -a

This will "symlink" all available packages into your buildroot

2) The "ipkg" package manager was replaced by "opkg". It has the same syntax as former ipkg.

~ JoW
