So, I wanted to install OpenWrt or DD-WRT on my TP-Link TL-WA901ND 300 mbps access point, but unfortunately I ran into the famous error 18005:

Error code: 18005 Upgrade unsuccessfully because the version of the upgraded file was incorrect. Please check the file name.

After a lot of search, I realized it had something to do with new US regulations for routers, thus they would not allow other firmware to be installed. So, I found the factory-to-ddwrt-US.bin which is recommended, but when I used it it gave me a new error 18001: 

Error code: 18001 Upgrade unsuccessfully because the length of the upgraded file is incorrect. Please check the file name.

Then I stumbled upon a thread http://www.dd-wrt.com/phpBB2/viewtopic. … 65#1033565 which had a link to the ftp site of beta releases of ddwrt ftp://ftp.dd-wrt.com/betas/2016/, and I picked the latest beta release for my router ftp://ftp.dd-wrt.com/betas/2016/07-01-2 … wa901ndv3/ and it worked perfectly!

Just thought to share this as not a lot of info is available for this on the internet I believe similar TP Link routers can be flashed with the latest beta releases.
