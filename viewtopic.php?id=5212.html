I've read ( http://forum.openwrt.org/viewtopic.php?id=4832 ) that using OpenVPN cuts throughput down from about 600 Kbytes/sec to 300 KBytes/sec.  Vincentfix has kindly pointed out that the WRT platform doesn't have any special encryption hardware, but tosuja mentions that WPA2 traffic can run at up to 12 Mbps ~= 120 Kbytes/sec.

So I'm wondering if there are any stats on what the Linksys WRT4GS platform can do in terms of throughput in the various modes.  The ones I can think of are:

wired / no security
wired / openVPN
wireless / 54 / no security
wireless / 54 / openVPN
wireless / 54 / WEP64
wireless / 54 / WEP128
wireless / 54 / WPA
wireless / 54 / WPA2
wireless / 54 speedbooster / (etc)

A long time ago I read that speedbooster isn't supported in the OpenWrt world, yet the wl program seems to support it.  Will anyone describe the current situation?
