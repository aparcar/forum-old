In the LAN configuration, scroll down and make sure "ignore interface / disable DHCP" is checked.  Physical settings should have both the ethernet and the wifi in the lan.  You can delete any wan interface.

The addresses you set only affect the router itself accessing the Internet.  Whatever network you plug into will transfer to wireless users without needing to reconfigure the router's address.
