I have Comcast w/ speedtierextreme. Some wireless devices can get full throughput, and others cannot. All devices are roughly the same distance from the router (usually no more then one room apart). Here are the devices that get full throughput (89Mbps/12Mbps):

  • Two Raspberry PI's (Wired)

  • MI Android Tablet (Chinese)

  • Sony Vaio Labtop


Devices that cannot reach full speed:

  • Samsung Galaxy S3 (21Mbps/12)

  • Ipad (21Mbps/12)

  • Iphone 6 (21Mbps/12)


So I'm kind of perplexed why some devices can, and some cannot. All devices are connected on the 5Ghz band. I need wireless config examples, particularly from folks that have a mix of Apple and Android devices that can reach full throughput.


My wireless config:

config wifi-device  radio0
        option type     mac80211
        option channel  10
        option hwmode   11g
        option path     'platform/ar934x_wmac'
        option htmode   HT20
        option 'noscan' '1'
       # REMOVE THIS LINE TO ENABLE WIFI:
       # option disabled 1

config wifi-iface
        option device 'radio0'
        option network 'lan'
        option mode 'ap'
        option ssid 'sr_guy-2.4Ghz'
        option key 'PW'
        option encryption 'psk2'

config wifi-device  radio1
        option type     mac80211
        option channel  161
        option hwmode   11a
        option path     'pci0000:00/0000:00:00.0'
        option 'noscan' '1'
        option htmode   HT40

config wifi-iface
        option device 'radio1'
        option network 'lan'
        option mode 'ap'
        option ssid 'sr_guy-5Ghz'
        option key 'PW'
        option encryption 'psk2'
