Found open project - A5V11 power / USB / RS232 shield board: https://github.com/rororor/a5v11-mod-01
Contains KiCAD sources and Gerber files.

Uses Fe1.1s USB hub chip, 3 ports are free, 1 port used for USB-RS232 (CH340G).
5V generated by cheap MP1584/MP2307 DC-DC module.
