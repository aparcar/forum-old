This has been covered in the forum before.  It's a bug in the diag module code that's fixed in CVS. 

You can check out the new diag.c and build your own version off of Rodent's buildroot, try FreiFunk's build, or mbm's experimental build.  Any of these work on the newer units, and have the diag bug fixed.
