If you're moving between different hardware its not that simple as backup and restore.

Look at config from my router :

config wifi-device 'radio0'
        option type 'mac80211'
        option channel '11'
        option hwmode '11g'
>>>        option path 'pci0000:00/0000:00:11.0'
        option country '00'
        option txpower '19'
        option htmode 'HT20'
        option beacon_int '256'

config wifi-device 'radio1'
        option type 'mac80211'
        option hwmode '11a'
>>>        option path 'pci0000:00/0000:00:12.0'
        option htmode 'HT40'
        option disabled '0'
        option country '00'
        option channel '44'
        option txpower '17'
        option beacon_int '128'

PCI numbers are there. They can be different on another harware.
