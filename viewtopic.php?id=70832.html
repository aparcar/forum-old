SCP not FTP.  SCP server is built in to dropbear SSH server running on the router, you don't need to install anything.  You do need a password set so the router will accept SSH logins.  In other words if you can SSH you can also initiate a SCP from your PC and "push" the firmware file to the /tmp directory on the router.  It is important to specify /tmp, which is in RAM, because if you try to put it in a flash portion of the filesystem obviously there is not 4 MB of space.
