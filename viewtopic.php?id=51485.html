A common problem, really. Both just won't fit into a 4 MB flash device, there's no "trick" to install them both so they will fit.

There's two solutions:
a) use extroot to extend your router's memory with an external device
b) build or compile a custom image that contains both LuCI and OpenVPN -- when built into an image, packages get compressed and will fit
