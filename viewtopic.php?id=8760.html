Although it might not be needed as you plan on using an USB connection to connect to the internet I would advice you to set some variables in nvram to get the device working as planned. This to avoid problems due to configuration differences in the WL500g Premium.

From the wiki:

nvram set vlan1ports="0 5"
nvram set wan_ifname=vlan1
nvram set lan_ifnames="vlan0 eth2"
nvram set lan_ifname=br0
nvram commit
