I face the same problem with a HP Photosmart C410. It seems to be absolutely impossible to connect the device via Wifi, the stupid printer always says that the MAC adress filter is enabled. A lot of people seem to have problems with those HP printers and Wifi, especially in combination with Verizon routers.
In a posting a HP support guy mentioned that the "MAC filter enabled" error is the very last error the printer shows. So if the printer has no clue what the error is it shows the "MAC filter enabled" error what caused a lot of confusion, at least for me ..... like " else { print('MAC filter enabled')".

Unfortunately I did not find a solution. Neither changing encryption, setting an easier WPA key, playing around with Wifi bands, .... worked. On my old WRT54gl running stock Linksys firmware it worked without any problems.

I would also be extremely thankful if anyone finds a solution.
