I try to

# umount /dev/mtdblock3
# mount -t tmpfs tmpfs /overlay

But then I find new file in / still be wroten into /dev/mtdblock3( rootfs_data block, even it had been umounted! ).

I also tried to edit /etc/config/fstab and mount /overlay as tmpfs but after reboot , /dev/mtdblock3 is still on /overlay.(But it  works when mount a usb block to /overlay in an other router )

It seems overlay will be in /tmp/overlay when failsafe. But how to do it( or similar thing, use tmpfs as overlay ) in a normal state?
