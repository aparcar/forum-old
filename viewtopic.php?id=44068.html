Go to http://downloads.openwrt.org/

Choose Attitude Adjustment

Choose 12.09

Choose brcm47xx

Choose generic

Choose openwrt-brcm47xx-squashfs.trx

That's how you navigate the openWrt downloads tree to find the file you want for the release and architecture you need.
