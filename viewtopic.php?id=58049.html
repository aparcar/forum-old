Ok, so.
It fails because you try to build gcc. If you don't need to build gcc, just unselect gcc from your config (make menuconfig) or build with IGNORE_ERRORS=1 make <make options>.

If you need gcc, you need to replace the config.sub and config.guess from some dirs in the gcc source with the config.{sub,guess} that are at the root of the gcc build dir.

If you need more specific help I can give you more specific pointers, but with those instruction you should work it out.
