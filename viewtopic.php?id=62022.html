Hello,

not sure if you already made it through but it's pretty easy:

Install required packages:

opkg update
opkg install curl python ffmpeg

(additional generic might be needed - I'm too lazy for end-to-end testing :-))

Download youtube-dl as per generic instructions (http://rg3.github.io/youtube-dl/download.html):

curl https://yt-dl.org/downloads/2016.04.06/youtube-dl -o /usr/bin/youtube-dl
chmod 755 /usr/bin/youtube-dl

Just make sure you have at least 64 MB of RAM and/or swap on relatively fast device and enough time for waiting as the downloaded tracks are transcoded with ffmpeg which is obviously both CPU and RAM intensive.
