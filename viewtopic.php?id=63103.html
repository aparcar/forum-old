The final 15.05 images come with firmware 10.2.4.45 and can be upgraded to 10.2.4.48 manually (I did that as well). However, as I stated here as well, a few days ago, new ath10k code was added to the 15.05 branch which now comes with firmware-5.bin_10.2.4.97-1.

I was a bit skeptical by just looking at the git changes. But on the weekend I compiled a new snapshot and can confirm, Chaos Calmer now actually loads firmware-5.bin (10.2.4.97-1) on an Archer C7 v2 while the kernel is still 3.18 (now at 3.18.27).

In order to use the newer firmware you'd either have to compile your own image or grab a recent one from the Community Releases section (or wait for 15.05.1 if that effort is going to be picked up again).
