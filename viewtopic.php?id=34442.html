As far as I remember, Juniiper SA series provides only HTTPS access. The communication between the client side (ActiveX or Java plugins) and the device is somehow proprietary HTTPS tunneled data.

Maybe someone can correct me if I'm wrong but those appliances support neither IPsec nor OpenVPN. Therefore no chance to build connections from OpenWrt. Usually in front of an SSL gateway you will find the "real" VPN site to site concentrator. If you really need the connection this should be your point of interest you want to connect to.
