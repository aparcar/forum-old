C7 is definitely 2.0, but not sure about C5. C5 1.0 works fine but not sure about any others.

That said, if you are trying to use QoS/SQM with shaping on a 200mbps internet connection, you will still struggle with the CPU in all of these. At this time, you'd probably need an x86 box to handle those speeds with traffic shaping/QoS/SQM. Without those features turned on, you *might* be able to get 200mbps out of an Archer C7.
