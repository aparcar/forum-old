I've got a wrt54 v4.0 that wouldn't boot... though after a long long while the DMZ light would turn on. To fix it I had to set my computer to a static IP and use another switch to connect to the router with a crossover cable. The computer was set to 192.168.1.2, and command prompt was open with tftp -i 192.168.1.1 PUT openwrt-wrt54g-squashfs.bin typed out. I unplugged the router, plugged it back in, waited about a second until the power light started flashing and then hit enter in the command prompt window. It worked for me.

Good luck!
