Post #4
Zajec
30 Dec 2013, 13:22

Colors are usually used to show user default roles of the ports. In most cases by default you have one port for WAN and rest for LANs. WIth OpenWrt you can re-arrange it the way you want/need.
For the database of routers you can see http://wiki.openwrt.org/toh/start . It's not perfect, but you should be able to find some router using it.
