Hey there.

You added eth0 to a bridge called "lan".  This means the interface you assigned the static IP address 192.168.1.1 to is not "eth0" but "br-lan".
So use "ifconfig br-lan" instead of "ifconfig eth0" and you'll see your IP address in place.

Do you have an actual problem with your configuration or did you just ask out of curiosity?
