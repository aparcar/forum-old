drawz
26 Nov 2014, 00:56

Please let us know exactly what files you are flashing (links please, not filenames).
If you can revert to stock, reflash with Barrier Breaker 14.07 final and use a "factory" file for your router. Make sure you have the correct hardware revision (v1 or v2).

If you are trying to flash from openwrt trunk to stable, then use a sysupgrade file and do not save settings. This assumes you used a factory file the first time around. It is ok to use a factory file again if you're not sure.

My point was that it's always going to say 'LuCI Trunk'.  The important part is that if it also has '14.07', you are running release Barrier Breaker.

Re-flashing like you did erases everything OpenWrt changed in your flash and replaces it completely with the new version.  Only the factory bootloader and default settings are left, and you don't want to erase those.  Those areas of flash were programmed at the factory and OpenWrt doesn't change them.
