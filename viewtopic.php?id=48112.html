Probably the flashing went ok, only the trunk release of openwrt doesn't come with a webgui.
You have to use cli instead: wiki.openwrt.org/doc/howto/user.beginner.cli#network

If you want to install the webgui luci, make sure the router has internet and issue the following commands:

opkg update
opkg install luci
