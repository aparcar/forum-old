I've been testing the Sparclan WUSB-505 V2 WiFi USB stick as a wireless access point with my RouterStation Pro and it's working great. The connection is very fast and very stable--faster than any other Atheros or Ralink based USB stick I've tried and comparable to the mini-pci 11n cards in my RS Pro. It uses the Ralink 28xx USB driver included in Openwrt. It is a 2 transmitter/2 receiver radio. The one issue is that 11a (5GHz) is still unstable when used as an access point, but 11n is rock solid. The oxford tec website shows it in bare metal but mine came with a normal white plastic case.

You could plug several of these USB wifi sticks into a powered USB hub and run multiple USB based wireless access points.

Here's my router's lsusb output:

Bus 001 Device 005: ID 148f:3572 Ralink Technology, Corp. RT3572 Wireless Adapter

RT3572 is an RT3072 with 5GHz dual band added (5GHz is flaky on OpenWrt). It's basically a combo RT2770 11gn & RT2750 11an radio.

I bought mine from oxfordtec for $43 + shipping here:

It's listed as an "integrators only" part but they will sell it to individuals.

The only thing required is to select Kernel Modules/Networking/RT28xxUSB in make menuconfig. It will then show up in your wireless config file just like any other wifi radio.
