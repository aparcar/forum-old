mk24
5 Feb 2017, 21:57

To follow those instructions, you will need to ssh to the router and execute the install from the command line.

Use scp to copy the firmware file to the router's /tmp directory
then cp to copy and rename it to the /lib/firmware... directory.

I use LEDE with the mainline firmware and ath10k, not Candela.  These can be installed directly.

If the radio is created use iw list to see its capabilities, particularly which channels are allowed,  Set a channel from that list.

dmesg will usually show an error if the wifi can't be started.   Note you will normally see lots of errors about loading firmware.  It tries several file names until finding one that exists (usually 'firmware2.bin').
