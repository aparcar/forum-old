Post #2
neryba
9 Jun 2015, 21:50

typically Sierra Wireless in QMI mode creates /dev/cdc-wdm0 interface. try  set this protocol.(uqmi -d /dev/cdc-wdm0 --get-sig)
or you can use gcom to get signal info. (gcom sig -d /dev/ttyUSB9) or try another ttyUSB interface.
