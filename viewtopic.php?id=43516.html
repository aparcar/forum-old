Guest network:
VLAN 7
Guest wireless network (no access to the lan ports or other VLAN.
IP-address: 10.0.8.2/24 (it should not be possible to acccess the wireless AP from this network)
Default gateway, DNS and DHCP server: 10.0.8.1

Client network:
VLAN 4

Wireless network should use this VLAN as well.
IP-address for the interface: 10.0.1.2/24 (should also be able to access the wireless AP from here)
