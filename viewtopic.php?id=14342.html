I have used OpenWrt (but not heavily) on some AR7s (DG834v2 is an AR7) and I have in general not had a problem, except for wireless.  On older versions of trunk I have got wireless to work, but current versions hang the router as soon as the module is inserted.

I have not had a problem with ADSL on the router, but as I said I have not used it heavily.

If you want to use wireless, though, I would not suggest an AR7, since the acx wireless driver is not stable yet.  (Of course some people seem to get it to work
