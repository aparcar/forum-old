Not showing up in luci is usually the result of misconfiguration. You could try moving it aside and generating a new one to see if you can get a correct one. Look in /rom/etc/uci-defaults

Here is relevant portion of /etc/config/network from a v1:

config switch
        option name 'switch0'
        option reset '1'
        option enable_vlan '1'

config switch_vlan
        option device 'switch0'
        option vlan '1'
        option ports '0 1 2 3 5'

config switch_vlan
        option device 'switch0'
        option vlan '2'
        option ports '4 6'

Adjust for the your device from switch layout.
