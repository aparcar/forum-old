Well I tested the ipv6 things i wanted 2 and already forgot with which browser i did the attempts but very likely most of them with firefox, I reverted to the Linksys firmware since if i understand it correctly you need more fine-tuning if you want wifi n enabled instead of g , and i am very novice to these thing. So for anyone wanting to revert.

To revert I done the following based on https://wiki.openwrt.org/doc/howto/generic.uninstall :
1- download the latest linksys  E2000 firmware let say "firmware.bin"
2- connected with winscp to E2000 with openwrt 14 installed
3- transfered "firmware.bin" to /tmp
4- opened ssh putty connection to the E2000
5- cd /tmp
6- dd bs=32 skip=1 if=firmware.bin of=firmware.trx
7- mtd -r write /tmp/firmware.trx firmware

Let it run, logged in with admin/admin to the original firmware and under administration updated the firmware again with firmware.bin .
