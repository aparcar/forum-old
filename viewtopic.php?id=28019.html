You can flash new firmware, but you have no other access to router. Correct?

You should maybe build a new firmware version, into which you would embedd working settings.

I am sure you have regularly made a backup copy of your settings, so you have a "last known good" settings copy somewhere.

Altenatively, browse source code repository for default settings for your model.

Embedd /etc/config/network (and also firewall settings and also possible the essential config files) from that backup copy into the new build, so that you can have a working config flashed in along the firmware.

As long as you have TFTP access, there is hope ;-)


Ps. I did not quite understand, if you had a working telnet access or not. If you can access it with telnet, then you could just edit /etc/config/network based on model/backup copy of the file.
