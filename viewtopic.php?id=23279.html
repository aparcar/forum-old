If u see Msg from 192.168.1.1: Please reset now, to enter Failsafe!
than ur brick is alive and ready to connect via telnet w/o passwords on 192.168.1.1

If the last action for configuring P334 was switch interfaces from static to DHCP - just plug it to the network with DHCP server and see what address will be assigned for ur box.
Than u can access it via Luci or SSH/Telnet again.
