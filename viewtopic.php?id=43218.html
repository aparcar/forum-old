31 Mar 2013, 16:58

You have probably forgot to remove uboot from tp-link firmware, whis is 131584 bytes long. If you have not restarted yet, strip the uboot first and then flash again:

dd if=wr740nv4_en_3_16_5_up_boot(130322).bin of=revert.bin skip=257 bs=512
mtd -r write /tmp/revert.bin firmware

If you have restarted, tplink is bricked and recovery is possible only with serial cable
