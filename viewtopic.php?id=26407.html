luci is pretty helpful, since you can configure the whole system with it. you dont need to install any packages at all - they are all included in the basic firmware. but you SHOULD probably telnet into your device and set a password after a fresh installation.

if you did that, go where jow pointed you in luci and you can setup all wireless features. luci is pretty simple - but sure it requires you to know what you are doing. what you perhaps miss is a "setup wizzard" - but thank god, openwrt doesnt have such crap.

another option (AFTER setting the password via telnet) - log into luci, switch from essentials to administration (upper right) and go to every menu topic available. oh - and allow javascript - else you might miss most submenus
