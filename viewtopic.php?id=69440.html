The package is currently at a state where I can release an alpha build, the driver requires heavy configuration to get it working currently so don't expect it to be easy to setup. You must manually set the country in the config file as well as setup the rest of the options. The firewall rules need to be setup as well after the driver is installed, as well as the dchp server.

Screenshots:
imgur dot com slash a slash PYRvM

WARNING: This is a pre-release version of the driver, I am not responsible for anything bad happening if you use it, It is largely untested proceed at your own risk. Source for the driver scripts will be released when the driver is more stable, if you want the source of the binaries please check the Quantenna code in netgear's firmware source files for the Netgear Nighthawk X4 R7500.
