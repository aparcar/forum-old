Are you absolutely sure that "kmod-sched" fits exactly to your running kernel?
Crashes like yours usually happen if a kmod is used which was compiled against a kernel with a different featureset, this causes slight abi mismatches which usually lead to bad memory accesses as soon as one tries to actually use the corresponding module.
