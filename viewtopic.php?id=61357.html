Hi,

I use a TP-Link TD-W8970 with my VDSL provider (italian "Telecom Fibra") and I had no issue for several months. In the last weeks I had a problem (the router was unable to obtain ip address) and after some config changes (with no results) I decided to flash the tplink with the last DD trunk. With DD I experienced a regression, DSL doesn't sync and ppoe negotiation obviously doesn't start. Trunk is not stable, i know, so I opened a ticket for this issue

https://dev.openwrt.org/ticket/21319#ticket

Then I flashed back to stable Chaos Calmer. With CC DSL syncing successfully, under luci wan interface correctly acquire ip address but under status -> overview IPv4 WAN Status was "not conntected", and infact tplink doesn't route anything to wan side..
