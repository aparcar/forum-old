Post your config for wireless and wpa_supplicant.
When you connect to a backend RADIUS server, you need a RADIUS shared secret. Is it possible the last file is the shared secret? As for the certificates, do not combine them. In the wpa_supplicant file you should only need to point to the root CA cert.
