OpenWrt wiki wrote:

A minimal example for a wifi-iface declaration is given below.

config 'wifi-iface'
        option 'device'     'wl0'
        option 'network'    'lan'
        option 'mode'       'ap'
        option 'ssid'       'MyWifiAP'
        option 'encryption' 'psk2'
        option 'key'        'secret passphrase'

option 'network'    'lan'

Your device are authenticating correctly, but they cannot obtain an IP address because you have not configured any IP address for the WLAN interface. You probably wanted to bridge the WLAN and LAN interfaces, but did not do it properly:

Max Hopper wrote:
OpenWrt wiki wrote:

        option 'device'     'wl0'
        option 'network'    'lan'

        option 'ssid'       'MyWifiAP'
        option 'encryption' 'psk2'
        option 'key'        'secret passphrase'

option 'network'    'lan'
