I'm using CC Image Generator to build a custom image for TL-WR703n with several extra packages. Unfortunately, they do not all fit on the small device, so I'm disabling the big stuff I don't need, and it seems the easiest way is to disable IPv6 (which I don't need at all in this setup) since the kmod-ipv6 package is pretty large.

I'm using the following make command:

$ make image PROFILE=TLWR703 PACKAGES="dump1090 openvpn-openssl -kmod-ipv6 -kmod-nf-conntrack6 -kmod-nf-ipt6 -libip6tc -odhcp6c -ip6tables" FILES=../files/

With the following changes in my config files:

  • firewall.defaults[0].disable_ipv6='1'

  • sysctl: net.ipv6.conf.all.disable_ipv6=1

  • sysctl: net.ipv6.disable=1

  • sysctl: ipv6.disable=1

  • No IPv6 ifaces in network config

However, the device still won't boot properly, and I can't tell what's missing. Failsafe still works and I can change configs.

What else am I missing in this image to get it to work?
