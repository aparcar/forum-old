  • 1. Web browsing, small HTTP traffic

  • 2. Video streaming, like Netflix

  • 3. Large HTTP traffic, downloads, torrent traffic

If it is too complicated to configure rules like this, it could be client bases as well; streaming devices like a Chromecast could be prioritized then.

Now, how should this be done? Which package is suitable?
