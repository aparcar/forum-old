Here is a way of installing owncloud or other CMS on your openwrt router automatically. no configuration are required. you just download the script, run it and you are done

wget http://www.gizfun.com/dl/web_for_openwrt.sh


sh web_for_openwrt.sh

after a minute or so, you can go to the browser and access the setup page to finish installing

Post #6
sin
17 Sep 2013, 05:04
written_direcon wrote:

Another idea for the script: It should be based on extroot instead of optware on OpenWrt :-)

yes, that can be done. but putting the whole rootfs on external storage can cause stability issues. for example, if your USB storage gets disconnected for some reason, the whole system will crash.  Running only non-system components from external storage can avoid that problem.

I think CPU and RAM of the RPi could be just fine. But I am under the impression that ownCloud is very database-io-intense. And running the database over USB2 is hardly optimal. If it is a marginal problem or a show stopper I don't know.

I use an old PowerPC based Mac (866Mhz/1GB) for ownCloud. I have an internal ATA-drive for OS/database, and an external Firewire drive for data storage. I guess that is quite much better than any storage for a Raspberry Pi.

Nevertheless, using RPi+OpenWRT for an ownCloud server is an interesting idea.

If I were to replace my PPC based ownCloud today I would buy a cheap Intel NUC and run Debian (since that is what I am familiar with).
Or a Banana Pi or a Cubieboard that has SATA.
