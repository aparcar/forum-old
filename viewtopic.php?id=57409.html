Hi,
uHTTPd is working fine but without data inside /www directory you will get "page not found".
Installing LuCI WEB UI will install index.html into /www directory.
Or put your WEB application into /www directory with index.html

Other possibility is a problem in your config above.
You missed the word "option" in front of "index_file"

Hope this helps
Christian
