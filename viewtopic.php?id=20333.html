I have a WRT54G moded with 2G SD card, It is running very good with only one problem, some of my init scripts won't run at boot up. scripts like qos-script, tinyproxy, and asterisk. From the log of asterisk, looks like it is related to SD mount not being ready at the time the script is running, my asterisk is installed in sd partition, same as the tinyproxy, but qos-script is installed in the flash, it too won't start at boot. Any suggestion on how to cope with this problem? Thanks!

You may try to customize the start level of your sd mount script. For my WRT54GS V1.1, I had to set S48 to my bootext mounting script. Otherwise, the default S40network script would not run. BTW, I am using 7.09, not 8.09.
