If you want to have your HooToo on the same subnet as CS50 then you want to use relayd (https://wiki.openwrt.org/doc/recipes/relayclient).

If you want your own private LAN on a separate subnet, then you pretty much just need to set the LAN side IP address to something other then CS50 and connect the WWAN via Luci-Wifi.

OpenWrt is the WLAN (AP) connection, and you want to keep this(for now), as well as add a NEW STAtion (Client). 

When you click the "Join" button, a new page opens.  At the top of the page is a check box which says "Replace wireless configuration".  UNCHECK this box, and you will create a NEW network. for the STAtion side.   Replace the security key and you probably can click through the other options, but if you are having an issue getting the STAtion to configure, take your time and read.  Note the tabs in the bottom section.

There are 2 processes, not documented anywhere that I know of, for "managing" wireless connections.  The results are the same, the actions are different.  The method in the video wipes ALL your connections, where as the method I was suggesting retains the existing WLAN config.  This is important if you are using a travel router and no ability to connect with a Ethernet cable.  Detailed steps are as follows:

  • 1. From Luci>>Network>>Wifi select the "Remove" for the Client, followed by "OK" to the "Really delete this wireless interface..." dialog box.
    2. Select "Scan"
    3. Select the "Join Network" button for the desired network.
    4. Uncheck the "Replace wireless configuration" check box. Enter the "WPA passphrase". Select "Submit".
    5. Click "Save & Apply"

The AP configuration is not impacted with this process.

I suggest you get and save a working config, and then familiarize yourself with the process for configuring your wireless connections if you plan on traveling with the device.

Regarding the channel, be careful.  You may or may not need\want this.  It depends on the device(s) you are connecting to.  The video demonstrates connecting to a device with a 40mhz band available.  It should still work with 20, but will not be as fast.  Not sure what will happen if you set this to 40 and the STAtion ONLY is in 20.  Did it not work if you accepted the default?
