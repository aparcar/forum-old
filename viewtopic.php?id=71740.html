You can use any router that runs OpenWrt / LEDE and has a USB port with that antenna.

Are these speeds by actual speedtest, or merely those reported by the driver?  The driver's reported speed isn't always correct.

You should definitely try the newer firmware.  LEDE 17.01.2 is the latest.  You will need to install the drivers (I think it is kmod-rt2800-usb) for the adapter.  These are not kept installed during an upgrade since kmods only work with one kernel.

Of course set the router's wifi to a channel that doesn't conflict (at least 5 channels away) with the channel the ISP uses.  Also it's a good idea to use a USB extension cord so you don't have two radios on the same band next to each other.
