Are those Tx-power values REAL or FAKE? If they are REAL how can I limit the Tx power to 100mW allowed in Europe?

Hi.

There was an option added for txpower on broadcom recently. Unfortunately it didn't made it into the RC1.
The command to run is:

iwconfig wl0 txpower 123dbm

Maybe put that in an init script.
