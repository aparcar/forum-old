- Open router
- Plug on serial cable (rx, tx, ground) - refer to router's model
- Open up serial console : eg : stty -F /dev/ttyUSB0 115200 && cu -l /dev/ttyUSB0
- Setup your computer with ethernet at 192.168.1.100
- Install tftp server, upload a working firmware file
- Type "tpl" at boot

tftpboot 0x81000000 openwrt-ar71xx-generic-tl-mr11u-v1-squashfs-factory.bin
erase 0x9f020000 +0x3c0000
cp.b 0x81000000 0x9f020000 0x3c0000
bootm 9f020000

- Have fun
