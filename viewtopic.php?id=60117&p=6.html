Post #143
tmo26
21 Oct 2017, 23:37

From https://lede-project.org/:

The LEDE Project (“Linux Embedded Development Environment”) is a Linux operating system based on OpenWrt.

There was a split of OpenWrt which resulted in LEDE.
Now, roughly 1.5 years after, LEDE and OpenWrt will be merging again, taking the LEDE source as base.
LEDE GUI = LuCI.

A general note on TL-WA901ND v4: It has only 4MB flash / 32MB RAM, which can be a problem.
Details see https://lede-project.org/meta/infobox/432_warning
