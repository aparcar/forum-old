I've compiled working backfire images for my dsm-g600 on Snow Leopard.  So, I don't think the problem is with OS X.  I've had to add a few additional symlinks (in addition to the ones mentioned in the howtos) to make it compile correctly.

sudo ln -s ginstall /opt/local/bin/install
sudo ln -s greadlink /opt/local/bin/readlink
sudo ln -s guname /opt/local/bin/uname
sudo ln -s gnutar /opt/local/bin/tar
