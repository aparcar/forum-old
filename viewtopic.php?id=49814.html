Can you still connect to your OpenWrt router at all? I sometimes had trouble when the WAN and LAN interface of the router were configured on the same subnet (192.168.1.x), so I usually change the IP of the LAN interface (which is also used for the wifi): 

uci set network.lan.ipaddr=192.168.2.1
uci commit network
reboot
