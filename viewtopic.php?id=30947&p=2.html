imam.h2002

hi,
only want to share how to upload firmware

i just upgrade hg553 with openwrt-HW553-squashfs-cfe.bin
it's not work well....

only red light,can't ping,no wifi

solution:
- go to fail safe mode (more info about failsafe http://wiki.openwrt.org/doc/howto/generic.failsafe )
- download original firmware 'image_b21_es.bin'
  try google to find this image


1. press reset button for 30s while powering device
2. wait a few minute, 5min
3. press wifi button
4. change comp ip address 192.168.1.x -> x=2,3,4,etc
5. http://192.168.1.1
6. upload firmware

welcome back hg553
