root@OpenWrt:~# /etc/init.d/uhttpd enable
root@OpenWrt:~# /etc/init.d/uhttpd start

-ash: /etc/init.d/uhttpd: not found
-ash: /etc/init.d/uhttpd: not found

IMO this clearly states that you simply don't have uhttpd webserver installed on your router by some reason. So unless you installed and configured some other web server (which I suppose you didn't) you have to (re)install uhttpd first in order to use luci GUI.
