Hi,

first of all, welcome to the community!

As far as I can see, there's a sysupgrade image available for your router model:

http://downloads.openwrt.org/barrier_br … pgrade.bin

The easiest way to do the upgrade is to flash this image using Luci's flash upgrade interface or the command line interface described here:

http://wiki.openwrt.org/doc/howto/generic.sysupgrade

IMPORTANT: please don't be angry with me if the upgrade doesn't succeed for some reason and your router doesn't boot afterwards, I have no experience at all with the router model you are using. Fortunately, OpenWrt never touches the bootloader's flash area so if you have a failed upgrade, you still have the option to flash a new image using the bootloader's capabilities (generally through TFTP or a serial connection). Reviving the device through TFTP seems to be quite easy to do using this method.
