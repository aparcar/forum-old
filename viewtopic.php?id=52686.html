Hi

I am running the BARRIER BREAKER (14.07-rc3, r42056) on a TP LINK MR3040v2 and i can not make the  ExtRoot work.

I did this:

opkg update
opkg install block-mount
mkdir -p /mnt/sda1
mount /dev/sda1 /mnt/sda1
tar -C /overlay -cvf - . | tar -C /mnt/sda1 -xf -

then modify my /etc/config/fstab to this:

        config 'global'
                option  anon_swap       '0'
                option  anon_mount      '0'
                option  auto_swap       '1'
                option  auto_mount      '1'
                option  delay_root      '20'
                option  check_fs        '0'

        config 'mount'
                option target        /overlay
                option device        /dev/sda1
                option fstype        ext4
                option options       rw,sync
                option enabled       1
                option enabled_fsck  0
                option  uuid    'cf9d8f61-8214-4520-9c89-24d55b9cbb46'

then enable fstab by

/etc/init.d/fstab enable

and reboot

but the new overlay is not mounted. I can still mount the sda1 in another mount point but can't on overlay while boot.

Any idea?

Thanks
