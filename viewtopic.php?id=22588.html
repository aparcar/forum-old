It is generally better to install an image directly from X-Wrt.org. Uninstalling LuCI with opkg will just mark it as removed but not actually free any space.
Also each time you reset your firmware image, LuCI will be active again and X-Wrt is gone - so use an image with X-Wrt embedded in the rom part.

You can switch the image while keeping your configuration by using "sysupgrade" .

Regards,
JoW
