I want to mount a folder from my Windows server on my OpenWrt device.
Therefore I generated a folder /test.
I installed cifs on my OpenWrt system.
and I tried the following command
mount -t cifs //192.168.1.1/folder /test
All I got is an error saying invalid argument.

But I can not recall how I did this.

Ok I figured it out now by my self (I still don't know why it was not working in the first place)
mount -t cifs //192.168.1.1/folder /test2 -o nounix,noserveroserverino,user=myname
I have added this line now to my fstab file for future use

config mount
        option target  /test2
        option device  //192.168.1.1/folder
        option fstype  cifs
        option options nounis,noserverino,username=myname
        option enabled 1

Can any of you confirm if this is the right way to do so
