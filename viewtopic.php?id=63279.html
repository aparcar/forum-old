You might experience some "side effects" - never tried using non linux fs as rootfs
- user and permission mappings may be different: IIRC msdos does not support multiple users (neither does vfat afaik)
- msdos does not support case sensitive file system (vfat isnt case sensitive either)
- msdos depends on vfat so you need that module too
