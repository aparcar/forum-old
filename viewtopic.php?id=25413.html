The architecure names where changed from cpu arch to boardname after 8.09, e.g. what formerly was "mipsel" is "brcm-2.4" now.
The 8.09 SDK uses old style arch names while trunk expects new style ones.

You can add such a line to opkg.conf:

arch mipsel 100

This will make the trunk opkg also accept packages made for "mipsel".
