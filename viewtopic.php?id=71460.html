From LuCI: System -> Backup/Flash Firmware -> Backup old configuration

This will give you a TAR archive containing all your configuration files. Then upgrade to a newer FW and clear the existing config. Now manually paste over those bits from the old config that you need.

Usually the config files are compatible between versions, but sometimes new things have been added and old things refactored so that the config files simply aren't compatible.
