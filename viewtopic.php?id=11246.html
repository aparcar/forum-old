Since I got it sorted you by researching forum, I thought it would benice to share with others. Problem was finally missing GPIO mask, soldering was after all ok. What I did was "echo "9c" > /proc/diag/gpio. After that 32M and 1 Gb SD card started working like angel with opteamized mmc -driver module that is available on this forum. Before fixing GPIO mask wrt54g v22 didn't really crash but it did run out of memory and did not response for anything for few minutes. However this is all now history and I'm running irssi and lighthttpd on my SD card. This is was afterall quite simple to task and the benefits are superb.
