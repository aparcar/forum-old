Do you have a Network -> Switch page?
It's reasonably self-explanatory.  On the left side, click the button at the bottom to create a row then edit the number to be 2.  Then place this tagged to the CPU port (it becomes eth0.2 or eth1.2), and untagged to one of the ethernet ports.  Turn off that port in all the other VLANs. Make sure you have the port number right by plugging and unplugging a live cable, that will cause the status icon to change from plugged to unplugged.
