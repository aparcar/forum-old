Flashing from the D-Link web interface is not possible, you get the below message as you already mentioned:

Just use the "failsafe" interface.

1.) Power off the device.
2.) Press the reset button.
3.) Power on the device while holding the reset button pressed.
4.) Wait for about 5 seconds, you'll notice the power led changes color.
5.) Open a web browser and go to http://192.168.1.1
     You will see this page:

6.) Upload the OpenWrt image.
7.) After about 3 minutes the device is ready.
