Dear all,

I flashed WRT160NL with latest backfire image with an ath9k driver.

I set the router as an AP, and one laptop as client.   I ran iperf UDP test from AP to the laptop.  They are placed pretty close to each other, and the throughput performance is not very good, around 20Mbps. 

I did the same experiment with the factory firmware, the UDP throughput is a little higher, around 24Mbps.

Anyone has some experience on ath9k performance at this moment?  I came across this ticket
https://dev.openwrt.org/ticket/6165,  back then there seemed to be some performance issue with ath9k

thanks a lot in advance!
