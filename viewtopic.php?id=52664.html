Ok. As expected your device use standard u-boot loader. You can see u-boot menu at the beginning of log. This menu contain emergency recovery, so I recommend to search description of this recovery procedure (just for garanted way to restore original zyxel firmware).
Next steps for flashing openwrt firmware:
1. Interrupt boot and get u-boot command promt (hit '4' in boot menu).
2. In u-boot console setup ip configuration. Usually '?' - generic help, printenv/setenv - see/show params. We interested in 'ipaddr' and 'serverip' - address of router and tftp-server (your PC).
3. Prepare your PC (setup static ip-address == serverip), install tftp-server (e.g tftp32.exe for windows) and put firmware (e.g 'firmware.bin') to tftp-server directory.
4. Prepare flash (typically 'erase linux' command - see output of '? erase' in u-boot promt).
5. Connect PC and router with ethernet cable.
6. With u-boot command 'tftpboot 80100000 firmware.bin' get firmware from PC to RAM.
7. With u-boot command 'cp.linux 80100000' write firmware fom RAM to flash chip.
8. Reboot.
WARNING!!! After successful step 4 your router loose any firmware so to restore you need get success in steps 5-7 or use emergency recovery procedure from zyxel!
