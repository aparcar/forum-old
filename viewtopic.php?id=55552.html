You tried to execute the URL as a command.  What zloop expected you to do is this:

git clone git://nbd.name/openwrt.git

If you still get timed out, test your internet connection, e.g.  

ping nbd.name
PING nbd.name (46.4.11.11) 56(84) bytes of data.
64 bytes from nbd.name (46.4.11.11): icmp_req=1 ttl=48 time=245 ms
64 bytes from nbd.name (46.4.11.11): icmp_req=2 ttl=48 time=244 ms

nbd.name and git.openwrt.org are both presently responding to pings from here.
