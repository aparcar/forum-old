there is already a binary newsgrabber (nzbget) available on openwrt. you can run it as daemon which is just monitoring a watch-dir or use ncurses-interface (it's some time ago I used this, because php was broken for my platform, and I had to modify the Makefile to compile it with ncurses) or you use the  nice php-webinterface (you might need extroot, or a router with 16MB flash because php5 is quite huge)
rar, and par2 are available, I'm not sure about yEnc.

did you check out nzbget, and it doesn't satisfy your needs, or are you used to sabnzbd?

[update] just checked: nzbget has an (fast) internal yEnc-decoder since 2008
