I got it, finally, after about a million attempts!

This is a Taiwan router.  I used a download for version DIR505A1_FW110B02.bin from the dlink.com.tw site.  I renamed it DIR505A1-image.bin (I doubt that part mattered).

From looking at the D-Link forum, it seemed to be a timeout problem.  They had a lot of hints for Windows, so I figured what the heck, I will try it on Wine.  I used wine regedit to change the Internet keep-alive to 3 minutes, following this article. https://support.microsoft.com/en-us/kb/813827

Then, I installed Opera version 11 from winetricks.  I don't really know how long it took, but it was at least 10 minutes before I saw the wired link disconnect.  I never did see the "success" page.
