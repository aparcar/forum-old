You can just use a 'generic usb sound device'. If it is supplied with drivers, it's probably not generic enough.

If you want to stream audio streams, the box may just lack the juice to do so. Most codecs need floating point support, and most routers don't have that, so soft floats are used, which is not very fast.

For playing mp3 libmad is available, which is an mp3 decoder which doesn't use floats. But I don't know if such implementations are also available for other codecs.
