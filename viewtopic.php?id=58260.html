Log into the router by wifi to do all this.  You will likely lose contact if you are connected by ethernet.

Network -- switch configuration.  Check the box to enable VLANs.  Make a vlan 1 which has ports 1 2 and 3 untagged and the CPU tagged.  Make another VLAN 2 which has 4 untagged and the CPU tagged.

Go to network - interfaces - lan - physical settings and put eth0.1 in the lan.  Plain eth0 should not be connected to anything.  In network - interfaces - wan have eth0.2 in the wan.  eth 1 is the hardware "WAN" port which is unusable.

Note that logical port 4 is labeled "1" on the router.  This will be your new WAN port, plug the modem in there.  (modem may also be damaged by storm, test it with a direct connection to a PC if necessary).  The other 3 yellow ports will still function as LAN ports.

also if you use ssh you can edit the following file:
vi /etc/config/network (press "i" to edit mode)
for example if you want change wan port to another port ( in my example 4 lan port changed to wan):

config switch_vlan
        option device 'switch0'
        option vlan '2'
        option ports '0t  5'

config switch_vlan
        option device 'switch0'
        option vlan '1'
        option ports '0t  2 3 4 '

(press esc and write :wq to write changes to file)
