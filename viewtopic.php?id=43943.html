When you first install Openwrt the lan interface is configured as 192.168.1.1/24.
I suspect thata you are using a machine that is expecting DHCP on its network interface.

If you are running windows, you will find and need to open the "Local Area Connection Properties" find the Internet Protocol (TCP/IP) entry select it

Which should bring you to this -

select "use the following IP address,
enter 192.168.1.5 in "IP address",
Accept the deafult "Subnet Mask",
enter 192.168.1.1 as Default gateway.
Click on OK. You should now be able to access your router on 192.168.1.1.

Once you have configured the router do not forget to repeat the above process EXCEPT this time select "Obtain an IP address automatically".

Good Luck.

169.254.XXX.XXX is the automatic prrivate ip range, this address will be randomly chosen as there is no DHCP server visible or no manual config is pressent.

Have you done anything more then changing the password?
Do other computers still connect fine to the router?

If all else fails you can try safemode and restore the router to default settings.
