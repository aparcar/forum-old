For the records, here is the struct that I use for WLC_GET_RSSI call :

typedef struct {
  uint32          rssi;        /* RSSI */
  struct ether_addr ea;        /* MAC address */
  uint16          unknown;    /* 41 then 00 */
} wlc_rssi_t;

unknown field is set to 0x0041.
