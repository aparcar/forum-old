John I know yours is an old post and you may have moved on, but for others searching like I am.  So far I have found the downloaded *.bin files for the WRT300Nv1.1 to not properly identify and activate the ethernet nic or wireless when installed and booted.  If, however, you build the "same" *.bin file with the Image Generator this will have the correct modules and give you a working unit.  So far this is true with the brcm2-4 generator.  I have not found the magic mix of flags and profiles to build a working 2.6 *.bin.
