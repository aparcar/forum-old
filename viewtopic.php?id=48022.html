I don't know if you can disable firewall for some intrface only, but if you specify option input accept in /etc/config/firewall for the interface, it will work the same way. This behaves as firewall is turned off in zone lan:

config zone                           
        option name             lan   
        list   network          'lan' 
        option input            ACCEPT
        option output           ACCEPT
        option forward          ACCEPT
