Since switching from Webif to LuCi I often get uci "failure to read config file" error messages if I edit them using filezilla which uses Windows "Write" program. Write offers to "save as" in several txt formats including msdos and unicode formats. What file format is LuCi using when it edits the config files?

I have been using the router's "nano" editor to make some changes and it has solved the issue with LuCi but it would be nice to be able to use filezilla and Write.

The problem here is most likely the difference between cr/lf and just lf
Windows wants to put cr/lf or chr(13) and chr(10) at the end of every line, *nix only wants lf or chr(10)
Since you're having this problem I'll assume you're using a windows system.
Visit http://projects.gnome.org/gedit/  and download the windows version and then change your settings in filezilla to use it as your default text editor. That should solve the problem for you.

It is nice using filezilla to just edit right in ftp
