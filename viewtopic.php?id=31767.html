I had it working with  ATTITUDE ADJUSTMENT (bleeding edge, r28151) but I needed to do some updates, got in a mess so went for a re-install.

I have installed ATTITUDE ADJUSTMENT (bleeding edge, r28173) but it does not have mjpg-streamer so I did:
opkg install http://downloads.openwrt.org/backfire/1 â€¦ ar71xx.ipk
