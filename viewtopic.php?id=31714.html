What you describe should work and fit on a TL-WR1043ND (although I'm a little worried about the OpenVPN requirement - what about a WNDR3700 or one of the Buffalo devices?).

However, on any of these devices you're not going to have a good time running DansGuardian with 30 concurrent users. Your users will certainly see some slowdown if you're on a reasonably fast connection. For a handful of users you'll be okay, but 30 is quite a lot. I would suggest using something more powerful if you really need to proxy all of this traffic.

Would be interested in other people's opinions on this too.

Thanks,

Sam
