TP Link TL-WR2543ND router

I have eventually managed to get the internet connection working, so for the benefit of any one else with the same setup:

The IP address of the ambit 256 modem is 192.168.100.1 so I set the lan address of the router to 192.168.100.2 via telnet

uci.set network.lan.ipaddr="192.168.100.2"
uci commit network
sync && reboot

and rebooted the modem.

I read somewhere that it can take several reboots of both the router and the modem for the router to be able to pickup a wan address so it may just be that it had nothing to do with the IP address change.

To install the LUCI web interface telnet into the router then run the following commands:
opkg update
opkg install luci-ssl
/etc/init.d/uhttpd enable
/etc/init.d/uhttpd start

Misc
------
If you want to see your config from telnet run: uci show
To do a factory reset of openwrt on the device power cycle it and immediately start pressing the other button (not power) on and off
