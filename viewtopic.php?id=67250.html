EDIT: The problem is partitially solved - Here's how I did it:

A) Had to compile UBoot with CONFIG_HTTPD commented
B) Compiled an arv7525pw compatible image with the 15.05 ImageBuilder... when not including LUCI packages it fits the NOR flash just fine.

download: h t t p s : / / ufile.io/f6110

-> Boot uboot.asc via HTerm (send over the ascii file via serial)
-> Serve uboot-bootstrap.bin and respective image via TFTP
-> Exec following commands in uboot-prompt

run update_uboot
run update_openwrt

Note: you might have to unprotect the flash manually for the uboot update to succeed. Dont forget to re-protect it!
