I want to install the Stable Branch of OpenWRT on a TP-Link WL-WR1043ND Revision 3. However, there only seems to be a Trunk (Designated Driver) for this HW revision as can be seen on the Wiki Page: https://wiki.openwrt.org/toh/tp-link/tl-wr1043nd

Hardware versions 2 and 3 of this device are identical, they only differ in the stock firmware. However the stock firmware won't let me install the image for v2, saying that I don't have the right firmware version.

From my understanding, the v2 OpenWRT image should run just fine. I only need the stock firmware to accept it. How can I accomplish this? Do I have to build an image myself?

The wiki says to follow https://wiki.openwrt.org/doc/howto/generic.flashing, so if Method 1 does not work then you need to move on to Method 2 or Method 3 and so on.

The wiki also says that v3 has Chaos Calmer 15.05.1 support, and it would seem that v2 flash is to be used.

Also a note in the wiki says

:!: NOTE: For revision v3.0, web UI on stock firmware 3.16.9 Build 151021 or newer won't accept OpenWRT image. I got it working by connecting a tftp server at 192.168.0.66 to a LAN port, copying the OpenWRT image to filename wr1043v3_tp_recovery.bin in the tftp server root, then powering on with the reset button held down for ~20sec.
