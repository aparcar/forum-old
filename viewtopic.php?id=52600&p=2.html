Post #28
lizby
13 Sep 2014, 19:13

Oops, didn't see last post.

Q1:  For smtp, I use whatever is the smtp for the provider at my locale.  I haven't tried smtp.gmail.com.

Q2:  password is not needed, because sender email address is not verified (can be spoofed).

Q3:  The text following "-sub" is the subject.

Q4:  Any appropriate encryption (or none) can be used.

Q5:  You should be able to connect to a DHCP server.

Q6 & Q7:  You should be able to skip the lan and wan setups when connecting via serial for configuration.

Q8:  It could be done, but the program would be somewhat more fiddly if a configuration file were used on usb rather than the script.  It would be a matter of a number of "if then" statements, which to me, are frustratingly obscure in ash/bash scripts.  The "read" statement would be used to get the values, e.g., "read ssid mode encrypt  key", then "  uci set wireless.@wifi-iface[0].ssid=$ssid", etc.

You need to be connected to the internet in order to use opkg to install packages.  You will need the mailsend package and all of the usb-storage packages I had indicated above.  I'd also install nano--a much easier editor to use than the built-in vi.
