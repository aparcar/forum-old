with RC4, I am fairly sure you can grab the micro version and just do an upgrade via the web interface under "firmware upgrade".  I know on my first go at this, I actually grabbed the dd-wrt mini version first, and upgraded to that firmware via the linksys interface.  Then I realized dd-wrt wasn't everything I wanted and came to OpenWrt.  Once I had the dd-wrt one installed, I was able to use the dd-wrt's interface to install the full version of OpenWrt.  I think you should be able to grab the OpenWrt Micro firmware and once that is installed from the linksys interface, grab the full version and tftp it up there.

I am using a WRT54GS v4 like yourself.
