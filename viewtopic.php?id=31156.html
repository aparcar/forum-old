If someone is willing to donate a WR841N to the developers then you may get it supoorted. The 16MB RAM issue shouldn't be insurmountable. Note that it also uses a much newer chipset, one that hasn't been given support in OpenWRT yet.

For my part, I have samples of the new TL-WR741ND (v4) and TL-WR703N arriving soon and I will be donating these to the developers straight away. These run the AR9331 chipset, so this may help here too.

Incidentally, I've had confirmation that the new TL-WR741ND will retain 4M flash and 32M RAM. So you may wish to look towards that device if you're unsure what version of the 841 you will get.

Thanks

Sam
