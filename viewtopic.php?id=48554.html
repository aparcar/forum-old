Hate to answer my own question.  But leaving a trail of bread crumbs in case others are curious.  As of yesterday (2/10/14), the openwrt firmware that was tweaked to be compatible with the WD N750....

Does NOT include LUCI

So...you have to telnet and ssh into the router to mess around at the command line level to get LUCI installed.  It's actually pretty easy if you are used to messing around with linux configuration in the pre-ubuntu period...

I have not had much time to test the openwrt firmware.  (just installed it yesterday).  But so far, my impression is that it is at least 1000% more stable than the factory firmware.   I base this impression on...

The router with factory firmware would need to be reset at least every hour or three, due to losing Internet connection.  This WD N750 with openwrt has maintained a solid openvpn connection all day so far (about 7 hours).  So it's already doing a lot more work than it was before...and it is (so far) proving reliable...

Time will tell if the hardware really holds up well.  But so far at least, it looks like openwrt turned a 30 buck piece of garbage into something that is actually kind of useful.
