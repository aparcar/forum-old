I'll post my experience with Sagem F@st 2604... There are 2 images in the trunk but none of them worked for me. There is a note on the wiki page that latest image from trunk should work but you have to manually type firstboot command via serial console which requires special cable and opening device http://wiki.openwrt.org/toh/sagem/fast2404#notes.

I stumbled upon a version that worked without using serial cable. It is a firmware from some Spanish forum (http://foro.seguridadwireless.net/openw … m-fst2604/). It is important that you get to the bootloader firmware upload page as described on wiki page (power led light turns red). Flash this firmware:
openwrt-F@ST2604-squashfs-cfe_r33081.bin (2.3 MB)
https://mega.co.nz/#!ItV0wI7R!cGgdDSYOv … 6QXoOrwA9Q
Then connect with putty (telnet) to 192.168.1.1 and change your password. After that, you can connect via SSH and do whatever you want. LuCI interface is not backed in the firmware but if you get this router to internet (I managed after setting gateway and dns in the /etc/config/network) you get install any package, LuCI also.
