Post #9
evertw
27 Feb 2008, 16:37

A work-around is the following:

- download the readline package from your nearest GNU mirror.
- unpack and configure with the proper settings for your crosscompiler:
        ./configure --target=i386-linux-uclibc --host=i386-linux-gnu --prefix=<target directory>
- Copy the libreadline* files from <target directory>/lib to /usr/lib on the embedded machine, using e.g. scp or an NFS share.
- install your package with the --force-depends option, e.g.
        ipkg --force-depends install sqlite3-cli

It is not pretty, but got me going again. Would be better to fix the bug, though...
