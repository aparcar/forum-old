Download putty and use the telnet function
telnet to 192.168.1.1

once you're in, type in: passwd
it will ask you for a new password

Now you have to install Luci, the webinterface for OpenWRT
(openwrt usually comes with web interface.. so no wonder you can't access it with google chrome)

Check this out how to enable it:
http://wiki.openwrt.org/doc/howto/luci.essentials
