I am currently using a 10 node wireless mesh network with Ubiquity SR2 (for Access Point) and SR5 (for Mesh) radios. This is working very well. The Ubiquity radios are end of life.

I have built nodes with the Silex SX-10WAN radio. This uses the Atheros (AR9160) chipset (ath9k-htc driver). The olsrd comes up. All the neighbors are visible with an ETX of 1.0. But I CANNOT PING any mesh node. “netstat –rn” shows the remote nodes with same data as when I use the ubiquity boards.

The Atheros Access Point antenna works well. Any decide logged into the Access Point for a node and ping each other (wirelessly or via Eth1) . They are all visible.

The configuration for both systems looks identical. The radios “appear” to be working well based on the access point tests. I am using a bridged interface (alan0) with a Mesh Radio, Access Point Radio and Eth1. I have tried the mesh network using different channels and modes (b, g, a etc..) The results are always identical.

Any help anyone gives me I GREATLY APPRECIATE.
