mkdir /mnt/media
mount /dev/sda1 /mnt/media

cd mount/media/boot/grub

vi menu.lst

then the modifications are :

default 0
timeout 0

title   OpenWrt
root    (hd0,0)
kernel  /boot/vmlinuz root=/dev/sda2 rootfstype=ext4 rootwait console=tty0 console=ttyS0,38400n8 noinitrd reboot=acpi
boot

title   OpenWrt (failsafe)
root    (hd0,0)
kernel  /boot/vmlinuz failsafe=true root=/dev/sda2 rootfstype=ext4 rootwait console=tty0 console=ttyS0,38400n8 noinitrd reboot=acpi
boot

now all works like it should
