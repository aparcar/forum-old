If that's the only message you see, it means that the USB core enumerated the phone, but the CDC / RNDIS driver did not recognize that it should attach to it.  You must of course have USB tethering turned on in the phone's settings.

Some drivers have an internal list of manufacuter:device USB ID's but I don't know if RNDIS is like that.
