to set a Client (repeater) + AP
first, set the AP, SSID, password etc. then scan for networks, select desire, then Click Join network, on next secreen, first line Replace network configuration UNTICK check mark,

subirimagenes
next line write network to join password, go to bottom SUBMIT, on next screen, is just to check you had set the right password, if you need to see it, click the 2 green arrows, SAVE, and SAVE APPLY.
thas all you need to do.
