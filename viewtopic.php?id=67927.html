If the OpenWrt is configured as a "dumb AP" (https://wiki.openwrt.org/doc/recipes/dumbap), then you have all you need for the "Private WiFi" to work ("DHCP passthrough" just works because LAN and WAN are bridged together). Then you add the "Guest WiFi" (which will be isolated and require a DHCP server on the OpenWrt device, as per the instructions on the guide), and you do not need a VLAN at all.

Set up vlans with each wifi network on its own vlan and with its own SSID.

Then you can firewall each vlan from the others and set up dnsmasq to hand out DHCP addresses on each vlan from a different subnet.
