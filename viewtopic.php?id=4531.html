I am interested in running OpenWrt on my Netgear WG602 wireless access point, so I've been trying to find out more about it. I've created an entry on the OpenWrt wiki, based on other sources on the web ( http://www.seattlewireless.net/NetgearWG602 and http://kbserver.netgear.com/products/WG602v1.asp among them) and my own fiddling.

I've been trying to put the precompiled OpenWrt .trx firmware images onto the device without succes. I've tried, in a spririt of newbie-ish experimentation/optimism, copying various lengths of the header from the official firmware to a WhiteRussian release of OpenWrt, which has at least been written to flash, but predictably made a brick of my WG602 (as detailed in the wiki entry, it's easily recoverable)

The current version of the official Netgear firmware at time of writing is 1.7.15 available from http://kbserver.netgear.com/support_det â€¦ dnldID=741
Previous versions had a telnet interface, but this was removed for security reasons. Portscanning with nmap reveals that the current version has only a web-based UI, and an ftp server (vsftpd) which gives access to the entire OpenRG FS, however many files are not viewable, and none seem writable. I'm currently not sure of the purpose of this ftp service, certainly it does not *appear* to offer any facility for uploading custom firmware versions.

I'm making this post in the hope of encouraging discussion among other people with this device.
