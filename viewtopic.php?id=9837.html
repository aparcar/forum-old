hatim wrote:

ok, thanks.
Is this an option already existing in Kamikaze or must I developpe it?
does it work with any type of AP supporting Kamikaze?

I believe all you have to do is:

# wlanconfig ath create wlandev wifi0 wlanmode ap

If no wireless devices are configured, this will make ath0. Running the command again will make ath1, ath2, and so on. You can then set the configuration for each one you create separately.

Hope this helps!
