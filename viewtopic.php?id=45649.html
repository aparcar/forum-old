Any OpenWrt supported router with an USB port can theoretically be used in conjunction with USB 3g modems, however in practise there are often hardware issues due to USB power limits which might lead to instable connections. In many cases it helps to use an externally powered USB hub but this might not be an option for a pure mobile scenario.

Also all OpenWrt supported Atheros routers can do simulateneous AP + Client ("Repeater") mode.
