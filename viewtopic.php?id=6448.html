Is your external disk VFAT?

root@OpenWrt:~# mount
/dev/root on /rom type squashfs (ro)
...
/dev/scsi/host0/bus0/target0/lun0/part1 on /mnt/disc0_1 type vfat (rw)

If yes, then it won't support symbolic links.  If you don't share this drive with a Windows system, reformatting as ext3 should fix it.

Hope this helps,
Charlie
