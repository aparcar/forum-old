Hi, I just flashed my WA830RE v1 with the trunk snapshot openwrt-ar71xx-generic-tl-wa830re-v1-squashfs-factory.bin from yesterday (27-Mar-2013).

You can easily do this by using the original firmware's web interface, where you can simply upload the bin file.

It looks like everything is working nice.

One note: there is no LuCI installed in this build. So if you want a WebUI, you have to install it via shell (LuCI Essentials).

Update: I am happy with the WA830RE on OpenWRT. I am using it simply as a WDS Repeater providing also Internet at its RJ45 Port for my mediacenter. I could have done this with the original firmware, but it works much more stable now than with the original. The original firmware jammed my whole wifi network totally every 4 hours or something like that. Now it is running for nearly two days without any problem.

The only issue I recognized was, that LuCI generated a 500 Server Error with the message "could not allocate memory" after I played a lot with the interface configuration. I decided to not autostart LuCI because it eats 4 MB of the 28 MB total memory. And OpenWRT alone needs about 20 MB.
