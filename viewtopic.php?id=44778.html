Sorry, you can't do that Dave

Gargoyle is not a package you can just add, Gargoyle is forked from OpenWrt with lots of their own customizations so only way to use Gargoyle is to install Gargoyle firmware.

Unfortunately Gargoyle only supports Atheros, Broadcom and x86 based routers, and Carambola is based upod Ralink chipset.

http://www.gargoyle-router.com/download.php
