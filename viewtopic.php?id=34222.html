Gave me trying to make PPTP work on OpenWrt myself. But it's okay to use a different subnet for PPTP, I did the same thing while using DD-Wrt.

You probably have to setup these rules though...

iptables        -A forwarding_rule -s 172.16.1.0/24 -d 192.168.1.0/24 -j ACCEPT
iptables        -A forwarding_rule -s 192.168.1.0/24 -d 172.16.1.0/24 -j ACCEPT

Using the 192.168.1.x subnet for VPN is generally not recommended as it collide easily with the roadwarriors' networks (e.g. hotel WiFi) but, YMMV.
