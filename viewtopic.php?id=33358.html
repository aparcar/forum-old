Of course I do. But I found an alternative. Holding the QSS button at boot time, when the system led blinks, in a few seconds will make the system led blink way faster. At this point I could telnet into 192.168.1.1 with no user/pass, and able to do a "mtd -r erase rootfs_data" resulting in a clean default system. After that I rebuilt my configs and everything rocks now!
