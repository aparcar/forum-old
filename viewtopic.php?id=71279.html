I used OLSR to build Ad-hoc network, following this tutorial (sorry I only found a webpage in Chinese).

Now I can get this in OLSR-Viz:

(192.168.2.11) --------- 1.000 ------------- (192.168.2.12)

it says X and Y are in the same Ad-hoc network.

But I want computer A and B communicate via this network, I tried to setup network bridge between lan and wlan of routers, but this will make the Ad-hoc invalid.
