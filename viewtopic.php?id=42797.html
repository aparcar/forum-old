Main problem seems to be that everything is in German.

If by "latest" you mean you installed trunk, you should be aware that there have been some recent (past week or two) changes to the underlying firewall packages so things are somewhat in flux.  So it's possible that nothing "went wrong" specifically, but that there's just a bug in those pages in Luci right now.

If you're not looking to tinker w/ the system just  install attitude_adjustment rc1 where things are more stable.
