The wifi of the DIR-825 does not work with RC6 and will probably not work with the stable release of Backfire, since it requires a trunk patch which has not been backported yet.

I'm running two of those with trunk as a 5GHz wifi bridge. Runs fine for over a month now. Had some stability issues when running 2,4GHz on one at the routers at the same time, guess that could be traced back to it being trunk.


So you would probably be better off with a router that's supported by Backfire, or you use trunk for now and wait for the Backfire successor's RC.
