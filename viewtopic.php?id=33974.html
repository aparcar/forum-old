Post #2
fyi
7 Jan 2012, 19:32 
uci set network.wan.macaddr=00:11:22:33:44:55
uci commit network
ifdown wan
ifup wan
opkg update
opkg install luci
/etc/init.d/uhttpd enable
/etc/init.d/uhttpd start
