Download the stock firmware from Linksys.

Convert the ".bin" file to a ".trx" by stripping off the first 32 bytes.  Here's a tip for that: http://www.pixelchaos.net/2007/11/04/co â€¦ in-to-trx/

Now just use mtd to install your new trx file.  Oh, and be sure that when you copy the file to your router, you copy it to the RAM disk "/tmp".

"mtd -r write Linksys_firmware.trx linux"

login to your router, then...

cd /tmp
wget http://downloads.openwrt.org/kamikaze/7.09/brcm-2.4/openwrt-brcm-2.4-squashfs.trx
mtd -r write openwrt-brcm-2.4-squashfs.trx linux

After your router boots, it will have default values.  You will have to completely reconfigure.

root@OpenWrt:~# cd /tmp/
root@OpenWrt:~# wget http://some.host/firmware.bin
root@OpenWrt:~# dd if=firmware.bin of=firmware.trx bs=32 skip=1
root@OpenWrt:~# mtd -r write firmware.trx linux
