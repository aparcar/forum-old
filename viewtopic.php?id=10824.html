Hi Jim,
This works for me-

#!/bin/ash

ARRAY=" \
http://openwrt.org \
http://wiki.openwrt.org \
"
i=1

for j in $ARRAY
do
        echo $i: $j
        i=`expr $i + 1`
done

Hope this helps,
Charlie
