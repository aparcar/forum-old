cptdondo wrote:

I am looking for a simple, small box that can run OpenVPN.

Any device capable of running OpenWrt can run OpenVPN.

Keep in mind that if you get

a) a device with only one ethernet port, it will be harder to set up if you want the LAN portion of your device behind OpenWrt's firewall

b) a device with only 4 MB of flash memory, it will be harder to set up because you can't just install the necessary packages, you will have to create your own image to flash

c) a device with a slower CPU, you will not get a lot of throughput, for example, a 400 MHz Atheros CPU can only realistically deliver about 30 to 40 mbit through OpenVPN
