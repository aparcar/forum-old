I noticed this against 9626; it seems to be a problem with all versions since the shift from dropbear 0.49 to 0.50.

However, and this seems completely reproducible, if I ssh in as root, then su to another user, things work just fine.

I'm going to assume it must be permissions-related somehow, but dang if I can figure out how.
