This is a simple port forwarding taken from live router:

config redirect
        option target 'DNAT'
        option src 'wan'
        option dest 'lan'
        option proto 'tcp'
        option src_dport '2006'
        option dest_ip '192.168.8.254'
        option dest_port '2006'
        option name 'myport'

Add to the /etc/config/firewall
