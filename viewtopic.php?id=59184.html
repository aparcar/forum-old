In newer Openwrt versions, the button hotplug scripts are in /etc/rc.button.

root@OpenWrt2:/# ls /etc/rc.button/
BTN_2     failsafe  power     reset     rfkill    wps

Which router do you have?
And which Openwrt version you are using?

Only the files named exactly as the button definitions for your router have any effect.
E.g. for my WNDR3700 the correct buttons are reset, wps and BTN_2. The others have no impact.
