Hi,

I have set up barrier breaker with a tunnelbroker.net tunnel. If I switch on router announcements on the LAN interface my clients pick up the v6 address of my router as a DNS server. I would like however a server on my LAN network to handle DNS requests (and forward them externally).
How do I override the standard DNS assignment?

Post #2
mcfly9
16 Feb 2015, 00:01

Found it. Adding the following to dhcp config seems to override the default entry:

list dns '<ipv6 of my internal dns>'
