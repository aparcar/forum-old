There are some reasons for me to use openssh and not dropbear.

1. ssh -g -d port did not work with dropbear.
2. building vpntunnel with dropbear did not work. Openssh has Option -w
3. agent-forwarding did not work with dropbear but works with openssh (I dont know, wether i really need that or not, but i dont like private keys on the openwrt-box)
4. Installing dropbear + socat + openssl + openvpn did not work on my box. I run out of space.

With openssh + socat + openssl i have more functionality.

I am not sure, what the errormessage in my first post does mean and if it is related to openssh.
