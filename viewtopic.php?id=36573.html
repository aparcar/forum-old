Hello friends!!

Excuse my bad english.. I'm newbie in openwrt but I receive help from friends, and I want to share and log my travel in openwrt configurations lololol

steps:

1) Install OpenWrt

2) configure network lan, pass etc..

3) enter in router by ssh and paste commands:

opkg update

opkg install nano comgt kmod-usb2 kmod-usb-ohci kmod-usb-uhci kmod-usb-serial kmod-usb-serial-option usb-modeswitch usb-modeswitch-data sdparm luci-proto-3g kmod-usb-core chat ppp usbutils

ps..
"nano" is one editor that I like to use
"usbutils" have some tools to identify devices etc..

4) plug 3g modem and enter in webinterface
network
interfaces

5) delete wan interface (where come ethernet wan cable)

6) add new interface

Name of the new interface =   wan
Protocol of the new interface = search for 3g umts etc..
config your 3g apn etc..

7) save apply

when reboot my internet is working great!!!
