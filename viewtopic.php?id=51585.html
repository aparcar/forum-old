You need to tell /etc/config/openvpn that you're using an external config file:

config openvpn [name]
   (...)
   option config /etc/openvpn/client.conf

(Or, as I do, replace /etc/config/openvpn with a small shell script that directly starts the openvpn binary with the config file as parameter. I figure it doesn't really make much sense to go through the UCI config file if I'm not using it to configure OpenVPN anyway.)
