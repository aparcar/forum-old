Selecting Y on the module means that it will be built into the firmware file, selecting M makes it a package which is installed with OPKG.

1) use the feeds program   ./scripts/feeds update    then    ./scripts/feeds search xxx    then ./scripts/feeds install xxx
the installed components should then be in your options when you do:    make menuconfig

2) Should be added, but you will usually get compile errors if you are missing something. compile with V=99 if you hit one, check the config.log file
