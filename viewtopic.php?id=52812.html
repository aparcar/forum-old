Hi

It could be a corrupt download or "something else"

Dowload again or roll your own image


When we are both on Skype, we will follow http://wiki.openwrt.org/doc/howto/buildroot.exigence so that you can create your own image from trunk and if necessary, change stuff.
I think y should be able to get the developement environment up on yr Ubuntu 10.04, or a virtual box and use ubuntu 14.04
In short

sudo apt-get update
sudo apt-get install build-essential subversion git-core libncurses5-dev zlib1g-dev gawk flex quilt libssl-dev xsltproc libxml-parser-perl mercurial bzr ecj cvs unzip
git clone git://git.openwrt.org/openwrt.git
cd openwrt
./scripts/feeds update -a
./scripts/feeds install -a
make defconfig         
make prereq
make menuconfig                  #This is where y choose board and router
make
