Today I bought another WDR4300 router, flashed with the latest firmware (trunk build "Barrier Breaker") and messed up my /etc/config/network. As a result I can't no longer access my router.

I made a change on vlan '1' from: 

option 'ports' '0t 2 3 4 5t'

to

option 'ports' '2 3 4 5t'

How can I reset my network settings?

I tried:
1. connecting LAN cable to all ports (WAN firewalled, wifi disabled by default). Naturally it didn't help

2. resetting the router based on http://wiki.openwrt.org/toh/tp-link/tl-wdr4300
a) I pressed reset button then power on... it didn't help. I believe the router got into failsafe mode (because it lid up 'tools' LED).
b) I pressed reset button for 8 seconds when I saw a star LED flashing... didn't help too

Correct me if I'm wrong the only way out of this is to order USB to UART adapter (someone on this forum suggested to buy cp2102) and follow failsafe procedures?

On a side note, when a router goes into failsafe mode, I read that it should be accessible on this IP 192.168.0.86? When, I believe, the router was in a failsafe mode and I tried pinging 192.168.0.86, it just didn't respond (my IP was set to 192.168.0.55)...

Thanks for your feedback
