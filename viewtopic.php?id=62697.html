option src_ip    192.168.1

.7 missed

and add it directly to the /etc/config/firewall

/etc/firewall.user is for iptables raw rules like

iptables -A zone_lan_forward -s 192.168.1.7/32 -p tcp -j zone_wan_dest_REJECT
iptables -A zone_lan_forward -s 192.168.1.7/32 -p udp -j zone_wan_dest_REJECT

not for UCI
