Unless you're willing to build OpenWrt yourself your best bet would be to pick up a pre-built image from the following location:  http://www.wolfteck.com/projects/candyhouse/openwrt/  I successfully loaded using the instructions on that page.   

If you build OpenWrt yourself you'll likely need serial access in order to install as I don't believe the build process creates the file format expected by the stock firmware.
