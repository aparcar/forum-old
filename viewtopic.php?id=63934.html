Never mind... seems that the problem is that for multiple ethernet devices, the environment variables ethNaddr have to be set to the MAC addresses, with N=1, 2, 3, ... (not 0 or nothing).

The ethprime gets N=1, the others 2, 3, etc. Nothing to do with linux.
