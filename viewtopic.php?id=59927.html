Hi All,

I've had a weird issue. I have a TPL-WDR-3600. A powered 4-port hub and USB printer connected to each usb port.

With the latest snapshot, and with previous trunk versions when I install the usb modules, and reboot multiple times, each time after the router reboots my disk mounts change positioning (ie /dev/sdb2 changes to /dev/sdc2 or vice versa). This becomes a real problem when setting up, and running extroot with packages installed. If I reboot, and the dsk mount changes to a partition that does not have all my openwrt packages installed everything reverts to first boot settings. This is especially bad, if I try to access my router remotely, and those settings change I then am locked out until I get home to correct the issue. I have not changed the partitions on the usb drives that are connected. I also have not added additional storage. This did not happen on previous snapshot versions, only the latest r47048.
