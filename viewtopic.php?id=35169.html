Thank you very much nebbia88, I solved the problem this morning but I didn't write here...

I solved this way:

I gain access to failsafe mode.
then:
telnet 192.168.1.1
cd /tmp
wget ftp://user:pass@192.168.1.2/openwrt-ar71xx-generic-tl-mr3220-v1-squashfs-sysupgrade.bin
mtd -r write /tmp/openwrt-ar71xx-generic-tl-mr3220-v1-squashfs-sysupgrade.bin firmware

and bingo!!!

then again telnet 192.168.1.1 and all is correct for me!!!

Thank you very much!
