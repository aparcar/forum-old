You are probably mixing system console and serial console.

For modifying many Openwrt config files, you will need an ssh connection to the system console (command line interface), where you can then use whichever editor you have installed to edit the config files.  (Or if there is no password yet, use telnet instead of ssh). http://wiki.openwrt.org/doc/howto/firstlogin

Serial console is the same, but instead of being connected via network, the connection is made via a serial cable to a header inside the router. That is usually only needed if the Openwrt system does not boot normally, or you want to debug the boot process. http://wiki.openwrt.org/doc/hardware/port.serial
