Update: everything seems to be working well. My code should find it's way to trunk soon.

I'm also packing libv4l which contains wrappers for v4l and v4l2 that implement missing features and fixes bugs for some drivers. For example, my gspca/spca561 based camera won't work properly with motion until I use...

 $ LD_PRELOAD=/usr/lib/libv4l/v4l1compat.so motion

As seen here...

Anyway, track my progress at http://etc.kupesoft.com/openwrt/gspca/
