Looks like a routine ath9k box.  If you have serial access you could try flashing images for similar models.  Check if the 941ND is the same CPU.  Maybe the HP just has RF amplifiers and larger memory.

Flashing through the bootloader avoids the model version check that the factory GUI does-- you would put a sysupgrade directly to flash then try to boot it.  TP Link builds should automatically probe the flash size.

If you build yourself an initrd image you can test it in RAM without touching the flash at all.   Once you have OpenWrt running (from RAM), use it to flash.  This is safer than cp.b in the bootloader where one wrong move can wipe the bootloader or the ART and leave the router severely bricked.

I see Amazon USA selling a WR-841HP but not the 941.
