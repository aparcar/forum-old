/etc/dnsmasq.conf is just a symlink to /rom/etc/dnsmasq.conf. To edit /etc/dnsmasq.conf your will first have to remove the symlink:
rm /etc/dnsmasq.conf
cp /rom/etc/dnsmasq.conf /etc/dnsmasq.conf
vi /etc/dnsmasq.conf

symlinks are true if squashfs is installed; jffs doesn't create symlinks; it writes it directly to the read/write filesystem.  In this case, it would be at /etc/dnsmasq.conf IF JFFS IS INSTALLED.
