Static wrote:

Will there ever be openWRT for the RT-AC66U? I want openWRT but I currently own a Asus RT-AC66U and was wondering if it will ever become available?

The RT-AC66U router has a BCM4360 and a BCM4331 wireless radio chip.
Atm both are not supported. It is unknown if / when they will get support from official Broadcom devs.

For chipsets like 4360 we need to add AC phy support to brcmsmac. So far
we do not have green light to do so.

There is no dev with a unit yet.
It might have a new Marvell chipset. However I did not see any Kernel drivers for that. (88W8864 has no good search results, Linux Marvell drivers are libertas or mwl8k for example)
If its a Broadcom you are in the same situation like before

Marvell uses a closed firmware (like currently QCA with ath10k).
A quick look at Marvell code (libertas, mwl8k) : It seems that it does not supporting all possible mode of operations.
Either AP mode seems to be missing or ADHOC/MESH.
(Searching for NL80211_IFTYPE_ADHOC , NL80211_IFTYPE_MESH, NL80211_IFTYPE_AP in the source is possibly a good way to do it)
