The correct way of enabling IPv6 routing is putting 

net.ipv6.conf.all.forwarding=1

in /etc/sysctl.conf. It is already default on trunk, but commented in backfire.

Not sure why you are talking about port forwarding, that is normally not done in IPv6.

You probably want to read http://wiki.openwrt.org/doc/howto/ipv6.essentials and http://wiki.openwrt.org/doc/howto/ipv6
