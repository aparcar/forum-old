I think they are used for some bridges in physical interfaces but I don't know more details and it is true that there is no information about them in the wiki.

Here is an example extracted from a network configuration file:

    config interface 'lan'
            option ifname 'eth0'
            option proto 'static'
            option ipaddr '192.168.1.1'
            option netmask '255.255.255.0'
            option type 'bridge'
            option _orig_ifname 'eth0 wlan0'
            option _orig_bridge 'true'


I know they are commonly used.
Hopefully someone else can add more information about these options
