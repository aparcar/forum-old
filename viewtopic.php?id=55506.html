The idea being building a tiny openwrt router with virtual dual nic then copying as many of them as needed to simulate a network of routers.

Performance really depends on emulation (see http://wiki.openwrt.org/doc/howto/qemu)
Several ppl have large deployments of OpenWrt with slightly different automatic hardware config
or offering different configuration via web interface (http://meshkit.freifunk.net/)


There is the click package / click project (http://www.read.cs.ucla.edu/click/ , https://github.com/kohler/click ) that can be used with OpenWrt.

Experiments with OpenWrt in larger deployments were done in at some Universities and Freifunk Installations or associated events ( http://battlemesh.org/ )
