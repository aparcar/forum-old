Hi KiaZ,

I'm fairly new to this myself, so I'm sure there's a better way to do this (but here I go anyways).

Run "make menuconfig", go into "Kernel modules"-->"Network Devices", and select "kmod-via-rhine" twice (so it shows an asterisk).\
Exit out, save the config and re-run "make world".

It worked when I just tested it now in my setup, so hopefully it'll work for you as wel.

-Robert
