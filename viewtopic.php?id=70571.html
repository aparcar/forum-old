You can follow these steps..
Connect the TP-Link CPE210 with your computer via ethernet
Set the IP address of your computer t0
Open your browser
Log in with user "admin" and password "admin", change the password in the next step
Go to the "System" tab and upload the OpenWrt firmware in the "Firmware Update" section and select "Restore" in the dialog promted
Follow official OpenWrt setup instructions (e.g. IP address of the device is now 192.168.1.1)
