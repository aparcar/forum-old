What I meant is strict dlna should be disabled. (ie 0).

Another wild guess, some files (mkv containers) have the aspect ratio flag set, and hence the player might know this info if implemented correctly. Maybe dlna doesn't pass it?

Whatever it is, this can all be changed in the player settings (ie your LG blu-ray), if it has aspect ratio set to "auto" or something like that, change it to "normal", which usually means never stretch videos.
