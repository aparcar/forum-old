I assume you mean the RouterStation Pro? I've been using one for quite a while now and I'm still very happy with its performance. I have a Mikrotik R52n mini-pci card in another router and it performs very well. Your situation doesn't call for the high power version. All the Sparklan Atheros cards have different antenna and mimo layouts. I really like the performance of my 3 antenna 3x3 Sparklan radio. I have 5 antennas hooked up to my 2 mini-pci cards in my RS Pro. I went with a 3x3 and a 2x2 card with 3 and 2 antennas respectively. The 3x3 card really outperforms the 2x2 card even though the 2x2 card has a high power radio.

I went with the big, 8dbi antennas because it's very efficient, no extra power needed for the stronger signal.

Bottom line is your described location doesn't require anything fancy so a basic Atheros 11n card with 2x2 and 2 antennas will work fine.

The RouterStation Pro is supported very well by OpenWrt so it's a great high performance OpenWrt router. I have a 16GB SD card plugged into its memory slot. It's formatted in ext4 to hold my web site. I also have a USB wireless lan stick acting as a third access point. It's my primary router and feeds two switches. I'm running X-Wrt, a web server and an ftp server. The RouterStation Pro handles all this without breaking a sweat.

I have a R52n running on routerstation pro (11na HT40) and there were a lot of problems and instablities, but since some time (1-2 month) backfire is quite stable (I build everything from svn) but I guess the lastest 10.3.1 release candidate should work fine. I use a second wificard, but I don't  remember which brand (gerneric 23dBm atheros 11g-card) for compabilitiy with my mobile, and old laptops.

I use the R52n with this antennas: http://www.joymax.com.tw/E/product-show … d-190.html
And I would recommend to get the R52Hn if you want to use 11na on 5GHz because the maxium txpower drops on higher MCS (MCS7 HT40 only 13dBm) so you get only real 11n-speed in the same room, because 5GHz signals don't like walls!  In the same room I get 10-15MByte/s, after one medium-thick wall I get a quite good connection 8-12MByte/s, after another thin wall only 5-8MByte/s but stable, and after another thin wall, no stable signal. The signal of the 11g card with 23dBm is still quite good/stable there.
The R52Hn has 17dBm at MCS7 HT40 and 4 dBi more are a lot.
So if you have some walls where the signal should get through get the 52Hn. If you don't want to use the 5Ghz band there is one need to get an 11na card. Then get something cheaper.
