Two ideas:
1) Set your PC's IP address manually to fixed 192.168.0.4, so that it is from the same subnet as the problematic router. Connect PC directly to a LAN port of the router. Then try telnet/SSH connection.

2) Try failsafe again.  Failsafe requires you to push the button during the correct 2 second period during the boot sequence. So, check your router's wiki page and search the forum for possible hints about the exact moment to push the button.  (And set PC to have a fixed IP address from the same subnet as the failsafe mode (192.168.1.x).)
Is it surely 702N? not 703? There is some advice for 703: http://wiki.openwrt.org/toh/tp-link/tl- … lsafe.mode
