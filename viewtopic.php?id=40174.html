Probably easier to understand in terms of internet connections

I think client mode is just for you in updating your system with opkg to get packages. and the cli programs like wget and using luci to update files with the gui. ( although there are probably other usages this one comes to mind )

Access point mode is for getting the internet to those who connect up to the router to gain internet access ( where that access is then served to clients who get the signal served to them via ethernet or wireless links that come into the router )   (WDS is just a button on the router to get them to hook up more easily with other things like wireless dongles example Dlink dwa125 uses one to hook to other Dlink routers (not 3g modems) without using ip addresses and those type of hassles )

Ad-Hoc is just to connect one computer to the router when using it like a server ie you want to transfer files internally to your intranet (not internet) via the router.  Although what pseudo means in this context i am unsure of.

Monitor maybe use in security context to duplicate the transmissions ?

glad to be corrected if i am wrong
