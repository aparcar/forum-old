Hi!

For a FAT partition you need to install the kmod-fs-vfat package. For ext3 of course the kmod-fs-ext3. But before mounting, you must formatting the partition for ext3. You can do this by installing the e2fsprogs package, and format with mkfs.ext3 /dev/sda1.
