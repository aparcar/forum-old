Don't know what a transparent bridge is, but... In my guide, I've made a second network (the guest) on the first router and made the second router "see" this second network. This is what you're doing?

If this router (your tp-link wr741nd v4.2) is not the one connected (the gateway), and it only knows your lan and nothing else... the only way (that I know) to make you guest network reach internet, is forwarding the guest traffic to the lan, but that means that anyone on the guest network would see the computers on the private network. If that's not a problem, you can go to Network, Firewall, and allow forwarding from the guest zone to the lan zone.

This router is your gateway to the internet? No? Is your gateway (the router connected to the internet) running openwrt?
