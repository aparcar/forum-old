You really don't need to use extroot if all you want to do is to use more space to install extra packages.  You can simply mount a usb drive.  Copy the major directories, e.g. /etc, /usr, /var, /lib, /bin and /sbin to the USB root.  Then do a mount -obind of these USB directories back to the corresponding directories.  For RAM, it is always possible to format a partition of the USB drive as swap space and do a swapon followed by the name of the USB swap partition.  Of course it will not be ideal to swap in and out too often but if only one program need a large chunk of ram but the others are not that active, hopefully the opportunity of read/write from/to the USB drive will not be very often.
I just installed Asterisk 1.8 using Mario's latest patch on the ASUS WL-502GU with a USB drive and it appears working well.  I don't see any additional delay at all.  There are up to 2M swap space were used.  However, since Asterisk need about 9M ram while 520GU has 16M RAM, I guess Asterisk can allocate all its needs inside the RAM.
