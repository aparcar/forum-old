Because it shall be a backbone network, there is need of full throughput, eg 54MBit/s. Therefor I want to use devices with several 802.11a interfaces, mostly 2-3. Then I want to set up dedicated connections between only 2 devices in one channel, so that there is maximum throughput. But I still don't know what antennas I should use. Because the network consists of several point-to-point links in different channels, it seems to use directional antennas.
