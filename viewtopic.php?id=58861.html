Yes there is a page typically under ip-address:8200 but this page is an overview about all items in the current db.
There is no link or something else to play that media directly from there.

If you want to play some media files then you need an renderer too.
Maybe gmediarenderer and a controll point like bubbleupnp via android.

But I would say that dlna is the wrong feature for your intention.
Write a little script with php, python or something else to bring up an webpage displaying your files and take some js snippets to play it.

Using VLC as a controllpoint is ok. I have this setup running.
