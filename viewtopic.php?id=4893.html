You should be able to do it if you setup separate vlans, one for each eth connection which goes to a pppoe device. I have two internet connections, one through cable and one through ADSL with pppoe. What I did is setup cable as wan, on the usual vlan1, and setup an additional port, vlan2, and configure wan2 as pppoe on vlan2. I think you should be able to do something similar if you setup more vlans and point each wan setup to a different vlan.
