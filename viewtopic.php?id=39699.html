Hi, I've just installed openwrt.

How do I change my router ip address, I mean, when I change it through luci web interface is easy, but how do I change it when I access the router through ssh?
eg. from 192.168.1.1 to 10.0.0.1

If I type
   "ifconfig eth0 10.0.0.1 netmask 255.255.255.0" it changes, but when router reboots it reconfigures the previous ip (192.168.1.1).

so, is there any command, as luci web interface does, that changes the ip even when routers reboots?

Post #2
jow
28 Sep 2012, 13:28

uci set network.lan.ipaddr=10.0.0.1
uci commit network
ifup lan
