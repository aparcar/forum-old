I wanted to add a Time based LUCI access to openwrt.
A user LOG IN. Then after some time of INACTIVITY lets say 60 seconds, the session should close and the User has to Re-login.

I modified the FOllowing FIle: "/etc/config/luci"

config internal 'sauth'
        option sessionpath '/tmp/luci-sessions'
        option sessiontime '10'                   -----> default entry was '3600'

However even after 10 seconds the session still exists.

How can this be achieved. Any pointers.

Post #2
RangerZ
28 Sep 2015, 14:54

LuCi will not close or auto refresh a page.  If you try to navigate to a new page you should be redirected to the login. 
You may need to reboot or restart some service for the change to take effect.
