regloss
25 Jun 2007, 03:55

I'd recommend not using the X-WRT images for Kamikaze.  I don't know if they go through the same level of testing the OpenWRT images have gone through.  I've had several strange problems using X-WRT Kamikaze images such as reboots and qos not working (using qos from the XWRT packages).  Install all the packages you need from the OpenWRT 7.06 packages (with the exception of the web interface). 

To get the X-WRT web interface, first install haserl (from the OpenWRT packages), then install the web interface from:

Yes, WhiteRussian has a built in webif.  X-WRT has webif that replaces the built in webif of WhiteRussian.  You can get by with the WhiteRussian built in webif - the X-WRT version expands its capabilities though. 

Kamikaze doesn't have a built in webif at this time - so you need to use the X-WRT webif or go line command mode.

I also had problems to reflash my WRT54GL running Kamikaze.
You can solve using TFTP, or alternatively, you can use Linksys utilities to recover or upgrade the firmware. You can find these official Linksys utilities somewhere on the web (I don't remember where I downloaded mine, I think on Linksys web site). This tool will install Linksys firmware on your AP. Then you can install OpenWrt.
But at first use TFTP. I don't think your AP to be bricked.
