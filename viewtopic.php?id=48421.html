Post #2
nozombian
14 Jan 2014, 16:28

There's no luci in trunk builds by default. You can try to run
opkg update
opkg install luci
But I'm afraid, that it won't fit anymore. To have luci, you will have to compile the build yourself and include in make menuconfig, then it will certainly fit, because it uses good compression.

But I'd recommend to configure router via ssh and enjoy the free space without luci I left luci long time ago and I'm glad I did that.
