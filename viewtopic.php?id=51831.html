This won't probably help you, but I always use option encryption 'psk2+ccmp' or 'psk+ccmp'. All my V8s work fine, V9 was added recently, maybe there's a bug in your build?

To display logs, run dmesg and logread commands. Then run wifi command and then check logs again, you should see new lines added with some description.
