* You might check that you have the newest version of your ssh client. (e.g. putty 0.67)
* Try with another ssh client to eliminate incompatibility
* Try login without public key, just using password.
* You might also let dropbear to re-generate its hostkeys by deleting the key files from /etc/dropbear:
   /etc/dropbear/dropbear_dss_host_key  /etc/dropbear/dropbear_rsa_host_key
* You might generate a new keypair for yourself.
* And of course, reset all router settings. That can be done from Luci.

(My guess is that the list of accepted key formats has changed since either the host keys or your personal keys were created. Or something like that)
