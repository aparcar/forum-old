Not any SOC Supported *WRT/lede device support near 1Gbit routeing speed with qos (sqm etc.)
HW acceleration support missing.

If you need it only way is a ITX mainboard with new Intel B150/H150 or higher with support linux kernel nic driver with vlan support.

For 1Gbit real speed you need for linux net stack >4Gbit bus speed to nic to cpu to nic again  (Up/Down  + overhead) only chose intel chips can give this performance

And you have to built your own *wrt/lede firmware
