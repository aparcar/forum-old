Solution

ip6tables -t nat -A PREROUTING -i br-lan -p udp --dport 53 -j DNAT --to-destination [fd04:ce7d:2425::1]
ip6tables -t nat -A PREROUTING -i br-lan -p tcp --dport 53 -j DNAT --to-destination [fd04:ce7d:2425::1]

workes well for tp-link 1043v2 trunk.
