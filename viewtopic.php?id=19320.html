if you are using ath5k, these ath0, ath1 names don't exist.  those names are for madwifi and were not generic (ath = Atheros).  now it is wlan0 for all devices (b43, ath5k, ath9k, etc,).

http://www.aircrack-ng.org/doku.php?id=ath5k

what version of aircrack-ng is OpenWrt on? notes say "requires aircrack-ng v1.0-rc1 or newer.".
