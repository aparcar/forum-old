For a single point to point link I would just use AP(WDS).  If you set one of the APs to WDS mode it can support both regular users and WDS links.  Then set up a STA(WDS) on a router that is not wired, to connect to that AP and bridge it into the LAN network.  Use the BSSID option on the STA if all your APs have the same SSID to force it to connect to one particular one.  This STA router can also offer a local AP, or you can use it only as a STA if you just need wired clients in that room.

For guest networks, lately I have been using gretap to tunnel the guests through the LAN back to the main router instead of using VLANs.  It works over AP-STA links, which do not support VLANnning.
