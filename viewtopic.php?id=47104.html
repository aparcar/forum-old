you need to open tcp 22 port on firewall. http://wiki.openwrt.org/doc/uci/firewall

config rule
    option target 'ACCEPT'
    option src 'wan'
    option proto 'tcp'
    option dest_port '22'
    option name 'ssh'

for dyndns install 

opkg update
opkg install ddns-scripts

or luci-app-ddns to configure via web interface. http://wiki.openwrt.org/doc/howto/ddns.client
