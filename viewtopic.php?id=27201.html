Try trunk ar7 build with  kmod-acx-mac80211 (not kmod-acx). I made it some time ago (r23391) and flashed my dsl-g624t. Wifi seems to work, i got a laptop authenticated with dhcp. No idea if it's stable, but would be worth a try?
