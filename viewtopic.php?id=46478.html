Post #4
FriedZombie
27 Sep 2013, 15:15

If the leasetime has run out, the client will do a new dhcp request and suggests the ip it allready has.
The dhcp server then can decline the suggestion and assign a new ip.

but the lease time will be reset again. (also the clients will get most likely the same ip, since there isn't a need for a different ip). This even goes for static leases.
