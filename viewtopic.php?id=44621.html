This may be a simple question to some, but I can't figure it out by myself:

I have a TP-Link wr741ND with the latest version of OpenWRT. It is connected to a router supplied by my ISP.

When I plug in the cable from the ISP router to the wan port, OpenWrt creates its own LAN and puts all the computers connected to it in that network, as it should. The problem with that configuration is that I can't share files between the computers connected to the OpenWRT and the ISP router, because the ISP router creates its own LAN.

What I want to do is somehow configure OpenWRT so that all computers connected to its LAN ports are on the same network as the WAN port, and the OpenWRT router is stil accessible on its own IP adress.

I've tried messing with the DHCP server and VLANs, but the only thing I've managed to do is to make the OpenWrt router invisible, as if it were a switch.

Any help?

You basically want to detach the internal switch and the SoC.

simple:
disable the VLANs
keep the bridge: br-lan, but instead of bridging eth0.1 and wlan0, let it bridge eth0 and wlan0 (if you want WLAN also in the same net)
disable DHCP-server on eth0/br-lan
enable DHCP-client to receive IP from other router
disable nat (woudn't work with one interface anyway ;-))
