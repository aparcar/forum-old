You are doing it the hard way.  You can build the certificates on a computer.  Go to the OpenVPN site and you should find links to EasyRSA or similar if I recall.  I do not recall using the challenge\export passwords.  You do not need to use real data in your key generation.  Make more keys than you think you will use, as you may add more devices in the future.

I prefer this tutorial
https://wiki.openwrt.org/doc/howto/open … rver-setup
