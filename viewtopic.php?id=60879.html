Post #2
hnyman
12 Nov 2015, 11:34

you should save the output of ./scripts/diffconfig.sh
That is your changes to the .config file compared to defaults.

If you save that output when you have a working config, you can then use that file as a seed to a new .config and expand it by "make defconfig" to a full .config

./scripts/diffconfig.sh > saved.config

*** Some changes to menuconfig leading to failure or you just want to lose the changes
*** Recovery with:

cp saved.config .config
make defconfig
