MarkMentovai

Run “scripts/feeds update packages”. Then run “make menuconfig” again. You’ll find tcpdump-mini and tcpdump under /Network.

Post #3

3 Mar 2011, 18:28 
./scripts/feeds update packages
./scripts/feeds install tcpdump
make menuconfig
make
