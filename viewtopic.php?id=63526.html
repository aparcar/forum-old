Hi,

I'm completely new to OpenWRT and I've set up a WDR4300 with OpenWRT Chaos Calmer 15.05.1. I require a captive portal and have chosen nodogsplash for it. It works fine so far, but I've trouble opening outgoing ports for email. I require ports 25, 993, 587 and 465.

In the nodogsplash config file I have:

  list authenticated_users 'allow tcp port 22'
  list authenticated_users 'allow tcp port 25'
  list authenticated_users 'allow tcp port 53'
  list authenticated_users 'allow udp port 53'
  list authenticated_users 'allow tcp port 80'
  list authenticated_users 'allow tcp port 443'
  list authenticated_users 'allow tcp port 465'
  list authenticated_users 'allow tcp port 857'
  list authenticated_users 'allow tcp port 993'

And I've set up a traffic rule like this:

Any traffic
From any host in lan
To any host, ports 25, 993, 587, 465 in wan, accept forward

However, the respective ports are still blocked after the user has accepted the agreement shown on the captive portal. I get ERR_UNSAFE_PORT when I probe portquiz.net with any of the mentioned ports. Port 80 works fine, though.

Can someone help me out with this?
