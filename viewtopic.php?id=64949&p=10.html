Post #238
davidc502
23 Sep 2016, 04:04

From what I understand there is no hardware difference between the 1900acs v1 and 1900acs v2.  I've read where people have been running v1 "shelby" on v2 with no issues.  I'm not saying you should or should not try it, but if you decide to do it, understand you are doing it at your own risk, since there is no official support for acs v2 yet.

Here is some information on Reddit about it.  https://www.reddit.com/r/openwrt/commen … 900acs_v2/
