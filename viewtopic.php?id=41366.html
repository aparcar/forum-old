1. Create a new file with name:

cron1.sh in /etc/ directory

2. In this file paste this:

# Access my Storage area from my computer
chmod 777 /mnt/sda3

3. Make this file executable with: 

chmod +x /etc/cron1.sh

4. Open /etc/crontabs/root

vi /etc/crontabs/root

5. before exit 0 add this:

@reboot /etc/cron1.sh
