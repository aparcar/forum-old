Copy file to your PC:
scp root@192.168.1.1:/etc/config/ssmtp.conf .
(The dot at the end is important, it tells scp to put the file into the current directory on the PC.  If you want it somewhere else, specify a local directory and/or file name instead of the dot).

Copy file back to the router:
scp ssmtp.conf root@192.168.1.1:/etc/config

It can take up to several seconds for encryption to be negotiated, then it will ask for the root password, which is the same one you use to log into LuCI.  I think the commands are the same in a DOS box under Windows.  Be careful editing files on Windows that you don't put DOS style line endings into them.

After changing a config file, go to System-->Startup and restart the service that is associated with it.  Or reboot the whole thing.
