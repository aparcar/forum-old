PowerPC -
Create a forward rule, then use the dropdown box and the add button to add 'protocol' and 'destination ports' to your rule.

Protocol: TCP
Destination ports: 8080
Forward to: <my web ip>

And note that the effects will only be visible for machines connected to the WAN port, in other words you won't be able to connect to the IP address from the LAN side.

---

You can also edit the firewall via /etc/firewall.user if you prefer iptables commandlines.
