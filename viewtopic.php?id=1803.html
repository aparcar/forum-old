Post #5
Duff
18 Jun 2005, 13:26

Try to change the lan ip-address.

Do:

nvram set lan_ipaddr=192.168.10.1
nvram commit
nvram_commit(): start
nvram_commit(): end
reboot

Now you can telnet to the routers ip-address 192.168.10.1.
