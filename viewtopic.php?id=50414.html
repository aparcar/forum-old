Well finally I've decided to stop fucking around and have bought a used Cisco sg300-10, a second network card for a then unused pc, and a TL-WN722N usb dongle to make a wifi AP.
Freebox now in bridge mode, and VLAN100 perfectly handled.

PC with ubuntu server 12.04, hostapd, dhcpd, squid and sarg. Everything is working perfectly now.

A tip for anyone that reads this topic. Having a dlna media server on a network range and a client on another network range is not a good idea.
In my case it was having a network for eth0 and another for wlan0. Routing upnp does not work. I've configured a bridge br0, and now everything works flawlessly.
