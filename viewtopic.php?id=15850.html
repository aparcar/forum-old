You can just comment out these tests in the configure script-

$diff -u2 configure.ORIG configure
--- configure.ORIG      2008-06-01 14:01:40.000000000 -0400
+++ configure   2008-06-01 14:04:13.000000000 -0400
@@ -20771,7 +20771,7 @@
 echo "$as_me: error: cannot run test program while cross compiling
 See \`config.log' for more details." >&2;}
-   { (exit 1); exit 1; }; }
+} #CROSS_COMPILE   { (exit 1); exit 1; }; }
...

BTW- Snort 2.8.2 mipsel packages are available here-

http://packetprotector.org/download/kamikaze/
