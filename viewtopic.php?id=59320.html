One can reset the time using the System=>Startup=>SYSNTPD => Restart.  It takes about a full minute to actually update the time.

The nomenclature here is a nightmare. 

SYSNTPD, which is installed can be called with NTPD at the command prompt (and params).

Installing the package called NTPD, uses the command NTPDATE (and parms).

That's confusing.
