Didn't want to brick this unit but decided to push ahead and compiled BB with r40804 and it did create firmware for the TL-WR710N.

Compiled and flashed it using BB r40804 from-factory image by using the upgrade option of the web interface. loaded luci onto it, but the wireless adapter was not recognised by luci.

Downloaded the latest snapshot image from the web and flashed the upgrade-image via web interface and it works.=)

Now I am trying to set up an alfa 1 AWUS036H on the usb port in client mode with the WR710N in AP mode (requires  rtl8187 and dependencies to be added). This would make a very compact repeater to drag in long distance signals using the Alfa 1 which has a sensitive receiver, for my hotel visits.
