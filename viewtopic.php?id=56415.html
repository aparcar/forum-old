I have a USB dongle (fat32) mounted on /mnt/usb, every time I do chmod -R 777 /mnt/usb the permission doesn't change, it's always set to 755. I want to have full access on my network sharing (/mnt/usb) how should I fix it?

hillz wrote:

I have a USB dongle (fat32) mounted on /mnt/usb, every time I do chmod -R 777 /mnt/usb the permission doesn't change, it's always set to 755. I want to have full access on my network sharing (/mnt/usb) how should I fix it?

fat32 doesn't support file permissions

you could either use the mount options dmask/fmask to set the desired permissions / configure your smbd/ftpd/whateverd properly or just use a different filesystem (like ext4 or so)
