What i did so far:
1. installed openwrt
2. installed luci

5. set up wlan interface to "client", interface: wlanclient
6. created a interface "relay bridge" between "wlan" and "wlanclient"

my clients connect to openwrt and receive ip adress from dhcp. they can connect to luci. but no connection to the internet.

any hints?
