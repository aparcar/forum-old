It works.  When a build for a particular model reaches the release stage, it's been well tested.  It's a generic ath9k machine so it is very similar to other models that have been in use for a long time.

The memory chips in that model are barely large enough, so you have to be careful of running out of memory by installing or running too much stuff.  In particular if you want to VPN you should look at the LEDE forum's community builds for 4/32 machines.
