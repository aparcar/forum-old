I just updated a router to this version and got a solid yellow light on my router.

If you want to unbrick via linux there is a sure way until Devels can improve the trunk firmware.

- Open two terminals

In terminal 1:
telnet 192.168.1.1
cd /dev
mtd erase mtd1

in terminal 2:
tftp 192.168.1.1
trace
verbose
binary
timeout 120

GO to terminal 1 and enter:
reboot

got to terminal 2 and enter:
put WNDR4000-lateststockfirmware.chk

WAIT for lights to return to normal and you're good to go.
