I have had a similar experience when I first started using OpenWRT but with time and patience I found the current arrangement productive.
There are tools such as Google Translate and others that allow me to get the gist of the code or implementation from others.

The danger I see in different languages is fragmentation.
Each group will stay in their own forum. This will reduce the amount of exposure to others and ideas.
It could reduce the richness and breadth that I currently see in these forums and perhaps the very strength of  OpenWRT/opensource.

On a constructive side perhaps a tool for translation or something similar or a better search tool.
