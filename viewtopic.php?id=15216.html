bunyman wrote:

I'm looking for informations on the wiki about port of OpenWrt on Linksys PAP2, and nothing.

Which version of PAP2? For a Linksys PAP2v1, you can forget about it.

For a Linksys PAP2v2, there may be a chance since its hardware resemblances to the hardware used by a Linksys RTP300 (minus the switches). You certainly can try to hack a Linksys PAP2v2 to run OpenWRT, but the end product will probably won't be fast mainly because the CPU is clocked @180MHz if not less. OTOH, you will probably need to port the Legarity voice module GPLed source to OpenWRT too so that it will be able to drive the PAP2v2 voice module.
