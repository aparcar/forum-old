Hi Everyone,

Hopefully someone can offer me some advise on this router, as I'd like to start making more use of it.

The Tecom AH2322W was distributed in Spain by Tele2, and I was given one recently. The stock firmware was not brilliant, and certainly the VoIP facilities were very limited.

The CPU is labelled as a Broadcom BCM6358VKFBG, which I understand is an SOC, integrating MIPS CPU, ADSL, VoIP and USB. There are also WiFi extensions in it as far as I know, and thats backed up by a BCM4318KFBG chip on the main board (not a daughter card) and that shows up with an lspci. The board ID shows as a 96358VW.

There is one external USB B port, which was used to connect a PC over usbnet, there are 2 LAN ports, a line in (FXO) and a line out (FXS). There is a 4mb flash (actually I think the physical chip is 8mb), and there is 32mb memory.

I'd like to achieve a few things and wondered if anyone can offer any pointers please?

I've flashed it with openwrt-96358GW-squashfs-dsl2740b-cfe.bin, and it boots fine. There is no ADSL, but as I understand it, Broadcom have not GPL'd their ADSL drivers? I have a spare modem, so thats not a concern. Eth1 picks up an IP address from the DHCP server on the LAN, and I have installed and accessed Luci.

I've tried installing kmod-usb-uhci and kmod-usb2. Both install and insmod, and I can see they are loaded. lsusb shows no devices connected though, and neither the (self powered) hub or devices on it wake up. A quick dmesg shows 2 ports detected, so any ideas why nothing is being detected on them?

Also, I really want to get the VoIP functions running, obviously I'd like to pivot boot at some point and use the USB disk as root so I can fit Asterisk on, but at the moment I'd be happy to have Asterisk on another machine on the network with the Tecom just providing FXO/FXS services over the network. Should I be looking at anything in particular to get these ports operating?

Many thanks for any help you can come up with, it's really appreciated,

Ian B
