Hi Everyone, I have a quick question. I have used OpenWrt to setup a router for my home, including a guest network that I setup almost exactly following the wiki instructions (https://wiki.openwrt.org/doc/recipes/guest-wlan). What I would like to allow is to open the firewall so that guests at my house can access my Plex server, which is accessible on my lan interface via https://192.168.2.2:32400. Any advice on what firewall rules I can use to make that happen? Thanks a lot.

Post #2
stangri
13 Jan 2016, 09:09

add the following to the /etc/config/firewall before the rule denying guest lan access:

config rule
        option name 'Allow Guest Plex Access'
        option src 'guest'
        option dest 'lan'
        option dest_ip '192.168.2.2'
        option dest_port '32400'
        option proto 'tcpudp'
        option family 'ipv4'
        option target 'ACCEPT'
