'vim' is an alias to 'vi':

root@wlanrouter:~# alias
more='less'
vim='vi'

Hmm, my /etc/profile has this line in it:

[..]
[ -x /usr/bin/vim ] && alias vi=vim || alias vim=vi
[..]

So, vim should be executed if /usr/bin/vim exists.
