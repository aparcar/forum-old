This is similar to what I have setup, I have one WZR-HP-G300NH running as a WDS AP and three WZR-HP-G300NH's running as WDS Stations, creating a 'wired' transparent bridge through the house. Since the entire location is within range of the primary router, there is no need to do any rebroadcasting of the wireless signals, which lowers the bandwidth available. Let me know if you need any more information.
