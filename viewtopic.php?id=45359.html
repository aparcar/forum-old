If you goto firewall, zone settings there should be a Zone=>Fowardings saying something on the lines of
"GuestWifi"=>"Wan"

And it should work.

To start with have default rules
"Input" - "Accept"
"Output" - "Accept"
"Forward" - "Reject"

So you have no issues with DHCP, and then later narrow it down "Input" - "Reject" and custom traffic rules for DHCP etc.

I also found issues at first setting up with the DHCP, a mixture of router reboot and deleting the DHCP leases on the Android device I was testing with sorted it.

James
