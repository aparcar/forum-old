The wrt1900ac and wrt1200ac are more or less the same creature, just the 1900 has a couple of more antennas on it.

What TFTP server are you trying to use?  Do you have anything else that is trying to use port 69?

To answer your question about the 24, that's the number of bits that are 'on' reading from left to right.  For every eight bits that are on, you have a value of 255.  That's why /24 means "255.255.255.0" because 24 of the 32 bits are on (set to '1').  It gets more complex than that for understanding how it works, but in short, you will commonly find 8/16/24 when done in the short notation (which is 255.0.0.0 for 8, 255.255.0.0 for 16 and 255.255.255.0 for 24).  Without getting all techie on you, it helps to define the network you are a part of.  With a network of 192.168.0.0/24, everything up to 192.168.0.255 is part of the same network, with 192.168.1.0 being a different network.  If it's 192.168.0.0/16 (255.255.0.0) then everything up to 192.168.255.255 is part of the same network.  That's a jump from 256 IP addresses (with only 254 actually being usable) to 65,536 addresses (with only 65,534 being usable).  If it doesn't make sense, don't worry.  Just know that it serves a purpose and there is a method to the madness.
