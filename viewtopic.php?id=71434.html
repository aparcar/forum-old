Patland wrote:

In the WAN interface, I have choosen Protocol Static address, IPv4: 192.168.2.12, IPv4 netmask: 255.255.255.0, IPv4 gateway: 192.168.2.1 .
With my prevoius Synology router and these parameters, I could connect to internet through the modem. With Linksys WRT3200ACM, it's not possible. Is there some parameters I forget.

You should likely define also DNS server in addition to the gateway. Unless you define that, the 3200ACM has no way to look up domains and clients connected to it get also left DNS-less.

You should probably put 192.168.2.1 there, as the modem likely acts as DNS server. Or input the same ISP server values that the modem has got via DHCP from ISP. (or put some open DNS server)
