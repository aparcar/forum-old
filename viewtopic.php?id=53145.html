Post #10
JonnyM
7 Oct 2014, 07:52

No, don't  use vendor/product with usbserial.

usb_wwan uses one single function of usbserial and could not access it:

"Sep  8 16:17:04 OpenWrt kern.warn kernel: [   10.800000] usb_wwan: Unknown symbol usb_serial_port_softint (err 0)"

so usb_wwan could not load which later on makes option driver complain over not being able to access usb_wwan functions.

You only need to modprobe (or insmod) the drivers in the right order:

modprobe usbserial
modprobe usb_wwan
modprobe option

check dmesg or with lsmod that the modules gets loaded without errors.
