ledo wrote:

example: i dont need a .trx version when revert to stock from openwrt but i need a .trx file for upgrade between two openwrt version to preserv settings. right?

No.
Old brcm47xx versions (up to 14.07 included) had support for .trx files only. No matter if you were reverting to the original firmware or doing sysupgrade.
New brcm47xx versions (CC and newer) have support for Linksys (.bin) and Netgear (.chk) firmwares. For both: reverthing to original and sysupgrade.
