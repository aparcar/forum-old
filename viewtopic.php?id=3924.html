you might want to use telnet to get into it.  have you read up on how OpenWrt works?  it's not something you can just flash and use a web frontend to change settings.  if you didn't use tftp and don't know how to use tftp, it's going to be difficult for you to get your old firmware back.  what version of OpenWrt did you install?  RC4 should have a web interface. if it is RC4, you may need to run firstboot when you telnet to it.  if you haven't read the documentation, now would be a good time to start as well.
