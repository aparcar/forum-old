Both the OpenWrt and X-Wrt 10.03 x86 release images should be suitable for installing within Vmware.

Download the ext2 image, gunzip it and convert it to vmdk with qemu-img:

qemu-img convert -O vmdk openwrt-x86-ext2.image openwrt.vmdk

~ Jow

If you build it yourself you can build luci right into it, you might be able to build xwrt/webif into it these days too. If you can't/won't build it yourself you can still make it happen. Just give your new OpenWRT guest access to the Internet and run "opkg update" followed by "opkg install webif" or "opkg install luci-admin-full".
