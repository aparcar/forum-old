The you can amend your /etc/config/firewall, locate the lan zone definition, add "option device ppp+" and set the forward policy to ACCEPT.

If you want to stick your pptp ifaces into a dedicated zone, declare a new one like this:

config zone
  option name vpn
  option input ACCEPT
  option output ACCEPT
  option forward ACCEPT
  list device 'ppp+'

And allow forwards from/to lan:

config forwarding
  option src vpn
  option dest lan

config forwarding
  option src lan
  option dest vpn
