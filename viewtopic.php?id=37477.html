Try disabling internal antenna:

uci set wireless.radio0.rxantenna=0x1
uci set wireless.radio0.txantenna=0x1
uci set wireless.radio0.diversity=0
uci commit wireless
reboot

Your older router, may not comply with your country regulation. Your new router does.

EUA wrote:

I don't aware if TL-MR3220 has internal antenna also...

Your router is aware:

iw phy phy0 info | grep -i ant
    Available Antennas: TX 0x1 RX 0x3
    Configured Antennas: TX 0x1 RX 0x3

EUA wrote:

Anyway, issuing commands at top looks doesn't solve the problem.

After issuing commands at top your router is instructed to use only the external antenna:

iw phy phy0 info | grep -i ant
    Available Antennas: TX 0x1 RX 0x3
    Configured Antennas: TX 0x1 RX 0x1

Now, unscrew the external antenna. YMMV

EUA wrote:

My both ADSL modems has much higher signal than MR3220.
Don't think if It's something related with country regulations.
