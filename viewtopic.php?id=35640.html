Thanks!
Reading the config file told me I had completely failed to notice that the "Target Profile" selection scrolled down! Doh! Having actually selected "TL-WR703" you get a valid build.
I have now successfully done two things:

1) Built a firmware image that was too big, necessitating soldering a connector to the serial pins in order to unbrick it (if the initial boot can't create the jffs filesystem, the network won't start).
2) Built a firmware that includes all of the features I want, and still have 700k left in /overlay!

So, thanks for the catalyst :-)
