I will answer myself.

You need openwrt-brcm-2.4-jffs2-4MB.trx image.

1. Start wl-hdd in faisafe mode (power off, press reset button, power on, release the button, the power led should be blinking)
2. Start Firmware Restoration tool (it's on WL-HDD CD), and try to upload the firmware - it will fail, that's fine
3. Now you can upload the OpenWRT image with tftp

You have to do step 2, otherwise you won't be able to upload the firmware via tftp in step 3.
