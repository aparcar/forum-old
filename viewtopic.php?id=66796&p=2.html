No one here tries to modify manufacturer's firmware.  The whole point of OpenWrt is to erase the manufacturer's firmware and completely replace it.

In OpenWrt if you want to change a logo or a bit of text or something, there are two approaches.

First you can do it while the router is running by going into the filesystem and changing a file.  This is almost never possible on manufacturer firmware because it won't let you have a command prompt and even if it did, the filesystem is read-only.

The other way is to modify it in the source code then build a new binary image and install it on the router.  This is seldom possible on manufacturer firmware because they won't give you the source code. No one here tries to break up a binary firmware image and modify something and put it back together, because the source code for OpenWrt is available.
