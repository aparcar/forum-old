I am a big fan of luci-app-sqm. It seems to do a better job of providing fair bandwidth sharing and consistent and low latency for all kinds of traffic than the built-in QoS of OpenWrt. See https://wiki.openwrt.org/doc/howto/sqm for details.

I also wrote a set of scripts for consistent configuration of OpenWrt (and collection of diagnostic info) that are at: https://github.com/richb-hanover/OpenWrtScripts

And if you also install netperf, you can use the betterspeedtest.sh script to measure bandwidth and latency at the same time.
