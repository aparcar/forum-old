arty.net wrote:

After downloading the tarball of the sources from kernel.org, I put in one folder the uncompressed kernel sources and the generic-3.18 and ar71xx-3.18 patches.

There is no need to do this manually. OpenWRT buildroot scripts will automatically do this for you.

Then after browsing into the kernel sources folder, I run a simple for control loop --dry-run one-liner :

for i in ../generic-3.18/*.patch
do
    patch -p1 --dry-run < $i
done

If the patches are original from OpenWRT, then again OpenWRT buildroot will do the patches automatically. If you wanna create your own patch, I strongly suggest you read this Working with patches and be familiar with quilt.
