try /tmp/resolve.conf.auto (make sure it has auto on the end and not just conf, both are present in that folder)
but it will reset on boot, not sure how to set it permanently, thats the file i use anyway for pushing my vpn's dns

you could also try /etc/resolv.conf but irc i think i might have had problems making that persistent aswell
also i think /etc/config/network might be another place to set it, using 'option dns 8.8.8.8' under one of your networks
