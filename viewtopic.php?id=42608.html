Started to find lots of useful detailed information now. I guess the most important discovery is that there's more than one way in which a new firmware can be developed.

In a thread on this forum about a year ago another router with the 6358 chip was discussed. It was the ct-6373.
https://forum.openwrt.org/viewtopic.php?id=34707

The OP wanted to know if an OpenWRT f/w would ever be available. Just a few months later it was all done and dusted. And at least one other forum user has since flashed their router with the image. At that time (it seems) nobody other than the OP had physical access to the ct-6373 hardware.

Some information about the MBRN3000 which might be helpful :
CFE board ID : 1.0.37-102.9
source:   https://forum.openwrt.org/viewtopic.php?id=35153
(this person was referring to DGN2200v1 I think)
Flash chip: (maybe) Macronix MX29LV640DBTC-90G
source:   http://wikidevi.com/wiki/Netgear_DGN2200v1
(I wish to confirm this brand)
For info, the wikidev for MBRN3000 is here :
http://www.wikidevi.com/wiki/Netgear_MBRN3000
Flash is 8MB, RAM is 32MB (but actual chips not named)

I haven't opened up my router but if that's necessary maybe that's what needs to happen.
All help appreciated !
