You can a) create your own image including NFS, SMB,... using the ImageBuilder. This is best because it compresses everything and makes it read-only. You can b) still write to the system though because OpenWrt uses an overlay filesystem, which means that your writes are transparently stored in the writeable /overlay partition and are visible as normal in /.
