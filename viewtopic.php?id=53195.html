Dear all,

A n00bish question, I'm sure. But I did search, and did look in the docs, but didn't find.

I have 2 identical APs (TP-LINK 4300's) with 14.07 BB installed. One is set as "Access Point (WDS)", and the other as "Client (WDS)". This works great, and stationary computers on cabled ethernet attached to my "Client (WDS)" bridge work brilliantly.

But, how can I assign a static IP address to the bridge "Client (WDS)" management interface, so that I can access LuCY and manage it?

When the "Access Point(WDS)" is turned off the "Client (WDS)" assumes IP 192.168.1.1, and thus I can access it and manage it. When the main "Access Point(WDS)" is on, it owns 192.168.1.1, and thus I can only manage the main AP then. But I'd like to also be able to manage the client part of the bridge while the main AP is turned on.

Thx!

Post #2
drawz
8 Oct 2014, 01:24

In Luci, go to Network->Interfaces->LAN then edit settings and you can set a static IP address there. I'd use 192.168.1.2, but anything outside of your DHCP range would work fine.
