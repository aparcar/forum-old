Post #5
jow
15 Oct 2013, 21:21

For this kind of changes to the ui I suggest working on a real target, e.g. an OpenWrt x86 qemu or VirtualBox image, then edit the gui files remotely, e.g. through SSHFS.

The SDK for the PC only provides a very minimal emulation of an OpenWrt system, many things will not work properly there.

Personally I use Komodo Edit and modify the .lua files directly on the Live OpenWrt system - you can find the files in /usr/lib/lua/luci/ on the target.

Since it is mostly plain .lua files or HTML / CSS there is no need for a compilation phase.
