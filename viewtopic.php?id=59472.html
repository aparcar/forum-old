But "sudo root 192.168.100.1" means: become root in this machine, and execute command "root" with parameter "192.168.100.1". Thus that password you are being required is the root password on your own machine; and even if you type it, next error will probably be "root: command not found".

I guess you meant to execute "ssh root@192.168.100.1".
