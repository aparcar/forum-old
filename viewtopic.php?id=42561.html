Hi,

here's situation:

Hardware: Siemens gigaset sx763 (DSL modem)
OpenWrt: OpenWrt Firmware Attitude Adjustment (r31118) /LuCI Trunk (trunk+svn8380)

Current setup: 2 VLANs, 1 for Internet and LAN (eth0.1), 1 for IPTV (eth0.2)

What I want is to add PPTP client to additional VLAN (only one physical LAN port). Possible? How?

If I create PPTP interface, all traffic goes trough PPTP. I cannot bridge PPTP with eg. eth0.3.

Regards

2 Mar 2013, 03:18

You cannot bridge something that isn't ethernet to an ethernet connection. You will need to route it.

I would set up an additional subnet on VLAN3 and have the router not bridge that to anything, then set up firewall rules to route traffic to/from the router's VLAN3 IP over the PPTP connection.
