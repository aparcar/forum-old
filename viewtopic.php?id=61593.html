I successfully flash stable version Chaos_Calmer 15.05  via tftp on my WNR2200. (I run before "mtd erase linux" in dd-wrt to get recovery mode)
Now I want to re-flash custom openwrt image  (with USB support) or any other image.
Sysupgrade method does not work (wiki openwrt).
I can not boot into recovery mode - reset button does not work (as described in wiki openwrt). Power led is always yellow and never blink.
wnr2200 is always boot in normal mode.
How to force boot recovery mode?
Will serial connection help me? Is it the only way?

This (not updatable) is fixed in latest openwrt sourcecode. Reason was:

https://dev.openwrt.org/changeset/48005

You installed your first image over tftp. Your updated images also could be installed over tftp .
For older images: Just backup before your config and load it again after flash.
