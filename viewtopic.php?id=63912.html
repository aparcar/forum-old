Have you tried using LuCI to set this up? I have a router setup with 2 VLANs (on separate ports) using LuCI. It wasn't that hard.

The trickiest part will be to not cut yourself off by reconfiguring the switch. I suggest you attach to VLAN 1 (the default) on port 1, and reconfigure the rest of the router from there.

HTH
