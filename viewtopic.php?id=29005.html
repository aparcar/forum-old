jow
10 Mar 2011, 17:17

Don't reference wireless ifnames in the network config. Their naming and ordering is not guaranteed in any way and additionally the hotplug handlers are written to ignore any wlan* in the network config.

Replace "option ifname wlan1" in the network wan config with "option network wan" in the wireless wifi-iface sta config.
After that recheck how the system behaves.
