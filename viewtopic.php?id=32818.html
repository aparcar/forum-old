You could use a VPN or SSH tunnel from your android phone to for example a wifi-router with OpenWrt firmware installed. Install a OpenVPN server on the router (or any computer which is on 24/7), and install a VPN client program on your smartphone, and make a VPN connection when using open wifi hotspots. A SSH tunnel might be easier to set up than VPN.
