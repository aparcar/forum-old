Privoxy in step one is a http filter proxy.
For socks5 proxy there are smaller/better solutions on the market.
If you really want to use Privoxy as socks5 proxy you should look at Privoxy's forum for configuration samples.
Have a look on CPU usage of Privoxy because on a router it can only handle a very small number of clients if running as http filter proxy.
