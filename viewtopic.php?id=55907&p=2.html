Post #44
mk24
18 Feb 2017, 02:56

Do you have good RF signal between 1-2 and 2-3?  Consider that they have directional antennas, so making a chain with one in the middle that can relay to both ends can be a problem.

Assuming the signal is OK, these are ath9k devices which have a very good mesh mode built into the open drivers so no problem to make it work on OpenWrt.  A mesh treats the units as equal peers so they are rather seamlessly bridged together.

Christoph, thx for mentioning the "EU" in the string - flashed it.

Apocalipsi, to get there, edit the mentioned file in your OpenWrt tree,
openwrt/tools/firmware-utils/src/tplink-safeloader.c
and add the following entry
"CPE210(TP-LINK|EU|N300-2):1.1\r\n"
somewhere amoungst the other similar entries.
Then recompile.
