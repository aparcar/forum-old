Post #6
vilpalu
6 Aug 2015, 08:59

firstly install https://github.com/openwisp/OpenWISP-Manager it's controller software.
secondly chose your wifi ap platform based on ar73xx. there provided fw package for wifi ap https://github.com/openwisp/OpenWISP-Firmware .
thirdly you may install openvpn on your controller or on another server ff you want use tunnel for connection between ap and controller. if you do not want use openvpn tunnels you must change fw package scripts for connecting directly to controller.
also if you interested in more details I can translate some my documentation in English.
