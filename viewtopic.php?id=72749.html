WWTK
10 Dec 2017, 17:29

it is normal for some packages to fail during compliation, especially if you build everything.  If that particular package failed and you specifically want it, try posting to the developer forum here or on the lede board.

If it isn't one you need, either deselect it in make menuconfig, or do your make with ignore errors.

IGNORE_ERRORS=nn make

where nn is some positive integer
