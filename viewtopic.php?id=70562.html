If you flashed a trunk or "snapshot" build it does not contain the web configuration system.  You can use the command line to sysupgrade to a release build.

Connect your PC to the Internet and download this file:
https://downloads.openwrt.org/chaos_cal â€¦ pgrade.bin

Now connect your PC back to the WR941 and scp it to the router.
scp <path to the file you downloaded on your PC> root@192.168.1.1:/tmp/firmware.bin

Then log into the router and flash the file.
ssh root@192.168.1.1
<router prompt appears>
sysupgrade -n /tmp/firmware.bin

When the flash completes and the router reboots, you should be running a Chaos Calmer install with the web interface operational at 192.168.1.1

The wiki page for this model https://wiki.openwrt.org/toh/tp-link/tl-wr941nd has instructions for reverting to stock TP-Link.

You do the scp while logged into your PC's DOS command prompt not the router's.  So the scp command runs on your PC and pushes the file to the router.  When you next log into the router the file will be there in its /tmp directory.

Also, I'm not sure how the DOS scp works with path quoting, it would be best to change directory to your C:\ directory and run the command with just the file name.
