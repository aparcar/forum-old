Indeed it needs 3,3 V in order to work properly. Loopback and serial communication with the router works perfect now. Here comes the color code for the genuine Nokia DKU-5:

Yellow: Irrelevant
Red: 3,3 Volt
Blue: Rx
White: Tx
Black: Ground
