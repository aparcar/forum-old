4 May 2012, 23:47

When the camera gives an uncompressed stream format YUV, for it to include JPEG-compression. Edit the file / etc / init.d / mjpg-streamer. We find the demon's command line:

$PROG -- --input "input_uvc.so --device $device --fps $fps --resolution $resolution" --output "output_http.so --port $port" &


Add the option - yuv to plug input_uvc :

$PROG -- --input "input_uvc.so --device $device --fps $fps --resolution $resolution --yuv" --output "output_http.so --port $port" &


Start the daemon:

$ /etc/init.d/mjpg-streamer start
