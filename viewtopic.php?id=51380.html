2 Jul 2014, 03:11

Remove the line:

        option ifname 'eth0'

Add the section:

config interface 'wan'
       option ifname 'eth0'
       option proto 'dhcp'

And modify as appropriate.

Post #9
phantanphu
2 Jul 2014, 05:07

I edited /etc/config/network like this:

config interface 'loopback'
        option ifname 'lo'
        option proto 'static'
        option ipaddr '127.0.0.1'
        option netmask '255.0.0.0'

config interface 'lan'
        option type 'bridge'
        option proto 'static'
        option ipaddr '192.168.1.1'
        option netmask '255.255.255.0'
config interface 'wan'
       option ifname 'eth0'
       option proto 'dhcp'

Ethenet port worked as WAN, thanks all so much.
