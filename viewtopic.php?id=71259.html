Since it's not possible to compile nano on OpenWrt with syntax color support, I've been compiling vi with it instead, however I came across Amir Salihefendic's VimRC project on GitHub, which I highly recommend utilizing.

  • While nano is definitely more user friendly, the lack of syntax support on OpenWrt is a major downside.

    • I've saved hours in troubleshooting due to syntax support, as it makes it far easier to spot an error, such as a forgotten closing operator, as well as making parsing a script or config fast & easy.

I believe the full version of vim must be compiled for runtime support, of which is easy enough for me to host on GitHub for users who don't have their own build environments. 

  • Amir's runtime folder is just shy of 22MBytes in size, so it will need to be saved to USB storage. 

    1. Edit: /mnt/sda1/.vim_runtime/install_awesome_vimrc.sh

    2. Change: ~ to /mnt/sda1

    3. Copy: cp /mnt/sda1/.vim_runtime/.vimrc ~
