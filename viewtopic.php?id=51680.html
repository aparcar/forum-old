I came across a Luxul XAP-1020 and discovered it runs MIPS OpenWrt Linux-3.3.8.  So I booted the device into failsafe mode, telneted into 192.168.1.1 changed the root password, and now I have access to ssh.  According to the opkg.conf file it runs attitude adjustment 12.09-rc1/ar71xx/generic. Packages will not install.  They complain missing dependencies. 

Just thought I'd share in case anyone else had one too.
The chip is an Atheros 7240-atha, 127009-1b
