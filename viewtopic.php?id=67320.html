According to the Table of Hardware the WRT610 v1&2 are only available in trunk, so i'm just guessing that maybe you mean there is no web interface to access at 192.168.1.1, if so you need to install Luci. Here's the wiki if you want to read up on it -

https://wiki.openwrt.org/doc/howto/luci.essentials

Establish an internet connection, ssh into your router, run 

opkg update

and then 

opkg install luci

At that point Luci will be installed and you can access it at 192.168.1.1
