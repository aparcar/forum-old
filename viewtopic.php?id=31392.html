chhatc wrote:

What is the suggested way for me to install this package and replace the one that is provided by the openwrt packages repository in "snapshots/trunk"?

This has worked for me:
1) Remove the old package installation either with opkg from console or with LuCI.
2) copy the new asterisk package and the index file (Packages.gz) to some location on the router
3) edit /etc/opkg.conf to point to the correct location (e.g. file:/mnt/usb instead of the official snapshot location http:/xxx.xxx)
4) update opkg package list. It should read your own index and show the new asterisk package on the list.
5) install the new package
