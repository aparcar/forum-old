Hello all.  I recently got a WRT54GS v1.0 from a thrift store for $5 and put OpenWRT on it as soon as I got it home.  One planned use for it is simply as a switch, connected to my 1043ND v2.0.  I was just about to assign the WAN port as a LAN port, but noticed it would not allow me to install packages over a LAN port.  Packages installed fine when using the WAN port (as a WAN port).

So, is there a way to install packages over LAN as opposed to WAN?

Thanks for any help

Post #2
dem
27 Nov 2014, 02:06

You probably just need to set your default gateway and DNS server on your WRT54S since normally these would be set via the WAN port (via DHCP for example)

If your 1043ND is your WAN router then just set the gateway and DNS server on you WRT54S to the LAN IP address of the 1043ND assuming that it is acting as a DNS server too (as in standard configuration)
