Hi forum...

I have collected information on this Zyxel Router, in the hope that OpenWRT will try to support it...8) 8)

When i tried to telnet to the router i discovered that the NBG-5715 is actually running OpenWRT,
so i thought that it might just be possible to run a new version of OpenWRT on it as well somehow  :?:  :?:
The OpenWRT that the router is running now, is a version 6.01 (the router say) but it is with a Zyxel-adapted LUCI web-interface on top...!?!

I have now collected a lot of informations on this router...
If more is needed, i will try to answer...

The FCC ID is I88NBG5715

I found internal pictures:
https://apps.fcc.gov/oetcf/eas/reports/ViewExhibitReport.cfm?mode=Exhibits&calledFromFrame=Y&application_id=193009&fcc_id='I88NBG5715'

I also found the sources for OpenWRT inside the router...
ftp://opensource.zyxel.com/NBG5715/
