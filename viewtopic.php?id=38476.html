What actually was the problem? These ports are not opened by default. And the web configuration (LuCI) has exactly what you need to open/close ports.

* ssh and http access shouldn't be opened by default but if you have it you can easily remove it using the Network -> Firewall -> Traffic Rules
* ssh access can be disable via System -> Administration

And for port forwarding, Network -> Firewall -> Port forwards
