in failsafe mode being able to access via telnet is by default: http://wiki.openwrt.org/doc/howto/generic.failsafe
But if you can access device in failsafe mode, before you reflash the whole thing I would recommend you to try the command:
firstboot
This will restore the default settings to the device.
