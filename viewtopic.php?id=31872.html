if you have "Image builder/generator",(just extract)  it can make new image with your packet which include to it with your command
command in terminal:
$ sudo make image PROFILE=<typerouter>  PACKAGES="" FILES=files/

<typerouter> can look with command
$ make info

you can include packet into image by PACKAGES="uhttpd webif" = icluding uhttpd and webif into image

sample:
$ sudo make image PROFILE=RB450G  PACKAGES="uhttpd webif"
