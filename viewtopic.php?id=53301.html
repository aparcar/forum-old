The advice about make oldconfig sounds a bit strange, when you are doing a new build.
1) Delete .config
2) do "make menuconfig" and select your architecture (ramips) and then the router. Exit menuconfig and save changes.

MD5 values are build-specific. You will never get the same values as the official build.

EDIT:
I am not sure if your architecture/router uses separate factory/sysupgrade files. That may depend on the embedded flash routine's checksum checks. There are no factory.img files for rt288x here: https://downloads.openwrt.org/barrier_b … ps/rt288x/
