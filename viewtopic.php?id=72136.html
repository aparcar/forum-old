Yes this is called a routed client.

Install OpenWrt.
(Network -- Interface (lan) Change the LAN IP to 192.168.2.1.  Reconnect your PC to the router with the new address.
(Network -- Wireless) Add a wifi client to connect to the other router.  Enter the encryption settings and key.
(Network -- Interface (wan) -- Physical) Attach this client to the wan network.  A wifi client cannot be in a bridge, so remove the Ethernet port from the wan network.

Of course you need adequate signal strength from the other router.

The DAP-1160 cannot run OpenWrt due to it having an unsupported Realtek CPU and only 16 MB RAM.  You may be able to do client mode with its stock firmware.
