I found all the documentation needed on repoflum's site. The JTAG dongle was home made, 5 resistors, wires from and IDE cable and a LPT socket.
http://wiki.openwrt.org/doc/hardware/po … unbuffered

The software is called alice03bis.exe, also from repoflum's site. Watch out, there are 2 versions, alice03.exe and alice3bis.exe. The second one is newer and is the only one that worked for me.

http://beghiero.myftp.org/modifiche/ali … 03_bis.zip

This is an console app. The CFE files (CFE.BIN, NVRAM.BIN, etc.) must be in the same folder as alice.exe

More info here:
http://www.ilpuntotecnicoeadsl.com/foru … 952.0.html

It's in italian, but Google Translate works pretty well.


P.S. I needed about 2 months to make this work.
P.S. Watch out for the JTAG connector layout on the PCB. In the second version they swaped the two columns. Google has lots of info about that. In italian.
