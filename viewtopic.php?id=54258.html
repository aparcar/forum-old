First set your router password to something hard to guess, and confirm on the status screen that your wan has a public IP.  If you are using DSL, you may need to use pppoe to connect to the DSL to get a public IP.

Then go to
Network -- Firewall -- Traffic Rules -- Open ports on router
Open port 22 (TCP) for ssh and port 80 (TCP) for web.  Never open a telnet port to the Internet because there will be no security. 

If you have open only the ssh port, you can ssh tunnel to port 80 to access the LuCI securely.  On your remote machine, use the command:
ssh root@<router public address> -L 8080:192.168.1.1:80
(If you have changed the router's LAN IP from 192.168.1.1, use whatever you changed it to)
After logging into the router, leave the ssh connection open and take your PC's web browser to http://localhost:8080 and it will be the router's web page.
This can also be used to administer printers, VoIP ATA's, etc. on the LAN that have a web interface.

The wiki has some advice on techniques to improve security.
