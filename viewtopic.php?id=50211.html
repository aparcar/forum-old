Hello.

I'm trying to wake on lan my desktop PC connected to OpenWRT router (DIR-320b1).
PC is configured correctly and I can wake it from notebook wired to OpenWRT box running wakeonlan "00:1D:67:2G:9A:F1"
I'm even can wake desktop from Android phone connected to OpenWRT box via WiFi.

But I'm unable to wake desktop from router itself (i.e. connecting to it via Telnet/SSH and running /usr/bin/etherwake -i br-lan -D "00:1D:67:2G:9A:F1" or wol "00:1D:67:2G:9A:F1"

Looking to router I can see that while it run etherwake or wol - Ethernet LED not blinking, but when I'm trying to wake from another device - it blinks and PC turns ON.

Tried etherwake on notebook - it doesn't work too (but led of desktop's ethernet port have blinked).

What can be wrong with my setup?
Thank You.

I have never used wakonlan. But try using the actual eth0.1 lan interface and see if it makes any difference.

uci get network.lan.ifname
