killerno wrote:

Hi,
thanks for the answer. so to reach the 192.168.1.1 network i have to set up a route for 192.168.0.2 on the adsl clients. How can i do that? By arp command? sorry for the newbie question.
thanks

That depends, what system Your ADLS clients are running.
If it's win32, You would use the following command:

route -p add 192.168.1.0 mask 255.255.255.0 192.168.0.2

If it's linux, you would use:

route add -net 192.168.1.0/24 gw 192.168.0.2
