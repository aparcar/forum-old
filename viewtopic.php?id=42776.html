Here are the free pin :

the three which are not readable are from top to bottom : 7,5,6.

And the other pins and matching :

GPIO Number - assignement - Identification on board

4,8      -    power led green, red      -     LED101 on board
11,2    -    Internet Led green, red  -     LED110 on Board
3         -    Adsl Led                          -     LED109 on Board
9         -    WPS Led                         -     LED107 on Board
24       -    WPS Button                    -     SW703 ( Warning : C750 near switch should be removed to free gpio NOT TESTED !!)
10       -    Wireless On/Off              -     SW704 ( Warning : C749 near green capacitor to be removed NOT TESTED !! )
23       -    Reset                              -     SW201

For the led, use the + sign pin to access corresponding gpio.

So if you uninstall led package :  10 free gpio pin, and with button packages removed : 13 !

Serial console : 155200,n,8,1

Serial console pin : GND - RX - TX - 3V3 (  the 4 wires on the picture )

OpenWRT flash : I use latest trunk ( own build ) : openwrt-DSL274XB-F1-EU-squashfs-cfe.bin

This board is not recognized by OpenWRT.
the trick :
- Boot with serial console,
- stop boot by pressing ESC.
- on prompt : CFE> type "b" to change board parameters. First choice is Board ID, choose 2 ( AW4339U )
- leave all other parameters with default configuration.
The board will have AW4339U code which is recognized by OpenWRT.

Attention : the switch will not work, in my use i don't care because i use WIFI
