Hey there,

You could try using rm with the recursive command '-r'.  The usage might be as follows, but maybe you should back it up first.

cp -r /etc/ppp/ /etc/pppBACKUP/
rm -r /etc/ppp

That should backup the directory ppp and remove the original.  Now if you reinstall anything it should recreate the ppp directory.

Before deleting make sure you are really deleting what you want.

I found out what the problem was.

Apparently when doing # rm -rf /etc/ppp, the shell thought for some reason that everything was an option.

Doing # rm -rf -- /etc/ppp worked.

Thanks to everyone who submitted an answer!
