1) YES! Use the linksys gui to upgrade! (there's no other SIMPLE way)
2) Yes,there's a gui, but i suggest to upgrade the gui with webif^2 (www.x-wrt.org)

Finally i suggest you (for the first time use) to install directly the image of Openwrt from x-wrt (the firmwares hosted on that site already come with webif installed).

It's up to you.... standard openwrt + your work to install webif or  openwrt with webif preinstalled (but I don't remember if it's the latest version of openwrt).

Happy OpenWRT'ing!
