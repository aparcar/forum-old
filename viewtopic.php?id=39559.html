The patch is only for if you compile OpenWrt packages yourself, which might be a bit complicated.
Instructions for the full build are available at: http://wiki.openwrt.org/doc/howto/buildroot.exigence and http://wiki.openwrt.org/doc/howto/build.
Or you could use the SDK http://wiki.openwrt.org/doc/howto/obtain.firmware.sdk.



After updating and installing the feeds, but before configuring and compiling, you would apply the patch like this:

patch -p0 < openwrt-libgphoto-Makefile-deti.patch
