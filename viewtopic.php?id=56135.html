Most scanning utilities use the MAC address to guess the manufacturer, as each manufacturer has a prefix assigned (see http://en.wikipedia.org/wiki/MAC_addres … ss_details for more info); so I guess you need to change the MAC address in your router to something else. You should use one of the "locally administered address" available (must be in the range x2-xx-xx-xx-xx-xx, x6-xx-xx-xx-xx-xx, xA-xx-xx-xx-xx-xx, or xE-xx-xx-xx-xx-xx). See http://wiki.openwrt.org/doc/uci/network … ocol_types to learn how to do that.
